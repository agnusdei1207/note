Atomic MVCC Sync
MVCC + Mutex eliminates concurrent TODO update conflicts. Every state change is cryptographically hashed.
Hook Orchestration
Priority-Phase Hook Registry with Topological Sort. Predictable, dependency-aware execution ordering.
Autonomous Recovery
Self-healing loops, stagnation detection, adaptive retry with exponential backoff.
Session Isolation
SessionPool with server-side compaction. Previous task context never leaks into new tasks.
Zero-Payload Turbo
system.transform unshifts agent instructions server-side. 90%+ payload reduction, no context fragmentation.
Hierarchical Memory
4-tier memory with EMA-based Context Gating. Preserves architectural truth across thousands of turns.

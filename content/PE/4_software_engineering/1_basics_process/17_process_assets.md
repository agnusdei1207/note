# 프로세스 자산 (Process Assets)

## 정의

**프로세스 자산(Process Assets)**은 조직이 프로젝트 수행 시 **재사용할 수 있는 프로세스 관련 자원**들의 집합이다. 표준 프로세스, 템플릿, 체크리스트, 지침서, 측정 데이터 등을 포함하며, 조직의 프로세스 역량을 체계적으로 축적하고 활용하기 위한 핵심 요소다.

│   │                                                         │ │

│   │              │  조직 프로세스 자산 라이브러리 │                │ │
│   │              │   (OPAL)                │                │ │

## 기술사 시험 포인트

### 1. 핵심 정의

> **프로세스 자산**: 조직이 재사용 가능한 프로세스 관련 자원
> **OSSP**: 조직 표준 프로세스 집합
> **PAL**: 프로세스 자산 라이브러리

### 2. 빈출 키워드

- **Tailoring**: 표준 프로세스를 프로젝트에 맞게 조정
- **Lessons Learned**: 프로젝트 교훈 및 경험
- **Best Practice**: 검증된 우수 실천 방법
- **Measurement Data**: 측정 데이터 및 베이스라인

### 3. 예상 문제

**Q1. 프로세스 자산의 구성요소를 설명하고 각각의 역할을 기술하시오.**

**Q2. 조직 표준 프로세스와 프로젝트 정의 프로세스의 관계를 설명하고 테일러링의 개념을 기술하시오.**

**Q3. 프로세스 자산 라이브러리(PAL)의 구축 및 관리 방안을 설명하시오.**

# 메타데이터 (Metadata)

## 정의

**메타데이터(Metadata)**는 **데이터에 대한 데이터**다. 데이터의 속성, 구조, 의미, 출처, 품질 등을 설명하는 정보로, 데이터의 효율적인 관리, 검색, 활용을 가능하게 한다. 시스템 카탈로그, 데이터 사전 등에서 저장·관리된다.

```
┌─────────────────────────────────────────────────────────────────┐
│           메타데이터 개념                                         │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   데이터 vs 메타데이터:                                          │
│   ┌─────────────────────────────────────────────────────────┐ │
│   │                                                         │ │
│   │   데이터:                                                │ │
│   │   "홍길동, 30세, 서울 거주"                              │ │
│   │                                                         │ │
│   │   메타데이터:                                            │ │
│   │   ┌───────────────────────────────────────────────┐    │ │
│   │   │ • 테이블명: 고객정보                            │    │ │
│   │   │ • 컬럼명: 이름, 나이, 주소                      │    │ │
│   │   │ • 데이터타입: VARCHAR(20), INT, VARCHAR(100)   │    │ │
│   │   │ • 제약조건: 이름 NOT NULL                       │    │ │
│   │   │ • 생성일: 2024-01-01                            │    │ │
│   │   │ • 소유자: DBA                                   │    │ │
│   │   └───────────────────────────────────────────────┘    │ │
│   │                                                         │ │
│   └─────────────────────────────────────────────────────────┘ │
│                                                                 │
│   DIKW 피라미드에서의 위치:                                      │
│   ┌─────────────────────────────────────────────────────────┐ │
│   │                                                         │ │
│   │                    ┌───────┐                            │ │

│   │                 │    지식     │ (Knowledge)              │ │
```

## 기술사 시험 포인트

### 1. 핵심 정의

> **메타데이터**: 데이터에 대한 데이터
> **Technical/Business/Operational**: 기술적, 비즈니스, 운영적 메타데이터
> **Data Catalog**: 메타데이터 기반 데이터 자산 검색 도구

### 2. 빈출 키워드

- **Data Dictionary**: 데이터 사전, 메타데이터 저장소
- **Schema Definition**: 스키마 정의 정보
- **Data Lineage**: 데이터 계보/이력
- **INFORMATION_SCHEMA**: SQL 표준 메타데이터 뷰

### 3. 예상 문제

**Q1. 메타데이터의 유형을 분류하고 각각의 예시를 들어 설명하시오.**

**Q2. 메타데이터가 쿼리 최적화에 미치는 영향을 기술하시오.**

**Q3. 데이터 거버넌스 관점에서 메타데이터 관리의 중요성을 설명하시오.**

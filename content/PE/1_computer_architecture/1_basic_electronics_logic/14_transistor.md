# 트랜지스터 (Transistor)

## 정의

**트랜지스터(Transistor)**는 **전류나 전압을 증폭하거나 스위칭**하는 3단자 반도체 소자다. 현대 전자공학의 핵심 소자로, CPU, 메모리 등 모든 디지털 회로의 기본 구성 요소다.

```
┌─────────────────────────────────────────────────────────────────┐
│                  트랜지스터의 두 가지 역할                        │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   ① 증폭 (Amplification):                                      │
│   ┌─────────────────────────────────────────────────────────┐ │
│   │                                                         │ │
│   │   약한 입력 신호        강한 출력 신호                    │ │
│   │        │                    ▲                            │ │
│   │        │     트랜지스터    │                             │ │
│   │        ▼     ┌─────┐      │                             │ │
│   │       ~~~    │     │     ~~~                            │ │
│   │      (작은)  │  ▲  │    (큰)                             │ │
│   │              │     │                                      │ │
│   │              └─────┘                                      │ │
│   │                                                         │ │
│   │   • 작은 신호로 큰 신호 제어                              │ │
│   │   • 오디오, RF 증폭기                                    │ │
│   └─────────────────────────────────────────────────────────┘ │
│                                                                 │
│   ② 스위칭 (Switching):                                        │
│   ┌─────────────────────────────────────────────────────────┐ │
│   │                                                         │ │
│   │   ON (포화):         OFF (차단):                         │ │
│   │   ┌─────┐           ┌─────┐                             │ │
│   │   │     │           │     │                             │ │
│   │   │  ●  │           │  ○  │                             │ │
│   │   │     │           │     │                             │ │
│   │   └─────┘           └─────┘                             │ │
│   │   전류 흐름          전류 차단                             │ │
│   │                                                         │ │
│   │   • 디지털 논리: 0과 1 표현                              │ │
│   │   • CPU, 메모리의 기본 소자                              │ │
│   └─────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

## 기술사 시험 포인트

### 1. 핵심 정의

> **트랜지스터**: 전류/전압을 증폭하거나 스위칭하는 3단자 반도체 소자
> **BJT**: 전류 제어, FET: 전압 제어

### 2. 빈출 키워드

- **증폭**: 작은 신호로 큰 신호 제어
- **스위칭**: ON/OFF 상태 전환
- **BJT 동작 영역**: 차단, 활성, 포화
- **CMOS**: NMOS + PMOS, 저전력 디지털 회로

### 3. 예상 문제

**Q1. BJT와 FET의 차이점을 설명하시오.**

**Q2. BJT의 동작 영역 3가지를 기술하고 각각의 용도를 설명하시오.**

**Q3. MOSFET가 디지털 IC의 핵심 소자인 이유를 설명하시오.**

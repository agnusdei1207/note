# 정류 회로 (Rectifier)

## 정의

**정류 회로(Rectifier)**는 AC(교류)를 DC(직류)로 변환하는 회로로, 다이오드의 **단방향 전도 특성**을 이용하여 양방향의 전압 파형을 단방향으로 변환한다.

```
┌─────────────────────────────────────────────────────────────────┐
│                    정류 회로 종류                              │
│                                                                 │
│   입력 (AC)                                                     │
│     │                                                          │
│     │    ┌────────────────────────────────────────────────────┐│
│     │    │                    정류                     ││
│     ├───▶│  ┌─────────────┐  ┌─────────────┐  ┌─────────────┐ ││
│     │    │  │ 1. 반파  │  │  │ 2. 풀파 │  │  │ 3. 브리지 │ │ │
│     │    │  └─────────────┘  └─────────────┘  └─────────────┘ ││
│     │    └────────────────────────────────────────────────────┘│
│     │                                                          │
│     ▼                                                          │
│   출력 (DC)                                                    │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

## 핵심 개념

### 1. 반파 정류 (Half-Wave Rectifier)

```
┌─────────────────────────────────────────────────────────────────┐
│                    반파 정류 회로                             │
│                                                                 │
│   입력 AC ───▶─┬───[D]───┬───────────────────────────────▶ 출력  │
│                 │       │       │                                │
│                 │       │       │                                │
│                 │       └───────┘                                │
│                 │                                               │
│                 └──▶ 부하 (R)                                        │
│                                                                 │
│   파형:                                                          │
│   ┌─────────────────────────────────────────────────────────┐ │
│   │                                                         │ │
│   │     ▲  ▲  ▲  ▲  ▲  ▲  ▲  ▲  ▲  ▲  ▲  ▲  ▲  ▲  ▲        │ │
│   │    ▼  ▼  ▼  ▼  ▼  ▼  ▼  ▼  ▼  ▼  ▼  ▼  ▼  ▼  ▼        │ │
│   │                                                         │ │
│   │   양의 반파만 통과 (D 도통 시)                                │ │
│   │   음의 반파는 차단 (D 비도통 시)                                │ │
│   │                                                         │ │
│   └─────────────────────────────────────────────────────────┘ │
│                                                                 │
│   리플 전압: V_ripple = V_peak / √2 × (1 - 0.637) ≈ 0.318 × V_peak        │
│   (실제로는 다이오드 전압 강하로 더 높음)                             │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### 2. 전파 정류 (Full-Wave Rectifier)

```
┌─────────────────────────────────────────────────────────────────┐
│                    전파 정류 회로                             │
│                                                                 │
│          D1                D2                D3                D4             │
│   ┌──┬──┐           ┌──┬──┐           ┌──┬──┐           ┌──┬──┐          │
│   │  │  │           │  │  │           │  │  │           │  │  │          │
│ ─┼──┼──┼───────────┼──┼──┼───────────┼──┼──┼──────────┼──┼──┼──▶ +Vdc │
│   │  │  │           │  │  │           │  │  │           │  │  │          │
│   └──┴──┘           └──┴──┘           └──┴──┘           └──┴──┘          │
│    │   │             │   │             │   │             │   │            │
│    └───┴─────────────┴───┴─────────────┴───┴─────────────┴───┘            │
│                         │                                      │
│   AC 입력 ─────────────┼────────────────────────────────────▶ DC 출력 │
│                         │                                      │
│   브리지 다이오드 브리지 (4개 다이오드)                           │
│                                                                 │
│   동작:                                                        │
│   • 정현 파: D1, D2 도통 / D3, D4 비도통                              │
│   • 부현 파: D1, D2 비도통 / D3, D4 도통                              │
│   • 양/음의 반파 모두 정류 → 리플 주파수 2배 (100Hz → 120Hz)           │
│                                                                 │
│   리플 전압: V_ripple ≈ V_peak / √2 × 0.48                  │
│   (반파 정류 대비 약 절반으로 감소)                             │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### 3. 브리지 정류 (Bridge Rectifier)

```
┌─────────────────────────────────────────────────────────────────┐
│                    브리지 정류 회로                             │
│                                                                 │
│         D1              D2              D3              D4            │
│        ┌──┴──┐        ┌──┴──┐        ┌──┴──┐        ┌──┴──┐          │
│        │  │  │        │  │  │        │  │  │        │  │  │          │
│  ─┬───┴──┴──┬────┬──┴──┴──┬────┬──┴──┴──┬────┬──┴──┴──┬──▶ +Vdc        │
│    │  │   │  │  │    │  │   │  │  │    │  │   │  │  │    │  │   │  │   │      │
│    └──┴───┴──┴──┘    └──┴───┴──┴──┘    └──┴───┴──┴──┘    └──┴───┴──┴──┘      │
│         │              │              │              │              │
│         └──────────────┴──────────────┴──────────────┘              │
│                                                                 │
│   AC 입력 ─────────────────────────────────────────────────▶ DC 출력 │
│                                                                 │
│   특징:                                                        │
│   • 4개 다이오드 사용                                          │
│   • 양/음의 반파 모두 정류                                      │
│   • 다이오드 2개씩 쌍으로 도통/비도통                                │
│   • 전파 정류와 동일한 리플 특성                                 │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

## 기술사 시험 포인트

### 1. 핵심 정의

> **정류**: AC를 DC로 변환하는 과정
> **반파/전파/브리지 정류**: 정류 방식 3가지

### 2. 빈출 키워드

- **리플 전압**: 정류 후 남은 AC 성분
- **평활**: 커패시터로 리플 제거
- **다이오드 브리지**: 4개 다이오드로 구성
- **PFC**: 역률 보정 회로

### 3. 예상 문제

**Q1. 반파 정류와 전파 정류의 차이점을 설명하시오.**

**Q2. 브리지 정류 회로의 장점을 기술하시오.**

**Q3. 정류 후 평활 회로가 필요한 이유를 설명하시오.**

# SR 래치 (Set-Reset Latch)

## 정의

**SR 래치(Set-Reset Latch)**는 **Set(설정)**과 **Reset(재설정)** 두 개의 입력으로 1비트 상태를 제어하는 가장 기본적인 래치다. NOR 게이트 또는 NAND 게이트의 크로스커플링으로 구성되며, S=1일 때 Set, R=1일 때 Reset 동작을 수행한다.

```
┌─────────────────────────────────────────────────────────────────┐
│           SR 래치 기본 구조                                       │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   NOR 게이트 SR 래치 (Active High):                              │
│   ┌─────────────────────────────────────────────────────────┐ │
│   │                                                         │ │
│   │   S ─────┐                                              │ │
│   │          │                                              │ │
│   │          ▼                                              │ │
│   │      ┌───────┐                                          │ │
│   │      │       │                                          │ │
│   │      │  NOR  │──────┬──────────▶ Q                      │ │
│   │      │       │      │                                   │ │
│   │      └───────┘      │                                    │ │
│   │          ▲          │                                    │ │
│   │          │          │                                    │ │
│   │          │    크로스커플링                               │ │
│   │          │          │                                    │ │
│   │      ┌───┴───┐      │                                    │ │
│   │      │       │      │                                    │ │
│   │      │  NOR  │──────┼──────────▶ Q'                     │ │
│   │      │       │      │                                    │ │
│   │      └───────┘      │                                    │ │
│   │          ▲          │                                    │ │
│   │          │          │                                    │ │
│   │   R ─────┘          │                                    │ │
│   │                      │                                   │ │
│   │   💡 Q과 Q'은 서로 반전                                 │ │
│   │                                                         │ │
│   └─────────────────────────────────────────────────────────┘ │
│                                                                 │
│   NAND 게이트 SR 래치 (Active Low, S'·R' 표기):                  │
│   ┌─────────────────────────────────────────────────────────┐ │
│   │                                                         │ │
│   │   S'────┐                                               │ │
│   │         │                                               │ │
│   │         ▼                                               │ │
│   │     ┌───────┐                                           │ │
│   │     │  NAND │──────┬─────────▶ Q                        │ │

│   │         │          │                                     │ │
│   │         │    크로스커플링                                │ │
│   │         │          │                                     │ │
│   │     ┌───┴───┐      │                                     │ │
│   │     │  NAND │──────┼─────────▶ Q'                       │ │
│   │     └───┬───┘      │                                     │ │
│   │         │          │                                     │ │
│   │         ▼          │                                     │ │
│   │   R'────┘          │                                     │ │
│   │                     │                                    │ │
│   │   💡 S'=R'=0일 때 금지 상태                               │ │
│   │                                                         │ │
│   └─────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

## 핵심 개념

### 1. NOR 게이트 SR 래치 특성

```
┌─────────────────────────────────────────────────────────────────┐
│           NOR SR 래치 특성 테이블                                 │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   ┌─────────────────────────────────────────────────────────┐ │
│   │                                                         │ │
│   │   입력         │ 출력      │                            │ │
│   │   S  │  R  │  Q  │ Q' │  동작                          │ │
│   │   ───┼─────┼─────┼─────┼────────────────────────        │ │
│   │    0 │  0  │  Q  │ Q'  │  상태 유지 (Memory)           │ │
│   │    0 │  1  │  0  │  1  │  Reset (Q=0)                  │ │
│   │    1 │  0  │  1  │  0  │  Set (Q=1)                    │ │
│   │    1 │  1  │  0  │  0  │  금지 상태 (Invalid)          │ │
│   │                  ↑    ↑                                  │ │
│   │                  └────┘ Q과 Q'이 같음 (모순)             │ │
│   │                                                         │ │
│   └─────────────────────────────────────────────────────────┘ │
│                                                                 │
│   동작 분석:                                                     │
│   ┌─────────────────────────────────────────────────────────┐ │
│   │                                                         │ │
│   │   S=1, R=0 (Set):                                       │ │
│   │   • 상위 NOR: S=1 → 출력=0 (Q')                         │ │
│   │   • 하위 NOR: R=0, Q'=0 → 출력=1 (Q)                    │ │
│   │   • 결과: Q=1, Q'=0 (Set 상태)                          │ │
│   │                                                         │ │
│   │   S=0, R=1 (Reset):                                     │ │
│   │   • 하위 NOR: R=1 → 출력=0 (Q)                          │ │
│   │   • 상위 NOR: S=0, Q=0 → 출력=1 (Q')                    │ │
│   │   • 결과: Q=0, Q'=1 (Reset 상태)                        │ │
│   │                                                         │ │
│   │   S=0, R=0 (Memory/Hold):                               │ │
│   │   • 두 NOR 모두 다른 입력에 의존                         │ │
│   │   • 이전 상태 유지                                       │ │
│   │                                                         │ │
│   │   S=1, R=1 (Invalid):                                   │ │
│   │   • 두 NOR 모두 0 출력                                   │ │
│   │   • Q=Q'=0 (논리적 모순)                                 │ │
│   │   • S, R이 동시에 0이 될 때 예측 불가                     │ │
│   │                                                         │ │
│   └─────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### 2. NAND 게이트 SR 래치 특성

```
┌─────────────────────────────────────────────────────────────────┐
│           NAND SR 래치 특성 테이블                                │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   ┌─────────────────────────────────────────────────────────┐ │
│   │                                                         │ │
│   │   입력 (Active Low)  │ 출력                              │ │
│   │   S' │  R' │  Q  │ Q' │  동작                          │ │
│   │   ───┼─────┼─────┼─────┼────────────────────────        │ │
│   │    0 │  0  │  1  │  1  │  금지 상태 (Invalid)          │ │
│   │    0 │  1  │  1  │  0  │  Set (Q=1)                    │ │
│   │    1 │  0  │  0  │  1  │  Reset (Q=0)                  │ │
│   │    1 │  1  │  Q  │ Q'  │  상태 유지 (Memory)           │ │
│   │                                                         │ │
│   │   💡 NAND 버전은 Active Low (0이 활성)                   │ │
│   │                                                         │ │
│   └─────────────────────────────────────────────────────────┘ │
│                                                                 │
│   NOR vs NAND SR 래치:                                          │
│   ┌─────────────────────────────────────────────────────────┐ │
│   │                                                         │ │
│   │   NOR SR 래치:                                          │ │
│   │   • Active High (1이 활성)                              │ │
│   │   • S=1: Set, R=1: Reset                               │ │
│   │   • S=R=1: 금지                                         │ │
│   │                                                         │ │
│   │   NAND SR 래치:                                          │ │
│   │   • Active Low (0가 활성)                               │ │
│   │   • S'=0: Set, R'=0: Reset                             │ │
│   │   • S'=R'=0: 금지                                       │ │
│   │                                                         │ │
│   └─────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### 3. 금지 상태 문제와 해결

```
┌─────────────────────────────────────────────────────────────────┐
│           금지 상태 (Invalid State) 문제                          │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   문제: S=R=1 (NOR) 또는 S'=R'=0 (NAND)일 때                    │
│   ┌─────────────────────────────────────────────────────────┐ │
│   │                                                         │ │
│   │   • Q=Q'=0 (NOR) 또는 Q=Q'=1 (NAND)                    │ │
│   │   • 논리적 모순 (Q와 Q'은 서로 반전이어야 함)             │ │
│   │   • S, R이 동시에 0이 될 때 최종 상태 예측 불가          │ │
│   │   • 경쟁 상황(Race Condition) 발생                      │ │
│   │                                                         │ │
│   │   S=R=1 → S=R=0 전이 시:                                │ │
│   │   ┌───────────────────────────────────────────────┐    │ │
│   │   │ • 게이트 지연 시간 차이                         │    │ │
│   │   │ • 어느 게이트가 먼저 반응하느냐에 따라 결과 다름│    │ │
│   │   │ • 비결정적 동작                                 │    │ │
│   │   └───────────────────────────────────────────────┘    │ │
│   │                                                         │ │
│   └─────────────────────────────────────────────────────────┘ │
│                                                                 │
│   해결 방안:                                                     │
│   ┌─────────────────────────────────────────────────────────┐ │
│   │                                                         │ │
│   │   ① 설계상 금지: S=R=1 입력 방지                        │ │
│   │                                                         │ │
│   │   ② D 래치 사용: 금지 상태 제거                         │ │
│   │      • D=1: S=1, R=0 (Set)                             │ │
│   │      • D=0: S=0, R=1 (Reset)                           │ │
│   │      • S=R=1 상태 불가능                                │ │
│   │                                                         │ │
│   │   ③ JK 플립플롭 사용: S=R=1일 때 토글 동작              │ │
│   │                                                         │ │
│   └─────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### 4. 게이트드 SR 래치

```
┌─────────────────────────────────────────────────────────────────┐
│           게이트드 SR 래치 (Gated/Clocked SR Latch)               │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   구조: SR 래치 + Enable 게이트                                  │
│   ┌─────────────────────────────────────────────────────────┐ │
│   │                                                         │ │
│   │   S ───▶ [AND] ──┐                                      │ │
│   │             ↑     │                                      │ │
│   │             │     │                                      │ │
│   │   EN ───────┤     ├──▶ S' ──▶ [SR 래치] ──▶ Q           │ │
│   │             │     │                    │                 │ │
│   │             │     │                    │                 │ │
│   │   R ───▶ [AND] ──┘         R' ────────┘                 │ │
│   │             ↑                                             │ │
│   │             │                                             │ │
│   │   EN (Enable/Clock)                                       │ │
│   │                                                         │ │
│   └─────────────────────────────────────────────────────────┘ │
│                                                                 │
│   동작:                                                         │
│   ┌─────────────────────────────────────────────────────────┐ │
│   │                                                         │ │
│   │   EN = 0:                                               │ │
│   │   • S' = R' = 0                                         │ │
│   │   • 상태 유지 (Hold)                                    │ │
│   │                                                         │ │
│   │   EN = 1:                                               │ │
│   │   • S' = S, R' = R                                      │ │
│   │   • S/R 입력에 따라 Set/Reset/Hold                      │ │
│   │                                                         │ │
│   │   특성 테이블 (EN=1일 때):                                │ │
│   │   ┌───────────────────────────────────────────────┐    │ │
│   │   │  EN │  S  │  R  │  Q(t+1)  │  동작           │    │ │
│   │   │─────┼─────┼─────┼──────────┼─────────────────│    │ │
│   │   │  0  │  X  │  X  │  Q(t)    │  Hold           │    │ │
│   │   │  1  │  0  │  0  │  Q(t)    │  Hold           │    │ │
│   │   │  1  │  0  │  1  │  0       │  Reset          │    │ │
│   │   │  1  │  1  │  0  │  1       │  Set            │    │ │
│   │   │  1  │  1  │  1  │  -       │  금지           │    │ │
│   │   └───────────────────────────────────────────────┘    │ │
│   │                                                         │ │
│   └─────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

## 기술사 시험 포인트

### 1. 핵심 정의

> **SR 래치**: Set과 Reset 입력으로 제어되는 기본 래치
> **금지 상태**: S=R=1 (NOR) 또는 S'=R'=0 (NAND) - 예측 불가
> **크로스커플링**: 상태 유지를 위한 피드백 연결

### 2. 빈출 키워드

- **Set/Reset**: 설정(1)/재설정(0) 동작
- **Invalid State**: 금지 상태 (S=R=1)
- **Active High/Low**: NOR(Active High), NAND(Active Low)
- **Gated SR Latch**: Enable이 추가된 SR 래치

### 3. 예상 문제

**Q1. NOR 게이트와 NAND 게이트로 구성한 SR 래치의 차이점을 설명하시오.**

**Q2. SR 래치의 금지 상태 문제와 이를 해결하는 방안을 기술하시오.**

**Q3. 게이트드 SR 래치의 구조와 동작을 설명하시오.**

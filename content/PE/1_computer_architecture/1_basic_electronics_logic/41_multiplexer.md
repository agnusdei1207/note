# 멀티플렉서 (Multiplexer, MUX)

## 정의

**멀티플렉서(Multiplexer, MUX)**는 여러 개의 입력 신호 중 **하나를 선택**하여 단일 출력으로 내보내는 조합 논리회로다. 데이터 선택기(Data Selector)라고도 한다. N개의 선택선(Select Line)으로 2^N개의 입력 중 하나를 선택한다.

```
┌─────────────────────────────────────────────────────────────────┐
│           4:1 멀티플렉서 구조                                     │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   입력선                                          │
│   ┌─────────────────────────────────────────────────────────┐ │
│   │                                                         │ │
│   │   D0 ─────────┐                                        │ │
│   │   D1 ─────────┼──▶ [ 4:1 MUX ] ──▶ Y (출력)            │ │
│   │   D2 ─────────┤      ▲   ▲                             │ │
│   │   D3 ─────────┘      │   │                             │ │
│   │                       │   │                             │ │
│   │   선택선:      S1 ───┘   │                             │ │
│   │                   S0 ─────┘                             │ │
│   │                                                         │ │
│   └─────────────────────────────────────────────────────────┘ │
│                                                                 │
│   선택 신호에 따른 출력:                                         │
│   ┌─────────────────────────────────────────────────────────┐ │
│   │   S1  S0  │  출력 Y                                     │ │
│   │   ────────┼──────────────                               │ │
│   │    0   0  │    D0                                       │ │
│   │    0   1  │    D1                                       │ │
│   │    1   0  │    D2                                       │ │
│   │    1   1  │    D3                                       │ │
│   └─────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

## 핵심 개념

### 1. 멀티플렉서 동작 원리

```
┌─────────────────────────────────────────────────────────────────┐
│           4:1 MUX 내부 회로 구현                                  │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   논리식: Y = D0·S1'·S0' + D1·S1'·S0 + D2·S1·S0' + D3·S1·S0   │
│                                                                 │
│   ┌─────────────────────────────────────────────────────────┐ │
│   │                                                         │ │
│   │   D0 ──┬──[AND]──┐                                     │ │
│   │        │    ↑     │                                     │ │
│   │   S0'──┘    │     │                                     │ │
│   │   S1'───────┘     │                                     │ │
│   │                    ├──[OR]──▶ Y                         │ │
│   │   D1 ──┬──[AND]──┐│                                     │ │
│   │        │    ↑     ││                                     │ │
│   │   S0 ──┘    │     ││                                     │ │
│   │   S1'───────┘     ││                                     │ │
│   │                    ││                                     │ │
│   │   D2 ──┬──[AND]──┐││                                     │ │
│   │        │    ↑     │││                                     │ │
│   │   S0'──┘    │     │││                                     │ │
│   │   S1 ───────┘     │││                                     │ │
│   │                    │││                                     │ │
│   │   D3 ──┬──[AND]──┐│││                                     │ │
│   │        │    ↑     ││││                                     │ │
│   │   S0 ──┘    │     ││││                                     │ │
│   │   S1 ───────┘     ││││                                     │ │
│   │                                                         │ │
│   └─────────────────────────────────────────────────────────┘ │
│                                                                 │
│   💡 각 입력은 해당 선택 조건에서만 AND 게이트를 통과            │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### 2. 멀티플렉서 종류

```
┌─────────────────────────────────────────────────────────────────┐
│           멀티플렉서 종류 및 특징                                  │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   ┌─────────────────────────────────────────────────────────┐ │
│   │  종류      │ 입력수 │ 선택선 │ 용도                      │ │
│   ├────────────┼────────┼────────┼───────────────────────────┤ │
│   │  2:1 MUX   │   2    │   1    │ 기본 선택                 │ │
│   │  4:1 MUX   │   4    │   2    │ 니블 선택                 │ │
│   │  8:1 MUX   │   8    │   3    │ 바이트 선택               │ │
│   │  16:1 MUX  │   16   │   4    │ 워드 선택                 │ │
│   │  32:1 MUX  │   32   │   5    │ 더블워드 선택             │ │
│   └────────────┴────────┴────────┴───────────────────────────┘ │
│                                                                 │
│   일반화: N:1 MUX는 ⌈log₂N⌉개의 선택선 필요                     │
│                                                                 │
│   2:1 MUX (기본 구성):                                          │
│   ┌─────────────────────────────────────────────────────────┐ │
│   │                                                         │ │
│   │   Y = D0·S' + D1·S                                      │ │
│   │                                                         │ │
│   │   D0 ───┬──[AND]────┐                                   │ │
│   │         │     ↑      │                                   │ │
│   │    S'───┘     │      │                                   │ │
│   │                ├──[OR]──▶ Y                             │ │
│   │   D1 ───┬──[AND]────┘                                   │ │
│   │         │     ↑                                          │ │
│   │    S ───┘                                                │ │
│   │                                                         │ │
│   └─────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### 3. 멀티플렉서 확장

```
┌─────────────────────────────────────────────────────────────────┐
│           멀티플렉서 확장 (Tree 구조)                              │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   8:1 MUX를 4:1 MUX 2개 + 2:1 MUX 1개로 구현:                   │
│                                                                 │
│   ┌─────────────────────────────────────────────────────────┐ │
│   │                                                         │ │
│   │   D0 ──┐                                                │ │
│   │   D1 ──┼──▶ [4:1 MUX] ──┐                              │ │
│   │   D2 ──┤        ↑       │                              │ │
│   │   D3 ──┘     S1,S0      │                              │ │
│   │                         ├──▶ [2:1 MUX] ──▶ Y           │ │
│   │   D4 ──┐        ↑       │        ↑                     │ │
│   │   D5 ──┼──▶ [4:1 MUX] ──┘       S2                     │ │
│   │   D6 ──┤        ↑                                     │ │
│   │   D7 ──┘     S1,S0                                    │ │
│   │                                                         │ │
│   │   선택선: S2, S1, S0 (3비트)                            │ │
│   │                                                         │ │
│   └─────────────────────────────────────────────────────────┘ │
│                                                                 │
│   다중 비트 MUX (버스 폭 확장):                                  │
│   ┌─────────────────────────────────────────────────────────┐ │
│   │                                                         │ │
│   │   4비트 4:1 MUX:                                        │ │
│   │   ┌───────────────────────────────────────────────┐    │ │
│   │   │   A[3:0]                                      │    │ │
│   │   │   B[3:0]  ──▶ [4:1 MUX × 4] ──▶ Y[3:0]       │    │ │
│   │   │   C[3:0]       (4개 MUX 병렬)                 │    │ │
│   │   │   D[3:0]                                      │    │ │
│   │   │                     ↑                         │    │ │
│   │   │               S1, S0 (공통)                   │    │ │
│   │   └───────────────────────────────────────────────┘    │ │
│   │                                                         │ │
│   └─────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

## 기술사 시험 포인트

### 1. 핵심 정의

> **멀티플렉서**: 다중 입력 중 하나를 선택하여 단일 출력으로 내보내는 회로
> **데이터 선택기**: MUX의 다른 명칭
> **선택선 수**: ⌈log₂(입력 수)⌉개 필요

### 2. 빈출 키워드

- **N:1 MUX**: N개 입력, 1개 출력
- **Select Line**: 입력 선택을 위한 제어선
- **Time Division Multiplexing**: MUX를 이용한 통신 방식
- **Bus Multiplexer**: CPU 데이터 패스에서 버스 선택

### 3. 예상 문제

**Q1. 4:1 멀티플렉서의 논리식을 유도하고 내부 회로를 설계하시오.**

**Q2. 멀티플렉서를 이용하여 XOR 게이트를 구현하는 방법을 기술하시오.**

**Q3. 멀티플렉서와 디멀티플렉서의 관계와 통신 시스템에서의 활용을 설명하시오.**

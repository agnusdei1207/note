# 다이오드 (Diode)

## 정의

**다이오드(Diode)**는 PN 접합으로 구성된 **2단자 반도체 소자**로, 한 방향으로만 전류가 흐르는 **정류 특성**을 가진다. 양극(Anode, +)과 음극(Cathode, -) 두 단자를 가지며, 순방향 바이어스에서만 전도된다.

```
┌─────────────────────────────────────────────────────────────────┐
│                    다이오드 기호와 구조                          │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   기호:                        구조:                           │
│                                                                 │
│       Anode (A)                 P형    │    N형               │
│           │                    ┌───────┴───────┐              │
│           │                    │               │              │
│        ───┼───▶               │   P    │    N   │              │
│           │  Cathode (K)       │   ┌────┴────┐   │              │
│           │                    │   │공핍영역 │   │              │
│           ▼                    │   └─────────┘   │              │
│                                 │               │              │
│                                 └───────┬───────┘              │
│                                         │                      │
│                                        V_bi (0.7V)             │
│                                                                 │
│   전류 방향: Anode → Cathode (화살표 방향)                       │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

## 핵심 개념

### 1. 다이오드 특성 곡선

```
┌─────────────────────────────────────────────────────────────────┐
│                 다이오드 V-I 특성 곡선                          │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   I (mA)                                                       │
│     ↑                                                          │
│     │                              ●                           │
│     │                           ●                              │
│     │                        ●                                 │
│     │                     ●                                    │
│     │                  ●                                       │
│     │               ●                                          │
│     │            ●                                             │
│   0 ─┼─┼─┼─┼─┼─┼─┼─┼─┼─┼─┼─┼─┼─┼─┼─┼─┼─┼─┼─┼─┼─┼─┼─┼─▶ V (V)  │
│     │ 0  0.3  0.5  0.7  1.0                                    │
│     │         ↑                                                │
│     │     Turn-on                                              │
│     │     Voltage                                              │
│     │                                                          │
│     │        ┌─ 역방향 포화 전류 (매우 작음, μA~nA)            │
│     │   ●●●●●│                                                │
│     │        └─ 항복 전압(V_B) 이상에서 급격한 전류 증가         │
│     │                                                          │
│   V_R ◀───────┼───────▶ V_B (항복 전압)                         │
│                                                                 │
│   순방향: V > 0.7V (Si)에서 전류 흐름                            │
│   역방향: V < 0에서 전류 거의 없음                               │
│           V < -V_B에서 항복 (Breakdown)                          │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### 2. 순방향 특성

```
┌─────────────────────────────────────────────────────────────────┐
│                    순방향 바이어스                               │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   전압-전류 관계 (Shockley 방정식):                              │
│                                                                 │
│             ┌─────────────────────────────────────────────┐     │
│             │                                             │     │
│             │   I = I_S × (e^(V/nV_T) - 1)               │     │
│             │                                             │     │
│             │   I_S: 포화 전류 (Saturation Current)       │     │
│             │   n  : 이상 계수 (1~2, 보통 1)               │     │
│             │   V_T: 열전압 = kT/q ≈ 26mV (상온)          │     │
│             │   V  : 인가 전압                              │     │
│             │                                             │     │
│             └─────────────────────────────────────────────┘     │
│                                                                 │
│   Turn-on 전압:                                                 │
│   ┌─────────────────────────────────────────────────────────┐ │
│   │  Si (실리콘): 0.6 ~ 0.7 V                               │ │
│   │  Ge (게르마늄): 0.2 ~ 0.3 V                             │ │
│   │  Schottky: 0.2 ~ 0.4 V                                 │ │
│   │  LED: 1.8 ~ 3.3 V (색상에 따라 다름)                    │ │
│   └─────────────────────────────────────────────────────────┘ │
│                                                                 │
│   동작 원리:                                                    │
│   • 외부 전압이 내부 전위 장벽(0.7V) 극복                       │
│   • 공핍 영역 축소                                              │
│   • 전자와 정공의 확산 전류 증가                                 │
│   • 정방향 전류 흐름                                             │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### 3. 역방향 특성

```
┌─────────────────────────────────────────────────────────────────┐
│                    역방향 바이어스                               │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   역방향 포화 전류 (Reverse Saturation Current, I_S):           │
│   ┌─────────────────────────────────────────────────────────┐ │
│   │  • 소수 캐리어에 의한 전류                               │ │
│   │  • 매우 작음 (nA ~ μA 수준)                              │ │
│   │  • 온도에 민감 (온도 ↑ → I_S ↑)                          │ │
│   │  • Si: ~nA, Ge: ~μA                                     │ │
│   └─────────────────────────────────────────────────────────┘ │
│                                                                 │
│   항복 (Breakdown):                                             │
│   ┌─────────────────────────────────────────────────────────┐ │
│   │                                                         │ │
│   │  ① 쇄빙 항복 (Avalanche Breakdown)                      │ │
│   │     • 고전압에서 전자 가속                               │ │
│   │     • 충돌 이온화로 전자-정공 쌍 생성                    │ │
│   │     • 전류 급격히 증가                                   │ │
│   │     • 파괴적 (돌이킬 수 없음)                            │ │
│   │                                                         │ │
│   │  ② 제너 항복 (Zener Breakdown)                          │ │
│   │     • 고농도 도핑 다이오드에서 발생                      │ │
│   │     • 양자 터널링 효과                                   │ │
│   │     • 낮은 전압에서 발생 (< 6V)                          │ │
│   │     • 비파괴적 (가역적)                                  │ │
│   │                                                         │ │
│   └─────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

## 다이오드 종류

```
┌─────────────────────────────────────────────────────────────────┐
│                    다이오드 종류                                 │
├─────────────────┬───────────────────────────────────────────────┤
│      종류       │                특징/용도                       │
├─────────────────┼───────────────────────────────────────────────┤
│  정류 다이오드  │  • 일반 PN 접합 다이오드                      │
│  (Rectifier)   │  • 전원 정류, 역류 방지                        │
│                │  • 1N4001~4007, 1N5400 시리즈                  │
├─────────────────┼───────────────────────────────────────────────┤
│  제너 다이오드  │  • 항복 영역에서 전압 일정 유지               │
│  (Zener)       │  • 전압 레퍼런스, 서지 보호                    │
│                │  • 3.3V, 5.1V, 12V 등 규격화                   │
├─────────────────┼───────────────────────────────────────────────┤
│ 쇼트키 다이오드 │  • 금속-반도체 접합                           │
│  (Schottky)    │  • 빠른 스위칭, 낮은 순방향 강하               │
│                │  • 고주파 정류, 클램핑                         │
├─────────────────┼───────────────────────────────────────────────┤
│     LED        │  • 전계 발광 (Electroluminescence)             │
│ (발광 다이오드) │  • 표시, 조명, 광 통신                        │
│                │  • 색상별 전압 상이                            │
├─────────────────┼───────────────────────────────────────────────┤
│  포토 다이오드  │  • 광전 효과 (Photovoltaic Effect)             │
│  (Photodiode)  │  • 광 검출, 광 통신 수신                       │
│                │  • 역바이어스에서 동작                         │
├─────────────────┼───────────────────────────────────────────────┤
│   바랙터       │  • 역바이어스에 따른 정전용량 변화             │
│  (Varactor)    │  • 전압 제어 발진기, 튜닝 회로                 │
│                │  • VCO, AFC 회로                              │
├─────────────────┼───────────────────────────────────────────────┤
│   터널 다이오드 │  • 중대역에서 음저항 특성                     │
│  (Tunnel)      │  • 초고속 스위칭, 발진                         │
│                │  • 마이크로파 응용                             │
└─────────────────┴───────────────────────────────────────────────┘
```

## 실무 적용

### 정류 회로

```
┌─────────────────────────────────────────────────────────────────┐
│                    반파 정류 vs 전파 정류                        │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│  ① 반파 정류 (Half-Wave Rectifier):                            │
│                                                                 │
│          ┌───┐                                                  │
│   AC ────┤ D ├───┬────────────▶ DC (반파)                       │
│          └───┘   │                                              │
│                  │                                              │
│                  └──[R Load]── GND                               │
│                                                                 │
│  • 한쪽 반파만 출력                                              │
│  • 효율 낮음, 리플 큼                                             │
│                                                                 │
│  ─────────────────────────────────────────────────────────────  │
│                                                                 │
│  ② 브리지 정류 (Bridge/Full-Wave Rectifier):                    │
│                                                                 │
│                D1         D2                                    │
│          ┌─────▶│────┬────│◀─────┐                              │
│          │          │    │       │                              │
│   AC ────┤          │    │       ├───▶ DC (전파)                │
│          │          │    │       │                              │
│          │     D3   │    │   D4  │                              │
│          └─────│◀───┴────▶│─────┘                              │
│                                                                 │
│  • 양쪽 반파 모두 출력                                           │
│  • 효율 높음, 리플 작음                                           │
│  • 다이오드 4개 필요                                              │
│                                                                 │
│  ─────────────────────────────────────────────────────────────  │
│                                                                 │
│  ③ 출력 파형:                                                   │
│                                                                 │
│   AC 입력:      반파 정류:       전파 정류:                     │
│      ∩∪∩          ∩  ∩           ∩∩∩∩                          │
│     ∕  ∕         ∕  ∕           ∕∥∥∥∥                          │
│    ∕  ∕         ∕  ∕           ∕ ∥∥∥∥                          │
│   ───────────  ───────────     ──∥∥∥∥──                         │
│    ∕  ∕                                       ∥∥               │
│   ∕  ∕         ∕  ∕           ∕ ∥∥∥∥                          │
│  ∕  ∕         ∕  ∕           ∕  ∥∥∥∥                          │
│  ∪∕∪∕∪         ∪  ∪           ∪∥∥∥∥∪                          │
│                                                                 │
│  평활 커패시터 추가로 DC 레벨 근접 가능                          │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### 전압 레퍼런스 (제너 다이오드)

```
┌─────────────────────────────────────────────────────────────────┐
│                 제너 다이오드 전압 안정화 회로                   │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│               R_s                                               │
│   Vin ─────/\/\/\────┬────────▶ Vout (= V_Z)                    │
│                      │                                          │
│                      │                                          │
│                      └──┬──┐                                    │
│                         │Z │                                    │
│                         └──┬──┘                                 │
│                            │                                    │
│                           GND                                   │
│                                                                 │
│   동작 원리:                                                    │
│   • Vin > V_Z: 제너 항복 영역에서 V_Z 일정 유지                  │
│   • 부하 전류 변화에도 Vout = V_Z                                │
│   • R_s: 과전류 방지, 부하 전류 분배                             │
│                                                                 │
│   설계 식:                                                      │
│   R_s = (Vin - V_Z) / (I_Z + I_L)                              │
│                                                                 │
│   I_Z: 제너 최소 동작 전류                                       │
│   I_L: 부하 전류                                                 │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

## 기술사 시험 포인트

### 1. 핵심 정의

> **다이오드**: PN 접합 2단자 소자, 순방향 전도, 역방향 차단
> **Turn-on 전압**: Si ≈ 0.7V, Ge ≈ 0.3V, LED ≈ 2-3V

### 2. 빈출 키워드

- **정류**: AC → DC 변환
- **순방향/역방향 바이어스**: 전도/차단 상태
- **제너/쇄빙 항복**: 가역적/비가역적 파괴
- **쇼트키**: 빠른 스위칭, 낮은 전압 강하
- **브리지 정류**: 다이오드 4개로 전파 정류

### 3. 예상 문제

**Q1. PN 접합 다이오드의 정류 원리를 설명하시오.**

**Q2. 제너 다이오드와 일반 다이오드의 차이점을 기술하시오.**

**Q3. 반파 정류와 전파 정류의 차이점을 비교 설명하시오.**

# 순차 논리회로 (Sequential Logic)

## 정의

**순차 논리회로(Sequential Logic Circuit)**는 현재 입력뿐만 아니라 **과거 상태(기억)**에도 출력이 의존하는 디지털 회로다. 메모리 요소(래치, 플립플롭)를 포함하며, 피드백 루프를 통해 상태를 유지한다.

```
┌─────────────────────────────────────────────────────────────────┐
│           순차 논리회로 vs 조합 논리회로                          │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   조합 논리회로:                                                 │
│   ┌─────────────────────────────────────────────────────────┐ │
│   │                                                         │ │
│   │   입력 ──▶ [조합 회로] ──▶ 출력                         │ │
│   │              (메모리 없음)                               │ │
│   │   출력 = f(현재 입력)                                    │ │
│   │                                                         │ │
│   └─────────────────────────────────────────────────────────┘ │
│                                                                 │
│   순차 논리회로:                                                 │
│   ┌─────────────────────────────────────────────────────────┐ │
│   │                                                         │ │
│   │   입력 ──┬──▶ [조합 회로] ──┬──▶ 출력                  │ │
│   │          │         ↑        │                           │ │
│   │          │         │        │                           │ │
│   │          │    [메모리 요소] │                           │ │
│   │          │         ↑        │                           │ │
│   │          └─────────┴────────┘                           │ │
│   │              (피드백 루프)                                │ │
│   │                                                         │ │
│   │   출력 = f(현재 입력, 현재 상태)                         │ │
│   │   다음 상태 = g(현재 입력, 현재 상태)                    │ │
│   │                                                         │ │
│   └─────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

## 핵심 개념

### 1. 순차 회로의 구성 요소

```
┌─────────────────────────────────────────────────────────────────┐
│           순차 회로 기본 구조                                     │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   ┌─────────────────────────────────────────────────────────┐ │
│   │                                                         │ │
│   │         입력 (X)                                        │ │
│   │           │                                             │ │
│   │           ▼                                             │ │
│   │   ┌───────────────────┐                                │ │
│   │   │                   │──────▶ 출력 (Y)                 │ │
│   │   │    조합 논리부     │                                │ │
│   │   │                   │                                │ │
│   │   └─────────┬─────────┘                                │ │
│   │             │                                           │ │
│   │             ▼  다음 상태 (Y)                            │ │
│   │   ┌───────────────────┐                                │ │
│   │   │                   │                                │ │
│   │   │    메모리 요소     │◀───── 클럭 (CLK)              │ │
│   │   │   (래치/플립플롭)  │                                │ │
│   │   │                   │                                │ │
│   │   └─────────┬─────────┘                                │ │
│   │             │                                           │ │
│   │             ▼  현재 상태 (Q)                            │ │
│   │             │                                           │ │
│   │             └──────────▶ 조합 논리부로 피드백           │ │
│   │                                                         │ │
│   └─────────────────────────────────────────────────────────┘ │
│                                                                 │
│   상태 변수: 메모리 요소에 저장된 정보                           │
│   상태 수: n개 플립플롭 → 2^n 개 상태                           │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### 2. 메모리 요소 종류

```
┌─────────────────────────────────────────────────────────────────┐
│           메모리 요소 분류                                        │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   ┌─────────────────────────────────────────────────────────┐ │
│   │                                                         │ │
│   │   ① 래치 (Latch): 비동기식, 레벨 트리거                  │ │
│   │   ┌───────────────────────────────────────────────┐    │ │
│   │   │ • Enable 신호가 활성 레벨일 때 입력 변화 반영  │    │ │
│   │   │ • 클럭 없이 동작                               │    │ │
│   │   │ • 투명(Transparent) 모드                       │    │ │
│   │   │ • SR 래치, D 래치                              │    │ │
│   │   └───────────────────────────────────────────────┘    │ │
│   │                                                         │ │
│   │   ② 플립플롭 (Flip-Flop): 동기식, 에지 트리거           │ │
│   │   ┌───────────────────────────────────────────────┐    │ │
│   │   │ • 클럭 에지에서만 상태 변화                    │    │ │
│   │   │ • 입력 변화에 민감하지 않음                    │    │ │
│   │   │ • SR, D, JK, T 플립플롭                        │    │ │
│   │   └───────────────────────────────────────────────┘    │ │
│   │                                                         │ │
│   └─────────────────────────────────────────────────────────┘ │
│                                                                 │
│   래치 vs 플립플롭 타이밍:                                       │
│   ┌─────────────────────────────────────────────────────────┐ │
│   │                                                         │ │
│   │   D 래치 (레벨 트리거):                                  │ │
│   │   CLK ────┐                                             │ │
│   │           │  Enable 구간 동안 입력 반영                  │ │
│   │   ──┐  ┌──┘                                             │ │
│   │     └──┘                                                │ │
│   │                                                         │ │
│   │   D 플립플롭 (에지 트리거):                              │ │
│   │   CLK ────┐                                             │ │
│   │           │  상승 에지에서만 입력 반영                   │ │
│   │   ────────┘                                             │ │
│   │            ↑                                            │ │
│   │                                                         │ │
│   └─────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

│   │   ② 상태 테이블 작성                                    │ │
│   │   ③ 상태 다이어그램 그리기                              │ │

### 4. 순차 회로 종류

```
┌─────────────────────────────────────────────────────────────────┐
│           순차 회로 응용                                          │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   ┌─────────────────────────────────────────────────────────┐ │
│   │                                                         │ │
│   │   ① 레지스터 (Register):                                │ │
│   │   • 데이터 저장                                         │ │
│   │   • 병렬 로드, 시프트 기능                              │ │
│   │   • n비트 = n개 플립플롭                                │ │
│   │                                                         │ │
│   │   ② 카운터 (Counter):                                   │ │
│   │   • 상태 순차적 변화                                    │ │
│   │   • 이진 카운터, 링 카운터, 존슨 카운터                 │ │
│   │   • 동기식/비동기식                                     │ │
│   │                                                         │ │
│   │   ③ 시프트 레지스터 (Shift Register):                   │ │
│   │   • 데이터 좌우 이동                                    │ │
│   │   • 직렬-병렬, 병렬-직렬 변환                           │ │
│   │   • SISO, SIPO, PISO, PIPO                              │ │
│   │                                                         │ │
│   │   ④ 상태 머신 (State Machine):                          │ │
│   │   • Moore 머신: 출력이 상태에만 의존                    │ │
│   │   • Mealy 머신: 출력이 상태와 입력에 의존               │ │
│   │                                                         │ │
│   └─────────────────────────────────────────────────────────┘ │
│                                                                 │
│   Moore vs Mealy 머신:                                          │
│   ┌─────────────────────────────────────────────────────────┐ │
│   │                                                         │ │
│   │   Moore: 입력 ──▶ [상태] ──▶ 출력                       │ │
│   │           (출력이 클럭에 동기)                          │ │
│   │                                                         │ │
│   │   Mealy: 입력 ──▶ [상태+입력] ──▶ 출력                  │ │
│   │           (출력이 비동기 가능)                          │ │
│   │                                                         │ │
│   └─────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

## 기술사 시험 포인트

### 1. 핵심 정의

> **순차 논리회로**: 현재 입력과 과거 상태에 의해 출력 결정
> **메모리 요소**: 상태 저장 (래치, 플립플롭)
> **피드백**: 출력을 입력으로 되돌리는 연결

### 2. 빈출 키워드

- **State Variable**: 메모리 요소에 저장된 상태 정보
- **State Table**: 상태 전이 표
- **State Diagram**: 상태 전이 그래프
- **Moore/Mealy Machine**: 순차 회로 모델

### 3. 예상 문제

**Q1. 조합 논리회로와 순차 논리회로의 차이점을 설명하고 순차 회로의 구성 요소를 기술하시오.**

**Q2. 래치와 플립플롭의 차이점을 타이밍도와 함께 설명하시오.**

**Q3. Moore 머신과 Mealy 머신의 차이점과 각각의 장단점을 기술하시오.**

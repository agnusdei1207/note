# 발광 다이오드 (LED)

## 정의

**발광 다이오드(LED, Light Emitting Diode)**는 순방향 바이어스 시 **전기 에너지를 빛 에너지로 변환**하는 반도체 소자다. PN 접합에서 전자와 정공이 재결합할 때 발생하는 에너지를 광자로 방출한다.

```
┌─────────────────────────────────────────────────────────────────┐
│                    LED의 발광 원리                              │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   전계발광 (Electroluminescence):                               │
│                                                                 │
│        (+) Anode                                              │
│           │                                                    │
│           ▼                                                    │
│       ┌───┴───┐                                               │
│       │ P │ N │  ◀── 전자-정공 재결합 영역                      │
│       └───┬───┘                                               │
│           │                                                    │
│           ▼                                                    │
│        (-) Cathode                                              │
│                                                                 │
│   전자 (●) ───▶ ───▶ ───▶ 정공 (○)                            │
│                           │                                    │
│                           ▼                                    │
│                      ★ 광자 방출                               │
│                     hν = E_gap                                 │
│                                                                 │
│   발광 파장: λ = hc/E_gap                                      │
│   • 적색: ~630nm (낮은 에너지)                                  │
│   • 녹색: ~530nm                                               │
│   • 청색: ~470nm (높은 에너지)                                  │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

## 핵심 개념

### 1. LED 소재와 색상

```
┌─────────────────────────────────────────────────────────────────┐
│                   LED 소재와 발광 색상                          │
├─────────────────┬─────────────────┬─────────────────────────────┤
│     소재       │   밴드갭 (eV)   │       발광 색상              │
├─────────────────┼─────────────────┼─────────────────────────────┤
│   GaAs          │     1.42        │     적외선 (~870nm)          │
│  (갈륨비소)      │                 │                           │
├─────────────────┼─────────────────┼─────────────────────────────┤
│   GaAsP         │   1.8-2.0       │     적색 (~630nm)            │
│  (갈륨비소인)    │                 │                           │
├─────────────────┼─────────────────┼─────────────────────────────┤
│   GaP           │     2.26        │     녹색 (~560nm)            │
│  (갈륨인)        │                 │                            │
├─────────────────┼─────────────────┼─────────────────────────────┤
│   GaN           │     3.4         │     청색 (~450nm)            │
│  (갈륨질화소)    │                 │                            │
├─────────────────┼─────────────────┼─────────────────────────────┤
│   InGaN         │    0.7-3.4      │     청색~녹색 (조절 가능)     │
│  (인듐갈륨질화소) │                 │     백색 LED (청+녹+적 혼합)  │
└─────────────────┴─────────────────┴─────────────────────────────┘
```

## 상세 설명

### LED 구동 회로

```
┌─────────────────────────────────────────────────────────────────┐
│                   LED 구동 방식                                 │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   ① 정전류 구동 (Constant Current):                           │
│   ┌─────────────────────────────────────────────────────────┐ │
│   │  VCC ───┬──[R]───┬──[LED]───┬── GND                       │ │
│   │         │         │           │                            │ │
│   │         │         └──▶ I = (VCC-Vf)/R (일정)              │ │
│   │         │                                                      │
│   │  • 간단, 저렴                                              │ │
│   │  • 전압 변화에 민감                                         │ │
│   └─────────────────────────────────────────────────────────┘ │
│                                                                 │
│   ② 정전류 소자 구동 (Constant Current Driver):               │
│   ┌─────────────────────────────────────────────────────────┐ │
│   │  VCC ───┬──[Driver IC]───┬──[LED]───┬── GND               │ │
│   │         │      │          │           │                    │ │
│   │         │   피드백         │           │                    │ │
│   │         │◀────────────────┘           │                    │ │
│   │         │                              │                    │ │
│   │  • 전류 일정 유지                                          │ │
│   │  • 밝기 균일, 신뢰성 높음                                   │ │
│   └─────────────────────────────────────────────────────────┘ │
│                                                                 │
│   ③ PWM 구동 (Pulse Width Modulation):                        │
│   ┌─────────────────────────────────────────────────────────┐ │
│   │                                                         │ │
│   │   Duty = Ton / (Ton + Toff)                              │ │
│   │                                                         │ │
│   │   ┌────┐   ┌────┐   ┌────┐                               │ │
│   │   │    │   │    │   │    │                              │ │
│   │ ──┘    └───┘    └───┘    └──                             │ │
│   │                                                         │ │
│   │   • 디밍 조절로 밝기 제어                                  │ │
│   │   • 고속 스위칭으로 깜빡임 없음                            │ │
│   │   • 디스플레이 백라이트에 주로 사용                        │ │
│   └─────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

```

```

## 기술사 시험 포인트

### 1. 핵심 정의

> **LED**: 전계발광 현상으로 빛을 내는 반도체 소자
> **전계발광**: 전자-정공 재결합 시 빛 에너지 방출

### 2. 빈출 키워드

- **밴드갭**: 에너지 간격, 색상 결정
- **발광 효율**: lm/W 단위, LED 80-200 lm/W
- **PWM**: 펄스 폭 변조, 밝기 제어
- **색온도**: 광원의 색감, 2700K(따뜻함)~6500K(차가움)

**Q1. LED의 발광 원리를 설명하시오.**

**Q2. LED가 백열등보다 효율이 높은 이유를 기술하시오.**

**Q3. 백색 LED를 구현하는 방법을 설명하시오.**

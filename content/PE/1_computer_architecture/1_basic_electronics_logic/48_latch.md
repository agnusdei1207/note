# 래치 (Latch)

## 정의

**래치(Latch)**는 **1비트 정보를 저장**하는 기본적인 순차 논리회로다. 레벨 트리거 방식으로 동작하며, Enable 신호가 활성화된 동안 입력 변화가 출력에 반영된다. 플립플롭의 기본 구성 요소이며 비동기식 메모리 요소다.

```
┌─────────────────────────────────────────────────────────────────┐
│           래치 기본 개념                                          │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   ┌─────────────────────────────────────────────────────────┐ │
│   │                                                         │ │
│   │   기본 구조:                                             │ │
│   │                                                         │ │
│   │           ┌─────────────────────┐                       │ │
│   │   입력 ──▶│                     │──▶ Q (출력)           │ │
│   │           │      래치          │                        │ │
│   │   Enable ─│    (Level Trigger) │                        │ │
│   │           │                     │                        │ │
│   │           └──────────┬──────────┘                       │ │
│   │                      │                                   │ │
│   │                      └──▶ Q' (반전 출력)                 │ │
│   │                                                         │ │
│   │   특징:                                                  │ │
│   │   ┌───────────────────────────────────────────────┐    │ │
│   │   │ • 1비트 저장                                    │    │ │
│   │   │ • 레벨 트리거 동작                              │    │ │
│   │   │ • Enable 신호로 제어                            │    │ │
│   │   │ • 피드백 루프 구조                              │    │ │
│   │   │ • 비동기식 (클럭 에지 불필요)                    │    │ │
│   │   └───────────────────────────────────────────────┘    │ │
│   │                                                         │ │
│   └─────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

## 핵심 개념

### 1. 래치 종류

```
┌─────────────────────────────────────────────────────────────────┐
│           래치 종류                                               │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   ┌─────────────────────────────────────────────────────────┐ │
│   │                                                         │ │
│   │   SR 래치 (Set-Reset Latch):                            │ │
│   │   ┌───────────────────────────────────────────────┐    │ │
│   │   │ • S(Set)와 R(Reset) 입력                       │    │ │
│   │   │ • S=1: Q=1 (Set)                               │    │ │
│   │   │ • R=1: Q=0 (Reset)                             │    │ │
│   │   │ • S=R=1: 금지 상태                              │    │ │
│   │   └───────────────────────────────────────────────┘    │ │
│   │                                                         │ │
│   │   D 래치 (Data Latch):                                  │ │
│   │   ┌───────────────────────────────────────────────┐    │ │
│   │   │ • D(Data)와 Enable 입력                        │    │ │
│   │   │ • Enable=1: Q=D                                │    │ │
│   │   │ • Enable=0: Q 유지                              │    │ │
│   │   │ • SR 래치의 금지 상태 해결                       │    │ │
│   │   └───────────────────────────────────────────────┘    │ │
│   │                                                         │ │
│   │   Gated SR 래치:                                         │ │
│   │   ┌───────────────────────────────────────────────┐    │ │
│   │   │ • SR 래치 + Enable 게이트                       │    │ │
│   │   │ • Enable=1일 때만 S/R 입력 반영                 │    │ │
│   │   └───────────────────────────────────────────────┘    │ │
│   │                                                         │ │
│   │   Gated D 래치:                                          │ │
│   │   ┌───────────────────────────────────────────────┐    │ │
│   │   │ • D 래치 + Enable 게이트                        │    │ │
│   │   │ • 가장 널리 사용되는 래치                       │    │ │
│   │   └───────────────────────────────────────────────┘    │ │
│   │                                                         │ │
│   └─────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

## 기술사 시험 포인트

### 1. 핵심 정의

> **래치**: 1비트를 저장하는 레벨 트리거 순차 회로
> **레벨 트리거**: 활성 레벨 구간 내 지속적 입력 반영
> **피드백 루프**: 상태 유지를 위한 크로스커플링 구조

### 2. 빈출 키워드

- **Level Trigger**: 레벨 트리거 동작
- **Transparent Mode**: 투명 모드 (Enable=1)
- **Cross-Coupling**: 크로스커플링 (피드백)
- **SR/D Latch**: 래치의 주요 종류

### 3. 예상 문제

**Q1. 래치와 플립플롭의 차이점을 비교하고 각각의 적합한 응용을 기술하시오.**

**Q2. SR 래치의 특성 테이블을 작성하고 NOR 게이트로 구현하시오.**

**Q3. 래치의 피드백 루프 구조가 상태 유지에 어떻게 기여하는지 설명하시오.**

# 에지 트리거 (Edge Trigger)

## 정의

**에지 트리거(Edge Trigger)**는 클럭 신호의 **상승 또는 하강 에지(전이 구간)**에서만 데이터를 샘플링하는 동기 방식이다. 신호가 특정 레벨이 아닌 변화하는 순간에 동작하므로 한 클럭 사이클에 한 번만 상태 변화가 발생한다.

```
┌─────────────────────────────────────────────────────────────────┐
│           에지 트리거 동작                                        │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   ┌─────────────────────────────────────────────────────────┐ │
│   │                                                         │ │
│   │   클럭 신호:                                             │ │
│   │                                                         │ │
│   │   전압                                                   │ │
│   │    Vdd ──┐   ┌───┐   ┌───┐   ┌───┐                    │ │
│   │           │   │   │   │   │   │   │                    │ │
│   │    0V  ──┘   └───┘   └───┘   └───┘                    │ │
│   │           │   │   │   │   │   │   │                    │ │
│   │           └───┘   └───┘   └───┘   └───                 │ │
│   │            ↑     ↑     ↑     ↑                         │ │
│   │         상승   하강  상승  하강                          │ │
│   │         에지   에지  에지  에지                          │ │
│   │           │     │     │     │                           │ │
│   │           ▼     ▼     ▼     ▼                           │ │
│   │         샘플링 샘플링 샘플링 샘플링                       │ │
│   │                                                         │ │
│   └─────────────────────────────────────────────────────────┘ │
│                                                                 │
│   에지 종류:                                                     │
│   ┌─────────────────────────────────────────────────────────┐ │
│   │   상승 에지 (Rising Edge / Positive Edge):              │ │
│   │   • 0 → 1 전이 시점                                    │ │
│   │   • 가장 일반적으로 사용                                │ │
│   │                                                         │ │
│   │   하강 에지 (Falling Edge / Negative Edge):             │ │
│   │   • 1 → 0 전이 시점                                    │ │
│   │   • 일부 시스템에서 사용                                │ │
│   └─────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

## 핵심 개념

### 1. 에지 트리거 동작 원리

```
┌─────────────────────────────────────────────────────────────────┐
│           D 플립플롭 에지 트리거                                  │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   상승 에지 트리거 D 플립플롭:                                   │
│   ┌─────────────────────────────────────────────────────────┐ │
│   │                                                         │ │
│   │   D ─────┐                                              │ │
│   │          │                                              │ │
│   │          │    ┌─────────────────┐                       │ │
│   │          └───▶│   D 플립플롭    │───▶ Q                 │ │
│   │               │  (상승 에지)   │                        │ │
│   │   CLK ───────▶│    ▲ 트리거    │                        │ │
│   │               └─────────────────┘                       │ │
│   │                                                         │ │
│   │   동작: 상승 에지에서 D → Q 전송                         │ │
│   │                                                         │ │
│   └─────────────────────────────────────────────────────────┘ │
│                                                                 │
│   타이밍 다이어그램:                                             │
│   ┌─────────────────────────────────────────────────────────┐ │
│   │                                                         │ │
│   │   CLK ────┐     ┌─────┐     ┌─────┐                    │ │
│   │            └─────┘     └─────┘     └─────               │ │
│   │            ↑           ↑                               │ │
│   │            │           │                               │ │
│   │   D   ─────┼───────────┼─────────────────────────      │ │
│   │        ────┘           └─────────────────               │ │
│   │            │           │                               │ │
│   │   Q   ─────┼───────────┼─────────────────               │ │
│   │        ────┘           └─────────────────               │ │
│   │            │           │                               │ │
│   │            ▼           ▼                               │ │
│   │         D 샘플링    D 샘플링                            │ │
│   │                                                         │ │
│   └─────────────────────────────────────────────────────────┘ │
│                                                                 │
│   💡 에지 순간의 D 값만 Q로 전송, 그 외 시점은 무시             │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### 2. 에지 트리거 vs 레벨 트리거

```
┌─────────────────────────────────────────────────────────────────┐
│           에지 트리거 vs 레벨 트리거                               │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   레벨 트리거 (Level Trigger):                                  │
│   ┌─────────────────────────────────────────────────────────┐ │
│   │                                                         │ │
│   │   CLK ────┐     ┌─────┐     ┌─────┐                    │ │
│   │            └─────┘     └─────┘     └─────               │ │
│   │            │←High→│   │←High→│                          │ │
│   │            │  구간 │   │  구간 │                         │ │
│   │            │       │   │       │                         │ │
│   │   D   ─────┼───┬───┼───┼───┬───┼───                     │ │
│   │        ────┘   │   └───┘   │   └───                     │ │
│   │                │           │                             │ │
│   │   Q   ─────────┼───────────┼───                         │ │
│   │        ────────┘           └───                         │ │
│   │                │           │                             │ │
│   │          High 구간 내내   High 구간 내내                  │ │
│   │          D 변화 반영      D 변화 반영                     │ │
│   │                                                         │ │
│   └─────────────────────────────────────────────────────────┘ │
│                                                                 │
│   비교:                                                         │
│   ┌─────────────────────────────────────────────────────────┐ │
│   │  구분        │ 에지 트리거        │ 레벨 트리거          │ │
│   ├──────────────┼─────────────────────┼────────────────────┤ │
│   │ 동작 시점    │ 전이 순간          │ 레벨 유지 구간      │ │
│   │ 샘플링 횟수  │ 사이클당 1회       │ 구간 내 지속        │ │
│   │ 입력 민감도  │ 순간만 민감         │ 구간 내 지속 민감   │ │
│   │ 회로 복잡도  │ 높음               │ 낮음                │ │
│   │ 노이즈 내성  │ 높음               │ 낮음                │ │
│   │ 적용         │ 플립플롭           │ 래치                │ │
│   └──────────────┴─────────────────────┴────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

│   │                                                         │ │

│                                                                 │
│   에지 트리거 효과:                                              │

## 기술사 시험 포인트

### 1. 핵심 정의

> **에지 트리거**: 클럭 전이 시점(상승/하강)에만 동작
> **상승 에지**: 0→1 전이 (가장 일반적)
> **하강 에지**: 1→0 전이

### 2. 빈출 키워드

- **Rising/Falling Edge**: 상승/하강 에지
- **One-shot**: 한 클럭에 한 번만 동작
- **Edge Detection**: 에지 검출 회로
- **Master-Slave**: 에지 트리거 효과를 위한 구조

### 3. 예상 문제

**Q1. 에지 트리거와 레벨 트리거의 차이점을 설명하고 각각의 장단점을 기술하시오.**

**Q2. 상승 에지 검출 회로의 동작 원리를 설명하시오.**

**Q3. 마스터-슬레이브 플립플롭이 에지 트리거 효과를 내는 원리를 설명하시오.**

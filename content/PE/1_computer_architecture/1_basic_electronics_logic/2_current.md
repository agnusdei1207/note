# 전류 (Current)

## 정의

**전류(Current)**는 도체 내에서 전하의 흐름 속도를 나타내는 물리량으로, 단위 시간당 특정 단면을 통과하는 전하의 양으로 정의된다. 국제단위계(SI)에서는 암페어(Ampere, 기호: A)를 사용하며, 1A는 1초당 1쿨롬(Coulomb)의 전하가 이동하는 전류량으로 정의된다.

```
수식: I = Q/t = dQ/dt
- I: 전류 (Ampere, A)
- Q: 전하량 (Coulomb, C)
- t: 시간 (second, s)
```

컴퓨터 아키텍처 관점에서 전류는 트랜지스터 스위칭 동작 시 발생하는 전하 이동을 의미하며, 전력 소모(Power)의 직접적인 구성 요소로 작용한다. 현대 고성능 CPU는 수십~수백 암페어의 전류를 소모하며, 이는 전원 공급 장치(PSU), 전압 조정기(VRM), 그리고 PCB 전원 평면 설계에 결정적인 영향을 미친다.

## 핵심 개념

### 1. 전류의 본질과 전하 이동

```
    [ 전류의 미시적 관점 ]

    도체 내부 구조:
    +------------------------------------------------------+
    |                                                      |
    |   +    -    +    -    +    -    +    -    +    -    |
    |   |    |    |    |    |    |    |    |    |    |    |
    |   |    |    |    |    |    |    |    |    |    |    |
    |   v    v    v    v    v    v    v    v    v    v    |
    |   e-   e-   e-   e-   e-   e-   e-   e-   e-   e-   | <-- 자유 전자
    |   |    |    |    |    |    |    |    |    |    |    |
    |   |    |    |    |    |    |    |    |    |    |    |
    |  (+)  (+)  (+)  (+)  (+)  (+)  (+)  (+)  (+)  (+)   | <-- 금속 원자핵
    |                                                      |
    +------------------------------------------------------+
          |                                         |
          |-------- 전자 이동 방향 (음극->양극) ------|
                    I (전류) 방향 (양극->음극)

    * 관습적 전류 방향: 양전하가 이동하는 방향 (+ → -)
    * 실제 전자 이동 방향: 음전하가 이동하는 방향 (- → +)
```

- **전자 전류(Electron Current)**: 실제 자유 전자가 이동하는 방향. 음극에서 양극으로 흐름
- **관습적 전류(Conventional Current)**: 양전하가 이동하는 것으로 가정한 방향. 양극에서 음극으로 흐름. 회로 이론에서 사용
- **정공(Hole)**: 반도체에서 전자가 빠져나간 자리로, 양전하처럼 행동하는 가상 입자. P형 반도체의 주요 전하 운반자

### 2. 전류의 종류

```
    [ 전류의 분류 ]

    1. 전도 전류 (Conduction Current)
       - 도체 내 자유 전자의 이동에 의한 전류
       - 금속 도선, 저항 등에서 발생
       - I = n * A * v * q (n: 전자 밀도, A: 단면적, v: 드리프트 속도, q: 전하량)

    2. 변위 전류 (Displacement Current)
       - 커패시터 내 전기장 변화에 의한 전류
       - 실제 전하 이동 없이 전기장 변화가 전류처럼 작용
       - I = C * dV/dt (C: 정전용량, dV/dt: 전압 변화율)

    3. 확산 전류 (Diffusion Current)
       - 농도 구배에 의한 전하 이동
       - 반도체 접합부(PN Junction)에서 중요

    4. 표류 전류 (Drift Current)
       - 전기장에 의한 전하 이동
       - 반도체 내 전자와 정공의 이동
```

| 전류 유형 | 발생 메커니즘 | 주요 발생 위치 |
|:---:|:---|:---|
| **전도 전류** | 자유 전자의 이동 | 금속 도선, 저항 |
| **변위 전류** | 전기장의 시간적 변화 | 커패시터 유전체 |
| **확산 전류** | 농도 구배에 의한 확산 | 반도체 접합부 |
| **표류 전류** | 전기장에 의한 가속 | 반도체 벌크 영역 |

### 3. 직류와 교류 전류

```
    [ DC 전류 vs AC 전류 ]

    직류 (DC)                              교류 (AC)
    +I |-----------------------          +Im |    /\    /\    /\
       |                                    |   /  \  /  \  /  \
    0  |-----------------------            |  /    \/    \/    \
       |                                 0  |----------------------
       |                                    |
    -I |                                 -Im |

    [ AC 전류의 주요 파라미터 ]

    i(t) = Im * sin(ωt + φ)
    - Im: 최대 전류 (Peak Current)
    - ω = 2πf: 각주파수 (Angular Frequency)
    - φ: 위상 (Phase)

    실효값 (RMS, Root Mean Square):
    I_rms = Im / √2 ≈ 0.707 * Im (정현파의 경우)
```

### 4. 전류 밀도 (Current Density)

```
    [ 전류 밀도의 개념 ]

    단면적 A를 통과하는 전류 I
    +------------------+
    |                  |  ^
    |   전류 흐름      |  | I (전류)
    |   방향 --->      |  |
    |                  |  v
    +------------------+
          단면적 A

    전류 밀도: J = I / A (A/m²)

    [ 반도체 소자에서의 전류 밀도 ]
    - MOSFET 채널: ~10⁶ A/cm²
    - 금속 배선(알루미늄): ~10⁵ A/cm²
    - 금속 배선(구리): ~10⁶ A/cm²
    - 탄소나노튜브: ~10⁷ A/cm²
```

## 상세 설명

### 1. 옴의 법칙과 전류-전압 관계

```
    [ 옴의 법칙 (Ohm's Law) ]

    선형 저항에서의 V-I 관계:
    V = I * R  또는  I = V / R

    전압(V)                              기울기 = 1/R (컨덕턴스)
      ^                                /
      |                              /
      |                            /
      |                          /
      |                        /
      |                      /
      |                    /
      +----------------------------> 전류(I)

    [ 비선형 소자의 V-I 특성 ]

    다이오드 (Diode)                   MOSFET (포화 영역)
    전압(V)                            전류(I)
      ^                                  ^        _______
      |                    /             |       /
      |                  /               |      /
      |                /                 |     /
      |              /                   |____/
      +-------------> 전류(I)            +--------------> 전압(V)
         순방향                        (V_gs > V_th)
```

- **옴의 법칙(Ohm's Law)**: V = IR. 선형 저항에서 전압과 전류의 정비례 관계
- **컨덕턴스(Conductance)**: G = 1/R. 전류가 흐르기 쉬운 정도. 단위: 지멘스(S)
- **비선형 소자**: 다이오드, 트랜지스터 등 옴의 법칙을 따르지 않는 소자

### 2. CMOS 회로에서의 전류 성분

```
    [ CMOS 인버터의 전류 성분 ]

    VDD
      |
    +-+-+
    |PMOS|  <-- 충전 전류 (Charge Current)
    +---+
      |--------+------> Vout
      |        |
    +-+---+    +---+
    | NMOS|    | CL |  <-- 부하 커패시턴스
    +-----+    +---+
      |          |
    GND        GND

    [ 스위칭 전류 파형 ]

    I
    ^            +--------+
    |           /          \
    |          /            \
    |   _____/              \_____
    +------------------------------> 시간
           충전    방전

    동적 전류 (스위칭 시):
    I_dynamic = C * dV/dt

    - C: 부하 커패시턴스
    - dV/dt: 출력 전압 변화율

    * 전류 스파이크 (Current Spike):
      - 출력 전압 천이 시 PMOS와 NMOS가 동시에 도통하는 순간
      - 관통 전류 (Shoot-through Current) 발생
      - 전원 노이즈의 주원인
```

**CMOS 전류 성분 분류**:

| 전류 성분 | 원인 | 수식/특성 | 설계 고려사항 |
|:---:|:---|:---|:---|
| **충전/방전 전류** | 커패시턴스 충방전 | I = C·dV/dt | 동적 전력 주원인 |
| **관통 전류** | 스위칭 천이 구간 | I_short-circuit | 트랜지스터 사이징 최적화 |
| **누설 전류** | 역바이어스 누설 | I_leak ∝ e^(Vgs-Vth) | 정적 전력 주원인 |
| **게이트 누설** | 터널링 효과 | I_gate ∝ e^(-tox) | 산화막 두께 영향 |

### 3. 전력 소모와 전류의 관계

```
    [ 전력 소모 공식 ]

    전력 = 전압 × 전류
    P = V × I

    저항에서: P = I²R = V²/R

    CMOS 동적 전력:
    P_dynamic = C × V² × f
              = (C × V) × V × f
              = Q × V × f
              = (I_avg × t) × V × f

    평균 전류:
    I_avg = C × V × f × α
    - C: 부하 커패시턴스
    - V: 전압
    - f: 클럭 주파수
    - α: 활동률 (Activity Factor)

    [ CPU 코어 전류 예시 ]

    코어 수: 8개
    클럭: 3.0 GHz
    V_core: 1.2V
    코어당 커패시턴스: ~10nF (추정)
    활동률: 0.3 (30%)

    I_avg = C × V × f × α × n
          = 10nF × 1.2V × 3GHz × 0.3 × 8
          = 10×10⁻⁹ × 1.2 × 3×10⁹ × 0.3 × 8
          = 86.4 A (코어만)
```

## 실무 적용

### 1. VRM(전압 조정기 모듈) 설계에서의 전류 고려사항

```
    [ 다상 VRM 구조 ]

    +----------------------------------------------------------+
    |                    다상(VRM) 구조                         |
    +----------------------------------------------------------+
    |                                                          |
    |    12V 입력                                               |
    |      |                                                    |
    |      +-------------+-------------+-------------+          |
    |      |             |             |             |          |
    |  +---+---+     +---+---+     +---+---+     +---+---+      |
    |  | 상 #1 |     | 상 #2 |     | 상 #3 |     | 상 #4 |      |
    |  |(MOSFET|     |(MOSFET|     |(MOSFET|     |(MOSFET|      |
    |  |+인덕터)|     |+인덕터)|     |+인덕터)|     |+인덕터)|      |
    |  +---+---+     +---+---+     +---+---+     +---+---+      |
    |      |             |             |             |          |
    |      +-------------+-------------+-------------+          |
    |                          |                                |
    |                     V_out (CPU)                           |
    |                          |                                |
    |                    +-----+-----+                          |
    |                    |    CPU    |                          |
    |                    +-----------+                          |
    +----------------------------------------------------------+

    [ 다상 VRM의 장점 ]

    1. 전류 분산: 각 상이 전체 전류의 1/N을 담당
       - 4상 VRM: 각 상이 25%씩 분담
       - 열 분산으로 효율 향상

    2. 리플 감소: 위상차에 의한 리플 상쇄
       - 4상: 리플 4배 감소 (이론적)
       - f_ripple = N * f_switching

    3. 과전류 대응: 단일 MOSFET 한계 극복
       - 고전류 애플리케이션 필수
```

**VRM 설계 체크리스트**:
| 항목 | 고려사항 | 일반적 요구사항 |
|:---:|:---|:---|
| **최대 전류** | CPU/GPU TDP 기준 | 100~400A+ |
| **전류 리플** | 출력 전압 안정성 | < 50mV_pp |
| **과전류 보호(OCP)** | 단락/과부하 보호 | 120~150% 정격 |
| **효율** | 발열 및 전력 손실 | 85~95% 이상 |
| **트랜션트 응답** | 급격한 부하 변화 대응 | < 10μs |

### 2. PCB 전원 설계에서의 전류 처리

```
    [ PCB 전원 평면 설계 ]

    +----------------------------------------------------------+
    |                      PCB 단면도                           |
    +----------------------------------------------------------+
    |                                                          |
    |    [전원 평면]         [접지 평면]                        |
    |    +-----------+       +-----------+                      |
    |    |    VDD    |       |    GND    |                      |
    |    |  (Plane)  |       |  (Plane)  |                      |
    |    +-----------+       +-----------+                      |
    |                                                          |
    |    전류 밀도 고려사항:                                    |
    |    - 트레이스 폭: I = k × ΔT^0.44 × A^0.725              |
    |    - 비아(Via) 개수: 전류 용량 확보                       |
    |    - 열 방출: 전원 평면의 열 설계                         |
    |                                                          |
    +----------------------------------------------------------+

    [ 전류 용량 계산 (IPC-2221 표준) ]

    외부 트레이스 폭 계산:
    A = (I / (k × ΔT^0.44))^1/0.725

    - I: 전류 (A)
    - k: 0.048 (외부 트레이스)
    - ΔT: 허용 온도 상승 (°C)
    - A: 단면적 (mil²)

    예: 10A, ΔT=10°C
    A = (10 / (0.048 × 10^0.44))^1/0.725
      ≈ 300 mil² ≈ 200 mm²
      ≈ 200mm 폭 (1oz 동박)
```

### 4. 전원 공급 장치(PSU) 전류 용량 산정

```
    [ 서버 PSU 용량 계산 예시 ]

    구성 요소           수량   전력(W)   12V 전류(A)   5V 전류(A)
```

## 기술사 시험 포인트

### 1. 핵심 정의 및 공식
- **전류 정의**: I = dQ/dt (단위 시간당 전하 이동량), 단위: 암페어(A)
- **옴의 법칙**: I = V/R (전류 = 전압/저항)
- **전류 밀도**: J = I/A (단위 면적당 전류, A/m²)
- **전력 공식**: P = VI = I²R (전류 제곱에 비례)

### 2. 시험 빈출 키워드
1. **전류 밀도(J)**: 단위 면적당 흐르는 전류. 반도체 소자 신뢰성의 핵심 지표
2. **누설 전류(Leakage Current)**: 꺼진 상태에서도 흐르는 미세 전류. 정적 전력 소모의 주원인
3. **관통 전류(Shoot-through Current)**: CMOS 스위칭 시 PMOS와 NMOS가 동시 도통하는 전류
4. **전동력(Electromotive Force, EMF)**: 전류를 흐르게 하는 힘. 전압과 같은 의미로 사용
5. **전자 이동도(Electron Mobility)**: 단위 전기장에서 전자의 드리프트 속도

### 3. 단답형 예상 문제
- **Q**: CMOS 회로에서 동적 전력 소모와 전류의 관계를 설명하시오.
- **A**: CMOS 회로의 동적 전력은 P = CV²f로 표현되며, 평균 전류는 I_avg = CVfα(α: 활동률)이다. 즉, 커패시턴스(C), 전압(V), 클럭 주파수(f)가 증가할수록 전류 소모가 증가한다.

### 4. 논술형 예상 문제
- **Q**: 고성능 CPU에서 전류 소모가 증가함에 따른 설계상의 문제점과 해결 방안을 기술하시오.
- **A**:
  1. **문제점**:
     - VRM 과부하 및 열 발생
     - PCB 전원 평면의 전류 밀도 초과
     - 전원 커넥터/케이블 용량 한계
     - 전압 강하(IR Drop)로 인한 안정성 저하

  2. **해결 방안**:
     - 다상 VRM 설계로 전류 분산
     - 전원/접지 평면 다층 설계
     - 고효율 전원 커넥터(12VHPWR 등) 도입
     - 적극적 전력 관리(DVFS, Power Gating)

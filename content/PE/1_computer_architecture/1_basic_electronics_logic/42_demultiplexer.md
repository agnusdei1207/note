# 디멀티플렉서 (Demultiplexer, DEMUX)

## 정의

**디멀티플렉서(Demultiplexer, DEMUX)**는 단일 입력 신호를 받아 **여러 출력선 중 하나로 전달**하는 조합 논리회로다. 멀티플렉서의 역동작이며, 데이터 분배기(Data Distributor)라고도 한다.

```
┌─────────────────────────────────────────────────────────────────┐
│           1:4 디멀티플렉서 구조                                   │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   ┌─────────────────────────────────────────────────────────┐ │
│   │                                                         │ │
│   │                  ┌──▶ Y0 (출력)                         │ │
│   │                  │                                      │ │
│   │   입력 D ──▶ [ 1:4 DEMUX ] ──▶ Y1 (출력)               │ │
│   │                   │   │                                 │ │
│   │                   │   └──▶ Y2 (출력)                    │ │
│   │                   │                                      │ │
│   │                   └───▶ Y3 (출력)                       │ │
│   │                    ▲   ▲                                 │ │
│   │                    │   │                                 │ │
│   │   선택선:         S1  S0                                 │ │
│   │                                                         │ │
│   └─────────────────────────────────────────────────────────┘ │
│                                                                 │
│   선택 신호에 따른 출력:                                         │
│   ┌─────────────────────────────────────────────────────────┐ │
│   │   S1  S0  │    활성 출력                                │ │
│   │   ────────┼─────────────────                            │ │
│   │    0   0  │    Y0 = D, 나머지 0                         │ │
│   │    0   1  │    Y1 = D, 나머지 0                         │ │
│   │    1   0  │    Y2 = D, 나머지 0                         │ │
│   │    1   1  │    Y3 = D, 나머지 0                         │ │
│   └─────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

## 핵심 개념

### 1. 디멀티플렉서 내부 회로

```
┌─────────────────────────────────────────────────────────────────┐
│           1:4 DEMUX 내부 회로 구현                                │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   논리식:                                                       │
│   Y0 = D · S1' · S0'                                           │
│   Y1 = D · S1' · S0                                            │
│   Y2 = D · S1 · S0'                                            │
│   Y3 = D · S1 · S0                                             │
│                                                                 │
│   ┌─────────────────────────────────────────────────────────┐ │
│   │                                                         │ │
│   │          ┌──[AND]──▶ Y0                                 │ │
│   │          │    ↑                                         │ │
│   │   D ─────┼──┬─S1'                                      │ │
│   │          │  └─S0'                                       │ │
│   │          │                                              │ │
│   │          ├──[AND]──▶ Y1                                 │ │
│   │          │    ↑                                         │ │
│   │          ├──┬─S1'                                       │ │
│   │          │  └─S0                                        │ │
│   │          │                                              │ │
│   │          ├──[AND]──▶ Y2                                 │ │
│   │          │    ↑                                         │ │
│   │          ├──┬─S1                                        │ │
│   │          │  └─S0'                                       │ │
│   │          │                                              │ │
│   │          └──[AND]──▶ Y3                                 │ │
│   │               ↑                                         │ │
│   │          ┌───┴───┐                                      │ │
│   │          S1      S0                                     │ │
│   │                                                         │ │
│   └─────────────────────────────────────────────────────────┘ │
│                                                                 │
│   💡 디코더와 유사: 입력 D가 Enable 역할                        │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### 2. DEMUX와 디코더 관계

```
┌─────────────────────────────────────────────────────────────────┐
│           DEMUX = Decoder + Enable                              │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   디코더에 Enable 입력 추가 = 디멀티플렉서                       │
│                                                                 │
│   ┌─────────────────────────────────────────────────────────┐ │
│   │                                                         │ │
│   │   1:4 DEMUX를 2:4 디코더로 구현:                         │ │
│   │                                                         │ │
│   │        D (데이터)                                       │ │
│   │         │                                               │ │
│   │         └──────▶ [ 2:4 Decoder ]                        │ │
│   │                        ↑                                │ │
│   │         Enable (D를 Enable로 사용)                      │ │
│   │                                                         │ │
│   │   S1 ───┐                                               │ │
│   │   S0 ───┴──▶ 선택 입력                                  │ │
│   │                                                         │ │
│   │   출력: Y0, Y1, Y2, Y3                                  │ │
│   │                                                         │ │
│   └─────────────────────────────────────────────────────────┘ │
│                                                                 │
│   1:8 DEMUX:                                                    │
│   ┌─────────────────────────────────────────────────────────┐ │
│   │                                                         │ │
│   │   D ──▶ [ 3:8 Decoder with Enable ] ──▶ Y0~Y7          │ │
│   │                   ↑                                     │ │
│   │         S2, S1, S0 (선택 입력)                          │ │
│   │                                                         │ │
│   └─────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### 3. DEMUX 종류

```
┌─────────────────────────────────────────────────────────────────┐
│           디멀티플렉서 종류                                       │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   ┌─────────────────────────────────────────────────────────┐ │
│   │  종류      │ 출력수 │ 선택선 │ 입력  │ 용도             │ │
│   ├────────────┼────────┼────────┼───────┼──────────────────┤ │
│   │  1:2 DEMUX │   2    │   1    │  1    │ 기본 분배        │ │
│   │  1:4 DEMUX │   4    │   2    │  1    │ 니블 분배        │ │
│   │  1:8 DEMUX │   8    │   3    │  1    │ 바이트 분배      │ │
│   │  1:16 DEMUX│   16   │   4    │  1    │ 워드 분배        │ │
│   └────────────┴────────┴────────┴───────┴──────────────────┘ │
│                                                                 │
│   1:2 DEMUX (기본):                                             │
│   ┌─────────────────────────────────────────────────────────┐ │
│   │                                                         │ │
│   │   Y0 = D · S'                                           │ │
│   │   Y1 = D · S                                            │ │
│   │                                                         │ │
│   │   D ───┬──[AND]──▶ Y0                                   │ │
│   │        │    ↑                                           │ │
│   │        └───S'                                           │ │
│   │                                                         │ │
│   │   D ───┬──[AND]──▶ Y1                                   │ │
│   │        │    ↑                                           │ │
│   │        └───S                                            │ │
│   │                                                         │ │
│   └─────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### 4. DEMUX 확장 및 응용

```
┌─────────────────────────────────────────────────────────────────┐
│           DEMUX 확장 (계층적 구조)                                │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   1:8 DEMUX를 1:4 DEMUX 2개 + 1:2 DEMUX 1개로 구현:             │
│                                                                 │
│   ┌─────────────────────────────────────────────────────────┐ │
│   │                                                         │ │
│   │                    ┌─▶ Y0~Y3                            │ │
│   │   D ──▶ [1:4 DEMUX]─┤     ↑                             │ │
│   │              ↑      └  S1,S0                            │ │
│   │              │                                          │ │
│   │          [1:2 DEMUX]                                    │ │
│   │              ↑        ┌─▶ Y4~Y7                         │ │
│   │              │        │    ↑                            │ │
│   │             S2 ──▶ ───┴─ S1,S0                          │ │
│   │                                                         │ │
│   └─────────────────────────────────────────────────────────┘ │
│                                                                 │
├─────────────────────────────────────────────────────────────────┤
│           DEMUX 응용 분야                                        │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   직렬-병렬 변환:                                                │
│   ┌─────────────────────────────────────────────────────────┐ │
│   │                                                         │ │
│   │   직렬 데이터 ──▶ [DEMUX] ──▶ 병렬 출력 (N비트)         │ │
│   │                   ↑                                     │ │
│   │              [카운터]                                    │ │
│   │                                                         │ │
│   │   D(시간순) ──▶ [1:8 DEMUX] ──▶ Y0~Y7 (동시 출력)       │ │
│   │                    ↑                                    │ │
│   │               [3비트 카운터]                             │ │
│   │                                                         │ │
│   └─────────────────────────────────────────────────────────┘ │
│                                                                 │
│   메모리 주소 디코딩:                                            │
│   ┌─────────────────────────────────────────────────────────┐ │
│   │                                                         │ │
│   │   상위 주소 ──▶ [DEMUX] ──▶ 메모리 뱅크 선택            │ │
│   │                   │                                     │ │
│   │                   ├──▶ Bank 0 (0-1KB)                   │ │
│   │                   ├──▶ Bank 1 (1-2KB)                   │ │
│   │                   ├──▶ Bank 2 (2-3KB)                   │ │
│   │                   └──▶ Bank 3 (3-4KB)                   │ │
│   │                                                         │ │
│   └─────────────────────────────────────────────────────────┘ │
│                                                                 │
│   I/O 장치 선택:                                                 │
│   ┌─────────────────────────────────────────────────────────┐ │
│   │                                                         │ │
│   │   CPU 데이터 ──▶ [DEMUX] ──▶ 장치별 데이터 라인         │ │
│   │                    │                                    │ │
│   │                    ├──▶ 장치 0                          │ │
│   │                    ├──▶ 장치 1                          │ │
│   │                    └──▶ 장치 2                          │ │
│   │                                                         │ │
│   └─────────────────────────────────────────────────────────┘ │
│                                                                 │
│   TDM 수신단:                                                    │
│   ┌─────────────────────────────────────────────────────────┐ │
│   │                                                         │ │
│   │   수신 데이터 ──▶ [DEMUX] ──▶ 채널별 분배               │ │
│   │                     │                                   │ │
│   │                     ├──▶ 채널 1                         │ │
│   │                     ├──▶ 채널 2                         │ │
│   │                     ├──▶ 채널 3                         │ │
│   │                     └──▶ 채널 4                         │ │
│   │                     ↑                                   │ │
│   │                   동기 신호                              │ │
│   │                                                         │ │
│   └─────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

## 기술사 시험 포인트

### 1. 핵심 정의

> **디멀티플렉서**: 단일 입력을 선택된 출력선으로 분배하는 회로
> **데이터 분배기**: DEMUX의 다른 명칭
> **MUX-DEMUX 관계**: MUX는 선택, DEMUX는 분배 (서로 역동작)

### 2. 빈출 키워드

- **1:N DEMUX**: 1개 입력, N개 출력 분배
- **Data Distributor**: 데이터 분배 기능
- **Decoder + Enable**: DEMUX 구현 방식
- **직렬-병렬 변환**: 통신 시스템 응용

### 3. 예상 문제

**Q1. 1:4 디멀티플렉서의 논리식을 유도하고 디코더와의 관계를 설명하시오.**

**Q2. 디멀티플렉서를 이용한 직렬-병렬 변환 회로를 설계하시오.**

**Q3. 멀티플렉서와 디멀티플렉서를 조합한 시분할 다중화(TDM) 시스템을 설명하시오.**

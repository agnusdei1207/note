# BJT (Bipolar Junction Transistor)

## 정의

**BJT(Bipolar Junction Transistor)**는 **전자와 정공 두 종류의 전하 운반자**를 사용하는 전류 제어형 트랜지스터다. 베이스(Base) 전류로 컬렉터(Collector) 전류를 제어하는 **전류 증폭 소자**다.

```
┌─────────────────────────────────────────────────────────────────┐
│                    BJT의 구조와 기호                            │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   NPN BJT:                                                     │
│                                                                 │
│              Collector (C)                                      │
│                   │                                             │
│              ┌────┴────┐                                        │
│              │    N    │ ←─── 전자(●) 공급                      │
│              ├─────────┤                                        │
│   Base (B) ──┤    P    │ ←─── 전자 통과 (얇은 층)              │
│              ├─────────┤                                        │
│              │    N    │ ←─── 전자 수집                          │
│              └────┬────┘                                        │
│                   │                                             │
│              Emitter (E)                                        │
│                                                                 │
│   기호:                                                         │
│                   C                                             │
│                   │                                             │
│                   │                                             │
│         B ───────┤                                              │
│                   │◀── E (NPN은 화살표 바깥쪽)                   │
│                   ▼                                             │
│                                                                 │
│   PNP BJT:                                                      │
│                   C                                             │
│                   │                                             │
│                   │                                             │
│         B ───────┤                                              │
│                   │──▶ E (PNP는 화살표 안쪽)                     │
│                   ▲                                             │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

## 핵심 개념

### 1. 전류 관계

```
┌─────────────────────────────────────────────────────────────────┐
│                   BJT 전류 관계                                 │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   NPN 트랜지스터 전류:                                          │
│                                                                 │
│                   Ic (컬렉터 전류)                              │
│                       │                                         │
│                       ▼                                         │
│              ┌────────────────┐                                 │
│              │                │                                 │
│              │     BJT        │                                 │
│              │                │                                 │
│              └────────────────┘                                 │
│                       ▲                                         │
│                       │                                         │
│          Ib ◀────────┴────────▶ Ie (이미터 전류)                │
│        (베이스      (이미터 전류)                                │
│         전류)                                                    │
│                                                                 │
│   전류 방정식:                                                  │
│   ┌─────────────────────────────────────────────────────────┐ │
│   │  Ie = Ib + Ic     (키르히호프 전류 법칙)                  │ │
│   │                                                         │ │
│   │  Ic = β × Ib        (β: 전류 증폭률, hFE)               │ │
│   │                                                         │ │
│   │  Ie = (1 + β) × Ib                                      │ │
│   │                                                         │ │
│   │  α = Ic / Ie        (α: 공통 베이스 증폭률)             │ │
│   │  β = α / (1 - α)                                        │ │
│   │  α ≈ 0.98 ~ 0.99, β ≈ 50 ~ 300                         │ │
│   └─────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### 2. 동작 모드

```
┌─────────────────────────────────────────────────────────────────┐
│                   BJT 동작 모드                                 │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   ┌─────────────────────────────────────────────────────────┐ │
│   │                                                         │ │
│   │   Ib                                                   │ │
│   │    │                                                    │ │
│   │    │      차단     활성      포화                       │ │
│   │    │       │       │         │                         │ │
│   │    │       │   ┌───┴───┐     │                         │ │
│   │    │       │   │       │     │                         │ │
│   │    ▼       ▼   ▼       ▼     ▼                         │ │
│   │   0 ──────┼───┼─────────┼─────▶ Ib                      │ │
│   │           │   │         │                               │ │
│   │           │   │         │                               │ │
│   │         Ib=0 Ib(on)   Ib(sat)                            │ │
│   │                                                         │ │
│   └─────────────────────────────────────────────────────────┘ │
│                                                                 │
│   ┌─────────────────────────────────────────────────────────┐ │
│   │  모드        │  조건              │  특성              │ │
│   ├───────────────┼────────────────────┼─────────────────────┤ │
│   │  차단 (Cutoff)│ Vbe < 0.5V         │ Ib ≈ 0, Ic ≈ 0     │ │
│   │               │ Vbc < 0            │ 스위치 OFF          │ │
│   │               │                    │ Vce ≈ Vcc          │ │
│   ├───────────────┼────────────────────┼─────────────────────┤ │
│   │  활성 (Active)│ Vbe ≈ 0.7V         │ Ic = β × Ib        │ │
│   │               │ Vbc < 0            │ 선형 증폭           │ │
│   │               │ Vce > Vce(sat)     │ 증폭기              │ │
│   ├───────────────┼────────────────────┼─────────────────────┤ │
│   │  포화 (Sat)   │ Vbe ≈ 0.7V         │ Ib > Ic/β          │ │
│   │               │ Vbc > 0            │ Vce ≈ 0.2V          │ │
│   │               │                    │ 스위치 ON           │ │
│   └───────────────┴────────────────────┴─────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

## 기술사 시험 포인트

### 1. 핵심 정의

> **BJT**: 전자와 정공 두 전하 운반자 사용, 전류 제어형 트랜지스터
> **β (전류 증폭률)**: Ic/Ib, 일반적으로 50~300

### 2. 빈출 키워드

- **α, β**: 공통 베이스/이미터 증폭률
- **동작 영역**: 차단, 활성, 포화
- **공통 이미터**: 가장 높은 이득, 위상 반전
- **Emitter Follower**: 전압 이득 ≈ 1, 임피던스 변환

### 3. 예상 문제

**Q1. BJT의 세 가지 동작 영역을 설명하고 각각의 용도를 기술하시오.**

**Q2. 공통 이미터 증폭기의 특징을 설명하시오.**

**Q3. BJT와 FET의 차이점을 비교하시오.**

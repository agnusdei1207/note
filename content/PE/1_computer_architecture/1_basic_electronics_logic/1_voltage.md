# 전압 (Voltage)

## 정의

**전압(Voltage)**은 전기 회로 내에서 전하가 흐르도록 하는 전기적 위치 에너지 차이(Electric Potential Difference)를 의미하며, 두 점 사이의 전위 차(Electromotive Force, EMF)로 정량화된다. 국제단위계(SI)에서는 볼트(Volt, 기호: V)를 사용하며, 1V는 1쿨롬(Coulomb)의 전하를 이동시키는 데 1줄(Joule)의 일을 수행하는 전위차로 정의된다.

```
수식: V = W/Q
- V: 전압 (Volt)
- W: 일 (Joule)
- Q: 전하량 (Coulomb)
```

컴퓨터 아키텍처 관점에서 전압은 트랜지스터의 스위칭 동작을 제어하는 핵심 물리량으로, 디지털 논리 회로에서 논리 '0'과 '1'을 구분하는 임계값(Threshold Voltage)을 결정하고, 전력 소모(Power Consumption) 및 열 발생(Heat Dissipation)에 직접적인 영향을 미친다.

## 핵심 개념

### 1. 전위와 전위차

```
    [ 전위(Potential)의 개념 ]

    높은 전위 (+)                                              낮은 전위 (-)
         |                                                         |
         |    +++++  전하 이동 방향 (전류의 반대)              -----|
         |    +++++  ---------------------------------->     -----|
         |                                                         |
         +--------------------- 도선 ------------------------------+

    * 전위: 전기장 내에서 단위 전하당 가지는 위치 에너지
    * 전위차(=전압): 두 점 사이의 전위 차이
    * 전하는 높은 전위에서 낮은 전위로 이동하려는 성질
```

- **전위(Electric Potential)**: 전기장 내의 한 점에서 단위 양전하(+)가 가지는 전위 에너지. 기준점(보통 무한원 또는 접지)에서의 전위를 0V로 정의
- **전위차(Voltage)**: 회로 내 두 지점 간의 전위 차이. 이 차이가 전하의 흐름(전류)을 유발하는 원동력이 됨
- **접지(Ground, GND)**: 회로의 기준 전위(0V)를 제공하는 접속점. 회로 설계에서 공통 반환 경로 역할

### 2. 전압의 종류

```
    [ 직류(DC) vs 교류(AC) 파형 ]

    직류 (DC: Direct Current)              교류 (AC: Alternating Current)
    +V |-----------------------           +V |    /\    /\    /\
      |      일정한 전압                     |   /  \  /  \  /  \
    0 |------------------------> 시간    0 |--/----\/----\/----\--> 시간
      |                                      |
    -V |                                    -V |

    * 컴퓨터 내부: DC 전압 사용 (+12V, +5V, +3.3V, +1.8V, +1.2V 등)
    * 전원 공급 장치(PSU): AC 220V 입력 -> DC 다중 레일 출력 변환
```

| 구분 | 직류(DC) | 교류(AC) |
|:---:|:---:|:---:|
| **정의** | 시간에 따라 극성이 변하지 않는 일정한 전압 | 시간에 따라 극성과 크기가 주기적으로 변화 |
| **파형** | 수평 직선 | 정현파(Sine Wave) |
| **용도** | 컴퓨터 내부 회로, 반도체, 배터리 기기 | 송전망, 가정용 콘센트 |
| **변환** | AC→DC: 정류기(Rectifier) | DC→AC: 인버터(Inverter) |

### 3. 디지털 회로에서의 전압 레벨

```
    [ CMOS 디지털 논리 레벨 (VDD = 3.3V 예시) ]

    VDD (3.3V) |=================================  (Power Rail)
               |
               |   VOH (2.4V) -------------------  High 출력 최소값
               |           HIGH (Logic 1) 영역
    VIH (2.0V) |--------------------------------  High 입력 최소값
               |   불확정 영역 (Undefined)        <- 노이즈 마진
    VIL (0.8V) |--------------------------------  Low 입력 최대값
               |           LOW (Logic 0) 영역
               |   VOL (0.4V) -------------------  Low 출력 최대값
    GND (0V)   |=================================  (Ground Rail)

    * 노이즈 마진 (Noise Margin):
      NMH = VIH - VOH (High 노이즈 마진)
      NML = VIL - VOL (Low 노이즈 마진)
```

- **VDD/VCC**: 회로의 전원 공급 전압 (Positive Supply Voltage)
- **GND/VSS**: 접지 전압 (0V 기준점)
- **VIH (Voltage Input High)**: 논리 '1'로 인식되는 최소 입력 전압
- **VIL (Voltage Input Low)**: 논리 '0'으로 인식되는 최대 입력 전압
- **VOH (Voltage Output High)**: 논리 '1' 출력 시 보장되는 최소 출력 전압
- **VOL (Voltage Output Low)**: 논리 '0' 출력 시 보장되는 최대 출력 전압
- **노이즈 마진(Noise Margin)**: 외부 간섭에도 논리 상태가 오동작하지 않는 전압 여유분

## 상세 설명

### 1. 전압과 전력 소모의 관계

컴퓨터 시스템에서 전압은 전력 소모와 밀접한 관계를 가진다. CMOS 회로의 전력 소모는 동적 전력( Dynamic Power)과 정적 전력(Static Power)으로 구분된다.

```
    [ CMOS 전력 소모 모델 ]

    동적 전력 (Dynamic Power):
    P_dynamic = a * C * V^2 * f

    - a: 활동률 (Activity Factor, 0~1)
    - C: 부하 정전용량 (Load Capacitance, Farad)
    - V: 공급 전압 (Voltage)
    - f: 클럭 주파수 (Frequency)

    정적 전력 (Static Power - 누설 전류):
    P_static = V * I_leak

    - I_leak: 누설 전류 (Leakage Current)

    ==> 전압(V)은 제곱으로 전력에 영향!
        V를 10% 낮추면 → 동적 전력은 약 19% 감소
```

**핵심 인사이트**: 전압을 낮추면 전력 소모가 제곱으로 감소하지만, 트랜지스터의 스위칭 속도도 느려져 최대 동작 클럭 주파수가 하락한다. 이것이 **전력-성능 트레이드오프(Power-Performance Tradeoff)**의 근본 원인이다.

### 2. 전압 조절 기술들

```
    [ 현대 CPU 전압 관리 구조 ]

    +--------------------------------------------------+
    |            전압 조정기 모듈 (VRM)                  |
    +--------------------------------------------------+
    |  +-------------+    +-------------+              |
    |  | PWM 컨트롤러 | -> | 다상 DC-DC  | -> V_core   |
    |  +-------------+    |  컨버터     |              |
    |         ^           +-------------+              |
    |         |                                        |
    |    VID (Voltage ID) 신호                        |
    |         |                                        |
    |  +------|-----------------------------------+   |
    |  |      v            CPU                    |   |
    |  |  +----------+   +------------------+     |   |
    |  |  | 전압 센서 | <-| DVFS/P-State     |     |   |
    |  |  +----------+   | 컨트롤러         |     |   |
    |  |                 +------------------+     |   |
    |  +------------------------------------------+   |
    +--------------------------------------------------+

    * VID (Voltage ID): CPU가 VRM에 요청하는 목표 전압값
    * DVFS (Dynamic Voltage/Frequency Scaling):
      - 부하에 따라 전압과 주파수를 실시간 조절
      - Intel SpeedStep, AMD Cool'n'Quiet 기술
```

| 기술명 | 설명 | 효과 |
|:---:|:---|:---|
| **DVFS** | 부하에 따라 전압과 클럭을 동적으로 스케일링 | 유휴 시 전력 절감 30~70% |
| **Power Gating** | 사용하지 않는 블록의 전원을 완전 차단 | 누설 전력 제거 |
| **AVS (Adaptive Voltage Scaling)** | 온도와 프로세스 편차에 따라 개별 칩 최적화 | 같은 성능에서 전력 10~20% 절감 |
| **Undervolting** | 안정성을 해치지 않는 범위에서 전압을 낮춤 | 발열 및 전력 소모 감소 |

### 3. CPU/GPU 전압 레일 구조

```
    [ 현대 프로세서 다중 전압 레일 ]

    +--------------------------------------------------------+
    |                    프로세서 패키지                       |
    +--------------------------------------------------------+
    |                                                        |
    |   V_core (0.8V ~ 1.4V)    <-- 코어 로직 전원            |
    |   +-----------------+                                  |
    |   | CPU 코어들      |                                  |
    |   | (연산 유닛)     |                                  |
    |   +-----------------+                                  |
    |                                                        |
    |   V_cache (0.9V ~ 1.1V)   <-- 캐시 메모리 전원          |
    |   +-----------------+                                  |
    |   | L2/L3 Cache     |                                  |
    |   +-----------------+                                  |
    |                                                        |
    |   V_gpu (0.8V ~ 1.2V)     <-- 내장 그래픽 전원          |
    |   +-----------------+                                  |
    |   | iGPU 유닛       |                                  |
    |   +-----------------+                                  |
    |                                                        |
    |   V_io (1.05V ~ 1.8V)    <-- I/O 인터페이스 전원        |
    |   +-----------------+                                  |
    |   | PCIe, DDR CTRL  |                                  |
    |   +-----------------+                                  |
    |                                                        |
    |   V_soc (0.75V ~ 1.0V)   <-- 시스템 온칩 전원           |
    |   +-----------------+                                  |
    |   | 미디어, 보안 등  |                                  |
    |   +-----------------+                                  |
    +--------------------------------------------------------+

    * 전압 레일 분리 이유:
      1. 각 블록별 독립적인 전력 관리 가능
      2. 노이즈 격리 (코어 스위칭 노이즈가 I/O에 영향 X)
      3. 최적의 전압 레벨 적용 (성능/전력 효율)
```

### 4. 전압 품질과 신호 무결성

```
    [ 전압 리플과 노이즈 ]

    이상적인 DC 전압                    실제 DC 전압 (리플 포함)
    +V |-----------------------      +V |~~~~~~~~~~~~~~~~~~~~~~~
       |                                |  /\  /\  /\  /\
    0V |-----------------------      0V |~~\/~~\/~~\/~~\~~~~~
       +-------------> 시간             +-------------> 시간
                                         리플 (Ripple)

    [ 전압 강하 (Voltage Drop) ]

    전원 공급점                              부하 (CPU/GPU)
    +V (12.0V) ----[---- 저항 ----]----> +V (11.7V)
                   PCB 트레이스            전압 강하!

    V_drop = I * R (옴의 법칙)
    * 대전류 경로에서 PCB 트레이스 저항으로 인한 전압 강하 발생
    * CPU V-core의 경우 수십~수백 암페어 흐름 -> 설계 중요
```

**전압 품질 지표**:
- **리플(Ripple)**: DC 전압 위에 겹쳐진 AC 성분. 스위칭 전원 특성상 불가피하며 mV 단위로 관리
- **전압 강하(IR Drop)**: 전류가 도체를 흐르며 발생하는 전압 손실. PCB 설계에서 전원 평면 설계가 핵심
- **트랜션트 응답(Transient Response)**: 부하 급변 시 전압이 복구되는 속도. VRM 성능의 핵심 지표

### 5. 전압과 반도체 공정의 관계

```
    [ 공정 축소에 따른 전압 변화 추세 ]

    전압(V)
    5.0 | *
        |   *
    3.3 |     *
        |       *
    2.5 |         *
        |           *
    1.8 |             *
        |               *
    1.2 |                 *              <-- 현대 공정
        |                   *
    0.8 |                     *          <-- 최신 공정
        +-------------------------> 공정(nm)
             350  250  180  130  90  65  45  28  14  7  5

    * Dennard Scaling (데나드 스케일링):
      - 트랜지스터 크기가 1/k로 작아지면 전압도 1/k로 낮아짐
      - 하지만 65nm 이후 이 법칙 붕괴 (누설 전류 증가)
      -> Dark Silicon 문제 대두
```

## 실무 적용

### 1. 서버 전원 설계에서의 전압 고려사항

```
    [ 서버급 다중 VRM 설계 ]

    메인보드
    +----------------------------------------------------------+
    |  +-------+   +-------+   +-------+   +-------+           |
    |  | VRM 1 |   | VRM 2 |   | VRM 3 |   | VRM 4 |           |
    |  |(상 #1)|   |(상 #2)|   |(상 #3)|   |(상 #4)|           |
    |  +---+---+   +---+---+   +---+---+   +---+---+           |
    |      |           |           |           |               |
    |      +-----------+-----------+-----------+               |
    |                        |                                 |
    |                   V_core (0.8~1.4V)                      |
    |                        |                                 |
    |                  +-----+-----+                           |
    |                  |   CPU     |                           |
    |                  +-----------+                           |
    +----------------------------------------------------------+

    * 다상(Multi-phase) VRM:
      - 각 상(Phase)이 교대로 전류 공급
      - 리플 감소, 열 분산, 과전류 대응
      - 하이엔드 서버: 16~24상 VRM 사용
```

**서버 전압 설계 체크리스트**:
1. 최대 부하 시에도 허용 오차 범위 내 유지 (예: +/- 5%)
2. VRM 효율 곡선 확인 (80 Plus 효율 등급)
3. hot-swap 지원 시 inrush current 제한 회로
4. redundant PSU 구성 시 전압 평형 유지

### 2. 오버클러킹과 전압 튜닝

```
    [ 클럭 vs 전압의 관계 ]

    클럭(GHz)                              안정 동작 영역
        ^                                 +++++++++++++
        |                            +++++            +++++
    5.0 |                       +++++                     +++
        |                   +++++                             ++ (과전압 영역)
    4.5 |               +++++                                   +
        |            ++++                                        +
    4.0 |        ++++        <-- 일반 동작점                      +
        |     ++++                                              +
    3.5 |  ++++                                                 +
        |++                                                     +
        +--------------------------------------------------------> 전압(V)
              1.0    1.1    1.2    1.3    1.4    1.5

    * 전압을 높이면 더 높은 클럭에서 안정 동작 가능
    * 하지만 발열과 전력 소모 급증, 수명 단축 위험
```

| 모니터링 항목 | 정상 범위 예시 | 경고 임계값 | 위험 수준 |

| Vcore | BIOS 설정값 +/- 2% | +/- 5% | +/- 10% |

## 기술사 시험 포인트

### 1. 핵심 정의 및 공식
- **전압 정의**: 전위차 = 일/전하량 (V = W/Q), 단위: 볼트(V)
- **옴의 법칙**: V = I * R (전압 = 전류 * 저항)
- **전력 공식**: P = V * I = V²/R = I²*R
- **동적 전력 공식**: P_dynamic = aCV²f (전압의 제곱에 비례)

### 2. 시험 빈출 키워드
1. **DVFS (Dynamic Voltage and Frequency Scaling)**: 부하에 따른 전압/주파수 동적 조절
2. **노이즈 마진**: 디지털 회로의 오동작 방지를 위한 전압 여유분
3. **IR Drop**: 전류 경로의 저항으로 인한 전압 강하
4. **Dennard Scaling**: 공정 미세화에 따른 전압 스케일링 법칙 (현대에 붕괴)
5. **Dark Silicon**: 누설 전류 증가로 인해 사용할 수 없는 칩 영역

### 3. 단답형 예상 문제
- **Q**: CMOS 회로에서 전력 소모를 줄이기 위해 전압을 낮추면 어떤 부작용이 발생하는가?
- **A**: 트랜지스터의 스위칭 속도가 느려져 최대 동작 클럭 주파수가 하락하며, 잡음 마진이 감소하여 신호 무결성 문제가 발생할 수 있다.

### 4. 논술형 예상 문제
- **Q**: 현대 데이터센터에서 에너지 효율을 높이기 위한 전압 관리 기술들을 설명하시오.
- **A**:
  1. **DVFS**: 부하에 따라 실시간으로 전압과 주파수를 조절
  2. **Power Gating**: 미사용 코어/블록의 전원 차단으로 누설 전력 제거
  3. **Adaptive Voltage Scaling**: 온도와 공정 편차를 고려한 칩별 맞춤 전압
  4. **다중 전압 도메인**: 각 기능 블록별 최적 전압 적용

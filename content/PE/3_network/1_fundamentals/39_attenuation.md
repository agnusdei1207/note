# 감쇠 (Attenuation), 데시벨(dB) 측정

## 핵심 인사이트 (3줄 요약)
> 1. **본질**: 감쇠는 **"신호 **전송 **중 **전력**이 **감소**하는 **현상\\\"**으로, **전자기적 **특성**과 **매체 **저항**으로 **인해 **발생**하며 **거리**에 **따라 **지수적으로 **증가**한다.
> 2. **데시벨(dB)**: **로그 **스케일**로 **전력 **비**를 **표현**하며 **dB = 10log₁₀(P₁/P₀)** **공식**을 **사용**하고 **음수**값은 **손실**, **양수**값은 **이득**을 **의미**한다.
> 3. **보상**: **증폭기**(Amplifier), **리피터**(Repeater), **중계기**가 **감쇠**를 **보상**하며 **링크 **예산**(Link Budget) **계산**으로 **필요 **전송 **전력**을 **결정**한다.

---

## Ⅰ. 개요 (Context & Background)

### 개념
감쇠는 **"신호 약화"**이다.

**감쇠 원인**:
| 원인 | 설명 | 주파수 의존성 |
|------|------|---------------|
| **도체 손실** | 저항에 의한 열 발생 | 모든 주파수 |
| **유전체 손실** | 유전체의 불완전 | 주파수 증가 |
| **복사 손실** | 에너지 방사 | 주파수 증가 |
| **절연 손실** | 누설 전류 | 고주파수 |

### 💡 비유
감쇠는 **물이 파이프를 지나며 줄어드는 것**과 같다.
- **신호 세기**: 물의 양
- **감쇠**: 파이프 마찰
- **증폭기**: 펌프

## Ⅱ. 아키텍처 및 핵심 원리

### 감쇠의 종류

```
┌─────────────────────────────────────────────────────────────────────────────────────────┐
│                         Attenuation Types                                                      │
└─────────────────────────────────────────────────────────────────────────────────────────┘

    1. Conductor Loss (도체 손실):
    ┌──────────────────────────────────────────────────────────────────────────────────────────┐
    │  Cause: Resistance of conductor (skin effect at high frequency)                         │  │
    │  Formula: αc = R / (2Z₀)                                                               │  │
    │  Where:                                                                                 │  │
    │  • αc: Conductor attenuation constant (Np/m)                                            │  │
    │  • R: Resistance per unit length                                                        │  │
    │  • Z₀: Characteristic impedance                                                         │  │
    │                                                                                         │  │
    │  Skin Effect:                                                                           │  │
    │  ┌──────────────────────────────────────────────────────────────────────────────────────┐  │  │
    │  │  High frequency → current flows on surface only                                      │  │  │
    │  │  Effective cross-section decreases → Resistance increases                           │  │  │
    │  │  Skin depth δ = √(2ρ/ωμ)                                                             │  │  │
    │  │  • ρ: Resistivity, ω: Frequency, μ: Permeability                                    │  │  │
    │  └──────────────────────────────────────────────────────────────────────────────────────┘  │  │
    └──────────────────────────────────────────────────────────────────────────────────────────┘

    2. Dielectric Loss (유전체 손실):
    ┌──────────────────────────────────────────────────────────────────────────────────────────┐
    │  Cause: Dielectric absorption and conductivity                                           │  │
    │  Formula: αd = (πf / c) × (√εʳ) × tan δ                                                 │  │
    │  Where:                                                                                 │  │
    │  • αd: Dielectric attenuation constant                                                   │  │
    │  • f: Frequency, c: Speed of light                                                      │  │
    │  • εʳ: Relative permittivity                                                             │  │
    │  • tan δ: Loss tangent (dissipation factor)                                             │  │
    │                                                                                         │  │
    │  Loss Tangent by Material:                                                               │  │
    │  ┌──────────────────────────────────────────────────────────────────────────────────────┐  │  │
    │  │  Material       │  tan δ      │  Application                  │                       │  │  │
    │  │  ─────────────────────────────────────────────────────────────────────────────────────│  │  │
    │  │  Air            │  ~0         │  Open wire, waveguide          │                       │  │  │
    │  │  Polyethylene    │  0.0002     │  Coaxial cable                │                       │  │  │
    │  │  Teflon (PTFE)  │  0.00015    │  High-frequency cable         │                       │  │  │
    │  │  FR-4 (PCB)     │  0.02       │  Low-frequency board          │                       │  │  │
    │  └──────────────────────────────────────────────────────────────────────────────────────┘  │  │
    └──────────────────────────────────────────────────────────────────────────────────────────┘

    3. Radiation Loss (복사 손실):
    ┌──────────────────────────────────────────────────────────────────────────────────────────┐
    │  Cause: Signal radiates from transmission line acting as antenna                         │  │
    │  Factors:                                                                               │  │
    │  • Discontinuities (connectors, bends)                                                  │  │
    │  • Unshielded cables                                                                   │  │
    │  • High frequency (> 1 GHz)                                                             │  │
    │                                                                                         │  │
    │  Minimization:                                                                          │  │
    │  • Proper shielding                                                                     │  │
    │  • Impedance matching                                                                   │  │
    │  • Smooth bends                                                                         │  │
    └──────────────────────────────────────────────────────────────────────────────────────────┘
```

### 데시벨(dB) 계산

```
┌─────────────────────────────────────────────────────────────────────────────────────────┐
│                         Decibel (dB) Calculations                                              │
└─────────────────────────────────────────────────────────────────────────────────────────┘

    Power Ratio (전력비):
    ┌──────────────────────────────────────────────────────────────────────────────────────────┐
    │  dB = 10 × log₁₀(P₁ / P₀)                                                             │  │
    │                                                                                         │  │
    │  Where:                                                                                 │  │
    │  • P₁: Measured power                                                                   │  │
    │  • P₀: Reference power                                                                  │  │
    │                                                                                         │  │
    │  Example:                                                                               │  │
    │  ┌──────────────────────────────────────────────────────────────────────────────────────┐  │  │
    │  │  P₁ = 100 mW, P₀ = 1 mW                                                               │  │  │
    │  │  dB = 10 × log₁₀(100/1) = 10 × 2 = 20 dB                                            │  │  │
    │  │                                                                                      │  │  │
    │  │  Common values:                                                                      │  │  │
    │  │  • 3 dB: Power doubles (×2)                                                          │  │  │
    │  │  • 6 dB: Power quadruples (×4), Voltage doubles (×2)                                │  │  │
    │  │  • 10 dB: Power ×10                                                                  │  │  │
    │  │  • 20 dB: Power ×100, Voltage ×10                                                   │  │  │
    │  │  • -3 dB: Power halves (×0.5)                                                        │  │  │
    │  │  │  -10 dB: Power ×0.1                                                               │  │  │
    │  └──────────────────────────────────────────────────────────────────────────────────────┘  │  │
    └──────────────────────────────────────────────────────────────────────────────────────────┘

    Voltage Ratio (전압비):
    ┌──────────────────────────────────────────────────────────────────────────────────────────┐
    │  dB = 20 × log₁₀(V₁ / V₀)                                                             │  │
    │                                                                                         │  │
    │  Note: 20 instead of 10 because P ∝ V²                                                │  │
    │                                                                                         │  │
    │  Example:                                                                               │  │
    │  ┌──────────────────────────────────────────────────────────────────────────────────────┐  │  │
    │  │  V₁ = 2V, V₀ = 1V                                                                    │  │  │
    │  │  dB = 20 × log₁₀(2/1) = 20 × 0.301 = 6.02 dB ≈ 6 dB                               │  │  │
    │  └──────────────────────────────────────────────────────────────────────────────────────┘  │  │
    └──────────────────────────────────────────────────────────────────────────────────────────┘

    dBm, dBW (Absolute Power):
    ┌──────────────────────────────────────────────────────────────────────────────────────────┐
    │  dBm: Power relative to 1 mW                                                             │  │
    │  ┌──────────────────────────────────────────────────────────────────────────────────────┐  │  │
    │  │  dBm = 10 × log₁₀(P / 1mW)                                                           │  │  │
    │  │                                                                                      │  │  │
    │  │  Common values:                                                                      │  │  │
    │  │  • 0 dBm = 1 mW                                                                      │  │  │
    │  │  • 10 dBm = 10 mW                                                                    │  │  │
    │  │  • 20 dBm = 100 mW                                                                   │  │  │
    │  │  • 30 dBm = 1 W                                                                      │  │  │
    │  │  • -10 dBm = 0.1 mW                                                                 │  │  │
    │  │  • -60 dBm = 1 nW (noise floor)                                                      │  │  │
    │  └──────────────────────────────────────────────────────────────────────────────────────┘  │  │
    │                                                                                         │  │
    │  dBW: Power relative to 1 W                                                              │  │
    │  ┌──────────────────────────────────────────────────────────────────────────────────────┐  │  │
    │  │  dBW = 10 × log₁₀(P / 1W)                                                            │  │  │
    │  │  • 0 dBW = 1 W                                                                       │  │  │
    │  │  • 10 dBW = 10 W                                                                    │  │  │
    │  │  • 30 dBW = 1000 W (1 kW)                                                           │  │  │
    │  │  • -10 dBW = 0.1 W                                                                  │  │  │
    │  └──────────────────────────────────────────────────────────────────────────────────────┘  │  │
    │                                                                                         │  │
    │  Conversion:                                                                            │  │
    │  dBm = dBW + 30 (since 1 W = 1000 mW = 10log₁₀(1000) = 30 dBm)                        │  │
    └──────────────────────────────────────────────────────────────────────────────────────────┘
```

### 감쇠와 거리

```
┌─────────────────────────────────────────────────────────────────────────────────────────┐
│                         Attenuation vs Distance                                                │
└─────────────────────────────────────────────────────────────────────────────────────────┘

    Exponential Decay:
    ┌──────────────────────────────────────────────────────────────────────────────────────────┐
    │  P(d) = P₀ × 10^(-αd/10)                                                               │  │
    │                                                                                         │  │
    │  Where:                                                                                 │  │
    │  • P(d): Power at distance d                                                             │  │
    │  • P₀: Initial power                                                                    │  │
    │  • α: Attenuation constant (dB/km or dB/100m)                                          │  │
    │  • d: Distance                                                                           │  │
    │                                                                                         │  │
    │  Example: Fiber optic cable                                                              │  │
    │  ┌──────────────────────────────────────────────────────────────────────────────────────┐  │  │
    │  │  α = 0.2 dB/km @ 1550 nm                                                            │  │  │
    │  │  P₀ = 1 mW (0 dBm)                                                                   │  │  │
    │  │                                                                                      │  │  │
    │  │  Distance  │  Attenuation │  Power (dBm) │  Power (mW)                              │  │  │
    │  │  ─────────────────────────────────────────────────────────────────────────────────────│  │  │
    │  │  0 km      │  0 dB        │  0 dBm       │  1.00                                  │  │  │
    │  │  10 km     │  2 dB        │  -2 dBm      │  0.63                                  │  │  │
    │  │  20 km     │  4 dB        │  -4 dBm      │  0.40                                  │  │  │
    │  │  50 km     │  10 dB       │  -10 dBm     │  0.10                                  │  │  │
    │  │  100 km    │  20 dB       │  -20 dBm     │  0.01                                  │  │  │
    │  └──────────────────────────────────────────────────────────────────────────────────────┘  │  │
    └──────────────────────────────────────────────────────────────────────────────────────────┘

    Free Space Path Loss (FSPL):
    ┌──────────────────────────────────────────────────────────────────────────────────────────┐
    │  FSPL(dB) = 20log₁₀(d) + 20log₁₀(f) + 20log₁₀(4π/c)                                     │  │
    │                                                                                         │  │
    │  Simplified: FSPL(dB) = 20log₁₀(d) + 20log₁₀(f) - 147.55 (d in km, f in Hz)            │  │
    │  Or: FSPL(dB) = 20log₁₀(d) + 20log₁₀(f) + 32.44 (d in km, f in MHz)                    │  │
    │                                                                                         │  │
    │  Example: WiFi @ 2.4 GHz, 100 m                                                         │  │
    │  ┌──────────────────────────────────────────────────────────────────────────────────────┐  │  │
    │  │  d = 0.1 km, f = 2400 MHz                                                            │  │  │
    │  │  FSPL = 20log₁₀(0.1) + 20log₁₀(2400) + 32.44                                      │  │  │
    │  │       = -20 + 67.6 + 32.44 = 80.04 dB                                               │  │  │
    │  │                                                                                      │  │  │
    │  │  If transmit power = 20 dBm (100 mW)                                                 │  │  │
    │  │  Received power = 20 - 80.04 = -60.04 dBm                                            │  │  │
    │  └──────────────────────────────────────────────────────────────────────────────────────┘  │  │
    └──────────────────────────────────────────────────────────────────────────────────────────┘
```

## Ⅳ. 실무 적용 및 기술사적 판단

### 기술사적 판단 (실무 시나리오)

#### 시나리오: 링크 예산 계산
**상황**: 10km 광섬유 링크 설계
**판단**: 송신기 전력, 증폭기 필요성

```
┌─────────────────────────────────────────────────────────────────────────────────────────┐
│                         Link Budget Calculation                                                │
└─────────────────────────────────────────────────────────────────────────────────────────┘

    Given:
    • Distance: 10 km
    • Fiber attenuation: 0.2 dB/km @ 1550 nm
    • Connector loss: 0.5 dB each (2 connectors)
    • Splice loss: 0.1 dB each (5 splices)
    • Receiver sensitivity: -28 dBm (minimum detectable signal)
    • Required margin: 3 dB

    Link Budget:
    ┌──────────────────────────────────────────────────────────────────────────────────────────┐
    │  Transmit Power:                                                                      │  │
    │  ┌──────────────────────────────────────────────────────────────────────────────────────┐  │  │
    │  │  Laser output: +3 dBm                                                               │  │  │
    │  └──────────────────────────────────────────────────────────────────────────────────────┘  │  │
    │                                                                                         │  │
    │  Losses:                                                                                │  │
    │  ┌──────────────────────────────────────────────────────────────────────────────────────┐  │  │
    │  │  Fiber attenuation: 10 km × 0.2 dB/km = 2 dB                                       │  │  │
    │  │  Connector loss: 2 × 0.5 dB = 1 dB                                                   │  │  │
    │  │  Splice loss: 5 × 0.1 dB = 0.5 dB                                                    │  │  │
    │  │  System margin: 3 dB                                                                │  │  │
    │  │  Total loss: 2 + 1 + 0.5 + 3 = 6.5 dB                                               │  │  │
    │  └──────────────────────────────────────────────────────────────────────────────────────┘  │  │
    │                                                                                         │  │
    │  Received Power:                                                                        │  │
    │  ┌──────────────────────────────────────────────────────────────────────────────────────┐  │  │
    │  │  P_RX = P_TX - Loss = 3 dBm - 6.5 dB = -3.5 dBm                                    │  │  │
    │  │                                                                                      │  │  │
    │  │  Margin vs Sensitivity:                                                              │  │  │
    │  │  Margin = P_RX - Sensitivity = -3.5 - (-28) = 24.5 dB                                │  │  │
    │  │  → 24.5 dB above minimum detectable signal                                          │  │  │
    │  │  → Link feasible, no amplifier needed                                                 │  │  │
    │  └──────────────────────────────────────────────────────────────────────────────────────┘  │  │
    │                                                                                         │  │
    │  If distance = 80 km (without repeater):                                                │  │
    │  ┌──────────────────────────────────────────────────────────────────────────────────────┐  │  │
    │  │  Fiber attenuation: 80 km × 0.2 dB/km = 16 dB                                      │  │  │
    │  │  Total loss: 16 + 1 + 0.5 + 3 = 20.5 dB                                              │  │  │
    │  │  P_RX = 3 dBm - 20.5 dB = -17.5 dBm                                                   │  │  │
    │  │  Margin = -17.5 - (-28) = 10.5 dB                                                    │  │  │
    │  │  → Still feasible, but marginal                                                       │  │  │
    │  │  → Consider adding optical amplifier at 40 km                                         │  │  │
    │  └──────────────────────────────────────────────────────────────────────────────────────┘  │  │
    └──────────────────────────────────────────────────────────────────────────────────────────┘
```

### 감쇠 보상 방법

```
┌─────────────────────────────────────────────────────────────────────────────────────────┐
│                         Attenuation Compensation                                           │
└─────────────────────────────────────────────────────────────────────────────────────────┘

    1. Amplifiers (증폭기):
    ┌──────────────────────────────────────────────────────────────────────────────────────────┐
    │  Optical Amplifier (EDFA - Erbium-Doped Fiber Amplifier):                              │  │
    │  ┌──────────────────────────────────────────────────────────────────────────────────────┐  │  │
    │  │  • Gain: 20-30 dB typical                                                           │  │  │
    │  │  • Noise Figure: 4-6 dB                                                              │  │  │
    │  │  • Placement: Every 80-100 km in long-haul systems                                   │  │  │
    │  │  • Wavelength: C-band (1530-1565 nm)                                                  │  │  │
    │  │                                                                                      │  │  │
    │  │  Example: 100 km link with -3 dBm transmit, need -28 dBm at receiver                  │  │  │
    │  │  Loss: 100 km × 0.2 dB/km = 20 dB                                                   │  │  │
    │  │  Without amp: -3 dBm - 20 dB = -23 dBm (feasible)                                   │  │  │
    │  │  With amp at 50 km:                                                                  │  │  │
    │  │    -3 dBm - 10 dB + 20 dB = +7 dBm (after amp)                                     │  │  │
    │  │    +7 dBm - 10 dB = -3 dBm (at receiver)                                             │  │  │
    │  └──────────────────────────────────────────────────────────────────────────────────────┘  │  │
    └──────────────────────────────────────────────────────────────────────────────────────────┘

    2. Repeaters (리피터):
    ┌──────────────────────────────────────────────────────────────────────────────────────────┐
    │  • Regenerate signal (3R: Re-amplify, Reshape, Retime)                                  │  │
    │  • Used in copper-based systems                                                          │  │
    │  • Typical spacing: 100m for UTP Ethernet                                                 │  │
    │                                                                                         │  │
    │  Ethernet Repeater Hubs:                                                                 │  │
    │  ┌──────────────────────────────────────────────────────────────────────────────────────┐  │  │
    │  │  ┌─────────┐   ┌─────────┐   ┌─────────┐   ┌─────────┐   ┌─────────┐              │  │  │
    │  │  │ Station │──│ Repeater │──│ Repeater │──│ Repeater │──│ Station │              │  │  │
    │  │  └─────────┘   └─────────┘   └─────────┘   └─────────┘   └─────────┘              │  │  │
    │  │       100m          100m          100m          100m                               │  │  │
    │  │  → Max 500m with 4 repeaters (for 10BASE-T)                                        │  │  │
    │  └──────────────────────────────────────────────────────────────────────────────────────┘  │  │
    └──────────────────────────────────────────────────────────────────────────────────────────┘

    3. Equalizers (이퀄라이저):
    ┌──────────────────────────────────────────────────────────────────────────────────────────┐
    │  • Compensate for frequency-dependent attenuation                                        │  │
    │  • Boost high frequencies more than low frequencies                                       │  │
    │  • Used in wired communication (DSL, cable modems)                                       │  │
    └──────────────────────────────────────────────────────────────────────────────────────────┘
```

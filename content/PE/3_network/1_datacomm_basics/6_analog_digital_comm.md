# 아날로그 통신 vs 디지털 통신 (Analog vs Digital Communication)

## 정의

**아날로그 통신**은 연속적인 신호를 전송하는 방식이며, **디지털 통신**은 이산적인 신호(0과 1)를 전송하는 방식이다. 현대 통신 시스템은 대부분 디지털 방식을 사용하며, 잡음 내성, 보안성, 처리 효율성 등의 장점을 가진다.

```
┌─────────────────────────────────────────────────────────────────┐
│           아날로그 vs 디지털 신호                                  │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   아날로그 신호 (연속적):                                        │
│   ┌─────────────────────────────────────────────────────────┐ │
│   │                                                         │ │
│   │   진폭                                                   │ │
│   │      ∧                                                   │ │
│   │     ╱ ╲     ╱╲    ╱╲                                    │ │
│   │    ╱   ╲   ╱  ╲  ╱  ╲    연속적으로 변화               │ │
│   │   ╱     ╲ ╱    ╲╱    ╲                                  │ │
│   │  ╱       V              ╲                                │ │
│   │ ──────────────────────────────▶ 시간                    │ │
│   │                                                         │ │
│   │   예: 음성, 온도, 조도 등 자연 신호                      │ │
│   │                                                         │ │
│   └─────────────────────────────────────────────────────────┘ │
│                                                                 │
│   디지털 신호 (이산적):                                          │
│   ┌─────────────────────────────────────────────────────────┐ │
│   │                                                         │ │
│   │   진폭                                                   │ │
│   │   Vcc ───┐   ┌───┐   ┌───┐                             │ │
│   │          │   │   │   │   │  유한한 레벨만 존재         │ │
│   │          │   │   │   │   │  (보통 2레벨: 0, 1)         │ │
│   │   0V  ───┘   └───┘   └───┘                              │ │
│   │          ─────────────────────▶ 시간                    │ │
│   │                                                         │ │
│   │   예: 컴퓨터 데이터, 디지털 오디오/비디오                │ │
│   │                                                         │ │
│   └─────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

## 핵심 개념

### 1. 특성 비교

```
┌─────────────────────────────────────────────────────────────────┐
│           아날로그 vs 디지털 통신 비교                             │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   ┌─────────────────────────────────────────────────────────┐ │
│   │  구분          │ 아날로그 통신      │ 디지털 통신        │ │
│   ├────────────────┼─────────────────────┼───────────────────┤ │
│   │ 신호 형태      │ 연속적             │ 이산적 (0, 1)     │ │
│   │ 잡음 영향      │ 누적됨             │ 재생 가능         │ │
│   │ 전송 거리      │ 제한적             │ 중계기로 무제한   │ │

│   │ 호환성         │ 낮음               │ 높음 (디지털화)   │ │

└─────────────────────────────────────────────────────────────────┘
```

### 4. 아날로그-디지털 변환

```
┌─────────────────────────────────────────────────────────────────┐
│           A/D 및 D/A 변환                                        │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   PCM (Pulse Code Modulation) 과정:                             │
│   ┌─────────────────────────────────────────────────────────┐ │
│   │                                                         │ │
│   │   아날로그     표본화     양자화     부호화     디지털   │ │
│   │     신호   ──▶(Sampling)──▶(Quantizing)──▶(Encoding)──▶신호│ │
│   │                                                         │ │
│   └─────────────────────────────────────────────────────────┘ │
│                                                                 │
│   표본화 (Sampling):                                            │
│   ┌─────────────────────────────────────────────────────────┐ │
│   │                                                         │ │
│   │   • 연속 신호를 일정 간격으로 샘플링                     │ │
│   │   • 나이퀴스트 정리: fs ≥ 2 × fmax                      │ │
│   │   • 예: 음성 (4kHz) → fs ≥ 8kHz (실제 8kHz 사용)        │ │
│   │                                                         │ │
│   │   아날로그    샘플링된                                   │ │
│   │     ∧          ∧ ∧ ∧  ∧  ∧                             │ │
│   │    ╱ ╲        │ │ │  │  │                              │ │
│   │   ╱   ╲  ──▶  │ │ │  │  │                              │ │
│   │  ╱     ╲      │ │ │  │  │                              │ │
│   │ ─────────     ─┴─┴─┴──┴──┴──                           │ │
│   │                                                         │ │
│   └─────────────────────────────────────────────────────────┘ │
│                                                                 │
│   양자화 (Quantization):                                        │
│   ┌─────────────────────────────────────────────────────────┐ │
│   │                                                         │ │
│   │   • 샘플링된 값을 유한한 레벨로 반올림                   │ │
│   │   • 양자화 오차(Noise) 발생                              │ │
│   │   • 비트 수가 많을수록 오차 감소                         │ │
│   │                                                         │ │
│   │   8비트 양자화: 256 레벨                                 │ │
│   │   16비트 양자화: 65,536 레벨                             │ │
│   │                                                         │ │
│   └─────────────────────────────────────────────────────────┘ │
│                                                                 │
│   부호화 (Encoding):                                            │
│   ┌─────────────────────────────────────────────────────────┐ │
│   │                                                         │ │
│   │   • 양자화된 값을 이진 코드로 변환                       │ │
│   │   • 예: 레벨 100 → 01100100 (8비트)                     │ │
│   │                                                         │ │
│   └─────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

## 기술사 시험 포인트

### 1. 핵심 정의

> **아날로그 통신**: 연속적 신호 전송 방식
> **디지털 통신**: 이산적 신호(0, 1) 전송 방식
> **PCM**: 아날로그를 디지털로 변환하는 표준 방식

### 2. 빈출 키워드

- **Signal Regeneration**: 디지털 신호 재생 (잡음 제거)
- **Quantization Error**: 양자화 과정에서 발생하는 오차
- **Nyquist Sampling**: fs ≥ 2 × fmax
- **Error Detection/Correction**: 디지털 통신의 에러 제어

### 3. 예상 문제

**Q1. 아날로그 통신과 디지털 통신의 장단점을 비교하고 현대 통신이 디지털화되는 이유를 설명하시오.**

**Q2. PCM(Pulse Code Modulation)의 변환 과정을 단계별로 설명하시오.**

**Q3. 디지털 통신에서 신호 재생(Regeneration)의 개념과 잡음 내성을 설명하시오.**

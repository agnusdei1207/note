# 전파 지연 (Propagation Delay)

## 정의

**전파 지연(Propagation Delay)**은 신호가 **송신지에서 수신지까지 물리적으로 이동하는 데 걸리는 시간**이다. 전파 속도와 거리에 의해 결정되며, 매체의 종류와 전송 거리에 따라 달라진다.

```
┌─────────────────────────────────────────────────────────────────┐
│           전파 지연 기본 개념                                     │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   ┌─────────────────────────────────────────────────────────┐ │
│   │                                                         │ │
│   │   정의:                                                   │ │
│   │                                                         │ │
│   │   전파 지연 = 거리 / 전파 속도                            │ │
│   │              d    /    v                                │ │
│   │                                                         │ │
│   │   송신자                        매체                    수신자│ │
│   │   ┌─────┐                  ┌───────────┐            ┌─────┐│ │
│   │   │     │══════════════════│═══════════│════════════│     ││ │
│   │   │  A  │   ▶── 신호 ───▶  │  매체    │  ▶── 신호─▶│  B  ││ │
│   │   │     │                  │           │            │     ││ │
│   │   └─────┘                  └───────────┘            └─────┘│ │
│   │        │◀────────────── 거리 d ────────────────────▶│     │ │
│   │                                                         │ │
│   │   💡 물리적 거리에 비례, 전파 속도에 반비례               │ │
│   │                                                         │ │
│   └─────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

## 핵심 개념

### 1. 전파 속도와 매체

```
┌─────────────────────────────────────────────────────────────────┐
│           매체별 전파 속도                                        │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   ┌─────────────────────────────────────────────────────────┐ │
│   │                                                         │ │
│   │   전파 속도 비교:                                         │ │
│   │   ┌───────────────────────────────────────────────┐    │ │
│   │   │                                                 │    │ │
│   │   │   매체            │ 전파 속도      │ 속도비     │    │ │
│   │   │   ────────────────┼────────────────┼─────────  │    │ │
│   │   │   진공 (빛)       │ 3×10⁸ m/s     │ 100%      │    │ │
│   │   │   광케이블 (코어) │ 2×10⁸ m/s     │ ~67%      │    │ │
│   │   │   구리선 (UTP)    │ 2.3×10⁸ m/s   │ ~77%      │    │ │
│   │   │   동축 케이블     │ 2×10⁸ m/s     │ ~67%      │    │ │
│   │   │   무선 (공기)     │ 3×10⁸ m/s     │ ~100%     │    │ │
│   │   │   위성 (우주)     │ 3×10⁸ m/s     │ 100%      │    │ │
│   │   │                                                 │    │ │
│   │   │   💡 매체 굴절률(n)에 의해 속도 결정              │    │ │
│   │   │      v = c / n (c = 빛의 속도)                  │    │ │
│   │   │                                                 │    │ │
│   │   └───────────────────────────────────────────────┘    │ │
│   │                                                         │ │
│   │   전파 지연 계산 예시:                                    │ │
│   │   ┌───────────────────────────────────────────────┐    │ │
│   │   │                                                 │    │ │
│   │   │   ① 100km 광케이블:                             │    │ │
│   │   │      d_prop = 100×10³ / 2×10⁸ = 0.5ms         │    │ │
│   │   │                                                 │    │ │
│   │   │   ② 5000km 광케이블 (대륙 간):                   │    │ │
│   │   │      d_prop = 5×10⁶ / 2×10⁸ = 25ms            │    │ │
│   │   │                                                 │    │ │
│   │   │   ③ GEO 위성 (35,786km 왕복):                   │    │ │
│   │   │      d_prop = (2 × 35,786×10³) / 3×10⁸        │    │ │
│   │   │            = 239ms (왕복)                      │    │ │
│   │   │                                                 │    │ │
│   │   └───────────────────────────────────────────────┘    │ │
│   │                                                         │ │
│   └─────────────────────────────────────────────────────────┘ │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

## 기술사 시험 포인트

### 1. 핵심 정의

> **전파 지연**: 거리/전파속도, 신호 이동 시간
> **물리적 한계**: 빛의 속도로 제한, 개선 불가
> **매체별 속도**: 진공 > UTP > 광케이블

### 2. 빈출 키워드

- **Propagation Delay = Distance / Velocity**: 전파 지연 공식
- **GEO Satellite Latency**: ~239ms 왕복
- **Speed of Light**: 3×10⁸ m/s (진공)
- **Fiber Optic**: ~2×10⁸ m/s (굴절률 영향)

### 3. 예상 문제

**Q1. 전파 지연의 정의와 영향 요소를 설명하시오.**

**Q2. GEO 위성 통신의 전파 지연을 계산하고 TCP 성능에 미치는 영향을 분석하시오.**

**Q3. 대역폭 증가가 전파 지연에 미치는 영향과 전파 지연을 최소화하는 방안을 기술하시오.**

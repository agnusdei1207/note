# 공공 빅데이터

## 핵심 인사이트 (3줄 요약)

> 1. **본질**: 공공 빅데이터는 정부와 공공기관이 보유한 데이터를 개방하여 시민, 기업, 연구자가 활용할 수 있도록 하는 정책과 기술 체계이다.
> 2. **가치**: 공공 빅데이터는 행정 효율성 향상, 시민 서비스 개선, 민간 혁신 생태계 조성을 통해 연간 약 50조 원의 경제적 가치를 창출한다.
> 3. **융합**: 스마트시티, 교통, 보건의료, 재난안전 분야와 결합하여 데이터 기반 공공 정책의 근거를 제공한다.

## Ⅰ. 개요 (Context & Background)

### 개념 정의

공공 빅데이터(Public Big Data)는 공공기관이 업무 수행 과정에서 생성·수집한 대용량 데이터를 의미하며, 이를 개방(Open Data)하여 민간이 활용할 수 있도록 하는 제도적·기술적 체계를 포함한다. 한국은 공공데이터포털(data.go.kr)을 통해 4만 여 개의 데이터셋을 제공한다.

### 💡 비유

공공 빅데이터는 "공공 도서관"에 비유할 수 있다. 도서관이 책을 모아 누구나 무료로 볼 수 있게 하듯, 공공 빅데이터는 정부가 가진 정보를 모아 기업과 시민이 자유롭게 활용할 수 있게 한다. 이를 통해 새로운 서비스가 탄생하고 사회가 발전한다.

### 등장 배경 및 발전 과정

1. **2011년**: 미국 Obama 행정부 "Digital Government" 전략, Data.gov 시작
2. **2013년**: 한국 공공데이터포털 개설 (data.go.kr)
3. **2015년**: 공공데이터의 제공 및 이용 활성화에 관한 법률 제정
4. **2018년**: 한국 데이터 3법, 공공데이터 지원 센터 설립
5. **2021년**: 디지털플랫폼정부 추진, 공공 마이데이터 시작

### Open API 활용 예시

```python
import requests
import pandas as pd
from typing import Dict, List
import json

class PublicDataAPI:
    """공공데이터포털 API 클라이언트"""

    def __init__(self, api_key: str):
        self.api_key = api_key
        self.base_url = "http://apis.data.go.kr"

    def get_subway_passenger(
        self,
        start_date: str,
        end_date: str,
        station_name: str = None
    ) -> pd.DataFrame:
        """지하철 승하차 인원 조회 (서울교통공사)"""

        endpoint = "/B553766/subway-passenger/getSubwayTraffic"
        params = {
            "serviceKey": self.api_key,
            "type": "json",
            "CRTR_YM": start_date[:6],  # YYYYMM
            "SBWY_ROUT_LN_NM": station_name
        }

        response = requests.get(
            f"{self.base_url}{endpoint}",
            params=params,
            verify=False
        )

        data = response.json()
        return pd.DataFrame(data.get("items", []))

    def get_real_estate_transaction(
        self,
        lawd_cd: str,      # 지역코드
        deal_ymd: str,     # 계약월 (YYYYMM)
        service_type: str = "apartment"  # 아파트, 연립, 단독
    ) -> pd.DataFrame:
        """실거래가 조회"""

        endpoints = {
            "apartment": "/B161000000/ArtrRltrQk",
            "townhouse": "/B161000000/TwhsRltrQk",
            "detached": "/B161000000/ScsbRltrQk"
        }

        endpoint = endpoints.get(service_type, endpoints["apartment"])
        params = {
            "serviceKey": self.api_key,
            "LAWD_CD": lawd_cd,
            "DEAL_YMD": deal_ymd,
            "numOfRows": 1000,
            "pageNo": 1
        }

        response = requests.get(
            f"{self.base_url}{endpoint}",
            params=params,
            verify=False
        )

        data = response.json()
        return pd.DataFrame(data.get("items", {}).get("item", []))

    def get_weather_forecast(
        self,
        nx: int,           # X 좌표
        ny: int,           # Y 좌표
        base_date: str,    # 기준 날짜 (YYYYMMDD)
        base_time: str     # 기준 시간 (HHMM)
    ) -> Dict:
        """단기예보 조회"""

        endpoint = "/B136000000/AsosHourlyInfoService/getWthrDataList"
        params = {
            "serviceKey": self.api_key,
            "dataType": "json",
            "dataCd": "ASOS",
            "dateCd": "HR",
            "startDt": base_date,
            "startHh": base_time[:2],
            "endDt": base_date,
            "endHh": "23"
        }

        response = requests.get(
            f"{self.base_url}{endpoint}",
            params=params,
            verify=False
        )

        return response.json()


# 활용 예시
if __name__ == "__main__":
    api = PublicDataAPI("your-api-key")

    # 지하철 승하차 인원 분석
    subway_df = api.get_subway_passenger(
        start_date="20240101",
        end_date="20240131",
        station_name="강남"
    )
    print(f"지하철 데이터: {len(subway_df)}건")

    # 실거래가 분석
    real_estate_df = api.get_real_estate_transaction(
        lawd_cd="11680",  # 강남구
        deal_ymd="202401",
        service_type="apartment"
    )
    print(f"실거래가 데이터: {len(real_estate_df)}건")
```

## 👶 어린이를 위한 3줄 비유

**1단계 (무엇인가요?)**: 공공 빅데이터는 정부가 가진 정보를 모두에게 공개하는 거예요. 날씨, 교통, 집 값 같은 정보를 누구나 볼 수 있게 해요.

**2단계 (어떻게 쓰나요?)**: 이 정보를 이용해서 앱을 만들 수 있어요. 예를 들어, 버스가 언제 오는지 알려주는 앱은 공공 정보를 사용해요. 무료로 가져다 쓸 수 있어요!

**3단계 (왜 중요한가요?)**: 공공 빅데이터가 있으면 더 좋은 서비스를 싸게 만들 수 있어요. 또 정부가 어떤 일을 하는지 투명하게 알 수 있어요. 시민, 기업, 정부 모두에게 좋아요!

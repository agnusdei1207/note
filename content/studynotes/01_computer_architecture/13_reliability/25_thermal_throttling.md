# 473. 서멀 스로틀링 (Thermal Throttling)

## 핵심 인사이트 (3줄 요약)
> 1. **본질**: 서멀 스로틀링은 CPU/GPU가 과열(TjMax 근접) 시 자동으로 주파수와 전압을 낮추어 발열을 줄이고 칩을 보호하는 자가 보호 메커니즘이다.
> 2. **가치**: 100°C에 도달하면 CPU는 스로틀링을 시작하여 80~90°C 수준을 유지하며, 적절한 쿨링으로 스로틀링을 방지하면 성능 저하(최대 50%+)를 막을 수 있다.
> 3. **융합**: PROCHOT# 신호, T-state, P-state 조절로 구현되며, 모바일 기기에서는 배터리 보호, 서버에서는 무중단 운영을 위해 필수적이다.

---

### I. 개요

#### 개념 정의

**서멀 스로틀링(Thermal Throttling)**은 프로세서가 과열되었을 때 자동으로 성능(주파수, 전압)을 낮추어 발열을 줄이는 기능이다. 칩 손상을 방지하기 위한 자가 보호 메커니즘이다.

```
┌─────────────────────────────────────────────────────────────────┐
│                서멀 스로틀링 동작                               │
└─────────────────────────────────────────────────────────────────┘

온도 구간:
┌─────────────────────────────────────────────────────────────────┐
│                                                                 │
│  0°C ─────────────────────────────────────────────────────────  │
│                          정상 구간                              │
│                          최대 성능                              │
│  80°C ────────────────────────────────────────────────────────  │
│                          주의 구간                              │
│                        (Turbo 제한)                             │
│  95°C ────────────────────────────────────────────────────────  │
│                        스로틀링 시작                            │
│                       (주파수/전압 하락)                        │
│  100°C (TjMax) ───────────────────────────────────────────────  │
│                      심각 스로틀링                              │
│                     (대폭 성능 저하)                            │
│  105°C ────────────────────────────────────────────────────────  │
│                       Thermal Shutdown                          │
│                      (시스템 강제 종료)                         │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

#### 비유

> **서멀 스로틀링은 "마라토너가 더위를 느껴서 속도를 줄이는 것"과 같다.**
>
> 마라톤 중에 너무 더우면 몸을 보호하기 위해 속도를 줄인다. 계속 전력 질주하면 쓰러질 수 있으니까! CPU도 과열하면 속도를 줄여서 쓰러지지(고장 나지) 않게 한다.

---

### II. 아키텍처 및 핵심 원리

#### 스로틀링 메커니즘

```
┌─────────────────────────────────────────────────────────────────┐
│                Thermal Throttling 단계                          │
└─────────────────────────────────────────────────────────────────┘

1. PROCHOT# (Processor Hot):
   · 온도 센서가 TjMax 근접 감지
   · PROCHOT# 신호 Active
   · 하드웨어 자동 스로틀링 시작

2. P-state 하향 조절:
   · 높은 P0 (3.0GHz) → 낮은 P2 (2.0GHz)
   · 전압도 함께 하향
   · 발열 감소

3. T-state (Throttle State):
   · 클럭 주기적 차단 (Duty Cycle 조절)
   · 12.5%, 25%, 50% Duty
   · 급격한 온도 하락 유도

4. Thermal Shutdown:
   · 105~110°C 도달 시
   · 시스템 강제 종료
   · 칩 손상 방지
```

#### 스로틀링 성능 영향

| 온도 | 상태 | 성능 |
|------|------|------|
| 70°C | 정상 | 100% |
| 85°C | Turbo 제한 | ~90% |
| 95°C | 경미 스로틀 | ~80% |
| 100°C | 중간 스로틀 | ~60% |
| 103°C | 심각 스로틀 | ~40% |

### III. 융합 비교 및 다각도 분석

#### 스로틀링 방지 방법

| 방법 | 효과 | 비용 |
|------|------|------|
| **팬 속도 증가** | 중간 | 낮음 |
| **쿨러 업그레이드** | 높음 | 중간 |
| **서멀 구리스 재도포** | 중간 | 낮음 |
| **케이스 공기 흐름 개선** | 중간 | 낮음 |
| **언더볼팅** | 높음 | 없음 (리스크 있음) |

---

### IV. 실무 적용

#### 실무 시나리오: 노트북 과열 문제

```
증상:
· 팬 소음 증가
· 성능 급감 (게임/렌더링 시)
· FPS 60 → 30으로 하락

원인 분석:
· 서멀 구리스 건조 (2년 이상)
· 먼지 누적 (히트싱크 막힘)
· 팬 고장/속도 저하

해결:
1. 서멀 구리스 재도포 (IC Diamond, Arctic MX-4)
2. 압축 공기로 먼지 제거
3. 팬 속도 프로파일 조절 (PWM)
4. 노트북 쿨링 패드 사용

결과:
· 온도: 100°C → 80°C
· 성능: 스로틀링 해제 → 100% 복원
```

---

### V. 기대효과 및 결론

서멀 스로틀링은 과열로 인한 칩 손상을 방지. 적절한 쿨링으로 스로틀링을 방지하면 성능 유지 가능.

### 어린이를 위한 3줄 비유 설명

**서멀 스로틀링은 "더우면 천천히 달리는 마라토너"와 같아요!**

1. 마라톤 선수가 너무 더우면 속도를 줄여요. 계속 빨리 달리면 쓰러질 수 있으니까요!

2. 컴퓨터도 너무 뜨거워지면 스스로 속도를 줄여요. "잠깐, 너무 뜨거워! 좀 천천히 할게!"라고 하면서요.

3. 그래서 게임하다가 갑자기 느려지면, 컴퓨터가 뜨거워서 쉬고 있는 걸 수 있어요. 쿨링 팬이 잘 작동하는지 확인해보세요!

# 466. 전력 소모 (Power Consumption)

## 핵심 인사이트 (3줄 요약)
> 1. **본질**: 전력 소모는 동적 전력(P_dynamic = αCV²f)과 정적 전력(P_static = V·I_leak)의 합으로 구성되며, 현대 프로세서에서 전력은 성능, 발열, 비용에 직접적 영향을 미친다.
> 2. **가치**: 데이터센터 전력 비용은 연간 수십억 원 규모이며, 1W 절감 = 연간 약 8,760Wh = 약 100만 원(전력+냉각) 절감 효과가 있다.
> 3. **융합**: DVFS, Clock Gating, Power Gating, Dark Silicon 등의 기술로 전력을 최적화하며, PUE(Power Usage Effectiveness)는 데이터센터 효율성의 핵심 지표다.

---

### I. 개요 (Context & Background)

#### 개념 정의

**전력 소모(Power Consumption)**는 컴퓨터 시스템이 단위 시간당 소비하는 전기 에너지다. 와트(W) 단위로 측정되며, 동적 전력(스위칭)과 정적 전력(누설)으로 나뉜다.

```
┌─────────────────────────────────────────────────────────────────┐
│                  전력 소모 공식                                 │
└─────────────────────────────────────────────────────────────────┘

총 전력 = 동적 전력 + 정적 전력 + 단락 전력

P_total = P_dynamic + P_static + P_short

┌─────────────────────────────────────────────────────────────────┐
│  동적 전력 (Dynamic Power):                                     │
│                                                                 │
│  P_dynamic = α × C × V² × f                                     │
│                                                                 │
│  α: 활동 계수 (Activity Factor, 0~1)                           │
│  C: 부하 커패시턴스 (Load Capacitance)                         │
│  V: 전압 (Voltage)                                              │
│  f: 주파수 (Frequency)                                          │
│                                                                 │
│  · 트랜지스터 스위칭 시 발생                                    │
│  · V를 10% 낮추면 전력 19% 감소 (V² 효과)                      │
│  · f를 10% 낮추면 전력 10% 감소                                 │
│                                                                 │
├─────────────────────────────────────────────────────────────────┤
│  정적 전력 (Static Power / Leakage Power):                      │
│                                                                 │
│  P_static = V × I_leak                                          │
│                                                                 │
│  I_leak: 누설 전류 (Leakage Current)                            │
│                                                                 │
│  · 트랜지스터가 꺼져 있어도 흐르는 전류                         │
│  · 온도에 지수함수적 비례                                       │
│  · 공정 미세화할수록 비중 증가                                  │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘

예시 (3nm 공정 CPU):
· TDP: 250W
· 동적 전력: ~175W (70%)
· 정적 전력: ~75W (30%)
```

#### 비유

> **전력 소모는 "자동차 연료 소비"와 같다.**
>
> - 동적 전력 = 주행 중 연료 소모 (속도가 빠를수록, 차가 클수록 많이 듦)
> - 정적 전력 = 공회전 시 연료 소모 (달리지 않아도 새어 나감)
> - 전압 = 엔진 압축비 (높을수록 파워는 세지만 연료 많이 듦)
> - 주파수 = RPM (빠를수록 연료 많이 듦)

#### 전력 절감 기법

```
┌─────────────────────────────────────────────────────────────────┐
│                    전력 절감 기법                               │
└─────────────────────────────────────────────────────────────────┘

1. DVFS (Dynamic Voltage and Frequency Scaling)
┌─────────────────────────────────────────────────────────────────┐
│                                                                 │
│  P_state  │  Frequency  │  Voltage  │  Power                   │
│  ─────────┼─────────────┼───────────┼─────────────────         │
│  P0       │  3.0 GHz    │  1.2V     │  100%                    │
│  P1       │  2.5 GHz    │  1.1V     │  ~70%                    │
│  P2       │  2.0 GHz    │  1.0V     │  ~45%                    │
│  P3       │  1.5 GHz    │  0.9V     │  ~28%                    │
│                                                                 │
│  · 부하에 따라 V/f 동적 조절                                    │
│  · OS/하드웨어가 자동 수행                                      │
│  · Intel SpeedStep, AMD Cool'n'Quiet                           │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘

2. Clock Gating (클럭 게이팅)
┌─────────────────────────────────────────────────────────────────┐
│                                                                 │
│  사용하지 않는 모듈에 클럭 차단                                 │
│                                                                 │
│  ┌─────────┐    ┌─────────┐    ┌─────────┐                    │
│  │  Clock  │───▶│   AND   │───▶│  Module │                    │
│  │ Source  │    │  Gate   │    │   (Idle)│                    │
│  └─────────┘    └────┬────┘    └─────────┘                    │
│                      │                                          │
│                Enable │ (0 = Gate)                              │
│                      ▼                                          │
│              ┌─────────────┐                                    │
│              │ Clock Gated │ ← 동적 전력 차단                   │
│              └─────────────┘                                    │
│                                                                 │
│  · 모듈 단위로 클럭 차단                                        │
│  · 누설 전력은 여전히 존재                                      │
│  · Fine-grained / Coarse-grained                               │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘

3. Power Gating (전력 게이팅)
┌─────────────────────────────────────────────────────────────────┐
│                                                                 │
│  사용하지 않는 모듈에 전원 자체 차단                            │
│                                                                 │
│  ┌─────────┐    ┌─────────┐    ┌─────────┐                    │
│  │  VDD    │───▶│  Sleep  │───▶│  Module │                    │
│  │ (Power) │    │Transistor│   │  (Off)  │                    │
│  └─────────┘    └────┬────┘    └─────────┘                    │
│                      │                                          │
│                Sleep │ (1 = Power Off)                          │
│                      ▼                                          │
│              ┌─────────────┐                                    │
│              │Power Gated  │ ← 정적 전력도 차단                 │
│              └─────────────┘                                    │
│                                                                 │
│  · 누설 전력까지 차단                                           │
│  · Wake-up 지연 시간 존재                                       │
│  · 상태 저장/복원 필요                                          │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### III. 융합 비교 및 다각도 분석

#### 전력 vs 성능 트레이드오프

| 기술 | 전력 절감 | 성능 영향 | 복구 시간 |
|------|-----------|-----------|-----------|
| **DVFS** | 30~50% | 선형 감소 | ns |
| **Clock Gating** | 20~40% | 없음 | ns |
| **Power Gating** | 90%+ | 없음 | us~ms |

### IV. 실무 적용 및 기술사적 판단

#### 실무 시나리오: 데이터센터 전력 비용

```
┌─────────────────────────────────────────────────────────────────┐
│              데이터센터 전력 비용 계산                          │
└─────────────────────────────────────────────────────────────────┘

서버 1,000대, 평균 500W/대:
· 총 IT 전력: 500kW
· 연간 전력: 500kW × 8760h = 4,380,000 kWh
· PUE 1.5 (냉각 포함): 6,570,000 kWh
· 전력 단가 150원/kWh: 985,500,000원/년 (약 10억 원)

전력 10% 절감 시:
· 절감 전력: 657,000 kWh/년
· 절감 비용: 약 1억 원/년
```

### 어린이를 위한 3줄 비유 설명

**전력 소모는 "자동차 연료"와 같아요!**

1. 자동차가 빨리 달릴수록 연료를 더 많이 써요. 컴퓨터도 빠르게 계산할수록 전기를 더 많이 써요!

2. 자동차가 멈춰 있어도 엔진이 켜있으면 연료가 조금씩 새어 나가요. 컴퓨터도 쉬고 있어도 전기가 조금씩 새어 나가요!

3. 데이터센터라는 거대한 컴퓨터 방은 전기를 아주 많이 써요. 그래서 전기를 아끼는 게 돈도 아끼고, 지구도 지키는 일이에요!

# 472. TDP (Thermal Design Power, 열 설계 전력)

## 핵심 인사이트 (3줄 요약)
> 1. **본질**: TDP는 CPU/GPU가 일반적인 부하 상태에서 발생시키는 최대 열 에너지로, 쿨링 시스템(히트싱크, 팬) 설계의 기준이 되는 와트(W) 단위의 지표이다.
> 2. **가치**: TDP 250W CPU는 250W급 쿨링 솔루션이 필요하며, TDP를 초과하는 발열은 서멀 스로틀링(성능 저하)이나 시스템 다운을 유발한다.
> 3. **융합**: TDP는 최대 소비 전력과 다르며, Intel/AMD는 PL1(=TDP)과 PL2(Turbo Power)를 구분하고, 모바일/서버/데스크탑마다 다른 TDP 등급을 사용한다.

---

### I. 개요

#### 개념 정의

**TDP(Thermal Design Power)**는 프로세서가 일반적인 작업 부하에서 방출하는 열 에너지의 최댓값이다. 쿨링 시스템이 처리해야 할 열량을 나타내며, 와트(W) 단위로 표기된다.

```
┌─────────────────────────────────────────────────────────────────┐
│                    TDP vs 소비 전력                             │
└─────────────────────────────────────────────────────────────────┘

TDP (Thermal Design Power):
· "쿨링 시스템이 처리해야 할 열량"
· 일반적 부하(Nominal) 상태 기준
· 설계 지표 (Design Metric)
· 예: 65W, 125W, 250W

최대 소비 전력 (Max Power):
· "실제 최대로 소비하는 전력"
· 극한 부하 시 순간 전력
· PL2, Turbo Power 등으로 불림
· TDP의 1.5~2.5배 가능
```

#### 비유

> **TDP는 "에어컨 용량"과 같다.**
>
> 25평 아파트에 필요한 에어컨은 10평형(약 2.5kW)이다. 이게 TDP다. 하지만 한여름 한낮에는 순간적으로 더 많은 냉방 능력이 필요할 수 있다(=Turbo Power).

---

### II. 아키텍처 및 핵심 원리

#### Intel Power Limit

```
┌─────────────────────────────────────────────────────────────────┐
│              Intel Power Limit (PL1, PL2)                       │
└─────────────────────────────────────────────────────────────────┘

PL1 (Sustained Power):
· = TDP
· 지속 가능한 최대 전력
· 무제한 시간 유지 가능
· 쿨링 설계 기준

PL2 (Turbo Power):
· 순간 최대 전력
· Turbo 시간 (Tau) 동안만 허용
· TDP의 1.5~2.5배
· 보통 56초 (조절 가능)

예시 (Intel Core i9-13900K):
· PL1 (TDP): 125W
· PL2 (Turbo): 253W
· 최대 Turbo 시간: 56초

Power Timeline:
    ┌─────────────────────────────────────────────────────────────
    │                    PL2 (253W)
    │    ┌────────────────────────────────────┐
    │    │         Tau = 56s                 │
P   │    │                                    │
o   │    └────────────────────────────────────┘
w   │                    │
e   │                    ▼
r   │    ┌─────────────────────────────────────────────────────
    │    │              PL1 (125W) - Sustained
    │    │
    └────┴──────────────────────────────────────────────────────
         0        56s                              Time
```

#### TDP 등급별 쿨링 요구사항

| TDP | 쿨링 타입 | 적용 |
|-----|-----------|------|
| 15W | 패시브/소형 팬 | 울트라북 |
| 35W | 소형 히트싱크 | 미니 PC |
| 65W | 표준 히트싱크 + 팬 | 일반 데스크탑 |
| 125W | 대형 히트싱크 | 하이엔드 PC |
| 250W+ | AIO 수냉 / 공냉 | 워크스테이션 |
| 350W+ | 서버급 쿨링 | 데이터센터 |

### IV. 실무 적용

#### 실무 시나리오: 데이터센터 냉각 설계

```
서버 1대당 TDP: 500W (CPU 250W + GPU 250W)
랙당 서버: 40대
랙당 열 부하: 500W × 40 = 20,000W = 20kW

냉각 요구사항:
· CRAC (Computer Room Air Conditioner) 용량 계산
· 냉각 톤(Refrigeration Ton): 1 RT = 3.517kW
· 필요 용량: 20kW / 3.517 ≈ 6 RT

PUE (Power Usage Effectiveness):
· PUE = 총 전력 / IT 전력
· 목표: PUE < 1.5
· 쿨링 효율이 비용 직결
```

---

### V. 기대효과 및 결론

TDP는 쿨링 설계의 핵심 지표. PL2(Turbo)는 TDP보다 높을 수 있음을 인지해야.

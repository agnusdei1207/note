# 데이터 거버넌스 (Data Governance)

## 핵심 인사이트 (3줄 요약)
> 1. **본질**: 데이터 거버넌스는 조직의 데이터 자산을 효율적으로 관리하기 위한 정책, 프로세스, 표준, 역할의 체계입니다.
> 2. **가치**: 데이터 품질, 보안, 컴플라이언스(규제 준수)를 보장하고, 데이터 기반 의사결정을 지원합니다.
> 3. **융합**: DAMA-DMBOK 프레임워크가 표준이며, 데이터 카탈로그, 리니지, 마스터 데이터 관리(MDM)가 핵심 구성요소입니다.

---

### Ⅰ. 개요

#### 1. 핵심 영역 (DAMA-DMBOK)
- 데이터 아키텍처
- 데이터 품질
- 데이터 보안
- 메타데이터 관리
- 데이터 리니지

### Ⅲ. 주요 활동

- 데이터 표준 정의
- 데이터 품질 모니터링
- 접근 권한 관리
- 규제 준수 (GDPR, CCPA)

---

### Ⅳ. 결론

데이터 거버넌스는 데이터를 전략적 자산으로 관리하기 위한 필수 체계입니다.

1. **학교 규칙**: 학교에 규칙이 있듯이 데이터에도 규칙이 있어요.

# 데이터 패브릭 (Data Fabric)

## 핵심 인사이트 (3줄 요약)
> 1. **본질**: 데이터 패브릭은 물리적으로 분산된 멀티 클라우드/온프레미스 데이터 소스를 가상화 계층으로 연결하여, 데이터 이동 없이 단일 뷰로 접근할 수 있는 통합 아키텍처입니다.
> 2. **가치**: 데이터 복제/이동을 최소화하고, AI 기반 메타데이터로 지능적 데이터 발견과 통합을 제공합니다.
> 3. **융합**: 데이터 메시와 상호 보완적이며, 분산형 오너십(메시) + 통합형 인프라(패브릭) 조합으로 활용됩니다.

---

### Ⅰ. 개요

#### 1. 핵심 구성요소
- **데이터 가상화**: 물리적 이동 없는 논리적 통합
- **지능형 메타데이터**: AI 기반 데이터 발견
- **연방 쿼리**: 이기종 DB 동시 쿼리

---

### Ⅲ. 핵심 기술

- **Trino/Presto**: 연방 쿼리 엔진
- **Denodo**: 데이터 가상화
- **Data Virtualization**: 논리적 데이터 통합

---

### Ⅳ. 결론

데이터 패브릭은 멀티 클라우드 환경에서 데이터 접근성을 높이는 아키텍처입니다.

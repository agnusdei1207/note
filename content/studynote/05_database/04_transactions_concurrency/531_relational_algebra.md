# 관계 대수 (Relational Algebra)

## 핵심 인사이트 (3줄 요약)
> 1. **본질**: 관계 대수 (Relational Algebra)는 관계형 데이터베이스에서 데이터를 操作する수학적 연산들의 집합으로,SQL의理论基础가 된다.
> 2. **가치**: 관계 대수를 통해 관계형 데이터 모델의 操作 의미를正式하게定義하고, 쿼리의 equivalence를 증명하고, 최적화의理论基础를 제공한다.
> 3. **융합**: 관계 대수는 관계 해석 (Relational Calculus)과 함께 관계형 데이터베이스의形式적根基이며, SQL 옵티마이저의 내부 동작原理와 밀접하게 연결된다.

---

## Ⅰ. 개요 및 필요성 (Context & Necessity)

### 개념 정의

관계 대수 (Relational Algebra)는 **관계형 데이터 모델에서 정의된 Operanden으로, 테이블에서 데이터를 추출하고 조작하기 위한수학적 연산들의 집합**이다.

### 기본 연산자

| 연산자 | 기호 | 설명 |
|:---|:---|:---|
| **선택 (Selection)** | σ (sigma) | 행을 필터링 |
| **프로젝션 (Projection)** | π (pi) | 열을 선택 |
| **합집합 (Union)** | ∪ | 두 관계의 합집합 |
| **차집합 (Difference)** | − | 두 관계의 차집합 |
| **카티션 프로덕트 (Cartesian Product)** | × | 두 관계의 조합 |
| **리네임 (Rename)** | ρ (rho) | 관계 또는 속성 이름 변경 |
| **조인 (Join)** | ⋈ | 결합 연산 (파생) |

### 비유

관계 대수는 **수학의集合論**과 같다. 합집합, 차집합, 교집합 등의集合演算이 테이블에 적용되며, σ와 π는 테이블에서行과열을 선택하는 기능이다.

- **📢 섹션 요약 비유:**수학에서 집합 A와 B의 합집합, 차집합을 구하는 것처럼, 관계 대수에서도 테이블 간의集合演算을 수행합니다.

## Ⅱ. 주요 연산자

### 1. 선택 (Selection) σ

행 (row)를 조건에 따라 필터링한다.

```
σ_city='서울' (customers)

[결과]
┌────────┬────────┐
│ 고객ID  │ 도시   │
├────────┼────────┤
│ C001   │ 서울   │
│ C003   │ 서울   │
└────────┴────────┘
```

### 2. 프로젝션 (Projection) π

지정한 열만 선택한다.

```
π_name, city (customers)

[결과]
┌────────┬────────┐
│ 이름    │ 도시   │
├────────┼────────┤
│ 김철수  │ 서울   │
│ 이영희  │ 부산   │
│ 박민수  │ 서울   │
└────────┴────────┘
```

### 3. 합집합 (Union) ∪

두 관계의 합집합을 구한다. 단, 스키마가 동일해야 한다.

```
R ∪ S = { t | t ∈ R or t ∈ S }
```

### 4. 조인 (Join) ⋈

두 관계를 결합한다.

```
customers ⋈ orders
= σ_customers.customer_id=orders.customer_id (customers × orders)
```

```
┌─────────────────────────────────────────────────────────────┐
│                    조인 (Join) 동작 예시                            │
├─────────────────────────────────────────────────────────────┤
│                                                                     │
│   customers:              orders:                                 │
│   ┌────────┬──────┐       ┌────────┬──────┐               │
│   │ C001   │ 김철수│       │ O001   │ C001 │               │
│   │ C002   │ 이영희│       │ O002   │ C002 │               │
│   └────────┴──────┘       └────────┴──────┘               │
│                                                                     │
│   customers ⋈ orders:                                             │
│   ┌────────┬──────┬────────┬──────┐                         │
│   │ C001   │ 김철수│ O001   │ C001 │                         │
│   │ C002   │ 이영희│ O002   │ C002 │                         │
│   └────────┴──────┴────────┴──────┘                         │
│                                                                     │
└─────────────────────────────────────────────────────────────┘
```

## 👶 어린이를 위한 3줄 비유 설명

1. 관계 대수는 **수학 시간에 배운 집합 연산**과 같아요. 합집합, 차집합, 교집합처럼 테이블도 연산할 수 있어요.
2. σ (시그마)는 "서울에 사는 사람만 골라"이고, π (파이)는 "이름과 주소만 보여줘"예요.
3. 관계 대수를 알면 SQL이 어떻게 작동하는지 더 깊이 이해할 수 있어요!

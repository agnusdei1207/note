## 핵심 인사이트 (3줄 요약)

> 1. **본질**: 정점 커버(Vertex Cover)는 그래프의 모든 에지(Edge)에 대해 적어도 한 끝점이 포함되는 최소 정점 집합을 찾는 문제로, 결정 버전은 NP-Complete이다.
> 2. **가치**: 2-근사 알고리즘(최대 매칭의 양 끝점 선택)이 존재하며, 이분 그래프(Bipartite Graph)에서는 쾨니히 정리(König's Theorem)에 의해 최대 매칭 = 최소 정점 커버로 다항 시간에 정확히 풀린다.
> 3. **판단 포인트**: 정점 커버 = n - 독립 집합(Independent Set) — 이 보완 관계가 복잡도·알고리즘 설계에서 핵심 환산 도구다.

---

## Ⅰ. 개요 및 필요성

**정점 커버(Vertex Cover)**는 그래프 G = (V, E)에서 모든 에지 (u, v) ∈ E에 대해 u 또는 v 중 적어도 하나가 포함되는 정점 집합 C ⊆ V를 찾는 문제다.

- **결정 문제**: "크기 k 이하의 정점 커버가 존재하는가?" → NP-Complete.
- **최소 정점 커버**: 가장 작은 크기의 커버 → NP-Hard.
- **직관적 의미**: 도로 교차로 CCTV 설치 — 모든 도로(에지)를 감시하기 위한 최소 교차로(정점) 집합.

📢 **섹션 요약 비유**: 정점 커버는 "도시의 모든 골목이 적어도 한 카메라에 잡히게 하는 최소 카메라 위치" — 최소화가 어려운 게 핵심이다.

---

## Ⅱ. 아키텍처 및 핵심 원리

### 정점 커버 예시

```
    1 ── 2 ── 3
    │         │
    4 ──────── 5

정점 커버 {2, 4, 5}:
  에지 (1,2): 2 포함 ✓
  에지 (2,3): 2 포함 ✓
  에지 (1,4): 4 포함 ✓
  에지 (4,5): 4,5 포함 ✓
  에지 (3,5): 5 포함 ✓
최소 정점 커버: {2, 4} 또는 {2, 5} 등 크기 2짜리 탐색 필요
```

### 2-근사 알고리즘

```
┌─────────────────────────────────────────────────┐
│  2-Approximation for Vertex Cover               │
│                                                 │
│  C ← ∅                                         │
│  E' ← E                                        │
│  while E' ≠ ∅:                                 │
│      임의 에지 (u, v) ∈ E' 선택                │
│      C ← C ∪ {u, v}          ← 양 끝점 모두 추가│
│      u, v에 인접한 모든 에지 E'에서 제거        │
│  return C                                       │
│                                                 │
│  보장: |C| ≤ 2 · OPT                           │
│  이유: 선택된 에지들은 서로 매칭(Matching) 관계│
│       OPT ≥ 매칭 크기 ≥ |C|/2                  │
└─────────────────────────────────────────────────┘
```

| 알고리즘 | 복잡도 | 근사 보장 | 조건 |
|:---|:---:|:---:|:---|
| 2-근사 (최대 매칭 기반) | O(E) | 2·OPT | 일반 그래프 |
| 쾨니히 정리 (Hopcroft-Karp) | O(E·√V) | 정확 | 이분 그래프만 |
| 반정수 LP 반올림 | O(LP 시간) | 2·OPT | 일반 그래프 |
| FPT (k-Vertex Cover) | O(2ᵏ · n) | 정확 | k 작을 때 |

📢 **섹션 요약 비유**: 2-근사는 "에지마다 양쪽 다 감시" — 최적보다 최대 2배 많은 카메라를 설치하지만, 빠르게 모든 도로를 커버할 수 있다.

---

## Ⅲ. 비교 및 연결

### 쾨니히 정리 (König's Theorem)

> **이분 그래프(Bipartite Graph)에서**:  
> 최소 정점 커버(Min Vertex Cover) = 최대 매칭(Max Matching)

이 정리 덕분에 이분 그래프에서는 Hopcroft-Karp 알고리즘으로 O(E·√V)에 정확한 최소 정점 커버를 구할 수 있다.

### 정점 커버·독립 집합·클리크 관계

```
┌──────────────────────────────────────────────────────┐
│  G = (V, E)에서                                      │
│                                                      │
│  S가 Vertex Cover                                    │
│  ⟺  V \ S가 Independent Set                         │
│                                                      │
│  따라서:                                             │
│  Min Vertex Cover + Max Independent Set = |V|        │
│                                                      │
│  G의 보완 그래프(Ḡ)에서:                            │
│  Independent Set(G) = Clique(Ḡ)                      │
└──────────────────────────────────────────────────────┘
```

| 문제 | 복잡도 | 비고 |
|:---|:---:|:---|
| Min Vertex Cover (일반) | NP-Hard | 2-근사 존재 |
| Min Vertex Cover (이분) | P | 쾨니히 정리 |
| Max Independent Set | NP-Hard | 근사 매우 어려움 |
| k-Vertex Cover (FPT) | O(2ᵏ·n) | 파라미터화 다항 시간 |

📢 **섹션 요약 비유**: 정점 커버와 독립 집합은 동전의 앞뒤 — 한 쪽이 "감시하는 정점", 다른 쪽이 "감시 안 받는 정점"이다.

## Ⅳ. 실무 적용 및 기술사 판단

| 분야 | 활용 |
|:---|:---|
| 네트워크 모니터링 | 최소 라우터 배치로 전체 링크 감시 |
| 단백질 상호작용 | 모든 상호작용 쌍을 커버하는 최소 단백질 집합 |
| 컴파일러 최적화 | 레지스터 할당 (독립 집합 변환) |
| 보안 카메라 배치 | 최소 CCTV로 모든 구간 커버 |

기술사 핵심 포인트:
1. 이분 그래프에서 쾨니히 정리로 정확해 가능함을 명시.
2. 일반 그래프에서 2-근사 알고리즘의 증명 논리(매칭 크기 ≥ OPT/2) 설명.
3. FPT 알고리즘(O(2ᵏ·n))으로 k가 작은 실무 시나리오 처리 가능.

📢 **섹션 요약 비유**: 정점 커버 실무는 "예산이 한정된 CCTV 설치 계획" — 이론적 최적은 어렵지만, 2배 예산이면 반드시 모든 골목을 커버할 수 있다.

---

## Ⅴ. 기대효과 및 결론

정점 커버 문제를 이해하면:
- **2-근사 알고리즘의 정당성**: 매칭 기반 근사 비율 증명으로 알고리즘 설계 철학 학습.
- **쾨니히 정리의 실용성**: 이분 그래프를 인식하고 정확 해법을 적용하는 능력.
- **복잡도 삼각형(VC-IS-Clique) 완성**: 세 문제의 상호 환산으로 NP-Complete 생태계 전체 조망.

📢 **섹션 요약 비유**: 정점 커버는 "그래프 세계의 경비원 배치 문제" — 최소 경비원으로 최대 보안을 달성하는 최적화의 전형이다.

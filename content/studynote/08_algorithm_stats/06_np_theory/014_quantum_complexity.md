## 핵심 인사이트 (3줄 요약)

> 1. **본질**: BQP(Bounded-error Quantum Polynomial time, 유계 오류 양자 다항 시간)는 양자 컴퓨터가 다항 시간에 효율적으로 풀 수 있는 문제 집합으로, P ⊆ BQP ⊆ PSPACE가 알려져 있다.
> 2. **가치**: 쇼어(Shor) 알고리즘은 정수 인수분해를 다항 시간에 해결해 RSA를 위협하고, 그로버(Grover) 알고리즘은 비정렬 탐색을 O(√n)으로 개선한다.
> 3. **판단 포인트**: BQP ≠ NP로 추정 — NP-Complete 문제를 양자 컴퓨터가 다항 시간에 풀 수 없으므로, RSA 이후의 보안은 포스트 양자 암호(PQC, Post-Quantum Cryptography)로 전환이 필요하다.

---

## Ⅰ. 개요 및 필요성

**양자 컴퓨팅(Quantum Computing)**은 중첩(Superposition)·얽힘(Entanglement)·간섭(Interference)을 이용해 고전 컴퓨터로는 비효율적인 특정 문제를 다항 시간에 해결한다.

- **중첩**: 큐비트(Qubit)는 0과 1을 동시에 가질 수 있어 병렬 탐색이 가능.
- **얽힘**: 두 큐비트의 상태가 물리적으로 분리되어도 상관관계 유지.
- **간섭**: 정답으로 가는 경로의 확률 진폭은 증폭, 오답은 상쇄.
- **BQP**: 2/3 이상의 확률로 정확한 답을 내는 양자 알고리즘이 다항 시간에 존재하는 문제 집합.

📢 **섹션 요약 비유**: 양자 컴퓨터는 미로의 모든 경로를 동시에 탐색하는 "평행우주 탐정" — 고전 컴퓨터는 한 경로씩 시도하지만, 양자는 모든 경로를 동시에 열어본다.

### 주요 양자 알고리즘

| 알고리즘 | 문제 | 고전 최선 | 양자 | 비고 |
|:---|:---|:---:|:---:|:---|
| Shor (1994) | 정수 인수분해 | O(e^(n^(1/3))) | O(n³) | RSA 위협 |
| Grover (1996) | 비정렬 탐색 | O(n) | O(√n) | 암호 해독 가속 |
| HHL (2009) | 선형 방정식 | O(n³) | O(log n) | 조건부 지수 가속 |
| VQE | 분자 에너지 | 지수 | 다항(추정) | 양자 화학 |
| QAOA | 최적화 근사 | — | 근사 | NP-Hard 문제 |

### Shor 알고리즘 핵심 원리

```
┌──────────────────────────────────────────────────────────┐
│  Shor's Algorithm: N 인수분해 (N = p·q)                  │
│                                                          │
│  1. 고전: 랜덤 a 선택 (1 < a < N), gcd(a,N) 확인        │
│           gcd ≠ 1 → 인수 발견, 종료                      │
│         │                                                │
│         ▼                                                │
│  2. 양자: f(x) = aˣ mod N 의 주기 r 탐색                │
│           QFT (Quantum Fourier Transform) 이용 O(n²)    │
│         │                                                │
│         ▼                                                │
│  3. 고전: r이 짝수이고 a^(r/2) ≢ -1 (mod N) 이면        │
│           gcd(a^(r/2)±1, N)으로 인수 계산                │
│                                                          │
│  전체: O(n² log n log log n) ← 다항 시간                │
└──────────────────────────────────────────────────────────┘
```

📢 **섹션 요약 비유**: Shor 알고리즘은 "자물쇠의 패턴(주기)을 양자 간섭으로 순식간에 찾아내는 마법" — 고전 컴퓨터가 수천 년 걸릴 계산을 몇 분에 처리한다.

## Ⅲ. 비교 및 연결

### 포스트 양자 암호 (PQC, Post-Quantum Cryptography)

| 현재 암호 | 양자 위협 | PQC 대안 |
|:---|:---|:---|
| RSA | Shor로 다항 시간 인수분해 | CRYSTALS-Kyber (격자 기반) |
| ECC | Shor로 이산 로그 | CRYSTALS-Dilithium (격자 기반) |
| AES-128 | Grover로 키 탐색 2^64 → 2^64 | AES-256 (키 길이 두 배) |
| SHA-256 | Grover로 역상 2^128 → 2^64 | SHA-384/512 |

NIST(National Institute of Standards and Technology)는 2022~2024년 PQC 표준 4개를 선정:
- **CRYSTALS-Kyber**: KEM(Key Encapsulation Mechanism) 표준.
- **CRYSTALS-Dilithium, FALCON, SPHINCS+**: 전자서명 표준.

### QMA (Quantum Merlin-Arthur)

**QMA**는 NP의 양자 대응물 — 양자 증인(Witness)을 양자 검증자가 다항 시간에 검증 가능한 문제.  
- QMA-Complete 문제: 로컬 해밀토니안 문제(Local Hamiltonian, 양자 물리 시뮬레이션).  
- P ⊆ NP ⊆ QMA로 추정 (QMA = NP인지 미증명).

📢 **섹션 요약 비유**: QMA는 "답을 양자 상태로 제출하면, 양자 검사관이 확인" — NP가 고전 증인이라면, QMA는 양자 증인을 쓰는 업그레이드 버전이다.

## Ⅳ. 실무 적용 및 기술사 판단

기술사 답안에서 양자 복잡도 핵심:

1. **BQP vs. NP 구분**: 양자 컴퓨터가 NP-Complete 문제를 다항 시간에 풀지 못한다 (BQP ⊄ NP로 추정) → 양자 = 만능 아님.
2. **암호 위협 논증**: Shor → RSA/ECC 위협, Grover → 대칭키 강도 절반 → AES-256, SHA-384 권장.
3. **NIST PQC 표준**: 격자 기반(Lattice-Based) 암호가 현재 PQC 주류 — 기술사 IT 보안 논술에 필수.
4. **양자 우위(Quantum Advantage)**: Google 2019 수이카모어(Sycamore) — 특정 랜덤 회로 샘플링에서 고전 최강 대비 양자 우위 주장.

📢 **섹션 요약 비유**: 양자 컴퓨터는 "특수 열쇠" — 몇 가지 자물쇠(RSA, 탐색)는 순식간에 열지만, 일반 자물쇠(NP-Complete)는 여전히 못 연다.

---

## Ⅴ. 기대효과 및 결론

양자 복잡도를 이해하면:
- **현대 암호 취약성 인식**: Shor 알고리즘의 위협과 PQC 전환 필요성 이해.
- **양자 우위의 범위 파악**: BQP와 NP의 관계로 "양자 컴퓨터의 한계"를 정확히 인식.
- **PQC 설계 근거**: 격자 문제(LWE), 코드 기반 암호 등이 양자 내성인 이유 설명.

양자 복잡도는 알고리즘과 보안의 미래를 동시에 바꾸는 이론·실용의 교차점이다.

📢 **섹션 요약 비유**: 양자 복잡도는 알고리즘 세계의 "핵에너지 발견" — 기존 규칙을 바꾸는 새로운 물리 법칙이지만, 모든 것을 해결하는 만능이 아니라 특정 영역에서만 강력하다.

### 📈 관련 키워드 및 발전 흐름도

```text
[고전 복잡도 이론 (P, NP) — 다항 시간 결정론적·비결정론적 계산 한계]
    │
    ▼
[BQP (Bounded-error Quantum Polynomial-time) — 양자 다항 시간 해결 가능 문제류]
    │
    ▼
[양자 우위 (Quantum Supremacy) — 특정 문제에서 고전 컴퓨터를 완전히 압도]
    │
    ▼
[쇼어 알고리즘 (Shor's Algorithm) — RSA 인수분해를 지수→다항 시간으로 돌파]
    │
    ▼
[양자 오류 정정 (QEC) — 큐비트 결어긋남 극복, 실용 양자 컴퓨터 조건]
```

이 흐름은 고전 복잡도 이론의 P/NP 한계에서 양자 복잡도 클래스 BQP가 등장하고, 양자 우위와 쇼어 알고리즘이 실용성을 증명하며 오류 정정 기술로 실용화를 향해 나아가는 양자 컴퓨팅 발전의 핵심 계보를 보여준다.

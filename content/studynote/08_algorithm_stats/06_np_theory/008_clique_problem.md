## 핵심 인사이트 (3줄 요약)

> 1. **본질**: 클리크(Clique)는 그래프에서 모든 정점 쌍이 연결된 완전 부분 그래프이며, k-클리크 존재 결정 문제는 NP-Complete이다.
> 2. **가치**: 클리크 문제는 3-SAT로부터 다항 시간 환산으로 NP-Complete가 증명되며, 독립 집합(Independent Set)·정점 커버(Vertex Cover)와 상호 환산 관계로 NP-Complete 생태계의 핵심 노드다.
> 3. **판단 포인트**: 최대 클리크(Max Clique)는 NP-Hard이며, n^(1-ε) 이내의 근사 알고리즘도 P ≠ NP이면 존재하지 않는 '근사 불가능(Inapproximable)' 문제다.

---

## Ⅰ. 개요 및 필요성

**클리크(Clique)**는 그래프 G = (V, E)에서 임의의 두 정점이 모두 연결된 정점 부분집합 S를 의미한다.

- **k-클리크 결정 문제(Decision)**: G에 크기 k인 클리크가 존재하는가? → NP-Complete.
- **최대 클리크(Max Clique)**: G에서 가장 큰 클리크의 크기/내용을 찾는 문제 → NP-Hard.
- **현실적 의미**: 소셜 네트워크에서 '모두 서로 친구인 그룹', 생물정보학에서 '모두 상호작용하는 단백질 집합' 탐색에 사용.

📢 **섹션 요약 비유**: 클리크는 파티에서 "서로 다 아는 사이"인 그룹 — 그 그룹이 k명 이상인지 확인하는 것조차 매우 어렵다.

---

## Ⅱ. 아키텍처 및 핵심 원리

### 클리크 예시 그래프

```
   1 ── 2
   │ ╲ ╱ │
   │  ╳  │
   │ ╱ ╲ │
   3 ── 4 ── 5
        │
        6

{1, 2, 3, 4}는 클리크? → 아니오 (3-4만 연결 안 됨 가정)
{1, 2, 4}가 클리크? → 1-2, 1-4, 2-4 모두 연결 시 YES
```

```
┌──────────────────────────────────────────────────┐
│  k-Clique 결정 알고리즘 (나이브 방법)            │
│                                                  │
│  입력: G = (V, E), k                             │
│    │                                             │
│    ▼                                             │
│  C(n, k)개의 부분집합 열거     O(nᵏ)             │
│    │                                             │
│    ▼                                             │
│  각 부분집합 S에 대해:                           │
│    S의 모든 쌍이 E에 있는지 확인  O(k²)          │
│    │                                             │
│    ▼                                             │
│  하나라도 YES → 출력 YES                         │
│  전부 NO  → 출력 NO                              │
└──────────────────────────────────────────────────┘
총 시간: O(nᵏ · k²)  →  k가 고정되면 다항, k가 변하면 NP-Complete
```

### NP-Complete 증명 (3-SAT ≤_p Clique)

| 단계 | 내용 |
|:---:|:---|
| ∈ NP | 크기 k 집합 S가 주어지면, S가 클리크인지 O(k²) 검증 가능 |
| NP-Hard | 3-SAT의 각 절 Cᵢ = (a ∨ b ∨ c)에서 3개 노드 생성 (절당 3개, m절이면 3m개 노드) |
| 환산 규칙 | 서로 다른 절의 노드 중, 서로 모순되지 않는 쌍(¬xᵢ와 xᵢ가 아닌)을 에지로 연결 |
| 핵심 | 3-SAT가 만족 가능 ⟺ 그래프에 m-클리크 존재 (각 절에서 하나씩 참인 리터럴 선택) |

📢 **섹션 요약 비유**: 3-SAT 환산은 마치 "각 팀에서 대표 한 명씩 뽑아 서로 충돌 없는 팀을 만들 수 있냐" — 충돌 없는 대표단이 클리크다.

## Ⅲ. 비교 및 연결

### 클리크·독립 집합·정점 커버의 삼각 관계

```
┌──────────────────────────────────────────────────────┐
│          그래프 G = (V, E)                           │
│                                                      │
│  Clique(G, k) ←→ Independent Set(Ḡ, k)              │
│       G의 보완 그래프(Ḡ)에서 Independent Set = Clique │
│                                                      │
│  Independent Set(G, k) ←→ Vertex Cover(G, n-k)      │
│       S가 Independent Set ⟺ V\S가 Vertex Cover       │
└──────────────────────────────────────────────────────┘
```

| 문제 | 관계 | 복잡도 |
|:---|:---|:---:|
| Clique (k) | G에서 크기 k 완전 부분그래프 | NP-Complete |
| Independent Set (k) | G에서 크기 k 독립 집합 | NP-Complete |
| Vertex Cover (k) | G의 n-k 정점 커버 | NP-Complete |
| Max Clique | 최대 클리크 크기 | NP-Hard |

### 근사 불가능성 (Inapproximability)

최대 클리크 문제는 P ≠ NP 가정 하에 n^(1-ε) 이내의 근사도 불가능하다 (Håstad 1999). 이는 대부분의 NP-Hard 문제와 달리 "좋은 근사 알고리즘조차 없다"는 매우 강한 결과다.

📢 **섹션 요약 비유**: 클리크 근사는 마치 "눈 가리고 소나기를 100% 피하라"는 요구 — 완벽한 해법은 물론이고, 웬만큼 좋은 해법조차 이론상 보장이 안 된다.

## Ⅳ. 실무 적용 및 기술사 판단

| 분야 | 활용 |
|:---|:---|
| 소셜 네트워크 분석 | 팔로워 완전 연결 그룹(파벌) 탐지 |
| 생물정보학 | 단백질 상호작용 네트워크에서 복합체 탐지 |
| 컴파일러 최적화 | 레지스터 할당 (독립 집합으로 변환) |
| 패턴 매칭 | 화학 구조 유사성 탐색 (서브그래프 동형) |

기술사 답안에서 핵심:
1. k-클리크 결정 = NP-Complete 증명 구조 서술 (∈NP + 3-SAT ≤_p Clique).
2. Max Clique = NP-Hard + 근사 불가능성 언급.
3. 독립 집합·정점 커버와의 상호 환산 관계 도식화.

📢 **섹션 요약 비유**: 클리크 문제는 NP 세계의 "만인이 서로 아는 집합" — 이를 찾는 것이 어렵기 때문에, 실무에서는 휴리스틱(탐욕 클리크, SA)으로 근사한다.

---

## Ⅴ. 기대효과 및 결론

클리크 문제를 이해하면:
- **NP-Complete 환산 기술 습득**: 3-SAT → Clique 환산을 통해 환산 패턴 이해.
- **그래프 문제 복잡도 네트워크 파악**: Clique-IS-VC 삼각 관계로 세 가지 문제를 동시에 이해.
- **근사 불가능성 인식**: 모든 NP-Hard 문제가 근사 알고리즘을 허용하지 않음을 학습.

📢 **섹션 요약 비유**: 클리크 문제는 NP-Complete 세계의 "허브 공항" — 독립 집합, 정점 커버, SAT가 모두 연결되는 중심 노드다.

| 개념 | 관계 |
|:---|:---|
| Clique (k) | NP-Complete (3-SAT ≤_p Clique) |
| Max Clique | NP-Hard, 근사 불가능 |
| Independent Set | 보완 그래프에서 Clique와 동치 |
| Vertex Cover | Independent Set의 여집합 |
| Karp 21 | Clique 포함 |
| 소셜 네트워크 | 파벌(팔로워 완전 연결 그룹) 탐지 |

### 📈 관련 키워드 및 발전 흐름도

```text
[그래프 이론 (Graph Theory) — 정점, 간선]
    │
    ▼
[클리크 문제 (Clique Problem) — NP-Complete]
    │
    ▼
[독립 집합 (Independent Set) / 정점 커버 (Vertex Cover)]
    │
    ▼
[3-SAT 환원 (3-SAT Reduction) — Karp 21 문제]
    │
    ▼
[근사 알고리즘 (Approximation Algorithm) — 최대 클리크 근사]
```

클리크 문제가 NP-Complete의 중심 노드로서 다른 NP 문제들과 환원 관계를 맺고 실용적 근사 해법으로 이어지는 흐름이다.

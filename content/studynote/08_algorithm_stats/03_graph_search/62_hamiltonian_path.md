## 핵심 인사이트 (3줄 요약)
1. **정점 중심 탐색**: 그래프의 모든 정점을 정확히 한 번씩만 방문하는 경로로, 오일러 경로(간선 중심)와 대조되는 개념이다.
2. **NP-완전 문제 (NP-Complete)**: 일반적인 그래프에서 해밀턴 경로의 존재 여부를 판별하는 것은 다항 시간 내에 해결 불가능한 대표적인 난제이다.
3. **최적화 확장**: 해밀턴 경로에 '최단 거리' 조건을 추가하면 외판원 문제(TSP)로 확장되며, 물류 및 회로 설계의 핵심 로직이 된다.

### Ⅰ. 개요 (Context & Background)
해밀턴 경로(Hamiltonian Path)는 19세기 수학자 윌리엄 로언 해밀턴이 제안한 '이코시언 게임'에서 유래했다. 그래프 이론에서 모든 간선을 지나는 '오일러 경로'는 차수(Degree) 계산만으로 쉽게 판별 가능하지만, 모든 정점을 지나는 해밀턴 경로는 효율적인 판별 알고리즘이 존재하지 않아 전수 조사나 백트래킹이 필수적이다. 이는 컴퓨터 과학의 복잡도 이론에서 P vs NP 문제를 설명하는 핵심 키워드 중 하나다.

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)
해밀턴 경로는 모든 정점 $V$를 방문 리스트에 포함하며 중복이 없어야 한다. 시작점과 끝점이 연결되어 사이클을 이루면 '해밀턴 회로(Hamiltonian Cycle)'라 부른다.

```text
[Hamiltonian Path vs Cycle Concept / 해밀턴 경로 vs 회로]

(A)---(B)           (A)---(B)
 |     |             |     |
(C)---(D)           (C)---(D)
 |                   |     |
(E)                 (E)---(F)

<Path Example>      <Cycle Example>
A -> B -> D -> C -> E (O)   A -> B -> D -> F -> E -> C -> A (O)
(All nodes visited once)    (Returns to Start Node)
```

**핵심 메커니즘:**
- **Backtracking (백트래킹)**: 현재 정점에서 방문하지 않은 인접 정점으로 이동하며, 모든 정점을 방문할 때까지 재귀적으로 탐색한다. 막히면 이전 단계로 되돌아간다.
- **Dirac's Theorem (디락의 정리)**: $n \ge 3$인 그래프에서 모든 정점의 차수가 $n/2$ 이상이면 해밀턴 회로가 존재한다는 충분 조건이다.
- **Bitmask DP**: $O(2^n \cdot n^2)$ 복잡도로 정점 방문 상태를 비트로 표현하여 중복 계산을 방지한다.

### Ⅲ. 융합 비교 및 다각도 분석 (Comparison & Synergy)

| 비교 항목 | 해밀턴 경로 (Hamiltonian Path) | 오일러 경로 (Eulerian Path) |
| :--- | :--- | :--- |
| **핵심 대상** | 모든 **정점(Vertex)** 방문 | 모든 **간선(Edge)** 방문 |
| **방문 횟수** | 각 정점을 **정확히 1회** | 각 간선을 **정확히 1회** |
| **판별 난이도** | **NP-Complete** (매우 어려움) | **P Class** (차수 확인으로 해결) |
| **필요 조건** | 충분 조건만 존재 (Dirac, Ore) | 차수(Degree)가 홀수인 정점이 0개 또는 2개 |
| **응용 분야** | 외판원 문제(TSP), 유전자 서열 정렬 | 네트워크 라우팅, 쓰레기 수거차 경로 |

### Ⅳ. 실무 적용 및 기술사적 판단 (Strategy & Decision)
기술사적 관점에서 해밀턴 경로는 단순한 경로 찾기가 아닌, **자원 제약 하의 최적 스케줄링** 문제로 접근해야 한다. 실무 적용 시 데이터 크기에 따라 전략을 달리해야 한다.
1. **소규모 데이터 (N < 20)**: 비트마스크를 활용한 동적 계획법(DP)으로 정확한 해를 구한다.
2. **대규모 데이터**: 근사 알고리즘(Approximation)이나 유전 알고리즘(GA), 타부 서치(Tabu Search) 등 메타 휴리스틱 기법을 적용하여 '충분히 좋은 해'를 빠르게 도출하는 것이 경제적이다.
3. **회로 설계 및 물류**: 드릴링 머신의 이동 최소화, 택배 배송 경로 최적화 등에서 Hamiltonian 기반 모델링을 통해 비용을 절감한다.

### Ⅴ. 기대효과 및 결론 (Future & Standard)
해밀턴 경로는 현대 물류 혁명의 수학적 기초다. AI와 결합하여 자율주행 로봇의 효율적인 동선 관리 및 스마트 팩토리의 공정 최적화에 기여하고 있다. 복잡도 이론의 한계에도 불구하고, 다양한 휴리스틱 기법의 발전으로 실무적 가치는 더욱 높아지고 있으며, 향후 양자 컴퓨팅이 상용화될 경우 NP 문제의 돌파구로서 다시금 주목받을 핵심 표준 모델이다.

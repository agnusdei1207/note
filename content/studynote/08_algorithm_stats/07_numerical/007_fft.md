> **핵심 인사이트 3줄**
> 1. FFT(Fast Fourier Transform)는 DFT(Discrete Fourier Transform)를 O(N²)에서 O(N log N)으로 가속하는 분할정복 알고리즘으로, 신호 처리·오디오·이미지·다항식 곱셈의 핵심이다.
> 2. Cooley-Tukey 알고리즘은 DFT를 짝수 인덱스(DFT_even)와 홀수 인덱스(DFT_odd) 두 절반 크기 DFT로 재귀 분할하고, 버터플라이(butterfly) 연산으로 결합한다.
> 3. FFT는 O(N log N) 다항식 곱셈을 가능하게 해 경쟁 프로그래밍의 큰 수 곱셈과 신호 필터링에서 핵심 도구로 사용된다.

---

## Ⅰ. DFT와 FFT 개요

### 1.1 DFT 정의

X[k] = Σ_{n=0}^{N-1} x[n] · ω_N^{nk}  (ω_N = e^{-2πi/N}, k=0..N-1)

- 시간 도메인 신호 x[n] → 주파수 도메인 X[k]
- 나이브 구현: O(N²) — N=10^6이면 10^12 연산

### 1.2 FFT의 핵심 아이디어

```
DFT(N) = DFT_even(N/2) + ω^k · DFT_odd(N/2)
```

- N이 2의 거듭제곱이면 log₂N 번 재귀 분할 가능
- 총 복잡도: O(N log N)

| N      | DFT O(N²) | FFT O(N log N) |
|--------|-----------|----------------|
| 1,024  | 1,048,576 | 10,240         |
| 1,048,576 | ~10^12 | ~20,971,520    |

📢 **섹션 요약 비유**: DFT는 1000명의 악기 소리를 일일이 들어보는 것, FFT는 절반씩 나눠 들어보고 합치는 것 — 훨씬 빠르다.

## Ⅱ. Cooley-Tukey 알고리즘

### 2.1 분할 과정

```
x = [x0, x1, x2, x3, x4, x5, x6, x7]  (N=8)
  ↓ 짝수/홀수 분리
x_even = [x0, x2, x4, x6]
x_odd  = [x1, x3, x5, x7]
  ↓ 각각 DFT
DFT(x_even) = [E0, E1, E2, E3]
DFT(x_odd)  = [O0, O1, O2, O3]
  ↓ 버터플라이 결합
X[k]     = E[k] + ω^k · O[k]
X[k+N/2] = E[k] - ω^k · O[k]   (k=0..N/2-1)
```

### 2.2 버터플라이 다이어그램

```
E[k] ──┬──────── X[k]   = E[k] + ω^k·O[k]
       │ω^k
O[k] ──┴──(×)── X[k+4] = E[k] - ω^k·O[k]
```

각 단계에서 N/2 버터플라이 → log₂N 단계 → 총 O(N log N).

📢 **섹션 요약 비유**: 버터플라이는 두 집 사이 직통 전화 — 두 절반 결과를 한 번의 곱셈+덧셈으로 합쳐 4번 할 일을 2번에 끝낸다.

## Ⅲ. FFT 구현

### 3.1 재귀 FFT (Python)

```python
import cmath

def fft(x):
    N = len(x)
    if N <= 1:
        return x
    even = fft(x[0::2])
    odd  = fft(x[1::2])
    T = [cmath.exp(-2j * cmath.pi * k / N) * odd[k] for k in range(N // 2)]
    return [even[k] + T[k] for k in range(N // 2)] +            [even[k] - T[k] for k in range(N // 2)]
```

### 3.2 반복 FFT (Bit-Reversal)

재귀 대신 비트 역순 인덱싱으로 스택 오버헤드 제거:
```
인덱스: 000 001 010 011 100 101 110 111
비트역: 000 100 010 110 001 101 011 111 (= 0 4 2 6 1 5 3 7)
```

📢 **섹션 요약 비유**: 반복 FFT는 책장 책을 미리 역순으로 정렬해두는 것 — 재귀 왕복 대신 순서대로 한 번 훑어내려간다.

## Ⅳ. 다항식 곱셈 응용

### 4.1 FFT 기반 O(N log N) 곱셈

```
A(x) = a0 + a1·x + ... + a_{n-1}·x^{n-1}
B(x) = b0 + b1·x + ... + b_{n-1}·x^{n-1}
C(x) = A(x) · B(x)  → 차수 2n-2

일반 곱셈: O(N²)
FFT 곱셈:
  1. FFT(A), FFT(B)  O(N log N)
  2. 점별 곱셈       O(N)
  3. IFFT(결과)      O(N log N)
총: O(N log N)
```

### 4.2 큰 수 곱셈

각 자릿수를 다항식 계수로 표현 → FFT 곱셈 → 올림(carry) 처리.

📢 **섹션 요약 비유**: FFT 곱셈은 두 악보를 각각 주파수로 변환(FFT) → 주파수끼리 곱 → 다시 악보로(IFFT) — 음표 하나씩 비교하는 것보다 훨씬 빠르다.

## Ⅴ. 응용 분야

### 5.1 신호 처리

```
오디오 압축 (MP3):
시간 도메인 샘플 → FFT → 주파수 성분 → 중요 주파수만 보존 → IFFT
(사람 귀에 안 들리는 고주파 제거 → 파일 크기 감소)
```

### 5.2 이미지 처리

2D FFT를 이용한 주파수 필터링:
- 저주파 통과 필터(Low-Pass) → 블러
- 고주파 통과 필터(High-Pass) → 엣지 검출

### 5.3 과학 계산

- 미분방정식 수치 해법
- 상관(Correlation) 계산: 두 신호 유사도 → O(N log N)

📢 **섹션 요약 비유**: FFT는 "음악을 주파수로 쪼개는 프리즘" — 음악(신호)을 색(주파수)으로 나눠서 원하는 색만 남기거나 처리한다.

## 👶 어린이를 위한 3줄 비유 설명

1. FFT는 음악을 음계로 쪼개는 마법 — 1000개 음표를 모두 비교하는 대신 절반씩 나눠 빠르게 분석해.
2. 버터플라이는 두 신호를 순식간에 합치는 요령 — 4번 할 일을 2번에 끝내는 비법.
3. MP3 압축이 FFT 덕분에 가능해 — 사람 귀에 안 들리는 소리(고주파)를 주파수로 찾아서 버리는 거야.

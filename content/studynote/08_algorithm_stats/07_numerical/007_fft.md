## 핵심 인사이트 (3줄 요약)

> 1. **본질**: FFT(Fast Fourier Transform, 고속 푸리에 변환)는 DFT(Discrete Fourier Transform, 이산 푸리에 변환)를 분할 정복으로 O(n²)에서 O(n log n)으로 가속하는 알고리즘으로, 신호 처리·다항식 곱셈의 핵심이다.
> 2. **가치**: 두 다항식의 곱을 O(n log n)에 계산하고, 음성·이미지·통신 신호의 주파수 분석 및 압축에 직접 활용되며, NTT(Number Theoretic Transform)로 정수 곱셈에도 응용된다.
> 3. **판단 포인트**: FFT의 핵심은 DFT를 짝수·홀수 인덱스로 분리하는 재귀 구조(Cooley-Tukey)이며, 단위원의 거듭제곱(N-th Root of Unity, ωₙ)의 대칭성이 분할 정복을 가능하게 한다.

---

## Ⅰ. 개요 및 필요성

**DFT**는 길이 n인 수열 a₀, a₁, ..., aₙ₋₁을 주파수 영역(Frequency Domain)으로 변환한다:

Aₖ = Σⱼ₌₀ⁿ⁻¹ aⱼ · ωₙʲᵏ,  ωₙ = e^(2πi/n) (n번째 단위원)

나이브 DFT: O(n²). FFT(Cooley-Tukey, 1965): O(n log n).

- **다항식 곱셈**: P(x)·Q(x) 두 차수 n 다항식 → 나이브 O(n²) → FFT로 O(n log n).
- **신호 처리**: 오디오·이미지·통신에서 주파수 성분 분해.
- **PDE 풀기**: 편미분 방정식의 스펙트럼 방법.

📢 **섹션 요약 비유**: FFT는 "오케스트라 소리에서 각 악기의 음량을 순식간에 분리하는 마법" — 복잡한 신호를 단순한 주파수 성분들의 합으로 쪼개는 알고리즘이다.

---

## Ⅱ. 아키텍처 및 핵심 원리

### Cooley-Tukey 분할 정복

```
DFT[n] (짝수 n):
  ┌─────────────────────────────────────────────────┐
  │  A(x) = a₀ + a₁x + a₂x² + ... + aₙ₋₁xⁿ⁻¹      │
  │                                                  │
  │  분리:                                           │
  │  A_even(x) = a₀ + a₂x + a₄x² + ...  (짝수 항) │
  │  A_odd(x)  = a₁ + a₃x + a₅x² + ...  (홀수 항) │
  │                                                  │
  │  A(ωₙᵏ) = A_even(ωₙ/₂ᵏ) + ωₙᵏ · A_odd(ωₙ/₂ᵏ) │
  │  A(ωₙᵏ⁺ⁿ/²) = A_even(ωₙ/₂ᵏ) - ωₙᵏ · A_odd(ωₙ/₂ᵏ) │
  │                                                  │
  │  → 각 n/2 크기 DFT로 분리 → T(n) = 2T(n/2) + O(n) │
  │  → T(n) = O(n log n)                            │
  └─────────────────────────────────────────────────┘
```

### 버터플라이 다이어그램 (n=8 예시)

```
입력 (비트역순)      버터플라이 단계들            출력
a₀ ──────┐
          ×──── A₀
a₄ ──────┘  ...
a₂ ──────┐
          ×──── A₁
a₆ ──────┘
a₁ ──────┐
          ×──── A₂
a₅ ──────┘
a₃ ──────┐
          ×──── A₃
a₇ ──────┘
         단계1    단계2    단계3
         (n/8)    (n/4)    (n/2)
         크기2   크기4    크기8
```

버터플라이(Butterfly) 연산: (a, b) → (a + ωb, a - ωb), 각 단계 O(n), log n 단계 → O(n log n).

| 방법 | 시간 | 용도 |
|:---|:---:|:---|
| 나이브 DFT | O(n²) | 소규모 |
| Cooley-Tukey FFT | O(n log n) | 표준 |
| NTT (정수 FFT) | O(n log n) | 정수 다항식 곱, 정확 |
| IFFT (역 FFT) | O(n log n) | 주파수→시간 역변환 |

📢 **섹션 요약 비유**: 버터플라이 연산은 나비가 날개를 펼치는 것처럼 두 원소를 동시에 처리 — 로그 단계를 거치며 n개 원소 모두가 목적지에 도달한다.

## Ⅲ. 비교 및 연결

### 다항식 곱셈 via FFT

```
┌──────────────────────────────────────────────────────────┐
│  P(x) · Q(x) 곱셈  (O(n log n))                         │
│                                                          │
│  1. P, Q를 n = 2·(deg+1) 크기로 패딩                    │
│     (n은 2의 거듭제곱)                                   │
│         │                                               │
│         ▼                                               │
│  2. FFT(P) → P̂,  FFT(Q) → Q̂         O(n log n)        │
│         │                                               │
│         ▼                                               │
│  3. 점별 곱: R̂[k] = P̂[k] · Q̂[k]     O(n)              │
│         │                                               │
│         ▼                                               │
│  4. IFFT(R̂) → R = P·Q                 O(n log n)       │
│                                                          │
│  총: O(n log n) (나이브 O(n²) 대비)                     │
└──────────────────────────────────────────────────────────┘
```

### NTT (Number Theoretic Transform, 수론 변환)

NTT는 복소수 ωₙ = e^(2πi/n) 대신 모듈러 산술(mod p)의 원시근(Primitive Root)을 사용:
- 정수 다항식 곱에서 부동소수점 오차 완전 제거.
- 경쟁 프로그래밍·암호 구현(NTRU, CRYSTALS-Kyber)에 필수.

### 응용 분야

| 분야 | 활용 |
|:---|:---|
| 오디오 처리 | 주파수 필터, 압축(MP3) |
| 이미지 처리 | JPEG 압축 (DCT, FFT 기반) |
| 통신 | OFDM (LTE, 5G 다중 반송파) |
| 과학 계산 | PDE 스펙트럼 해법 |
| 대용량 정수 곱 | Schönhage-Strassen, NTT |
| 암호 (PQC) | CRYSTALS-Kyber KEM 핵심 |

📢 **섹션 요약 비유**: FFT를 통한 다항식 곱은 "두 신호를 주파수 도메인에서 점별로 곱한 후 시간 도메인으로 돌아오는 것" — 복잡한 시간 도메인 합성곱이 주파수 도메인에서는 단순한 곱셈이 된다.

---

## Ⅳ. 실무 적용 및 기술사 판단

기술사 핵심 포인트:
1. **FFT 원리**: DFT O(n²) → Cooley-Tukey FFT O(n log n), 버터플라이 구조.
2. **다항식 곱셈**: FFT 기반 O(n log n) → 나이브 O(n²) 대비 대규모에서 결정적 우위.
3. **NTT**: 정수 연산에서 FFT의 부동소수점 오차 제거 버전 → PQC 암호에서 핵심.
4. **OFDM**: 5G 통신의 다중 반송파 변조 방식이 FFT/IFFT로 구현.

📢 **섹션 요약 비유**: FFT는 현대 디지털 세계의 보이지 않는 엔진 — 스마트폰 통신(5G OFDM), 음악 파일(MP3), 사진 압축(JPEG)이 모두 FFT 없이는 작동하지 않는다.

---

## Ⅴ. 기대효과 및 결론

FFT를 이해하면:
- **신호 처리 기초**: 디지털 통신·오디오·이미지 처리의 수학적 기반 이해.
- **알고리즘 속도 향상**: 다항식 곱·합성곱의 O(n²) → O(n log n) 전환.
- **PQC 이해**: CRYSTALS-Kyber의 NTT 핵심 연산 이해 → 포스트 양자 암호 구현 기초.

📢 **섹션 요약 비유**: FFT는 알고리즘의 "만능 번역기" — 시간 도메인과 주파수 도메인 사이를 빛의 속도로 오가며, 복잡한 합성곱을 단순한 곱셈으로 바꿔준다.

### 📌 관련 개념 맵

| 개념 | 관계 |
|:---|:---|
| DFT | O(n²) 이산 푸리에 변환 |
| FFT (Cooley-Tukey) | DFT를 O(n log n)으로 가속 |
| 버터플라이 연산 | FFT의 기본 연산 단위 |
| IFFT | 역 FFT, 주파수→시간 변환 |
| NTT | 정수 모듈러 FFT, PQC 핵심 |
| OFDM | 5G 통신, IFFT/FFT로 구현 |
| CRYSTALS-Kyber | NTT 기반 PQC 표준 |

### 📈 관련 키워드 및 발전 흐름도

```text
[이산 푸리에 변환 (DFT, Discrete Fourier Transform)]
    │
    ▼
[고속 푸리에 변환 (FFT, Fast Fourier Transform)]
    │
    ▼
[쿨리-튜키 알고리즘 (Cooley-Tukey Algorithm)]
    │
    ▼
[신호 처리 (Signal Processing)]
    │
    ▼
[주파수 도메인 분석 (Frequency Domain Analysis)]
    │
    ▼
[GPU 병렬 FFT (Parallel FFT)]
```

고전 DFT의 O(N²) 복잡도를 O(N log N)으로 단축한 FFT가 신호·이미지 처리와 딥러닝 연산에 기반이 되는 흐름이다.

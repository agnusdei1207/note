## 핵심 인사이트 (3줄 요약)

> 1. **본질**: 가우스 소거법(Gaussian Elimination)은 n×n 연립 일차방정식을 행 연산(Row Operation)으로 상삼각 행렬(Upper Triangular Matrix)로 변환한 후 역대입(Back Substitution)으로 해를 구하는 O(n³) 알고리즘이다.
> 2. **가치**: 부분 피벗(Partial Pivoting)으로 수치 안정성을 확보하고, LU 분해(LU Decomposition)로 확장하면 같은 계수 행렬로 여러 우변 벡터를 효율적으로 처리한다.
> 3. **판단 포인트**: 부동소수점 오차 누적(Round-Off Error)과 조건수(Condition Number)가 수치 해석의 핵심 — 조건수가 크면 작은 입력 오차가 해에서 증폭된다.

---

## Ⅰ. 개요 및 필요성

**가우스 소거법**은 연립 일차방정식 Ax = b의 해를 구하는 가장 기본적인 알고리즘으로, 수치 선형 대수(Numerical Linear Algebra)의 핵심이다.

- **행 연산(Elementary Row Operations)**:
  1. 두 행 교환(Row Swap)
  2. 한 행에 비영(Non-zero) 스칼라 곱
  3. 한 행에 다른 행의 스칼라 배 더하기
- **전진 소거(Forward Elimination)**: 하삼각 부분을 0으로 만들어 상삼각 행렬 생성.
- **역대입(Back Substitution)**: 상삼각 행렬에서 마지막 방정식부터 역순으로 해 계산.

📢 **섹션 요약 비유**: 가우스 소거법은 방정식들을 계단처럼 정리하는 것 — "아래 수업이 위 수업보다 먼저 끝나도록" 단계적으로 정리하면, 마지막 수업부터 거슬러 올라가며 답을 찾는다.

## Ⅱ. 아키텍처 및 핵심 원리

### 가우스 소거 과정 예시

```
초기 증대 행렬 [A|b]:
┌                    ┐
│  2   1  -1  |  8  │
│ -3  -1   2  | -11 │
│ -2   1   2  | -3  │
└                    ┘

전진 소거 후 상삼각:
┌                    ┐
│  2   1  -1  |  8  │
│  0   1   1  |  1  │  ← R2 = R2 + (3/2)R1
│  0   0  -1  | -1  │  ← R3 연산 후
└                    ┘

역대입:
x₃ = 1
x₂ = 1 - 1·(1) = 0  → x₂ = 0  (오류 예시, 실제 계산은 달라짐)
x₁ = (8 - 1·0 + 1·1)/2 = 4.5  (예시)
```

### 알고리즘 및 복잡도

```
┌──────────────────────────────────────────────────────────┐
│  Gaussian Elimination with Partial Pivoting              │
│                                                          │
│  for k = 1 to n-1:              ← 열(Pivot 열) 반복     │
│      피벗 선택: |A[p][k]| 최대인 p 찾기 (k ≤ p ≤ n)    │
│      A[k] ↔ A[p] (행 교환)                              │
│                                                          │
│      for i = k+1 to n:          ← 아래 행 소거          │
│          m = A[i][k] / A[k][k]  (승수, Multiplier)      │
│          for j = k to n+1:                              │
│              A[i][j] -= m · A[k][j]                     │
│                                                          │
│  역대입: O(n²)                                           │
│  전체: O(n³)                                             │
└──────────────────────────────────────────────────────────┘
```

| 방법 | 시간 | 피벗 | 안정성 |
|:---|:---:|:---:|:---:|
| 나이브 소거 | O(n³) | 없음 | 불안정 |
| 부분 피벗 | O(n³) | 열 최대값 | 안정 |
| 완전 피벗 | O(n³) | 전체 최대값 | 최강 안정 |
| LU 분해 | O(n³) | 있음 | 안정 |

📢 **섹션 요약 비유**: 부분 피벗은 "가장 크고 명확한 선생님(피벗 행)을 먼저 앞에 내세우는 것" — 작은 수로 나누면 오차가 커지므로, 항상 가장 큰 값을 피벗으로 선택한다.

## Ⅲ. 비교 및 연결

### LU 분해 (LU Decomposition)

가우스 소거법의 행 연산을 행렬로 표현: **A = L·U**

- **L**: 하삼각 행렬(Lower Triangular), 소거 승수들 저장.
- **U**: 상삼각 행렬(Upper Triangular), 소거 결과.

```
장점:
  동일한 A로 여러 b 벡터 처리:
    A·x₁ = b₁, A·x₂ = b₂, ...
    L·y₁ = b₁ (전진 대입, O(n²))
    U·x₁ = y₁ (역대입, O(n²))
    → 최초 LU 분해 O(n³) 후 각 b는 O(n²)
```

### 수치 안정성과 조건수

**조건수(Condition Number)**: κ(A) = ||A|| · ||A⁻¹||

- κ가 크면: 작은 입력 오차 → 해에서 큰 오차 (ill-conditioned).
- κ ≈ 1: 수치적으로 안정 (well-conditioned).
- 부동소수점 정밀도 d자리 → 해의 정확도 약 d - log₁₀(κ)자리.

| 행렬 특성 | 조건수 | 예시 |
|:---|:---:|:---|
| 단위 행렬 | 1 | 가장 안정 |
| 잘 조건화된 행렬 | 10~100 | 일반 선형계 |
| 불량 조건화 행렬 | 10⁸ 이상 | 힐베르트 행렬 |

📢 **섹션 요약 비유**: 조건수는 "확대경의 배율" — 배율이 높을수록(조건수 큼) 원본의 작은 흠이 해에서 크게 보인다.

## Ⅳ. 실무 적용 및 기술사 판단

| 분야 | 활용 |
|:---|:---|
| 컴퓨터 그래픽스 | 좌표 변환 행렬 역행렬 |
| 머신러닝 | 선형 회귀의 정규 방정식 풀기 |
| 과학/공학 시뮬레이션 | 유한 요소법(FEM) 선형계 풀기 |
| 회로 해석 | 노드 전압법(KVL/KCL) 연립방정식 |
| 암호학 | 행렬 기반 암호 역행렬 계산 |

기술사 핵심:
1. 가우스 소거법 = O(n³), LU 분해로 여러 b 처리 O(n²).
2. 부분 피벗 → 수치 안정성 확보 필수.
3. 희소 행렬(Sparse Matrix): 특수 알고리즘(Sparse LU, 반복법 CG) 사용.

📢 **섹션 요약 비유**: LU 분해는 "자물쇠(A)를 한 번만 분해해서 캐비닛에 보관, 이후 여러 열쇠(b)에 빠르게 적용" — 한 번의 O(n³) 비용으로 이후 O(n²)씩 절약한다.

## Ⅴ. 기대효과 및 결론

가우스 소거법을 이해하면:
- **선형 대수 기초 완성**: 연립방정식, 역행렬, 행렬식 계산을 하나의 알고리즘으로 통합.
- **수치 안정성 인식**: 부동소수점 오차의 증폭 메커니즘과 피벗 전략 이해.
- **LU 분해 활용**: 시뮬레이션·ML 시스템에서 반복 선형계 풀기 최적화.

📢 **섹션 요약 비유**: 가우스 소거법은 수학의 "청소부 알고리즘" — 지저분한 연립방정식을 깔끔한 계단 형태로 치워서, 위에서 아래로 하나씩 해를 꺼낸다.

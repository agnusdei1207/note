## 핵심 인사이트 (3줄 요약)

> 1. **본질**: S&OP(Sales and Operations Planning)는 영업·마케팅의 수요 계획과 생산·물류의 공급 계획을 월간 단위로 통합 조정하여 수요-공급 불균형을 사전에 해소하는 경영 프로세스다.
> 2. **가치**: 수요 과잉 시 생산 선제 대응, 수요 부족 시 재고 처분 계획 수립이 가능하여 재고 최적화·고객 서비스 레벨 향상·운전자본(Working Capital) 절감을 동시에 달성한다.
> 3. **판단 포인트**: S&OP의 성공은 데이터 품질(수요 예측 정확도)보다 회의 참여자의 권한과 의사결정 문화에 더 의존하며, CFO·CSO·COO가 직접 참여하는 Executive S&OP가 핵심이다.

## Ⅰ. 개요 및 필요성

공급망 관리에서 수요와 공급의 불균형은 두 가지 반대 방향의 비용을 야기한다. 수요 > 공급이면 결품(Out-of-Stock)으로 매출 기회 손실, 고객 이탈. 수요 < 공급이면 과잉재고(Overstock)로 보관비·자본비·진부화 손실. S&OP는 이 불균형을 월 1회 공식 프로세스를 통해 사전에 인식하고 조정한다.

1970년대 Oliver Wight가 MPS(Master Production Schedule)와 판매 계획의 연계 필요성을 제기하면서 시작되어, 현재는 IBP(Integrated Business Planning)로 진화하고 있다. SAP IBP, Kinaxis RapidResponse, OMP 등이 대표 S&OP 지원 솔루션이다.

📢 **섹션 요약 비유**: S&OP는 월간 가족 회의처럼, 이번 달 지출(수요)과 수입(공급 능력)을 맞춰보고 어디서 조정할지 결정하는 정기 모임이다.

## Ⅱ. 아키텍처 및 핵심 원리

### S&OP 5단계 월간 프로세스

```
┌─────────────────────────────────────────────────────┐
│                S&OP 월간 프로세스                     │
├─────────────────────────────────────────────────────┤
│ 1단계: 데이터 수집 및 분석 (주 1~5일)               │
│   실적 리뷰, 재고 분석, 공급 능력 현황 업데이트        │
├─────────────────────────────────────────────────────┤
│ 2단계: 수요 계획 검토 (주 1~2주)                    │
│   영업·마케팅: 수요 예측 수립, 캠페인·프로모션 반영    │
│   산출물: 컨센서스 수요 계획 (Consensus Demand Plan)  │
├─────────────────────────────────────────────────────┤
│ 3단계: 공급 계획 검토 (주 2~3주)                    │
│   제조·물류: 생산 가능량, 병목 분석, 대안 수립        │
│   산출물: 공급 응답 계획 (Supply Response Plan)       │
├─────────────────────────────────────────────────────┤
│ 4단계: Pre-S&OP (주 3~4주)                         │
│   수요-공급 갭 분석, 조정 시나리오 수립               │
│   재무 영향 분석, 경영진 이슈 도출                   │
├─────────────────────────────────────────────────────┤
│ 5단계: Executive S&OP (월 마지막 주)               │
│   C-레벨 참여, 최종 의사결정, 전략 방향 설정          │
│   산출물: 승인된 수요-공급 계획, 실행 지시            │
└─────────────────────────────────────────────────────┘
```

### S&OP 주요 지표 (KPI)

| 지표 | 설명 | 목표 수준 |
|:---|:---|:---|
| 수요 예측 정확도 (MAPE) | |(실제-예측)|/실제 × 100 | < 15% (완성품) |
| 재고 회전율 | 연간 매출 / 평균 재고 | 업종별 목표 수준 |
| 서비스 레벨 (Fill Rate) | 주문 충족률 | > 95% |
| 공급 유연성 | 급격한 수요 변동 대응 기간 | ≤ 2주 |
| 운전자본 (Working Capital) | 재고 + 매출채권 - 매입채무 | 전년 대비 감소 |

📢 **섹션 요약 비유**: Executive S&OP는 회사 이사회처럼, 데이터(수요·공급 갭)를 가지고 CEO·CFO·COO가 함께 "어떻게 할지"를 결정하는 공식 의사결정 무대다.

## Ⅲ. 비교 및 연결

### S&OP vs IBP (Integrated Business Planning)

| 구분 | S&OP | IBP |
|:---|:---|:---|
| 범위 | 영업·생산·물류 | + 재무·전략·HR 통합 |
| 계획 호라이즌 | 3~18개월 | 1~5년 |
| 재무 통합 | 제한적 | 완전 통합 (P&L까지) |
| 의사결정 | 운영 수준 | 전략 수준 |
| 데이터 | 주로 단일 시나리오 | 복수 시나리오 (What-if) |
| 도구 | 기본 S&OP 솔루션 | SAP IBP, Kinaxis |

### S&OP와 채찍 효과의 관계

S&OP를 통해 공급망 전체의 수요 예측 합의가 이루어지면 각 단계가 독립적으로 수요를 예측하여 발생하는 채찍 효과(Bullwhip Effect)를 사전에 억제할 수 있다.

📢 **섹션 요약 비유**: S&OP와 IBP의 차이는 한 달 가계부(S&OP)와 5년 재무 계획(IBP)처럼, 운영 조정과 전략적 의사결정의 시간 지평이 다르다.

## Ⅳ. 실무 적용 및 기술사 판단

### S&OP 성숙도 단계

```
레벨 1 반응적 (Reactive):
  각 부서 독립 계획, 수요-공급 조정 사후 처리

레벨 2 통합 계획 (Integrated Planning):
  기본 S&OP 회의 체계 수립, 수요-공급 가시성 확보

레벨 3 협업적 (Collaborative):
  외부 파트너(공급업체/채널) 포함, 공동 예측

레벨 4 통합 비즈니스 계획 (IBP):
  재무·전략 통합, 시나리오 계획, 실시간 분석
```

### 실패 원인 및 해결 방안

| 실패 원인 | 해결 방안 |
|:---|:---|
| 경영진 불참 | Executive S&OP를 월 달력에 고정 예약 |
| 데이터 신뢰 부족 | 하나의 숫자(One Number) 원칙 → 영업·생산 공용 |
| 회의 과부하 | S&OP 준비 업무 자동화 (S&OP 솔루션) |
| 부서 이기주의 | 통합 KPI (재고+서비스레벨+비용) 설계 |

📢 **섹션 요약 비유**: S&OP의 "One Number" 원칙은 팀 전체가 같은 경기 스코어보드를 보는 것처럼, 영업과 생산이 같은 수요 예측 숫자를 공유해야 한다.

## Ⅴ. 기대효과 및 결론

성숙한 S&OP를 운영하는 기업은 재고 15~30% 감소, 고객 서비스 레벨 5~10% 향상, 생산 계획 안정화로 가동률 5~10% 향상, 운전자본 개선 효과를 거둔다. 코로나19와 같은 급격한 공급망 충격 시에도 S&OP가 성숙한 기업이 빠른 공급 재조정으로 더 빠른 회복(Supply Chain Resilience)을 보였다.

**한계**: S&OP는 기술보다 조직 문화 변화가 더 어렵다. 영업팀의 과대 예측 습관, 생산팀의 과소 약속 습관을 인센티브 체계로 교정해야 하며, 이는 단기간에 달성되지 않는다.

📢 **섹션 요약 비유**: S&OP는 악보처럼, 모든 연주자(부서)가 같은 악보를 보고 연주해야 아름다운 음악(효율적 공급망)이 나온다.

### 👶 어린이를 위한 3줄 비유 설명

1. S&OP는 가족 여행 계획처럼, 어디 갈지(수요)와 차가 몇 명 태울 수 있는지(공급)를 미리 맞춰보는 회의예요.
2. 이 회의가 없으면 영업팀은 "100개 팔겠다" 하고, 공장은 "50개밖에 못 만들어"라며 서로 맞지 않는 상황이 생겨요.
3. 매달 한 번 같이 앉아서 숫자를 맞추면, 창고에 물건이 넘치거나 부족한 일이 훨씬 줄어들어요.

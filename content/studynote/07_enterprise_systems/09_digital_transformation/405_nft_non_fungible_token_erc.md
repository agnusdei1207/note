## 핵심 인사이트 (3줄 요약)

> 1. **본질**: NFT(Non-Fungible Token)는 블록체인 위에서 각 토큰이 고유한 식별자를 가져 대체 불가능(Non-Fungible)한 디지털 자산의 소유권을 증명하는 ERC-721/ERC-1155 표준 스마트 컨트랙트다.
> 2. **가치**: 디지털 파일의 복사 가능성 문제를 해결하여 예술품·게임 아이템·부동산 권리 등에 희소성(Scarcity)과 소유권을 부여하고, 2차 판매 로열티 자동 지급을 가능하게 한다.
> 3. **판단 포인트**: NFT는 소유권을 증명하지 실제 파일을 블록체인에 저장하지 않으므로, 파일 링크(IPFS 해시)가 유실되면 NFT만 남고 콘텐츠가 사라지는 구조적 한계를 이해해야 한다.

## Ⅰ. 개요 및 필요성

디지털 아트, 게임 아이템, 음악은 무한 복사 가능하여 희소성이 없다. NFT는 특정 디지털 파일에 대한 소유권 증명을 블록체인에 기록하여 "이 토큰 소유자가 원본 소유자"임을 증명한다. 2017년 CryptoKitties 게임이 Ethereum ERC-721 표준을 최초 실용화했고, 2021년 Beeple의 디지털 아트가 $69M에 낙찰되며 주류 미디어에 알려졌다.

📢 **섹션 요약 비유**: NFT는 디지털 그림의 진품 증명서 — 그림(파일)은 누구나 복사할 수 있지만, 증명서(NFT)는 단 하나뿐이다.

## Ⅱ. 아키텍처 및 핵심 원리

```
NFT 소유권 구조:

블록체인 (Ethereum)
  ┌─────────────────────────────────────┐
  │ NFT Token ID: #1234                  │
  │ Owner: 0xABC...                      │
  │ Token URI: ipfs://QmXyz...           │  ──> IPFS (파일 저장)
  │ Contract: ERC-721 스마트 컨트랙트    │      ipfs://QmXyz... → art.jpg
  └─────────────────────────────────────┘

ERC 표준 비교:
  ERC-20  : 대체 가능 토큰 (1 ETH = 1 ETH)
  ERC-721 : 대체 불가능 토큰 (각 토큰 고유 ID)
  ERC-1155: 다중 토큰 (대체 가능 + 불가능 혼합)
```

| 구분 | ERC-20 | ERC-721 | ERC-1155 |
|:---|:---|:---|:---|
| 대체 가능성 | 가능 | 불가능 | 혼합 |
| 고유 ID | 없음 | 있음 | 있음 |
| 활용 | 암호화폐 | NFT 아트·아이템 | 게임 다중 자산 |
| 가스 효율 | 높음 | 중간 | 높음(배치 전송) |

📢 **섹션 요약 비유**: ERC-20은 지폐(모두 동일), ERC-721은 그림(각각 고유), ERC-1155는 카드 게임(일반 카드+희귀 카드 혼합)이다.

## Ⅲ. 비교 및 연결

IPFS(InterPlanetary File System): NFT가 가리키는 파일은 중앙 서버 대신 IPFS에 분산 저장하는 것이 권장된다. 중앙 서버가 다운되면 Token URI가 가리키는 파일이 사라질 수 있다(링크 로트, Link Rot). 로열티(ERC-2981): NFT 2차 판매 시 원작자에게 자동으로 로열티 지급.

📢 **섹션 요약 비유**: IPFS는 분산 클라우드 — 파일을 한 서버가 아닌 전 세계 수천 노드에 나눠 저장하여 서버 죽음(단일 장애점)을 방지한다.

## Ⅳ. 실무 적용 및 기술사 판단

**의사결정 포인트**:
- 디지털 자산 소유권 증명: ERC-721 NFT
- 게임 내 다중 자산 효율: ERC-1155 선택
- 파일 영속성: IPFS 또는 Arweave(영구 분산 저장) 사용
- 로열티 설계: ERC-2981 표준 구현
- 과세/규제: NFT 거래 시 자산 양도세 과세 국가 증가

📢 **섹션 요약 비유**: NFT 발행은 부동산 등기 — 내가 소유한다는 사실을 공개 장부에 등록하는 것과 같다.

## Ⅴ. 기대효과 및 결론

NFT는 디지털 경제에서 희소성·소유권·이전성을 구현하여 크리에이터 이코노미와 메타버스 자산 경제의 핵심 인프라가 된다. 단, 파일 영속성 문제, 투기적 가격 형성, 환경 부담(PoW 기반 민팅), 저작권 분쟁이 현실적 한계다. IPFS 저장, PoS 블록체인 활용, 명확한 법적 소유권 연계가 엔터프라이즈 NFT 도입의 필수 선결 조건이다.

📢 **섹션 요약 비유**: NFT는 디지털 세계의 소유권 혁명 — 처음으로 인터넷 파일에 "진품"과 "소유자"가 생겼다.

### 📌 관련 개념 맵

| 개념 | 관계 | 설명 |
|:---|:---|:---|
| ERC-721 | 핵심 표준 | 대체 불가능 토큰 Ethereum 표준 |
| ERC-1155 | 확장 표준 | 다중 토큰(가능+불가능 혼합) |
| IPFS | 파일 저장 | 분산 파일 시스템 |
| Minting | 발행 과정 | NFT 생성·블록체인 등록 |
| Royalty (ERC-2981) | 수익 분배 | 2차 판매 시 원작자 자동 로열티 |

### 👶 어린이를 위한 3줄 비유 설명

1. NFT는 디지털 그림의 소유권 스티커 — 그림은 누구나 볼 수 있지만, 스티커(NFT)는 단 한 사람만 가질 수 있어.
2. ERC-721은 야구 선수 카드 — 같은 선수 카드라도 번호(ID)가 달라서 각각 고유한 가치를 가져.
3. IPFS는 분산 USB — 한 USB(서버)가 망가져도 다른 USB에 파일이 있어서 영원히 보존돼!

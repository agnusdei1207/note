## 핵심 인사이트 (3줄 요약)

> 1. **본질**: 양자 컴퓨터(Quantum Computer)는 큐비트(Qubit)의 중첩(Superposition)·얽힘(Entanglement)·간섭(Interference) 원리로 고전 컴퓨터가 지수적 시간이 필요한 문제를 다항 시간으로 해결하며, Shor 알고리즘(소인수분해)·Grover 알고리즘(데이터베이스 검색)이 대표 응용이다.
> 2. **가치**: Shor 알고리즘은 RSA·ECC 암호를 효율적으로 해독할 수 있어 기존 공개 키 암호 체계를 위협하고, Grover 알고리즘은 AES-128 브루트포스를 AES-64 수준으로 약화시켜 대칭 키 암호의 키 길이 배증을 요구한다.
> 3. **판단 포인트**: 현재 양자 컴퓨터(NISQ: Noisy Intermediate-Scale Quantum)는 오류율이 높아 실용적 Shor 실행 불가능하나, 양자 내성 암호(PQC: Post-Quantum Cryptography)로의 전환이 현재 최우선 보안 과제다.

## Ⅰ. 개요 및 필요성

고전 컴퓨터의 비트는 0 또는 1이지만, 큐비트는 측정 전까지 0과 1의 확률적 중첩 상태에 있다. N큐비트 시스템은 2^N 상태를 동시에 표현하여 특정 문제에서 지수적 병렬성을 제공한다. Shor 알고리즘(1994)은 RSA 기반 인터넷 보안 인프라를 수년 내 위협할 수 있어, NIST는 2024년 양자 내성 암호 표준(ML-KEM, ML-DSA)을 발표했다.

📢 **섹션 요약 비유**: 큐비트는 동전 — 고전 비트는 앞/뒤만 있지만, 큐비트는 공중에 있는 동전(중첩) — 떨어지기 전까지는 앞과 뒤 동시에.

## Ⅱ. 아키텍처 및 핵심 원리

```
큐비트 특성:
  중첩 (Superposition): |ψ⟩ = α|0⟩ + β|1⟩
    → N 큐비트 = 2^N 상태 동시 표현
  
  얽힘 (Entanglement): 큐비트 쌍의 상관 관계
    → 한 큐비트 측정이 다른 큐비트 상태 즉시 결정
  
  간섭 (Interference): 오답 경로 소멸·정답 경로 증폭

알고리즘 비교:
  Shor 알고리즘 (소인수분해):
    고전: O(exp(n^(1/3))) → 2048비트 RSA 수천 년
    양자: O(n^3) → 수 시간 (이론적, 논리 큐비트 충분 시)
    위협: RSA, ECC, DH → PQC로 전환 필요

  Grover 알고리즘 (비정형 탐색):
    고전: O(N) → 완전 탐색
    양자: O(√N) → 제곱근 속도 향상
    영향: AES-128 → 64비트 수준 보안 (키 길이 배증 권고)
```

| 특성 | 고전 컴퓨터 | 양자 컴퓨터 |
|:---|:---|:---|
| 기본 단위 | 비트 (0/1) | 큐비트 (중첩) |
| 병렬성 | 제한적 | 지수적(특정 문제) |
| 오류 | 낮음 | 높음(NISQ) |
| 강점 분야 | 범용 | 최적화·인수분해·ML |
| 현재 상태 | 성숙 | NISQ(노이즈 임계 규모) |

📢 **섹션 요약 비유**: Shor 알고리즘은 자물쇠(RSA) 해독기 — 고전 컴퓨터가 수천 년 걸리는 자물쇠를 양자 컴퓨터는 수 시간에 여는 잠재력이 있다.

## Ⅲ. 비교 및 연결

PQC(Post-Quantum Cryptography, 양자 내성 암호): 양자 컴퓨터로도 해독이 어려운 수학적 문제(격자, 해시, 코드) 기반 암호. NIST PQC 표준(2024): CRYSTALS-Kyber(ML-KEM), CRYSTALS-Dilithium(ML-DSA), FALCON, SPHINCS+. 하이브리드 암호화: 전환 기간 동안 기존 RSA/ECC + PQC 병행 사용.

📢 **섹션 요약 비유**: PQC는 양자 컴퓨터에도 안전한 새 자물쇠 — RSA 자물쇠가 깨질 것을 대비해 미리 더 튼튼한 자물쇠(격자 암호)로 교체하는 것이다.

## Ⅳ. 실무 적용 및 기술사 판단

**의사결정 포인트**:
- Harvest Now, Decrypt Later 위협: 현재 암호화 데이터를 저장해두고 양자 컴퓨터 완성 후 해독 공격
- PQC 전환 긴급성: 의료·금융 등 장기 기밀 데이터 보호 위해 즉시 전환 계획 필요
- TLS 1.3 + PQC 혼합: Google Chrome, Cloudflare가 X25519Kyber768 하이브리드 실험
- 양자 유용성 분야: 물류 최적화, 신약 분자 시뮬레이션, 금융 포트폴리오 최적화

📢 **섹션 요약 비유**: Harvest Now, Decrypt Later는 편지 도둑 — 지금 당장 못 읽는 암호 편지를 훔쳐두고, 나중에 양자 컴퓨터로 해독하는 전략이다.

## Ⅴ. 기대효과 및 결론

양자 컴퓨팅은 약물 발견·재료 과학·금융 최적화에서 고전 컴퓨터 불가능 문제를 해결할 잠재력을 지니며, 암호학적으로는 기존 공개 키 암호 체계의 근본적 패러다임 전환을 요구한다. 실용적 폴트 톨러런트 양자 컴퓨터는 10~15년 후로 전망되지만, PQC 전환은 지금 당장 시작해야 하는 현재의 과제다.

📢 **섹션 요약 비유**: 양자 컴퓨팅은 디지털 세계의 원자폭탄 — 엄청난 잠재력과 위험을 동시에 가지고 있으며, 준비 없이 맞이하면 위협이, 대비하면 기회가 된다.

### 📌 관련 개념 맵

| 개념 | 관계 | 설명 |
|:---|:---|:---|
| Qubit | 기본 단위 | 중첩·얽힘 특성의 양자 비트 |
| Shor Algorithm | 핵심 응용 | 소인수분해 → RSA 위협 |
| Grover Algorithm | 핵심 응용 | 데이터베이스 검색 √N 가속 |
| PQC (Post-Quantum Cryptography) | 대응책 | 양자 내성 암호 표준 |
| NISQ | 현재 단계 | 노이즈 중간 규모 양자 컴퓨터 |

### 👶 어린이를 위한 3줄 비유 설명

1. 큐비트는 마법 동전 — 보기 전까지는 앞면이기도 하고 뒷면이기도 해서, 여러 경우를 동시에 계산할 수 있어.
2. Shor 알고리즘은 빠른 자물쇠 해독기 — 현재 인터넷 암호(RSA)를 양자 컴퓨터가 수 시간에 풀 수 있어서 새 자물쇠(PQC)가 필요해.
3. PQC는 양자 시대의 새 암호 — 양자 컴퓨터에도 안전한 수학 문제(격자)를 기반으로 만든 차세대 보안이야!

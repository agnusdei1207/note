## 핵심 인사이트 (3줄 요약)

> 1. **본질**: 스마트 그리드(Smart Grid)는 전력망에 ICT를 융합하여 발전-송전-배전-소비를 실시간 데이터로 지능적으로 관리하며, 마이크로그리드(Microgrid)와 ESS(Energy Storage System)는 분산 에너지 자원(DER)을 로컬에서 자율 운영하는 핵심 구성 요소다.
> 2. **가치**: 재생에너지(태양광·풍력)의 간헐성을 ESS로 보완하고, 수요 반응(DR: Demand Response)으로 피크 수요를 평준화하여 전력 효율성과 탄소 중립을 동시에 달성한다.
> 3. **판단 포인트**: AMI(Advanced Metering Infrastructure: 지능형 원격 검침)·SCADA·EMS(Energy Management System)의 통합이 스마트 그리드의 데이터 인프라이며, 사이버 보안이 핵심 취약점이다.

## Ⅰ. 개요 및 필요성

전통 전력망은 중앙 발전소에서 일방향으로 전력을 공급하는 구조로, 재생에너지 확산(분산 발전)·전기차 충전·스마트홈의 수요 패턴 변화에 대응이 어렵다. 스마트 그리드는 양방향 전력·데이터 흐름으로 수요-공급을 실시간 균형화하고, 고장 자동 격리·복구(FDIR), 전력 도난 탐지를 실현한다.

📢 **섹션 요약 비유**: 스마트 그리드는 전력의 인터넷 — 발전소(서버)와 가정(단말)이 양방향으로 소통하며 에너지를 효율적으로 주고받는다.

## Ⅱ. 아키텍처 및 핵심 원리

```
┌─────────────────────────────────────────────────────────┐
│              Smart Grid 계층 구조                         │
│                                                         │
│  발전 레이어: 중앙 발전소 + 태양광/풍력 DER               │
│       ↕ (양방향 전력+데이터)                              │
│  송변전 레이어: EMS + SCADA + 계통 안정화                 │
│       ↕                                                 │
│  배전 레이어: AMI(스마트 미터) + DR 프로그램              │
│       ↕                                                 │
│  소비/저장 레이어: 마이크로그리드 + ESS + EV 충전          │
└─────────────────────────────────────────────────────────┘

마이크로그리드:
  Grid-Connected 모드: 계통과 연결 운영
  Island 모드: 계통 장애 시 로컬 자율 운영
  구성: 태양광 + ESS(배터리) + 부하 + 컨트롤러
```

| 구성 요소 | 설명 | 역할 |
|:---|:---|:---|
| AMI (Advanced Metering Infrastructure) | 지능형 원격 검침 | 실시간 전력 사용 데이터 수집 |
| ESS (Energy Storage System) | 에너지 저장 | 재생에너지 간헐성 보완 |
| DR (Demand Response) | 수요 반응 | 피크 시 수요 자동 조절 |
| V2G (Vehicle-to-Grid) | 전기차 역방전 | EV 배터리 → 그리드 전력 공급 |
| FDIR | 자동 고장 격리 | 고장 구간 자동 검출·복구 |

📢 **섹션 요약 비유**: ESS는 전력망의 은행 — 전기가 남을 때(낮 태양광) 저축하고, 부족할 때(저녁 피크) 찾아 쓴다.

## Ⅲ. 비교 및 연결

V2G(Vehicle-to-Grid): 전기차 배터리를 그리드에 역방전하여 피크 수요 대응 및 재생에너지 균형화. 블록체인 + P2P 에너지 거래: 이웃 간 잉여 태양광 에너지를 블록체인 스마트 컨트랙트로 자동 거래(P2P Energy Trading).

📢 **섹션 요약 비유**: V2G는 전기차의 이중 역할 — 출퇴근용(이동 수단)이면서 피크 때는 가정에 전기를 공급하는 이동식 발전소다.

## Ⅳ. 실무 적용 및 기술사 판단

**의사결정 포인트**:
- 재생에너지 비중 30%+ 계통: ESS + 마이크로그리드 필수
- 데이터 통신: DLMS/COSEM(AMI 표준), IEC 61968(CIM), ANSI C12.22
- 사이버 보안: IEC 62351(전력 통신 보안), NERC CIP
- 마이크로그리드 Island 전환: <100ms 이내 자동 절환 요건

📢 **섹션 요약 비유**: 마이크로그리드는 에너지 자급자족 마을 — 외부 전력(계통)이 끊어져도 자체 발전+저장으로 버티는 독립 에너지 섬이다.

## Ⅴ. 기대효과 및 결론

스마트 그리드는 에너지 효율 20~30% 향상, 정전 복구 시간 80% 단축, 재생에너지 통합을 통한 탄소 중립 달성을 가능하게 한다. AMI 보급과 사이버 보안 강화가 핵심 투자 과제이며, V2G·P2P 에너지 거래·블록체인 통합이 차세대 스마트 그리드의 발전 방향이다.

📢 **섹션 요약 비유**: 스마트 그리드는 에너지의 민주화 — 모든 가정이 발전자(Prosumer: Producer+Consumer)가 되어 에너지를 생산하고 거래한다.

| 개념 | 관계 | 설명 |

### 👶 어린이를 위한 3줄 비유 설명

1. 스마트 그리드는 전기의 인터넷 — 발전소, 집, 전기차가 전기와 정보를 주고받으며 효율적으로 운영돼.
2. ESS는 전기 저금통 — 낮에 태양광으로 발전한 전기를 저금(저장)해두고, 저녁에 꺼내 쓴다.
3. 마이크로그리드는 에너지 자급자족 마을 — 외부 전기가 끊겨도 태양광+배터리로 자체 운영할 수 있어.

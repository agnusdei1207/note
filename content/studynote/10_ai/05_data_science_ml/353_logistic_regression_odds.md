## 핵심 인사이트 (3줄 요약)
> 1. **본질**: 로지스틱 회귀(Logistic Regression)는 선형 회귀의 출력을 오즈비(Odds Ratio)→로짓(Logit)→시그모이드(Sigmoid) 변환을 통해 [0,1] 확률로 매핑하여 이진 분류를 수행하는 확률적 분류기다.
> 2. **가치**: 계수(Coefficient)가 로그 오즈(Log-Odds)를 의미하므로 "나이가 1세 증가할 때 암 발생 오즈가 exp(β)배 변한다"는 직관적 해석이 가능해 의료·신용 평가 등 해석가능 AI(Explainable AI)에 강력하다.
> 3. **판단 포인트**: 손실 함수는 이진 교차 엔트로피(Binary Cross Entropy = Negative Log-Likelihood)이며, 결정 경계(Decision Boundary)는 w·x + b = 0 초평면으로 여전히 선형이다.

---

## Ⅰ. 개요 및 필요성

선형 회귀로 이진 분류를 하면 예측값이 0~1 범위를 벗어나 확률로 해석 불가하다. 로지스틱 회귀는 선형 조합 z = w·x + b를 시그모이드 σ(z) = 1/(1+e⁻ᶻ)에 통과시켜 P(y=1|x) ∈ (0,1)로 변환한다. 이름은 "회귀"지만 실제로는 분류 모델이며, 의료 진단(암 여부), 신용 평가(부도 여부), 스팸 분류 등에서 해석가능성 덕분에 여전히 광범위하게 사용된다.

📢 **섹션 요약 비유**: 로지스틱 회귀는 "저울 → 확률 변환기"다. 여러 증거를 저울에 올려 무게(선형 합)를 재고, 시그모이드라는 마법 변환기로 "암일 확률 73%"처럼 0~100% 확률로 변환해준다.

## Ⅱ. 아키텍처 및 핵심 원리

```
┌──────────────────────────────────────────────────────────┐
│         로지스틱 회귀 수식 전개                           │
├──────────────────────────────────────────────────────────┤
│                                                          │
│  오즈 (Odds):                                           │
│  odds = p / (1-p)    (성공 확률 / 실패 확률)            │
│                                                          │
│  로짓 (Logit) 변환:                                     │
│  logit(p) = log(p/(1-p)) = w·x + b                     │
│  → 선형 모델 = 로그 오즈 = 로짓                         │
│                                                          │
│  시그모이드 역변환:                                      │
│  p = σ(w·x+b) = 1 / (1 + e^(-(w·x+b)))               │
│                                                          │
│  손실 함수 (Binary Cross Entropy):                      │
│  L = -[y·log(p) + (1-y)·log(1-p)]                     │
│                                                          │
│  계수 해석: eʷ = Odds Ratio (오즈비)                   │
│  w=0.5 → eʷ=1.65 → 해당 특성 1 증가 시 오즈 1.65배↑  │
└──────────────────────────────────────────────────────────┘
```

| 변환 | 수식 | 범위 | 의미 |
|:---|:---|:---|:---|
| 확률 p | - | (0,1) | 이벤트 발생 확률 |
| 오즈 (Odds) | p/(1-p) | (0,∞) | 성공:실패 비율 |
| 로그 오즈 (Log-Odds) | log(p/(1-p)) | (-∞,+∞) | 선형 모델 = 로짓 |
| 오즈비 (Odds Ratio) | exp(w) | (0,∞) | 계수의 해석 단위 |

📢 **섹션 요약 비유**: 로짓 변환은 "담장 밖으로 나가는 터널"이다. 확률(0~1 범위)은 담장 안에 갇혀있어서 무한대로 더하거나 뺄 수 없다. 로그(log) 변환이 이 담장을 부수고 (-∞,+∞) 전체 실수 공간으로 확장시켜 선형 회귀와 연결한다.

## Ⅲ. 비교 및 연결

로지스틱 회귀 vs 선형 판별 분석(LDA, Linear Discriminant Analysis): LDA는 정규 분포 가정 하에 클래스 조건부 밀도로부터 결정 경계를 유도하고, 로지스틱 회귀는 P(y|x)를 직접 모델링(판별 모델)한다. 다중 클래스 확장은 소프트맥스 회귀(Softmax Regression = Multinomial Logistic Regression)로, K개 클래스의 로그 오즈를 각각 계산한다.

📢 **섹션 요약 비유**: 로지스틱 회귀(판별 모델)는 "경계선 그리기"고 LDA(생성 모델)는 "각 클래스 분포 모델링"이다. 경계선만 그리면 더 효율적이고, 분포 전체를 모델링하면 새 샘플 생성도 가능하다.

---

## Ⅳ. 실무 적용 및 기술사 판단

의료 AI에서 "나이(x₁)와 혈압(x₂)로 심장병(y=1) 예측" 시 로지스틱 회귀의 계수 w₁=0.03을 해석하면: "나이 1세 증가 → 심장병 오즈 exp(0.03)=1.03배 = 3% 증가". 이 해석 가능성이 의료·금융 규제 환경에서 블랙박스 딥러닝보다 선호되는 이유다. L2 정규화(Ridge Logistic)로 과적합을 방지하고 C=1/λ 파라미터로 조정한다.

📢 **섹션 요약 비유**: 로지스틱 회귀의 계수 해석은 "경제 기사의 통계"다. "금리 1% 상승 시 주택 구매 의향 30% 감소"처럼 한 요인의 변화가 결과에 미치는 영향을 오즈비로 명확히 설명할 수 있어, 판사와 의사가 AI 결정을 신뢰하게 만든다.

---

## Ⅴ. 기대효과 및 결론

로지스틱 회귀는 딥러닝이 범람하는 시대에도 해석가능성(Interpretability)과 계산 효율성 덕분에 금융(신용 평가), 의료(진단 보조), 마케팅(클릭 예측)에서 강력한 기준선(Baseline) 모델로 살아있다. 기술사 시험에서 오즈→로짓→시그모이드 변환 경로, BCE 손실 함수 유도, 계수의 오즈비 해석을 수식과 함께 설명하면 고득점이다.

📢 **섹션 요약 비유**: 로지스틱 회귀는 AI 세계의 "스위스 아미 나이프"다. 복잡한 딥러닝이 우수한 성능을 보일 때도, 법원이나 병원처럼 "왜 이렇게 판단했는지 설명하라"고 요구받는 고위험 환경에서는 해석 가능한 이 고전 도구가 여전히 황금 기준이다.

## 핵심 인사이트 (3줄 요약)
> 1. **본질**: SVD (Singular Value Decomposition, 특이값 분해) 는 임의의 m×n 행렬 A 를 A = UΣVᵀ 로 분해하는데, U 는 좌 특이벡터 (Left Singular Vector), Σ 는 특이값 (Singular Value) 대각 행렬, V 는 우 특이벡터 (Right Singular Vector) 행렬이다.
> 2. **가치**: 저랭크 근사 (Low-Rank Approximation) 를 통해 상위 k개 특이값만 사용하면 원 행렬을 최소 오차로 압축할 수 있고, 이것이 이미지 압축·LSA (Latent Semantic Analysis) 의 수학적 기반이 된다.
> 3. **판단 포인트**: SVD 는 EVD (Eigenvalue Decomposition) 의 일반화로, EVD 가 정방·대칭 행렬에 한정되는 반면 SVD 는 모든 실수 행렬에 적용 가능하다는 범용성이 핵심이다.

---

## Ⅰ. 개요 및 필요성

### SVD 의 필요성과 직관

현실 데이터(문서-단어 행렬, 사용자-아이템 행렬)는 대부분 직사각형 (m≠n) 행렬이다. EVD 는 정방 대칭 행렬에만 적용되므로, 이러한 일반 행렬을 분해하기 위한 도구가 SVD 다.

| 행렬 유형 | 분해 방법 | 결과 |
|:---|:---|:---|
| 정방·대칭 행렬 (n×n) | EVD: A = QΛQᵀ | 고유벡터·고유값 |
| 임의 행렬 (m×n) | SVD: A = UΣVᵀ | 좌·우 특이벡터·특이값 |
| 양의 준정부호 행렬 | EVD와 SVD 일치 | 고유값 = 특이값² |

📢 **섹션 요약 비유**: EVD 가 "정사각형 거울만 분해하는 도구"라면, SVD 는 "어떤 크기의 사진이든 분해하는 범용 분석기"다. 사진을 왼쪽·오른쪽 시각과 중간 중요도로 나눠 이해한다.

## Ⅱ. 아키텍처 및 핵심 원리

### SVD 분해 구조 다이어그램

```
  A (m×n) = U (m×m) · Σ (m×n) · Vᵀ (n×n)
  ┌─────────────────────────────────────────────────┐
  │        U           Σ           Vᵀ               │
  │  ┌──────────┐  ┌────────┐  ┌──────────┐        │
  │  │  u₁  u₂ │  │σ₁ 0 0 │  │  v₁ᵀ    │        │
  │  │  u₃  u₄ │  │ 0 σ₂ 0│  │  v₂ᵀ    │        │
  │  │  ...     │  │ 0  0 0│  │  ...     │        │
  │  └──────────┘  └────────┘  └──────────┘        │
  │  열: 좌 특이벡터  대각: 특이값  행: 우 특이벡터  │
  │  σ₁ ≥ σ₂ ≥ ... ≥ σᵣ ≥ 0  (r = rank(A))       │
  └─────────────────────────────────────────────────┘
```

### 저랭크 근사 (Low-Rank Approximation)

상위 k개 특이값으로 원 행렬을 근사하는 최적 압축 (Eckart-Young 정리):

```
  Aₖ = Σᵢ₌₁ᵏ σᵢ · uᵢ · vᵢᵀ   (외적 합산)

  원 행렬 재구성 오차:
  ||A - Aₖ||_F² = σₖ₊₁² + σₖ₊₂² + ... + σᵣ²

  k=3 으로 근사 시:
  ┌────────────────────────────────────────────────┐
  │ 원본 A (1000×500) = 500,000 값                 │
  │ 근사 Aₖ (k=3)  = 1000×3 + 3 + 3×500          │
  │               = 3,000 + 3 + 1,500 ≈ 4,503 값  │
  │ 압축률: 99.1%  (오차: σ₄² + ... + σᵣ²)        │
  └────────────────────────────────────────────────┘
```

### SVD 와 EVD 의 관계

```
  AᵀA = V Σᵀ Uᵀ U Σ Vᵀ = V (ΣᵀΣ) Vᵀ
  ─────────────────────────────────────────
  → V : AᵀA 의 고유벡터 = 우 특이벡터
  → σᵢ² = AᵀA 의 고유값  (특이값의 제곱)

  AAᵀ = U Σ Vᵀ V Σᵀ Uᵀ = U (ΣΣᵀ) Uᵀ
  → U : AAᵀ 의 고유벡터 = 좌 특이벡터
```

📢 **섹션 요약 비유**: SVD 는 "문서를 '주제(Σ)와 그 주제를 구성하는 단어 패턴(V)', 그리고 '문서가 주제를 얼마나 포함하는지(U)' 세 가지로 나누는" 정보 분해기다.

## Ⅲ. 비교 및 연결

### LSA (Latent Semantic Analysis, 잠재 의미 분석)

LSA 는 문서-단어 행렬 (Term-Document Matrix) 에 SVD 를 적용해 숨겨진 의미 구조를 추출한다.

```
  문서-단어 행렬 A (단어수 × 문서수):
  ┌───────────────────────────────────────┐
  │         문서1 문서2 문서3             │
  │  AI      3    1    0                  │
  │  딥러닝   2    0    1                  │
  │  주식     0    4    2                  │
  └───────────────────────────────────────┘
  SVD 적용 → 잠재 주제 (AI·기술 vs 금융) 자동 추출
```

### 이미지 압축 활용

- 512×512 흑백 이미지: 262,144 픽셀
- SVD 후 상위 50개 특이값만 사용: 512×50 + 50 + 50×512 = 51,250 값
- 압축률: 80.4%, 시각적 품질 충분히 유지

| 응용 분야 | 활용 방식 | 효과 |
|:---|:---|:---|
| 이미지 압축 | 픽셀 행렬 저랭크 근사 | 저장 공간 절감 |
| LSA (문서 유사도) | 문서-단어 행렬 SVD | 동의어·개념 파악 |
| 추천 시스템 | 사용자-아이템 행렬 SVD | 잠재 요인 추출 |
| 노이즈 제거 | 작은 특이값 제거 | 신호 강화 |

📢 **섹션 요약 비유**: SVD 의 저랭크 근사는 "수백만 화소 사진의 핵심 윤곽만 남기고 세부 잡음을 지우는 포토샵 필터"다. 특이값이 큰 성분이 "주요 윤곽", 작은 성분이 "잡음"에 해당한다.

## Ⅳ. 실무 적용 및 기술사 판단

### NumPy SVD 활용

```python
import numpy as np

A = np.random.randn(1000, 500)
U, s, Vt = np.linalg.svd(A, full_matrices=False)
# full_matrices=False: Thin SVD (계산 효율화)

# 상위 k개로 저랭크 근사
k = 50
A_approx = U[:, :k] @ np.diag(s[:k]) @ Vt[:k, :]

# 근사 오차 확인
error = np.linalg.norm(A - A_approx, 'fro')
```

### 기술사 출제 포인트

- A = UΣVᵀ 의 각 행렬 차원과 의미 (U: 행 공간, V: 열 공간, Σ: 특이값)
- Eckart-Young 정리: 저랭크 근사의 최소 오차 보장
- SVD vs EVD: 적용 행렬 범위의 차이 명확히 구별
- LSA 에서 SVD 의 역할: 문서-단어 행렬 → 잠재 의미 공간
- 특이값 감쇠 (Singular Value Decay) 패턴으로 랭크 추정

📢 **섹션 요약 비유**: SVD 의 U, Σ, Vᵀ 는 "영화(A) = 감독 스타일(U) × 장르 강도(Σ) × 배우 특성(Vᵀ)" 처럼 데이터를 세 요소로 나누는 것이다. 장르 강도(특이값)가 큰 성분만 남기면 핵심 패턴이 살아남는다.

## Ⅴ. 기대효과 및 결론

- **범용성**: 어떤 실수 행렬에도 적용 가능한 강력한 분해 도구
- **최적 압축**: Eckart-Young 정리 보장 최소 오차 저랭크 근사
- **의미 추출**: LSA, 협업 필터링에서 잠재 구조 자동 발견
- **수치 안정성**: 의사 역행렬 (Pseudoinverse) 계산 시 안전한 구현
- **한계**: 대규모 행렬에서 전체 SVD 는 O(min(m,n)·mn) 비용 발생

SVD 는 선형대수의 가장 강력하고 범용적인 분해 도구다. 기술사 시험에서는 A=UΣVᵀ 구조, 저랭크 근사의 수학적 의미, LSA 적용 방식, EVD 와의 비교를 체계적으로 서술하면 고득점 가능하다.

📢 **섹션 요약 비유**: SVD 는 "어떤 언어로 쓰인 책이든 번역할 수 있는 만능 통역기"다. 정사각·직사각 어떤 행렬이든 세 가지 순수한 성분으로 분해해 핵심 의미를 드러낸다.

| 개념 | 연관 키워드 | 관계 |
|:---|:---|:---|
| 특이값 (Singular Value) | σᵢ ≥ 0, 정보량 | A 의 중요도 순 성분 강도 |

| 우 특이벡터 (V) | 열 공간, AᵀA 고유벡터 | 입력 공간 정규 기저 |
| 저랭크 근사 | Eckart-Young 정리 | 최소 오차 행렬 압축 |
| LSA | 문서-단어 행렬, 잠재 의미 | SVD 의 NLP 응용 |
| EVD | 정방 대칭 행렬, 고유값 | SVD 의 특수 케이스 |

## 핵심 인사이트 (3줄 요약)
> 1. **본질**: RBF 커널(Radial Basis Function Kernel, 방사 기저 함수 커널) K(x,x') = exp(-γ||x-x'||²)은 두 점 사이의 거리를 무한 차원 특성 공간에서의 내적으로 암묵적으로 변환하는 커널 트릭(Kernel Trick)의 대표 구현이다.
> 2. **가치**: 원본 공간에서 선형 분리 불가능한 XOR 패턴도 가우시안 커널을 통해 무한 차원으로 매핑하면 선형 분리 가능해져, SVM이 어떤 복잡한 결정 경계도 학습할 수 있다.
> 3. **판단 포인트**: γ = 1/(2σ²)는 가우시안의 폭을 제어하며, γ가 크면(좁은 가우시안) → 결정 경계 복잡(과적합), γ가 작으면(넓은 가우시안) → 매끄러운 결정 경계(과소적합).

---

## Ⅰ. 개요 및 필요성

SVM 이중 문제(Dual Problem)에는 샘플 간 내적(x·x')만 나타난다. 커널 트릭은 원본 공간의 내적을 고차원 특성 공간에서의 내적으로 대체하는 수식 K(x,x') = φ(x)·φ(x')을 이용한다. 핵심은 φ(x)(명시적 고차원 변환)를 직접 계산하지 않고 K(x,x')만 계산하면 된다는 것이다. RBF 커널은 이 아이디어의 가장 강력한 구현으로, Taylor 급수 전개 시 무한 차원 내적으로 확장됨이 수학적으로 증명된다.

📢 **섹션 요약 비유**: RBF 커널은 "차원 이동 포탈"이다. 2D 평면에서 뒤엉킨 데이터(선형 분리 불가)를 무한 차원 공간으로 순간 이동시키면 그 공간에서는 초평면 하나로 깔끔하게 나눌 수 있다. 포탈을 직접 만들지 않고(명시적 변환 없이) 목적지에서의 거리(커널 값)만 계산하는 것이 트릭이다.

## Ⅱ. 아키텍처 및 핵심 원리

```
┌──────────────────────────────────────────────────────────┐
│           RBF 커널 수식 및 무한 차원 확장                │
├──────────────────────────────────────────────────────────┤
│  K(x,x') = exp(-γ||x-x'||²)  where γ = 1/(2σ²)        │
│                                                          │
│  Taylor 급수 전개 (1차원 예시):                         │
│  exp(-γ(x-x')²) = exp(-γx²)·exp(2γxx')·exp(-γx'²)     │
│  = Σₙ (2γ)ⁿ/n! · xⁿ·x'ⁿ · exp(-γx²)exp(-γx'²)       │
│  → φ(x) = exp(-γx²)·[1, √(2γ)x, (2γ)x²/√2!, ...]    │
│  → 무한 차원 특성 벡터!                                │
│                                                          │
│  γ 파라미터 효과:                                       │
│  γ 크면 → 각 샘플이 좁은 영역만 영향 → 복잡한 경계    │
│  γ 작으면 → 각 샘플이 넓은 영역 영향 → 매끄러운 경계 │
│                                                          │
│  커널 행렬 K_ij = K(xᵢ, xⱼ): 반드시 양 정치(PSD)      │
└──────────────────────────────────────────────────────────┘
```

| 커널 | 수식 | 특성 공간 | 하이퍼파라미터 |
|:---|:---|:---|:---|
| 선형 (Linear) | x·x' | 원본 공간 | 없음 |
| 다항 (Polynomial) | (x·x'+c)^d | d차 다항 | d, c |
| RBF (가우시안) | exp(-γ||x-x'||²) | 무한 차원 | γ |
| 시그모이드 | tanh(αx·x'+c) | - | α, c |

📢 **섹션 요약 비유**: γ는 "AI의 집중력 범위"다. γ가 크면 각 훈련 샘플이 아주 가까운 이웃에만 영향을 주어(집중력 좁음) 구불구불한 경계를 그린다. γ가 작으면 멀리까지 영향을 주어(집중력 넓음) 부드러운 경계를 그린다.

---

## Ⅲ. 비교 및 연결

Mercer 정리(Mercer's Theorem): 커널 함수가 양 정치(Positive Semi-Definite, PSD) 조건을 만족하면 대응하는 특성 공간 φ가 존재한다. RBF 커널은 PSD이므로 Mercer 정리를 만족한다. 이론적으로 RBF SVM은 충분한 C와 γ 조합에서 어떤 훈련 데이터도 완벽히 분류할 수 있지만 과적합 위험이 있다. 가우시안 프로세스(Gaussian Process)는 RBF 커널을 공분산 함수로 사용하는 베이즈 비모수 회귀 모델이다.

📢 **섹션 요약 비유**: Mercer 정리는 "커널이 합법적인 내적인지 검증하는 수학 법원"이다. PSD 조건을 만족해야만 고차원 특성 공간이 존재하고 커널 트릭이 합법적으로 작동한다. RBF는 이 법원의 심사를 통과한 합법 커널이다.

---

## Ⅳ. 실무 적용 및 기술사 판단

RBF SVM 하이퍼파라미터 튜닝 전략: C와 γ를 로그 스케일 격자(log₁₀ scale grid, 예: C=[0.01,0.1,1,10,100], γ=[0.001,0.01,0.1,1])로 2D GridSearchCV. C와 γ가 모두 크면 과적합, 모두 작으면 과소적합. 데이터 스케일링(StandardScaler) 필수. γ='scale'(1/(n_features·Var(X)))이나 γ='auto'(1/n_features)가 scikit-learn 기본값이다.

📢 **섹션 요약 비유**: C-γ 격자 탐색은 "냉장고 온도와 냉동실 온도 동시 최적화"다. 냉장고(C)와 냉동실(γ)을 각각 독립적으로 조절하며 음식이 가장 잘 보관되는(최적 성능) 조합을 찾는 2D 탐색이다.

---

## Ⅴ. 기대효과 및 결론

RBF 커널은 SVM을 "비선형 분류의 만능 도구"로 만드는 핵심이다. 명시적 고차원 변환 없이 커널 함수만으로 무한 차원에서의 내적을 효율적으로 계산하는 커널 트릭의 아름다운 구현이다. 기술사 시험에서 RBF 수식, γ 역할, Mercer 정리 조건을 간결하게 서술하면 이론적 깊이를 보여줄 수 있다.

📢 **섹션 요약 비유**: RBF 커널은 "마법 지도"다. 복잡하게 뒤엉킨 지형(비선형 데이터)을 위에서 내려다보면(무한 차원) 완전히 분리된 지역으로 보인다. 이 마법 지도를 만드는 비용은 놀랍도록 저렴하다(커널 함수 하나면 충분).

## 핵심 인사이트 (3줄 요약)
> 1. **본질**: GAT (Graph Attention Network, 그래프 어텐션 네트워크)는 GCN (Graph Convolutional Network)의 균일한 이웃 집계를 개선해, 메시지 패싱 (Message Passing) 시 이웃 노드마다 학습된 어텐션 계수 (Attention Coefficient)를 부여한다.
> 2. **가치**: 스펙트럼 기반 GCN과 달리 귀납적 (Inductive) 학습이 가능하고, 어텐션 가중치가 중요한 이웃을 명시적으로 포착하므로 해석 가능성이 높다.
> 3. **판단 포인트**: 멀티헤드 어텐션 (Multi-Head Attention)으로 다양한 이웃 중요도를 병렬 포착하며, GATv2는 원래 GAT의 표현력 제한(정적 어텐션)을 동적 어텐션으로 해결했다.

---

## Ⅰ. 개요 및 필요성

그래프 데이터 (소셜 네트워크, 분자 구조, 지식 그래프)는 유클리드 공간이 아닌 비정규 구조다. GNN (Graph Neural Network)은 이를 학습하지만, GCN은 모든 이웃을 동등하게 취급하는 단점이 있다.

현실에서 "나에게 더 중요한 이웃"이 있다. GAT는 이웃마다 다른 가중치를 학습해 GCN의 한계를 극복한다.

📢 **섹션 요약 비유**: GCN이 "동네 모든 집에서 똑같이 정보를 가져오는" 것이라면, GAT는 "신뢰하는 집은 더 많이, 낯선 집은 덜 듣는" 선택적 정보 수집이다.

---

## Ⅱ. 아키텍처 및 핵심 원리

### GAT 어텐션 계산

```
1단계 - 선형 변환:
   h'ᵢ = W · hᵢ   (W: 공유 가중치 행렬)

2단계 - 어텐션 계수 계산:
   eᵢⱼ = LeakyReLU(aᵀ [h'ᵢ || h'ⱼ])
   (a: 어텐션 파라미터 벡터, ||: 연결)

3단계 - 소프트맥스 정규화:
   αᵢⱼ = softmax_j(eᵢⱼ)
        = exp(eᵢⱼ) / Σ_{k∈Nᵢ} exp(eᵢₖ)

4단계 - 이웃 집계:
   h'ᵢ = σ(Σ_{j∈Nᵢ} αᵢⱼ · Wⱼ · hⱼ)
```

### 멀티헤드 어텐션

```
K개의 독립 어텐션 헤드:
h'ᵢ = ||_{k=1}^{K} σ(Σ_{j∈Nᵢ} α^k_{ij} · W^k · hⱼ)  (중간 레이어)
h'ᵢ = σ(1/K Σ_{k=1}^{K} Σ_{j∈Nᵢ} α^k_{ij} · W^k · hⱼ) (최종 레이어)
```

```
┌──────────────────────────────────────────────────────┐
│  노드 i의 이웃 집계                                   │
│                                                      │
│  j₁ ──(α₁=0.5)──┐                                   │
│  j₂ ──(α₂=0.3)──┼──► [집계] ──► h'ᵢ (업데이트)      │
│  j₃ ──(α₃=0.1)──┤                                   │
│  j₄ ──(α₄=0.1)──┘                                   │
│                                                      │
│  αᵢⱼ: 노드 j가 i에게 얼마나 중요한지 (학습됨)         │
│  Σαᵢⱼ = 1 (소프트맥스 정규화)                        │
└──────────────────────────────────────────────────────┘
```

| 모델 | 이웃 집계 | 그래프 구조 | 귀납적 학습 |
|:---|:---|:---|:---|
| GCN | 균일 평균 | 라플라시안 | ✗ (전이적) |
| GAT | 어텐션 가중치 | 인접 행렬 | ✓ |
| GraphSAGE | 샘플링 집계 | 인접 행렬 | ✓ |
| GATv2 | 동적 어텐션 | 인접 행렬 | ✓ |

### GATv2의 개선 (Brody et al., 2021)

```
원래 GAT: eᵢⱼ = aᵀ (W·hᵢ || W·hⱼ)  → 정적 어텐션
GATv2:   eᵢⱼ = aᵀ W(hᵢ || hⱼ)       → 동적 어텐션
→ 가중치 행렬이 두 노드의 쌍에 공동 적용 → 더 강한 표현력
```

📢 **섹션 요약 비유**: 원래 GAT는 "각 이웃의 점수를 개별로 계산 후 합산", GATv2는 "이웃 쌍의 관계를 함께 보고 점수 계산"하는 더 깊은 상호작용이다.

## Ⅲ. 비교 및 연결

**GNN 분류**:
- 스펙트럼 방법: GCN (라플라시안 기반, 전이적)
- 공간 방법: GAT, GraphSAGE (이웃 직접 집계, 귀납적)

**이질 그래프 (Heterogeneous Graph)**: HAN (Heterogeneous Attention Network) - 다종 노드/엣지에 GAT 확장

📢 **섹션 요약 비유**: 귀납적 학습은 "처음 보는 사람에게도 패턴을 적용하는" 능력이다. 전이적 GCN은 처음 보는 노드를 못 처리하지만, GAT는 처리 가능하다.

---

## Ⅳ. 실무 적용 및 기술사 판단

**응용 분야**:
- 분자 특성 예측 (약물 발견): 원자=노드, 결합=엣지
- 추천 시스템: 사용자/아이템=노드, 상호작용=엣지
- 지식 그래프 완성: 관계 추론
- 트래픽 예측: 도로망 그래프

기술사 포인트: GAT 4단계(선형변환→어텐션→소프트맥스→집계) 수식, 멀티헤드의 역할, 귀납/전이적 학습의 차이 설명.

📢 **섹션 요약 비유**: 분자 그래프 학습에서 GAT가 "어떤 원자 결합이 이 분자의 특성에 더 중요한지"를 어텐션으로 자동 발견한다.

---

## Ⅴ. 기대효과 및 결론

GAT는 그래프 학습에 어텐션 메커니즘을 결합해 GCN의 균일 집계 한계를 극복했다. 어텐션 계수의 시각화로 중요한 이웃 관계를 해석할 수 있어 과학적 발견에도 유용하다. GATv2의 동적 어텐션과 멀티헤드 확장으로 다양한 복잡한 그래프 태스크에서 뛰어난 성능을 보인다.

📢 **섹션 요약 비유**: GAT의 어텐션 계수 시각화는 "AI가 분자에서 어떤 결합을 주목하는지" 보여주는 과학자의 현미경이다.

### 👶 어린이를 위한 3줄 비유 설명
1. GAT는 "친한 친구의 말은 더 많이 듣고, 잘 모르는 사람의 말은 덜 듣는" 선택적 경청 방식이야.
2. 어텐션 계수는 "이 이웃이 나에게 얼마나 중요한지"를 점수로 매기는 거야. 높을수록 더 많이 참고해.
3. 멀티헤드는 여러 명의 친구가 각자 다른 기준으로 중요한 이웃을 고르고, 그 결과를 합쳐서 더 풍부한 정보를 얻는 방법이야.

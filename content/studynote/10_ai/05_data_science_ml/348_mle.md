## 핵심 인사이트 (3줄 요약)
> 1. **본질**: MLE(Maximum Likelihood Estimation, 최대 우도 추정)는 관측된 데이터가 나올 확률(우도)을 가장 크게 만드는 파라미터 θ를 찾는 통계 추정법이며, 딥러닝 손실 함수(MSE, 교차 엔트로피)의 수학적 기원이다.
> 2. **가치**: 가우시안 노이즈 가정 하에 MLE를 적용하면 MSE 손실이, 베르누이 분포 가정 하에 적용하면 이진 교차 엔트로피 손실이 자연스럽게 유도되어 손실 함수 선택의 이론적 근거를 제공한다.
> 3. **판단 포인트**: 계산 편의를 위해 log를 취한 로그 우도(Log-Likelihood)를 최대화하며, 이는 음의 로그 우도(NLL, Negative Log-Likelihood) 최소화 = 경험적 위험 최소화(ERM)와 동치다.

---

## Ⅰ. 개요 및 필요성

선형 회귀 모델이 주어진 데이터를 가장 잘 설명하는 가중치(w)를 어떻게 찾을까? MLE는 "관측 데이터 X = {x₁, x₂, ..., xₙ}이 모델 파라미터 θ 하에서 나올 확률(우도, Likelihood)을 최대화하는 θ를 찾는다"는 원리다. 확률 계산 편의상 독립 가정(i.i.d.) 하에 우도를 곱으로 표현하고, 로그를 씌워 합(Log-Sum)으로 변환한 뒤 미분하여 최적점을 구한다.

📢 **섹션 요약 비유**: MLE는 "범인 찾기"다. 여러 용의자(θ) 중에서 현장 증거(데이터 X)가 나올 가능성을 가장 크게 만드는 단 한 명의 용의자를 찾는 과정이다. 증거가 많을수록 확률이 정교해진다.

---

## Ⅱ. 아키텍처 및 핵심 원리

```
┌──────────────────────────────────────────────────────────┐
│              MLE → 손실 함수 유도 과정                    │
├──────────────────────────────────────────────────────────┤
│ 1. 우도 함수:  L(θ) = Π p(xᵢ | θ)                       │
│                i=1..n                                    │
│                                                          │
│ 2. 로그 우도: ℓ(θ) = Σ log p(xᵢ | θ)                   │
│                                                          │
│ 3. 가우시안 가정 (회귀):                                 │
│    p(y|x,w) = N(y; wᵀx, σ²)                            │
│    ℓ(w) = -n/2·log(2πσ²) - 1/(2σ²)·Σ(yᵢ-wᵀxᵢ)²       │
│    → 최대화 = MSE 최소화 ✓                               │
│                                                          │
│ 4. 베르누이 가정 (분류):                                 │
│    p(y|x,w) = σ(wᵀx)^y · (1-σ(wᵀx))^(1-y)            │
│    ℓ(w) = Σ [y·log σ + (1-y)·log(1-σ)]                 │
│    → 최대화 = BCE(Binary Cross Entropy) 최소화 ✓        │
└──────────────────────────────────────────────────────────┘
```

| 분포 가정 | MLE 목적 | 유도 손실 함수 |
|:---|:---|:---|
| 가우시안 (Gaussian) | 로그 우도 최대화 | MSE (평균 제곱 오차) |
| 베르누이 (Bernoulli) | 로그 우도 최대화 | 이진 교차 엔트로피 |
| 카테고리 (Categorical) | 로그 우도 최대화 | 다중 교차 엔트로피 |
| 라플라스 (Laplace) | 로그 우도 최대화 | MAE (평균 절대 오차) |

📢 **섹션 요약 비유**: MLE는 "피자 레시피 최적화"다. 손님 100명이 먹고 "맛있다/없다"를 평가(데이터)했을 때, 이 평가가 나올 가능성을 가장 높이는 밀가루·치즈 비율(θ)을 찾는 과정이다. 가우시안을 가정하면 평균과 표준편차(MSE), 베르누이를 가정하면 이진 분류(BCE)가 나온다.

---

## Ⅲ. 비교 및 연결

MLE와 MAP(Maximum A Posteriori, 최대 사후 확률 추정)의 차이는 사전 지식(Prior)의 유무다. MLE는 사전 확률을 무시(균등 Prior 가정)하고 우도만 최대화하는 반면, MAP는 베이즈 정리(Bayes Theorem)를 통해 사전 분포를 반영한다. 가우시안 Prior를 사용한 MAP = L2 정규화(Ridge), 라플라스 Prior = L1 정규화(Lasso)와 동치다.

📢 **섹션 요약 비유**: MLE는 "증거만 보는 판사"고 MAP는 "전과 기록(Prior)도 참고하는 판사"다. 전과 기록이 없으면(균등 Prior) 둘의 판결은 같아진다.

---

## Ⅳ. 실무 적용 및 기술사 판단

실무에서 손실 함수를 선택할 때 MLE 관점에서 노이즈 분포를 먼저 가정한다. 이상치(Outlier)가 많은 데이터라면 가우시안 대신 라플라스 분포를 가정하여 MAE 손실을 사용한다. 클래스 불균형(Class Imbalance) 상황에서는 Focal Loss로 우도 함수를 보정한다. 베이즈 딥러닝(Bayesian Deep Learning)에서는 ELBO(Evidence Lower Bound) 최대화가 MLE의 변분 근사(Variational Approximation)로 사용된다.

📢 **섹션 요약 비유**: 손실 함수 선택은 "측량 도구 선택"이다. 평탄한 지형(가우시안 노이즈)엔 자(MSE)를, 울퉁불퉁한 지형(이상치 많음)엔 줄자(MAE)를 쓰듯, 데이터 분포에 맞는 도구를 골라야 정확한 측량(학습)이 된다.

---

## Ⅴ. 기대효과 및 결론

MLE는 딥러닝의 모든 손실 함수를 통합하는 이론적 우산이다. "왜 회귀에 MSE를, 분류에 교차 엔트로피를 쓰는가?"라는 질문에 MLE가 단 하나의 원리로 답한다. 기술사 시험에서 MLE → 손실 함수 유도 과정(분포 가정 → 우도 → 로그 우도 → 최소화)을 수식과 함께 서술하면 최고 수준의 답안이 된다.

📢 **섹션 요약 비유**: MLE는 딥러닝 손실 함수의 "족보"다. MSE도, 교차 엔트로피도, MAE도 모두 이 족보의 자손이며, "어떤 분포 가정에서 태어났는가"를 따라가면 MLE라는 공통 조상을 만난다.

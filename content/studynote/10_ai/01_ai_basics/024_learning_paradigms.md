## 핵심 인사이트 (3줄 요약)
> 1. **본질**: 머신러닝(ML)의 3대 학습 패러다임은 지도 학습(Supervised Learning, 레이블 있는 데이터), 비지도 학습(Unsupervised Learning, 레이블 없는 데이터), 강화 학습(Reinforcement Learning, 보상 기반 환경 상호작용)으로 구분되며, 각각 다른 데이터 구조와 목적 함수를 갖는다.
> 2. **가치**: 이 3가지 패러다임의 경계를 이해하는 것이 ML 문제 정의의 첫 단계다 — 레이블 수집 비용, 데이터 구조, 목표(분류/군집/정책)에 따라 어떤 패러다임을 선택하느냐가 모델 설계 전체를 결정한다.
> 3. **판단 포인트**: 현대 AI는 순수한 3종 패러다임보다 반지도 학습(Semi-supervised), 자기 지도 학습(Self-supervised, GPT·BERT의 기반), 강화 학습+사람 피드백(RLHF)처럼 경계를 허무는 하이브리드 방식이 주류이므로, 패러다임의 조합이 실무 핵심이다.

---

## Ⅰ. 개요 및 필요성

머신러닝 문제를 정의할 때 가장 먼저 결정해야 할 것이 "어떤 데이터가 있고, 무엇을 학습시킬 것인가"이다. 이 결정이 학습 패러다임을 결정한다.

```text
┌────────────────────────────────────────────────────────────┐
│            3대 학습 패러다임 비교                             │
├──────────────┬────────────────────────┬────────────────────┤
│  지도 학습   │     비지도 학습          │    강화 학습        │
├──────────────┼────────────────────────┼────────────────────┤
│ 레이블 데이터 │ 레이블 없는 데이터       │ 환경·보상 신호      │
│ f(x) → y    │ 숨겨진 패턴 발견         │ 최적 정책 π 학습    │
│ 분류·회귀    │ 군집·차원 축소·생성      │ 게임·로봇·추천      │
│ CNN, SVM    │ K-means, VAE, GAN       │ DQN, PPO, AlphaGo  │
└──────────────┴────────────────────────┴────────────────────┘
```

- **📢 섹션 요약 비유**: 지도 학습은 정답이 있는 시험(레이블), 비지도 학습은 정답 없이 스스로 분류하는 탐구 활동, 강화 학습은 게임을 하며 점수로 전략을 배우는 것이다.

---

## Ⅱ. 아키텍처 및 핵심 원리

### 지도 학습 (Supervised Learning)

```text
입력 x → 모델 f → 예측 ŷ → 손실(Loss) = L(y, ŷ) → 역전파(Backprop) → 가중치 업데이트
레이블 y ─────────────────────────────────────────────↗
```

- 분류: 스팸 메일 탐지, 이미지 분류, 의료 진단
- 회귀: 집값 예측, 주가 예측, 수요 예측

### 비지도 학습 (Unsupervised Learning)

| 유형 | 목적 | 대표 알고리즘 |
|:---|:---|:---|
| **군집화 (Clustering)** | 유사한 데이터 그룹화 | K-means, DBSCAN |
| **차원 축소** | 고차원 데이터 시각화 | PCA, t-SNE, UMAP |
| **생성 모델** | 새 데이터 생성 | VAE, GAN, Diffusion |
| **이상 탐지** | 정상 패턴 학습 후 이상 탐지 | Autoencoder |

### 강화 학습 (Reinforcement Learning)

```text
[에이전트 (Agent)]
       │ 행동(Action) a
       ▼
[환경 (Environment)]
       │ 상태(State) s', 보상(Reward) r
       ▼
[에이전트] → 정책(Policy) π 업데이트
목표: 누적 보상(Cumulative Reward) 최대화
```

- **📢 섹션 요약 비유**: 강화 학습은 강아지 훈련이다. 올바른 행동(앉아!)에 간식(보상)을 주고, 잘못된 행동에는 보상을 주지 않으면서 개(에이전트)가 최적 행동 정책을 스스로 학습한다.

---

## Ⅲ. 비교 및 연결

| 항목 | 지도 학습 | 비지도 학습 | 강화 학습 |
|:---|:---|:---|:---|
| **데이터** | 레이블 필요 | 레이블 불필요 | 환경 시뮬레이션 |
| **목표** | 예측 함수 학습 | 구조 발견 | 최적 정책 학습 |
| **피드백** | 즉각 (레이블) | 없음 | 지연 (보상) |
| **난이도** | 낮음 | 중간 | 높음 |

자기 지도 학습(Self-supervised Learning)은 데이터 자체에서 레이블을 생성(ex: 문장 다음 단어 예측)하여 대규모 언어 모델(GPT, BERT)을 훈련하는 비지도 학습의 현대적 발전 형태다.

- **📢 섹션 요약 비유**: 자기 지도 학습은 책을 읽으며 스스로 퀴즈를 만들고 답하는 독학이다. 선생님(레이블)이 없어도 책(비레이블 데이터)에서 스스로 배운다.

---

## Ⅳ. 실무 적용 및 기술사 판단

### 실무 시나리오: 이커머스 추천 시스템 설계
- **비지도 학습 (1단계)**: 구매 이력 군집화 → 고객 세그먼트 발견.
- **지도 학습 (2단계)**: 세그먼트 레이블 + 구매 데이터 → 클릭률(CTR) 예측 모델.
- **강화 학습 (3단계)**: 추천 → 클릭 보상 → 정책 최적화 (장기 구매 전환 극대화).

### 안티패턴
- 레이블 없이 지도 학습을 시도하거나, 레이블이 풍부한데 비지도 학습을 선택하는 패러다임 불일치 안티패턴. 각 패러다임은 데이터 구조에 의해 결정되므로, 데이터 탐색(EDA)을 먼저 수행하고 패러다임을 선택해야 한다.

- **📢 섹션 요약 비유**: 정답지(레이블)가 있는데 스스로 탐구하는 것은 시험 답안지를 갖고도 스스로 풀겠다고 옆으로 치우는 것이다. 반대로 정답지가 없는데 지도 학습을 쓰는 건 없는 답안지를 만들어내는 헛수고다.

## Ⅴ. 기대효과 및 결론

| 패러다임 | 대표 성과 |
|:---|:---|
| **지도 학습** | 의료 AI 진단 정확도 95%+ |
| **비지도 학습** | 고객 세그먼트 자동 발견 |
| **강화 학습** | AlphaGo 세계 챔피언 격파, 로봇 제어 |

LLM (Large Language Model, 대형 언어 모델)의 등장으로 자기 지도 학습 + RLHF (Reinforcement Learning from Human Feedback)의 조합이 AI 패러다임의 새 표준이 되었으며, 3대 패러다임의 경계는 더욱 흐려지고 있다.

- **📢 섹션 요약 비유**: 3대 학습 패러다임은 인간 학습의 3가지 방식이다. 지도 학습은 선생님에게 배우기, 비지도 학습은 스스로 탐구하기, 강화 학습은 경험으로 깨닫기. 현대 AI는 이 세 가지를 모두 혼합하여 사용한다.

### 📌 관련 개념 맵

| 개념 | 연결 포인트 |
|:---|:---|
| **손실 함수 (Loss Function)** | 지도 학습의 학습 방향 결정 |
| **K-means / DBSCAN** | 비지도 군집화 대표 알고리즘 |
| **Q-Learning / PPO** | 강화 학습 정책 최적화 알고리즘 |
| **자기 지도 학습** | 비지도 학습의 현대적 발전; GPT, BERT |
| **RLHF** | 강화 학습 + 인간 피드백; ChatGPT 훈련 방법 |

### 📈 관련 키워드 및 발전 흐름도

```text
[지도 학습 — 레이블 데이터, 예측 함수 학습]
    │
    ▼
[비지도 학습 — 패턴·구조 발견, 생성 모델]
    │
    ▼
[강화 학습 — 환경 상호작용, 보상 기반 정책 최적화]
    │
    ▼
[자기 지도 학습 — 데이터 자체에서 레이블 생성 (GPT, BERT)]
    │
    ▼
[RLHF — 인간 피드백 강화 학습 (ChatGPT, Claude)]
```

### 👶 어린이를 위한 3줄 비유 설명

1. 지도 학습은 선생님이 정답을 알려주며 가르치는 것, 비지도 학습은 스스로 탐구하며 패턴을 찾는 것, 강화 학습은 게임을 하면서 점수를 올리는 법을 배우는 것이에요!
2. 스팸 메일 잡기는 지도 학습, 고객 취향 자동 분류는 비지도 학습, 바둑 AI(알파고)는 강화 학습이에요.
3. 요즘 챗GPT 같은 AI는 이 세 가지를 모두 섞어서 훨씬 더 똑똑해졌답니다!

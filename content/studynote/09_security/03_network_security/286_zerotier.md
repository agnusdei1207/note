## 핵심 인사이트 (3줄 요약)
> 1. **본질**: ZeroTier는 SDN (Software-Defined Networking) 원리를 P2P VPN에 적용하여, 중앙 서버 없이 분산된 Planet/Moon 네트워크를 통해 전 세계 어디서든 같은 가상 이더넷 세그먼트를 공유하는 혁신적 네트워킹 플랫폼이다.
> 2. **가치**: ZeroTier는 Layer 2 이더넷 프레임을 인터넷을 통해 전달하므로, 물리적으로 분산된 서버들이 동일 LAN에 있는 것처럼 동작할 수 있어 브로드캐스트·ARP가 필요한 레거시 애플리케이션도 원격에서 작동한다.
> 3. **판단 포인트**: Planet(공용 루트 서버)에 의존하는 구조는 ZeroTier Inc.의 인프라 가용성에 종속되는 문제가 있으며, Moon(자체 루트 서버)으로 일부 의존성을 줄일 수 있지만 완전한 자립은 어렵다.

## Ⅰ. 개요 및 필요성

ZeroTier는 Adam Ierymenko가 2011년 창업한 ZeroTier, Inc.가 개발한 오픈소스(BSL 1.1) 네트워킹 플랫폼이다. 전통적인 VPN은 중앙 서버(게이트웨이)를 통해 모든 트래픽이 흐르는 허브-스포크(Hub-and-Spoke) 구조인데 반해, ZeroTier는 피어 간 직접 통신(P2P)을 추구한다. 가능하면 직접 UDP 홀 펀칭(Hole Punching)으로 피어끼리 연결하고, 직접 연결이 불가능할 때만 DERP(분산 릴레이) 서버를 통해 중계한다.

ZeroTier의 핵심 차별점은 가상 이더넷 스위치를 소프트웨어로 구현한다는 것이다. 사용자는 ZeroTier 네트워크 ID(16자리 16진수)에 가입하면, 가상 NIC이 생성되고 해당 네트워크의 다른 모든 멤버와 동일한 Layer 2 도메인을 공유하게 된다. 이는 마치 전 세계에 분산된 서버들이 하나의 물리 스위치에 연결된 것과 같은 효과를 낸다.

📢 **섹션 요약 비유**: ZeroTier는 "세계 어디서나 같은 와이파이에 연결되는 마법 공유기"다. 서울에 있는 서버와 뉴욕에 있는 서버가 마치 같은 방에 있는 것처럼 서로 직접 대화할 수 있다.

## Ⅱ. 아키텍처 및 핵심 원리

### Planet / Moon / Leaf 3계층 구조

```
┌─────────────────────────────────────────────────────────┐
│  PLANET (공용 루트 서버, ZeroTier Inc. 운영)             │
│  전 세계 12개 노드, 모든 피어의 위치 정보 보유           │
└───────────────────────┬─────────────────────────────────┘
                        │ 피어 위치 조회
        ┌───────────────┼───────────────┐
        ↓                               ↓
┌───────────────┐               ┌───────────────┐
│  MOON          │               │  MOON          │
│ (자체 루트서버) │               │ (기업용 사설)  │
│ Planet 백업   │               │ 프라이빗 루팅  │
└───────┬───────┘               └───────┬───────┘
        │ 연결 중개                      │
   ┌────┴────┐                    ┌────┴────┐
   │  LEAF   │   ←─ P2P 직접 ─→  │  LEAF   │
   │ (단말)  │   UDP 홀 펀칭      │ (단말)  │
   └─────────┘                    └─────────┘
```

### ZeroTier 패킷 처리 흐름

```
애플리케이션 (10.144.0.2)
    |  ARP 요청 / IP 패킷
    ↓
ZeroTier 가상 이더넷 스위치 (VL2)
    |  Ethernet 프레임 처리
    |  목적지 MAC → 피어 매핑 테이블 조회
    ↓
ZeroTier 암호화 레이어 (Salsa20/Poly1305)
    |  패킷 암호화 + 서명
    ↓
UDP 전송 (직접 P2P 또는 DERP 릴레이)
    ↓
원격 피어 ZeroTier 스택
    |  복호화 + 검증
    ↓
상대방 애플리케이션 (10.144.0.3)
```

### 주요 구성 요소

| 구성 요소 | 설명 |
|:---|:---|
| Network ID | 16자리 16진수, 가상 네트워크 식별자 |
| Node ID | 40비트 공개키 기반, 피어 고유 식별자 |
| VL1 | 암호화된 P2P 전송 계층 |
| VL2 | 가상 이더넷 스위치 계층 |
| Controller | 네트워크 멤버 인증·IP 할당 관리 |
| Rules Engine | L2/L3/L4 수준의 트래픽 필터링 규칙 |

📢 **섹션 요약 비유**: ZeroTier의 Planet/Moon/Leaf는 "전국 도로망"과 같다. Planet은 고속도로 안내판(전국 위치 정보), Moon은 지역 도로 표지판(기업 내부 라우팅), Leaf는 실제 차량(단말)이다.

## Ⅲ. 비교 및 연결

| 항목 | ZeroTier | WireGuard | Tailscale |
|:---|:---|:---|:---|
| 계층 | Layer 2 (이더넷) | Layer 3 (IP) | Layer 3 (IP) |
| 토폴로지 | P2P + 릴레이 | 허브-스포크 또는 풀메시 | P2P + DERP 릴레이 |
| 중앙 서버 의존 | Planet (필수) | 없음 (설정만) | Coordination Server |
| 셀프호스팅 | Moon (부분 가능) | 완전 가능 | Headscale |
| 브로드캐스트 지원 | ✅ (L2) | ❌ (L3) | ❌ (L3) |
| 암호화 | Salsa20/Poly1305 | ChaCha20-Poly1305 | WireGuard 동일 |
| GUI 관리 | ZeroTier Central | ❌ (텍스트 설정) | ✅ (웹 대시보드) |
| 최대 권장 노드 | ~200 (L2 한계) | 제한 없음 | 제한 없음 |

📢 **섹션 요약 비유**: ZeroTier는 "모든 방이 같은 복도로 이어지는 대형 아파트", WireGuard는 "각 집이 개별 현관 열쇠를 가진 단독주택 단지", Tailscale은 "단독주택에 스마트 인터폰을 달아 관리해주는 서비스"다.

## Ⅳ. 실무 적용 및 기술사 판단

**사용 시나리오별 적합성:**

1. **홈랩/개발 환경**: 집에서 회사 서버, 클라우드 VM, 라즈베리파이를 같은 네트워크에 연결. 설정이 간단하고 무료 플랜(25노드)으로 충분.

2. **레거시 애플리케이션**: NetBIOS, mDNS, 브로드캐스트 기반 프로토콜이 필요한 레거시 시스템. L2 지원이 필수인 경우 ZeroTier만이 현실적 선택.

3. **게임 서버 / 소규모 협업**: 온라인 게임의 LAN 파티 구현, 소규모 팀 개발 환경. 방화벽 없이 간단하게 같은 네트워크 구성.

4. **엔터프라이즈 한계**: 200+ 노드에서 L2 브로드캐스트 스톰 위험. 컴플라이언스가 강한 환경에서 ZeroTier Planet 의존성이 문제. 이 경우 Tailscale Business 또는 WireGuard+Headscale이 적합.

5. **자체 컨트롤러**: `zerotier-one` + `ztncui`(오픈소스 컨트롤러 UI)를 자체 운영하면 Planet 의존성을 줄일 수 있지만 완전 오프라인은 Moon+자체 컨트롤러 조합으로만 가능.

📢 **섹션 요약 비유**: ZeroTier 컨트롤러 자립화는 "마트 대신 직접 농장을 차리는 것"이다. 제어권은 높아지지만 관리 부담도 함께 높아진다. 소규모에서는 마트(ZeroTier Central)가, 대규모에서는 직영 농장이 효율적이다.

## Ⅴ. 기대효과 및 결론

ZeroTier는 복잡한 VPN 설정 없이 글로벌 가상 LAN을 10분 안에 구축할 수 있다는 압도적인 편의성으로 개발자·홈랩·소규모 팀에서 큰 인기를 얻고 있다. Layer 2 지원은 WireGuard나 IPsec이 따라올 수 없는 고유 강점이다.

그러나 엔터프라이즈 환경에서는 Planet 의존성, L2 스케일 한계, 보안 감사 복잡성이 장벽이 된다. 기술사는 사용 규모, L2 필요 여부, 자립화 요구 수준에 따라 ZeroTier/WireGuard/Tailscale 중에서 적합한 솔루션을 선택하는 판단 능력이 중요하다.

📢 **섹션 요약 비유**: ZeroTier는 "미리 깔려 있는 마법 인터넷 케이블"이다. 가입 즉시 연결되는 편리함이 있지만, 케이블 회사(ZeroTier Inc.)가 어느 날 서비스를 중단하면 대체 방안이 필요하다.

### 👶 어린이를 위한 3줄 비유 설명
1. ZeroTier는 전 세계 친구들이 같은 마을에 사는 것처럼 연결해주는 마법의 네트워크야.
2. 중간에 우체국(Planet 서버)이 친구 집 주소를 알려주고, 이후에는 친구끼리 직접 편지를 주고받아.
3. 그냥 방 안에 있는 것처럼(Layer 2) 소리를 질러도(브로드캐스트) 다른 친구들에게 들릴 수 있는 게 특별한 점이야.

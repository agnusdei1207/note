## 핵심 인사이트 (3줄 요약)
> 1. **본질**: APT (Advanced Persistent Threat)는 고도의 기술력을 가진 위협 행위자(주로 국가 지원)가 특정 표적을 장기간에 걸쳐 은밀하게 침투·유지하며 정보를 수집하거나 파괴 공작을 수행하는 조직적 사이버 위협 캠페인이다.
> 2. **가치**: 단발성 공격이 아닌 지속적 캠페인이므로 탐지부터 대응까지 전통적 보안 체계(시그니처 기반 AV, 단순 방화벽)로는 한계가 있으며, 위협 인텔리전스(TI)와 헌팅(Threat Hunting)이 핵심 대응 방법론이다.
> 3. **판단 포인트**: MITRE ATT&CK 프레임워크로 APT 전술·기법(TTP)을 매핑하고, 킬체인(Kill Chain) 단계별 차단 포인트를 수립하는 것이 기술사 답안의 핵심이다.

---

## Ⅰ. 개요 및 필요성

APT (Advanced Persistent Threat)의 세 단어가 각각의 특성을 정의한다.  
- **Advanced(고도화)**: 제로데이(Zero-day) 익스플로잇, 커스텀 악성코드, 정교한 사회공학 등 고수준 기법 사용  
- **Persistent(지속성)**: 수개월~수년 동안 탐지되지 않고 잠복하며 목표 달성  
- **Threat(위협)**: 단순 자동화 공격이 아닌 인간 운영자에 의한 목적 지향적 공격  

APT 그룹은 주로 국가 정보기관, 군사 사이버 부대, 국가 지원 범죄 조직으로 구성된다. APT1(중국 PLA), Lazarus Group(북한), Cozy Bear/Fancy Bear(러시아 SVR/GRU) 등이 대표적이다. 민간 기업의 지적재산(IP) 탈취, 정부·군사 기밀 수집, 핵심 인프라 사전 침투가 주요 목적이다.

📢 **섹션 요약 비유**: 일반 사이버 공격이 소매치기라면, APT는 몇 달을 관찰하고 준비한 전문 스파이다.

## Ⅱ. 아키텍처 및 핵심 원리

**APT 킬체인 (Lockheed Martin Kill Chain)**

| 단계 | 행위 | 대응 포인트 |
|:---|:---|:---|
| 1. 정찰 | 표적 정보 수집 (LinkedIn, 쇼단 등) | 노출 정보 최소화 |
| 2. 무기화 | 익스플로잇+페이로드 패키징 | 위협 인텔리전스 |
| 3. 전달 | 피싱·드라이브바이·USB 등 | 이메일 필터·AV |
| 4. 익스플로잇 | 취약점 코드 실행 | 패치·EMET |
| 5. 설치 | 악성코드·백도어 설치 | EDR·AppLocker |
| 6. C2 | 공격자와 통신 채널 확립 | 네트워크 모니터링 |
| 7. 목표 달성 | 데이터 유출·파괴·권한 확대 | DLP·세그멘테이션 |

```text
┌──────────────────────────────────────────────────┐
│               APT 침투 타임라인 예시             │
├──────────────────────────────────────────────────┤
│                                                  │
│  0일: 스피어 피싱 이메일 → 초기 침투             │
│  1-7일: 내부 탐색, 권한 상승                     │
│  7-30일: 측면 이동, 추가 시스템 장악             │
│  30-180일: 데이터 수집, 은밀한 유출              │
│  180일+: 장기 잠복, 추가 임무 대기               │
│                                                  │
│  → 평균 탐지 시간: 약 197일 (Mandiant M-Trends)  │
└──────────────────────────────────────────────────┘
```

**MITRE ATT&CK 매핑**: APT 그룹별 TTP (Tactics, Techniques, Procedures)를 표준화된 프레임워크로 추적해 방어 갭을 시각화한다.

📢 **섹션 요약 비유**: APT는 거미줄처럼—탐지 못 하게 조용히 실을 뻗어 하나씩 연결하다가 먹잇감이 걸리는 순간 모든 것이 드러난다.

## Ⅲ. 비교 및 연결

| 항목 | APT | 일반 사이버 범죄 | 핵티비즘 |
|:---|:---|:---|:---|
| 동기 | 국가 전략·IP 탈취 | 금전 | 정치·이념 |
| 기술 수준 | 최고급 | 중급 | 중·하급 |
| 지속 기간 | 수개월~수년 | 수일~수주 | 단발 |
| 주요 피해 | 기밀 유출·인프라 파괴 | 금전 피해 | 웹사이트 변조 |
| 탐지 난이도 | 매우 어려움 | 중간 | 쉬움 |

📢 **섹션 요약 비유**: APT는 단거리 달리기 선수가 아니라 마라토너—눈에 띄지 않고 오래 달린다.

---

## Ⅳ. 실무 적용 및 기술사 판단

**APT 대응 방법론**  
1. **위협 인텔리전스(TI)**: ISAC·공개 CTI 피드·다크웹 모니터링으로 공격 조기 경보  
2. **위협 헌팅(Threat Hunting)**: 알려지지 않은 IOC (Indicator of Compromise) 능동적 탐색  
3. **제로트러스트 아키텍처**: 내부 신뢰 최소화, 모든 접근 지속 검증  
4. **네트워크 세그멘테이션**: 측면 이동 차단으로 피해 범위 제한  
5. **특권 계정 관리(PAM)**: 도메인 관리자 계정 탈취 후 전파 차단  
6. **SIEM + SOAR**: 로그 통합 분석 및 자동화된 대응  

**기술사 핵심**: "APT 대응은 예방이 아닌 탐지·대응 속도 경쟁"임을 강조한다.

📢 **섹션 요약 비유**: APT 방어는 집에 도둑이 들어올 것을 가정하고 CCTV와 경보를 설치하는 것—들어오지 못하게 막는 것만으로는 부족하다.

---

## Ⅴ. 기대효과 및 결론

APT는 현대 사이버 위협의 정점이다. 단일 기술 통제로는 방어가 불가능하며, 기술·프로세스·사람(인력)을 아우르는 심층 방어(Defense in Depth)와 지속적 모니터링이 필수다. MITRE ATT&CK 프레임워크를 기반으로 조직의 방어 갭을 주기적으로 평가하고, 사고 대응 훈련(CSIRT 演習)을 정기화하는 것이 장기적 APT 방어 체계의 핵심이다.

📢 **섹션 요약 비유**: APT 방어는 전쟁 준비—평소에 훈련하고, 인텔리전스를 수집하고, 방어선을 촘촘히 해야 실전에서 살아남는다.

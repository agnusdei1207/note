## 핵심 인사이트 (3줄 요약)
> 1. **본질**: AV(Anti-Virus)+EDR(Endpoint Detection and Response)+NDR(Network Detection and Response)+UEBA(User and Entity Behavior Analytics)는 각각의 탐지 공백을 상호 보완하는 4중 방어 레이어 조합이다.
> 2. **가치**: 단일 도구는 공격 킬 체인의 일부만 볼 수 있지만, 4가지 기술을 통합하면 초기 침투부터 데이터 유출까지 전 단계를 커버하는 포괄적 가시성을 확보한다.
> 3. **판단 포인트**: 통합 효과를 최대화하려면 각 솔루션의 이벤트를 SIEM(Security Information and Event Management) 또는 XDR 플랫폼에서 상관 분석해야 하며, 데이터 표준화(STIX/TAXII)가 선행되어야 한다.

---

## Ⅰ. 개요 및 필요성

현대 사이버 공격은 단일 벡터(Vector)로 이루어지지 않는다. 2024년 Verizon DBIR(Data Breach Investigations Report)에 따르면 대부분의 침해 사고는 ① 피싱·취약점 악용(초기 침투) → ② 악성코드 실행(엔드포인트 감염) → ③ 네트워크 측면 이동(Lateral Movement) → ④ 자격증명 탈취·데이터 유출의 복합 단계를 거친다.

각 단계를 탐지하는 데 최적화된 도구가 다르다. AV는 알려진 악성코드 파일을 막고, EDR은 엔드포인트의 프로세스 행위를 분석하며, NDR은 네트워크 트래픽의 이상을 감지하고, UEBA는 사용자·엔티티의 비정상적 행동 패턴을 탐지한다. 이 4가지를 조합하면 공격의 전체 킬 체인을 빈틈 없이 커버할 수 있다.

특히 내부자 위협(Insider Threat)과 계정 탈취(Account Takeover) 공격에는 UEBA가 결정적 역할을 한다. AV·EDR은 악성 파일·프로세스를 탐지하지만, 탈취된 계정으로 정상 도구를 사용해 데이터를 유출하는 경우는 포착하기 어렵다. UEBA는 이런 비정상적인 행동 패턴을 기준선(Baseline) 대비 이상치로 탐지한다.

📢 **섹션 요약 비유**: AV+EDR+NDR+UEBA 조합은 "공항 보안 4단계"다. 탑승권 확인(AV) → 짐 X레이(EDR) → 게이트 카메라(NDR) → 탑승자 행동 분석(UEBA)이 순서대로 작동한다.

## Ⅱ. 아키텍처 및 핵심 원리

### 4가지 솔루션 역할 분담

| 솔루션 | 탐지 대상 | 데이터 소스 | 탐지 공백 |
|:------|:---------|:----------|:---------|
| AV | 알려진 악성 파일 | 파일 시스템 | 제로데이, 파일리스 |
| EDR | 엔드포인트 프로세스 행위 | OS 이벤트, API 호출 | 네트워크 이동 |
| NDR | 네트워크 트래픽 이상 | 패킷, 플로우 데이터 | 호스트 내부 행위 |
| UEBA | 사용자·엔티티 행동 이상 | 로그, ID 이벤트 | 파일·네트워크 세부 |

```
┌──────────────────────────────────────────────────────┐
│        AV + EDR + NDR + UEBA 통합 방어 구조          │
│                                                      │
│  초기 침투    엔드포인트    네트워크    데이터유출    │
│  ┌──────┐   ┌──────┐    ┌──────┐   ┌──────┐        │
│  │  AV  │──▶│ EDR  │───▶│ NDR  │──▶│UEBA  │        │
│  └──────┘   └──────┘    └──────┘   └──────┘        │
│  악성파일   프로세스     트래픽      행동이상        │
│                                                      │
│              ┌─────────────────────┐                 │
│              │  SIEM / XDR 상관분석 │                 │
│              │  통합 인시던트 뷰    │                 │
│              └─────────────────────┘                 │
└──────────────────────────────────────────────────────┘
```

통합 운영의 핵심은 각 솔루션의 **이벤트를 공통 컨텍스트로 연결**하는 것이다. 엔드포인트 IP, 사용자 ID, 프로세스 해시가 공통 필드로 활용되어 SIEM·XDR에서 "A 사용자의 PC에서 악성 프로세스 실행 → 외부 C2 서버로 연결 → 비정상 시간대 데이터 다운로드" 같은 공격 스토리를 자동으로 구성할 수 있다.

📢 **섹션 요약 비유**: 4가지 솔루션 통합은 "탐정 팀"과 같다. 법의관(AV), 현장 형사(EDR), 교통 조사관(NDR), 프로파일러(UEBA)가 각자 전문 분야를 맡고 정보를 공유한다.

## Ⅲ. 비교 및 연결

### 공격 단계별 탐지 도구 매핑

| 공격 단계(MITRE ATT&CK) | 주요 탐지 도구 | 탐지 예시 |
|:----------------------|:-------------|:---------|
| Initial Access | AV, 이메일 보안 | 피싱 첨부파일, 악성 URL |
| Execution | EDR | 악성 스크립트 실행 |
| Persistence | EDR | 레지스트리 자동 실행 등록 |
| Lateral Movement | NDR | 비정상 SMB 연결 |
| Collection/Exfiltration | NDR, UEBA | 대량 파일 접근, 외부 전송 |

이 조합은 제로 트러스트(Zero Trust) 아키텍처와 자연스럽게 연계된다. UEBA의 신원 검증 정보는 ZTNA(Zero Trust Network Access) 접근 제어에 활용되고, EDR의 엔드포인트 상태는 마이크로 세그멘테이션(Micro-Segmentation) 정책에 반영된다.

📢 **섹션 요약 비유**: 공격 단계별 도구 매핑은 "마라톤 구간별 심판 배치"와 같다. 각 구간마다 전문 심판이 부정 행위를 감시하고, 전체 기록을 중앙에서 통합 관리한다.

## Ⅳ. 실무 적용 및 기술사 판단

**통합 구축 로드맵:**
1. AV/EPP 기반 구축 (즉시 예방 효과)
2. EDR 추가 (포렌식·행위 분석)
3. NDR 연동 (네트워크 가시성 확보)
4. UEBA 통합 (내부자 위협·계정 탈취 탐지)
5. SIEM/XDR 통합 (상관 분석·통합 대응)

**운영 효율화 포인트:**
- 공통 IOC(Indicator of Compromise) 피드로 4개 솔루션 동시 업데이트
- Alert Triage 자동화: 단일 이벤트 알람은 SOAR로 자동 처리
- 분기별 Red Team 연습으로 탐지 공백 점검

📢 **섹션 요약 비유**: 통합 구축 로드맵은 "집 방범 강화 순서"다. 자물쇠(AV) → CCTV(EDR) → 골목 감시카메라(NDR) → 이웃과의 연락망(UEBA) 순으로 단계별로 강화한다.

---

## Ⅴ. 기대효과 및 결론

AV+EDR+NDR+UEBA 통합 구축 시 공격 킬 체인의 전 단계를 커버할 수 있으며, 단일 솔루션 대비 위협 탐지율이 3배 이상 향상된다는 산업 보고서가 다수 발표되었다. 또한 각 솔루션의 상관 분석을 통해 오탐(False Positive)도 40% 이상 감소한다.

기술사 시험 서술 시 이 4가지 조합을 킬 체인 단계와 매핑하여 설명하면 높은 논리성 점수를 받을 수 있다. 실무에서는 예산과 운영 역량을 고려한 단계적 도입이 권고된다.

📢 **섹션 요약 비유**: AV+EDR+NDR+UEBA는 "철통 방어 팀"이다. 한 명이 놓치면 다른 한 명이 잡는 중첩 방어가 공격자의 성공 확률을 극적으로 낮춘다.

### 👶 어린이를 위한 3줄 비유 설명
1. AV는 문에서 나쁜 사람을 막고, EDR은 건물 안에서 감시하고, NDR은 복도 카메라가, UEBA는 행동 분석가가 맡아.
2. 각자 맡은 부분이 달라서 한 명이 놓쳐도 다음 사람이 잡을 수 있어.
3. 모두 합치면 어디서도 도망칠 수 없는 완벽한 감시망이 만들어져.

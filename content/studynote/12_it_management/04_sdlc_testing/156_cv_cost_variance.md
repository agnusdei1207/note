## 핵심 인사이트
> 1. **본질**: CV (Cost Variance, 비용 차이) = EV - AC로, 완료된 작업의 계획 예산과 실제 투입 비용의 차이를 나타낸다—양수(+)는 예산 절감, 음수(-)는 예산 초과를 의미한다.
> 2. **가치**: CV는 단순한 "예산 vs 지출" 비교가 아니라 "완료된 만큼을 기준으로 한 비용 차이"이므로, 작업이 계획보다 적게 완료된 경우에도 비용 초과를 정확히 진단한다.
> 3. **판단 포인트**: CV와 CPI를 함께 봐야 한다—CV가 같은 -100만원이어도 프로젝트 규모에 따라 의미가 다르므로 CV% = (CV/EV)×100으로 상대적 크기를 파악해야 한다.

---

## Ⅰ. 개요 및 필요성

CV (Cost Variance, 비용 차이)는 EVM (Earned Value Management)의 비용 성과 진단 지표다. 전통적인 비용 추적에서 "예산 500만원 중 600만원을 썼다"는 단순 비교는 작업 진척을 고려하지 않는다. EVM의 CV는 "600만원을 썼는데, 그 돈으로 500만원어치의 가치를 완성했는가"를 측정한다.

CV = EV - AC 공식에서 EV는 완료된 작업의 계획 예산 가치이고, AC는 실제 투입 비용이다. EV < AC이면 계획보다 비싸게 일한 것이므로 CV < 0 (비용 초과), EV > AC이면 계획보다 싸게 일한 것이므로 CV > 0 (비용 절감)이다.

CV는 프로젝트 종료 시까지 누적되어 최종 VAC (Variance at Completion) = BAC - EAC를 예측하는 기반이 된다. 따라서 CV 추이를 매 보고 주기마다 모니터링하고, |CV| > 임계값(예: EV의 10%)이면 시정 조치(Corrective Action)를 취해야 한다.

📢 **섹션 요약 비유**: CV는 음식점의 원가 관리와 같다. 오늘 팔린 음식(EV)의 계획 원가가 100원인데, 실제로 130원이 들었다면 CV = -30원(비용 초과)이다—많이 팔수록 손실이 커진다.

## Ⅱ. 아키텍처 및 핵심 원리

### CV 공식과 해석

```text
CV = EV - AC

예시:
  BAC = 1,000만원
  현재 완료율 = 50% → EV = 500만원
  실제 지출 AC = 600만원

  CV = 500 - 600 = -100만원 (비용 초과)

  해석:
  → 500만원어치 작업을 완료하는 데 600만원을 썼음
  → 100만원을 계획보다 더 지출

CV% = (CV / EV) × 100
     = (-100 / 500) × 100 = -20%
  → 완료된 가치의 20%만큼 예산 초과
```

### CV 상태 분류

```text
┌──────────────────────────────────────────────────────────────┐
│  CV 상태 분류 및 조치                                         │
├──────────────┬──────────────────┬────────────────────────────┤
│ CV 값        │ 상태             │ 권장 조치                  │
├──────────────┼──────────────────┼────────────────────────────┤
│ CV > 0       │ 예산 절감        │ 긍정적, 절감 원인 분석     │
│ (Over Budget)│ (Under Budget)   │ (품질 타협 여부 확인)      │
├──────────────┼──────────────────┼────────────────────────────┤
│ CV = 0       │ 계획대로         │ 유지                       │
├──────────────┼──────────────────┼────────────────────────────┤
│ 0 > CV       │ 경미한 초과      │ 원인 파악, 모니터링 강화   │
│ > -EV×10%    │                  │                            │
├──────────────┼──────────────────┼────────────────────────────┤
│ CV < -EV×10% │ 심각한 초과      │ 즉시 조치, 재계획 검토     │
│              │ (적색 경보)      │                            │
└──────────────┴──────────────────┴────────────────────────────┘
```

### CV와 CPI의 관계

```text
CV와 CPI는 같은 정보를 다른 형태로 표현:
  CV  = EV - AC   → 절대값 (원화 단위)
  CPI = EV / AC   → 비율 (단위 없음)

  CV가 같아도 CPI는 다를 수 있음:
  예1: EV=100, AC=110 → CV=-10, CPI=0.91
  예2: EV=1000, AC=1010 → CV=-10, CPI=0.99

  → CV 금액은 동일하지만
    예1은 10% 초과, 예2는 1% 초과로 심각도 다름
  → CPI가 상대적 효율을 더 잘 표현
```

📢 **섹션 요약 비유**: CV와 CPI의 관계는 "음식값이 200원 비쌌다"와 "음식값이 20% 비쌌다"의 차이다. 1000원짜리 음식에서 200원이 비싼 것(20%)과 10000원짜리 음식에서 200원이 비싼 것(2%)은 심각도가 다르다.

## Ⅲ. 비교 및 연결

### CV vs SV 비교

| 구분 | CV (Cost Variance) | SV (Schedule Variance) |
|:---|:---|:---|
| 공식 | EV - AC | EV - PV |
| 양수(+) 의미 | 비용 절감 (Under Budget) | 일정 선행 (Ahead of Schedule) |
| 음수(-) 의미 | 비용 초과 (Over Budget) | 일정 지연 (Behind Schedule) |
| 측정 대상 | 비용 성과 | 일정 성과 |
| 단위 | 화폐 | 화폐 (EV, PV 모두 화폐) |
| 한계 | 없음 | 프로젝트 완료 시 SV=0 수렴 |

### CV 개선 전략

| 전략 | 설명 | 단점 |
|:---|:---|:---|
| 비용 절감 | 저비용 자원 활용, 도구 효율화 | 품질 저하 위험 |
| 생산성 향상 | 프로세스 개선, 자동화 | 단기 투자 필요 |
| 범위 축소 | 저우선순위 기능 제거 | 발주자 승인 필요 |
| 예산 재조정 | EAC 재산정, BAC 증액 | 발주자 신뢰 저하 |

📢 **섹션 요약 비유**: CV 개선 전략은 가계부 적자 해결 방법과 같다. 수입 늘리기(생산성 향상), 지출 줄이기(비용 절감), 사고 싶은 것 포기(범위 축소)—모두 선택지지만 각각 부작용이 있다.

## Ⅳ. 실무 적용 및 기술사 판단

### CV 임계값 관리

```text
일반적 임계값 설정 예:
  황색 경보: |CV%| > 5% (원인 파악 요구)
  적색 경보: |CV%| > 10% (즉시 조치 요구)
  
  임계값은 조직/프로젝트 성격에 따라 조정:
  - 국방/항공: ±5% (엄격)
  - 일반 IT: ±10% (표준)
  - 스타트업: ±20% (유연)
```

### 기술사 시험 계산 예

```text
문제: 다음 상황에서 CV와 CV%를 계산하시오.
  BAC=500만원, 계획 완료율=60%, 실제 완료율=50%, AC=280만원

풀이:
  PV = 500 × 60% = 300만원
  EV = 500 × 50% = 250만원
  AC = 280만원

  CV = EV - AC = 250 - 280 = -30만원 (비용 초과)
  CV% = (-30/250) × 100 = -12%

  해석:
  → 12% 비용 초과로 적색 경보 기준 초과
  → CPI = 250/280 = 0.89 (11% 비효율)
  → EAC = 500/0.89 = 561만원 (61만원 초과 예상)
  → 즉시 원인 파악 및 시정 조치 필요
```

📢 **섹션 요약 비유**: CV 임계값 관리는 혈압 관리와 같다. 130/80이면 주의 단계, 140/90이면 고혈압으로 치료가 필요하듯, CV가 특정 임계값을 넘으면 "프로젝트 고혈압"으로 즉각 처방(시정 조치)이 필요하다.

## Ⅴ. 기대효과 및 결론

CV를 체계적으로 관리하면:

1. **비용 초과 조기 감지**: 작은 CV 누적이 큰 EAC 초과로 이어지기 전에 조치한다.
2. **원가 관리 근거**: 발주자에게 비용 성과를 수치로 투명하게 보고한다.
3. **자원 배분 최적화**: CV가 나쁜 Work Package에 자원을 집중해 효율을 개선한다.
4. **계약 분쟁 예방**: CV 추이 기록이 계약 변경 협의의 객관적 근거가 된다.

CV의 핵심 한계는 "왜 초과됐는가"를 설명하지 못한다는 점이다. CV < 0이 발견되면 반드시 원인 분석—재작업, 범위 확산, 추정 오류, 자원 비효율 중 어느 것인지—을 수행해야 적절한 대응이 가능하다.

📢 **섹션 요약 비유**: CV는 체온계와 같다. 38도(CV < 0)가 나왔다는 것은 이상이 있다는 신호지만, 왜 열이 났는지(바이러스인지, 세균 감염인지)는 추가 검사가 필요하다.

| 재작업 (Rework) | CV 악화의 주요 원인 | 품질 비용 |

### 👶 어린이를 위한 3줄 비유 설명

1. CV는 소풍 간식 예산과 같아요 — "아이스크림 5개 먹을 돈(EV)인데 6개 먹었다면(AC)" CV = -1개 분의 비용 초과예요.
2. CV%는 비율이에요 — 100원 예산에서 10원 초과(10%)와 1000원 예산에서 10원 초과(1%)는 심각도가 달라요.
3. CV가 나쁠 때 "왜 그랬나"를 꼭 찾아야 해요 — 그냥 놔두면 다음에도 같은 이유로 예산이 초과될 테니까요.

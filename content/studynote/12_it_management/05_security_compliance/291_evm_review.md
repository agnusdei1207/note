## 핵심 인사이트
> 1. **본질**: EVM (Earned Value Management, 획득 가치 관리)은 프로젝트의 진척도와 예산 소진을 금액 단위로 통합 통제하는 기법으로, PV·EV·AC 세 지표로 일정과 비용 상태를 동시에 파악한다.
> 2. **가치**: "예산대로 진행 중"이라는 주관적 보고 대신, CV·SV·CPI·SPI라는 정량 지표로 프로젝트 상태를 객관화하여 조기에 문제를 탐지한다.
> 3. **판단 포인트**: CPI < 1은 예산 초과, SPI < 1은 일정 지연. EAC(완료 시 예상 비용) = BAC/CPI로 최종 예산 초과액을 조기 예측한다.

## Ⅰ. 개요 및 필요성

EVM은 PMBOK의 프로젝트 원가 관리(Cost Management)와 일정 관리(Schedule Management)를 단일 프레임워크로 통합한다. 전통적 예산 관리는 "지출액 vs 계획 예산"만 비교했지만, EVM은 "지출액 vs 실제 완료된 작업의 가치"를 비교하여 진짜 효율을 측정한다.

📢 **섹션 요약 비유**: EVM은 자동차 계기판이다. 속도계(SPI)와 연료 게이지(CPI)를 동시에 보며 남은 여정(EAC)을 예측한다.

## Ⅱ. 아키텍처 및 핵심 원리

```
┌────────────────────────────────────────────┐
│  EVM 핵심 3 지표                           │
│  PV (Planned Value)  = 계획 작업 예산      │
│  EV (Earned Value)   = 완료 작업 예산 가치 │
│  AC (Actual Cost)    = 실제 투입 비용      │
├────────────────────────────────────────────┤
│  CV = EV - AC   (비용 편차, 양수 = 절감)  │
│  SV = EV - PV   (일정 편차, 양수 = 단축)  │
│  CPI= EV / AC   (1 초과 = 비용 효율적)    │
│  SPI= EV / PV   (1 초과 = 일정 단축)      │
│  EAC= BAC / CPI (완료 시 예상 비용)       │
└────────────────────────────────────────────┘
```

| 지표 | 공식 | 해석 |
|:---|:---|:---|
| CPI | EV/AC | >1: 예산 절감, <1: 초과 |
| SPI | EV/PV | >1: 일정 단축, <1: 지연 |
| EAC | BAC/CPI | 완료 시 예상 총 비용 |

📢 **섹션 요약 비유**: CPI=0.8은 1만원 일을 하는 데 1.25만원 쓴다는 뜻이다. 이 추세면 최종 비용은 예산의 125%가 된다.

## Ⅲ. 비교 및 연결

| 전통 관리 | EVM |
|:---|:---|
| 지출액 vs 계획 예산 | 지출액 vs 완료 작업 가치 |
| 일정·비용 별도 관리 | 통합 관리 |
| 사후 보고 | 조기 경보 |

📢 **섹션 요약 비유**: EVM은 GPS 내비게이션이다. 남은 거리(EAC)와 예상 도착 시간(ECD)을 실시간으로 알려준다.

## Ⅳ. 실무 적용 및 기술사 판단

ERP 구축 프로젝트 5개월 차: PV=5억, EV=4억, AC=5.5억 → SV=-1억(지연), CV=-1.5억(초과), SPI=0.8, CPI=0.73. BAC=10억이면 EAC=10억/0.73=13.7억 예측. PM은 이를 이사회에 보고하고 추가 예산 승인과 자원 보강을 요청한다.

📢 **섹션 요약 비유**: EVM 보고는 항공기 블랙박스처럼 현재 상태를 데이터로 기록하여 이후 의사결정의 근거가 된다.

## Ⅴ. 기대효과 및 결론

EVM 적용으로 프로젝트 문제의 조기 탐지(리드 타임 1개월 단축), 예산 초과 예측 정확도 향상, 프로젝트 포트폴리오 통합 대시보드 구현이 달성된다.

📢 **섹션 요약 비유**: EVM은 혈액 검사다. 겉보기엔 멀쩡해도 수치가 이상하면 조기에 치료를 시작할 수 있다.

### 👶 어린이를 위한 3줄 비유 설명
1. EVM은 숙제를 얼마나 했고, 시간은 얼마나 썼는지 동시에 체크하는 성적표예요.
2. CPI=0.8은 숙제 80%를 했는데 시간은 100% 썼다는 뜻이에요.
3. 이 속도로 계속하면 마감까지 얼마나 걸릴지 미리 계산할 수 있어요.

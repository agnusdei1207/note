## 핵심 인사이트
> 1. **본질**: ERP(Enterprise Resource Planning)는 재무·인사·생산·구매·판매·물류 등 전사 업무 프로세스를 단일 통합 시스템으로 연결해 실시간 데이터 정합성과 운영 효율을 실현하는 전사 플랫폼이다.
> 2. **가치**: ERP의 핵심 가치는 '데이터 사일로 제거'다. 모든 부서가 동일한 데이터베이스를 공유하므로 재무 보고·재고 관리·공급망 가시성이 실시간으로 연결된다.
> 3. **판단 포인트**: ERP 도입 실패의 80%는 기술이 아닌 조직 변화 관리(Change Management) 실패에서 비롯된다. BPR(Business Process Reengineering)과 함께 추진해야 한다.

## Ⅰ. 개요 및 필요성

ERP(Enterprise Resource Planning, 전사 자원 관리)는 1990년대 Gartner가 명명한 이후 SAP, Oracle, Microsoft Dynamics 등의 솔루션으로 전 세계 기업에 확산된 핵심 경영 정보 시스템이다. ERP 이전에는 재무팀의 엑셀, 생산팀의 자체 시스템, 인사팀의 별도 DB가 공존하며 데이터 불일치와 수작업 통합이 만연했다.

ERP는 이 모든 데이터를 단일 데이터베이스와 공통 프로세스 위에서 통합한다. 판매 주문이 입력되면 재고가 자동 차감되고, 생산 계획이 업데이트되며, 매출이 자동으로 재무에 반영된다. 이 실시간 연동이 ERP의 핵심 가치다.

📢 **섹션 요약 비유**: ERP는 병원의 통합 의료 시스템이다. 접수·진단·처방·수납이 하나의 시스템에서 연결되어, 어느 창구에서든 환자의 전체 이력을 볼 수 있다.

## Ⅱ. 아키텍처 및 핵심 원리

| ERP 모듈 | 주요 기능 | 연계 데이터 |
|:---|:---|:---|
| FI/CO (재무/관리회계) | 분개·결산·원가 분석 | 구매·판매 트랜잭션 |
| MM (자재 관리) | 구매·재고·입출고 | 생산·재무 |
| SD (판매·유통) | 주문·배송·청구 | 재고·재무 |
| PP (생산 계획) | 생산 오더·MRP | BOM·자재 |
| HR/HCM (인사) | 급여·채용·평가 | 재무·조직 |
| PS (프로젝트 관리) | 프로젝트 비용·일정 | 재무·인사 |

## Ⅲ. 비교 및 연결

| 항목 | 온프레미스 ERP | 클라우드 ERP (SaaS) |
|:---|:---|:---|
| 대표 솔루션 | SAP ECC, Oracle E-Business | SAP S/4HANA Cloud, Oracle Fusion |
| 초기 비용 | 높음 (라이선스·하드웨어) | 낮음 (구독형) |
| 커스터마이징 | 높음 | 제한적 |
| 업그레이드 | 수동, 비용 큼 | 자동 (벤더 관리) |
| 적합 기업 | 대기업, 복잡한 프로세스 | 중소기업, 성장 기업 |

📢 **섹션 요약 비유**: 온프레미스 ERP는 자체 주방을 운영하는 레스토랑이고, 클라우드 ERP는 공유 주방 서비스를 이용하는 것이다. 큰 레스토랑은 자체 주방이 효율적이지만, 소규모는 공유 주방이 낫다.

## Ⅳ. 실무 적용 및 기술사 판단

ERP 구축 방법론은 SAP의 ASAP(Accelerated SAP)이나 Activate, Oracle의 AIM(Application Implementation Methodology) 등 벤더별 표준 방법론을 따른다. 핵심 성공 요인은 ① 경영진 스폰서십, ② 프로세스 표준화(BPR), ③ 데이터 마이그레이션 품질, ④ 사용자 교육·변화 관리, ⑤ 단계적 구축(Phased Rollout)이다. ERP 고도화 트렌드로는 S/4HANA 인메모리 DB, Intelligent ERP(AI·ML 내장), API 기반 Best-of-Breed 통합이 주목받는다.

📢 **섹션 요약 비유**: ERP 도입은 도시 재개발처럼, 잘못된 설계로 진행하면 더 큰 혼란이 오지만, 잘 계획하면 도시 전체가 효율화된다. 설계 단계가 구현보다 더 중요하다.

## Ⅴ. 기대효과 및 결론

성공적인 ERP 구축은 재고 비용 10~20% 감소, 재무 결산 기간 50% 단축, 업무 자동화로 운영 인력 10~15% 효율화 등의 효과를 가져온다. 디지털 전환 시대에 ERP는 AI·IoT·클라우드와 연결되는 '디지털 코어'로 진화하며, 실시간 경영 의사결정의 기반 인프라로 자리를 굳히고 있다.

📢 **섹션 요약 비유**: ERP는 기업의 중추 신경계다. 모든 부서가 동일한 신경망으로 연결되어야 기업 몸 전체가 하나로 움직인다.

### 📌 관련 개념 맵
| 개념 | 설명 | 연관 키워드 |
|:---|:---|:---|
| BPR (Business Process Reengineering) | 업무 프로세스 근본적 재설계 | ERP 도입 전 필수 |
| MRP (Material Requirements Planning) | 자재 소요 계획, ERP 전신 | PP 모듈, 생산 |
| Intelligent ERP | AI·ML 내장 차세대 ERP | SAP S/4HANA, Oracle Fusion |
| Best-of-Breed | 기능별 최적 솔루션 조합 | API 통합, EAI |
| Change Management | ERP 도입 시 조직 변화 관리 | 성공 요인 |

### 👶 어린이를 위한 3줄 비유 설명
1. ERP는 학교에서 출결·성적·급식을 각각 다른 공책에 적는 대신 하나의 컴퓨터로 통합 관리하는 것이에요.
2. 선생님이 어느 부분을 봐도 같은 학생 정보가 연결되어 있어서 혼란이 없어요.
3. 처음 도입할 때는 모두가 새 시스템에 익숙해져야 하므로 교육과 적응 기간이 중요해요.

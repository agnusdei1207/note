## 핵심 인사이트
> 1. **본질**: IT 투자 타당성 분석은 ROI(투자수익률), NPV(순현재가치), IRR(내부수익률), PP(회수기간) 4개 지표를 종합해 투자 의사결정의 객관성을 확보한다.
> 2. **가치**: 단일 지표만 사용하면 편향이 생기며, 4개 지표의 조합이 서로의 약점을 보완한다. NPV는 절대적 가치, IRR은 효율성, PP는 유동성, ROI는 비율 성과를 측정한다.
> 3. **판단 포인트**: NPV > 0이고 IRR > 자본비용이면 투자 타당성이 있다. ROI는 비율이 높아도 절대 금액이 작을 수 있어 NPV와 병행이 필수다.

## Ⅰ. 개요 및 필요성

IT 투자 타당성 분석은 디지털 전환·시스템 구축·인프라 교체 등 IT 프로젝트가 경제적으로 정당화되는지를 검증하는 재무 분석이다. 경영진이 'IT 투자가 비용인가 자산인가'를 판단할 때 객관적 수치가 없으면 감에 의존한 의사결정이 된다. 4개 지표는 각각 다른 각도에서 투자를 평가하므로 종합 사용이 권장된다.

IT 투자의 편익(Benefit)은 정량적(비용 절감, 수익 증가)과 정성적(고객 만족, 리스크 감소) 두 가지로 나뉘며, 정성적 편익을 화폐 가치로 환산하는 방법론(TCO, EVA 등)을 함께 활용한다.

📢 **섹션 요약 비유**: IT 투자 분석은 집을 살 때 가격(NPV), 대출 이자 대비 임대 수익률(IRR), 월세 회수 기간(PP), 총 투자 대비 수익(ROI)을 모두 따지는 것과 같다.

## Ⅱ. 아키텍처 및 핵심 원리

| 지표 | 정의 | 공식 | 판단 기준 |
|:---|:---|:---|:---|
| ROI (Return on Investment, 투자수익률) | 투자 대비 순이익 비율 | (순이익 / 투자비용) × 100% | ROI > 0 (양수) |
| NPV (Net Present Value, 순현재가치) | 미래 현금흐름의 현재 가치 합계 - 초기 투자 | Σ(현금흐름/(1+r)^t) - 초기투자 | NPV > 0 |
| IRR (Internal Rate of Return, 내부수익률) | NPV = 0 되는 할인율 | NPV = 0인 r | IRR > 자본비용(WACC) |
| PP (Payback Period, 회수기간) | 초기 투자를 회수하는 기간 | 초기투자 / 연간 순현금흐름 | PP < 목표 기간 |

```
┌─────────────────────────────────────────────────────────────────┐
│          IT 투자 타당성 분석 종합 평가 프레임워크               │
│                                                                 │
│  투자안 A: IT 시스템 신규 구축 (초기 투자 10억, 5년 운영)      │
│                                                                 │
│  연도  현금흐름   현재가치(r=10%)   누적 현재가치               │
│  0     -10억원    -10억원            -10억원                    │
│  1     +3억원     +2.73억원          -7.27억원                  │
│  2     +3억원     +2.48억원          -4.79억원                  │
│  3     +3억원     +2.25억원          -2.54억원                  │
│  4     +3억원     +2.05억원          -0.49억원                  │
│  5     +3억원     +1.86억원          +1.37억원 ← NPV > 0 ✓     │
│                                                                 │
│  ROI = (5억 순이익) / 10억 × 100% = 50%                        │
│  PP  = 10억 / 3억 = 3.33년                                     │
│  IRR = NPV=0이 되는 할인율 ≈ 15.2%  (자본비용 10% 초과 ✓)     │
└─────────────────────────────────────────────────────────────────┘
```

📢 **섹션 요약 비유**: 투자 분석은 요리 레시피 평가와 같다. 맛(ROI)만 봐선 안 되고, 원가(NPV), 요리 시간(PP), 마진율(IRR)을 함께 봐야 실제로 팔만한지 알 수 있다.

## Ⅲ. 비교 및 연결

| 항목 | ROI | NPV | IRR | PP |
|:---|:---|:---|:---|:---|
| 측정 대상 | 투자 효율성 (%) | 절대적 가치 창출 | 투자 효율성 (수익률) | 유동성·위험 |
| 시간 가치 반영 | 미반영 | 반영 (할인율 사용) | 반영 | 미반영 (단순형) |
| 주요 한계 | 규모 무시 | 할인율 민감도 | 복수 해 가능 | 이후 현금흐름 무시 |
| 단독 사용 적합성 | 낮음 | 중간 | 중간 | 낮음 |
| 종합 평가 역할 | 성과 비율 | 핵심 판단 기준 | 효율 비교 | 유동성 점검 |

📢 **섹션 요약 비유**: NPV가 '이 투자로 얼마 버는가'를 알려준다면, IRR은 '효율이 은행 이자보다 좋은가', PP는 '얼마나 빨리 원금을 뽑을 수 있는가'를 보여준다.

## Ⅳ. 실무 적용 및 기술사 판단

IT 투자 제안서에는 ROI, NPV, IRR, PP를 모두 제시하고, 할인율 가정·편익 산정 근거를 명시해야 경영진 설득력이 높다. 정성적 편익(보안 강화, 컴플라이언스 준수, 직원 만족)은 위험 비용 절감액이나 과태료 회피액으로 환산해 정량화한다. 클라우드 전환 프로젝트에서는 OPEX 전환 효과(자본 해방)를 NPV에 별도 반영해야 실제 가치를 정확히 나타낸다.

📢 **섹션 요약 비유**: IT 투자 분석은 공인 중개사의 매물 평가처럼, 감정(ROI 직관)만이 아닌 시세(NPV), 수익률(IRR), 투자 회수(PP)를 종합해야 실수 없는 결정을 한다.

## Ⅴ. 기대효과 및 결론

IT 투자 타당성 분석의 4대 지표를 체계적으로 활용하면 IT 프로젝트의 비즈니스 정당성을 확보하고, 한정된 IT 예산의 배분 우선순위를 객관화할 수 있다. CIO·CDO가 이사회에 IT 투자를 설득할 때 이 4개 지표는 '언어'의 역할을 한다.

📢 **섹션 요약 비유**: ROI·NPV·IRR·PP는 투자 결정의 사분면 나침반이다. 하나가 좋다고 투자하지 말고, 네 방향 모두를 확인해야 길을 잃지 않는다.

### 📌 관련 개념 맵
| 개념 | 설명 | 연관 키워드 |
|:---|:---|:---|
| TCO (Total Cost of Ownership) | 총 소유 비용 분석 | IT 투자, 숨겨진 비용 |
| WACC (Weighted Average Cost of Capital) | 가중평균자본비용, IRR 비교 기준 | IRR, 자본비용 |
| EVA (Economic Value Added) | 경제적 부가가치 | IT 성과 측정 |
| 민감도 분석 (Sensitivity Analysis) | 가정 변경 시 NPV 변화 분석 | 리스크 평가 |
| OPEX vs CAPEX | 운영비 vs 자본비 전환 효과 | 클라우드 전환, ROI |

### 👶 어린이를 위한 3줄 비유 설명
1. ROI는 용돈을 투자해서 얼마나 더 벌었는지 비율이에요.
2. NPV는 미래에 받을 돈을 지금 가치로 바꿔서 '정말 이득인지' 확인하는 방법이에요.
3. PP는 투자한 돈을 언제 다 돌려받는지 알려주는 '원금 회수 날짜'예요.

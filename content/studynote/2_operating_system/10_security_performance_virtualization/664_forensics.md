# 디지털 포렌식 (Digital Forensics)

## 핵심 인사이트 (3줄 요약)
> 1. **본질**: 디지털 포렌식은 **사이버 범죄 증거를 수집, 보존, 분석, 제시**하는 과학적 절차로, 법적 증거 능력을 가진다.
> 2. **가치**: "누가, 언제, 무엇을 했는지" 재구성하여 **기업 내부 조사, 범죄 수사, eDiscovery**에 활용된다.
> 3. **융합**:**즉시 기록(Live Forensics)**, **사후 기록(Dead Forensics)**, **메모리 포렌식** 등 다양한 기법이 존재하며, **도구 체인(CoC)**이 필수적이다.

---

## Ⅰ. 디지털 포렌식의 개요

### 1. 정의
- 디지털 포렌식은 디지털 증거를 **법적으로 인정 가능한 형태**로 수집/분석하는 학문이다.

### 2. 등장 배경
- 1980~90년대 컴퓨터 범죄 증가

### 3. 💡 비유: 'CSI 데이터 수사'
- 디지털 포렌식은 **"범죄 현장에서 지문/흔적 수집"**과 같다.
- 흔적을 훼손하지 않고, **Chain of Custody**(증거 관리)를 준수한다.

---

## Ⅱ. 포렌식 절차 (Deep Dive)

### 1. 4단계
```text
    ┌─────────────────────────────────────────────────────────────────┐
    │                 디지털 포렌식 4단계                             │
    ├─────────────────────────────────────────────────────────────────┤
    │                                                                 │
    │  [1단계] 수집 (Acquisition)                                     │
    │   - 디스크 이미징 (dd, E01, AFF)                               │
    │   - write-blocker로 쓰기 방지                                   │
    │   - 해시(Hash)로 무결성 확보                                   │
    │                                                                 │
    │  [2단계] 검사 (Examination)                                     │
    │   - 파일 시스템 분석                                           │
    │   - 삭제 파일 복구                                             │
    │   - 메타데이터 추출                                           │
    │                                                                 │
    │  [3단계] 분석 (Analysis)                                        │
    │   - 타임라인 구축                                              │
    │   - 관계성 분석                                                │
    │   - 재구성                                                    │
    │                                                                 │
    │  [4단계] 보고 (Reporting)                                      │
    │   - 법적 보고서 작성                                           │
    │   - 전문가 증언                                               │
    └─────────────────────────────────────────────────────────────────┘
```

---

## Ⅲ. 주요 도구

### 1. 이미징
```bash
# dd로 디스크 이미징
dd if=/dev/sda of=disk.img bs=4M status=progress

# 해시 계산
sha256sum disk.img
```

### 2. 분석 도구
- **Autopsy**: 오픈소스
- **EnCase**: 상용
- **FTK**: 상용

## 👶 어린이를 위한 3줄 비유 설명
1. 디지털 포렌식은 **"컴퓨터 범죄 수사대"** 같아요.
2. 컴퓨터에 남은 흔적을 모아서 누가 나쁜 짓을 했는지 찾아내죠.
3. 마치 CSI에서 지문을 찾는 것처럼, 디지털 흔적도 찾아낼 수 있답니다!

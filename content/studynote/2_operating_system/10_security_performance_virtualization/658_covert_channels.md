# 커버트 채널 (Covert Channels)

## 핵심 인사이트 (3줄 요약)
> 1. **본질**: 커버트 채널은 **시스템의 설계 의도가 아닌 경로를 통해 정보를 전송**하는 통신 경로로, **격리된 프로세스/VM 간 데이터 유출**에 악용된다.
> 2. **가치**: "완전히 격리된 시스템은 없다"는 것을 보여주며, **CPU 캐시, shared resources**를 활용한 **cross-VM 공격**이 가능하다.
> 3. **융합**:** Storage Channel**과 **Timing Channel**이 있으며, **채널 용량 저감, 잡음 추가**로 완화한다.

---

## Ⅰ. 커버트 채널의 개요

### 1. 정의
- 커버트 채널은 보안 정책을 우회하여 정보를 전송하는 비밀 경로이다.

### 2. 등장 배경
- 1970년대 Lampson: **Confined 프로그램의 정보 유출** 문제 제기

### 3. 💡 비유: '감옥의 탈옥 메시지'
- 커버트 채널은 **"감옥의 수감자가 벽에 노크하여 메시지 전달"**과 같다.
- 경비원이 메시지를 전달하지 않아도, 수감자 간 소통 가능.

---

## Ⅱ. 커버트 채널 유형 (Deep Dive)

### 1. Storage Channel
```text
    ┌─────────────────────────────────────────────────────────────────┐
    │                  Storage Channel 예시                            │
    ├─────────────────────────────────────────────────────────────────┤
    │                                                                 │
    │   [High Process]          [Low Process]                         │
    │      │                        │                               │
    │      │ 파일 생성 "file_1"      │                               │
    │      ▼                        │                               │
    │   /tmp/file_1                │                               │
    │      │                        │                               │
    │      │                        │ 파일 존재 확인?               │
    │      │                        ▼                               │
    │      │                    if exists: 1                         │
    │      │                    else: 0                            │
    │                                                                 │
    │  * 공유 자원(파일 시스템)을 통해 정보 전달                    │
    └─────────────────────────────────────────────────────────────────┘
```

### 2. Timing Channel
- 리소스 경쟁 시간으로 정보 전달

### 3. 캐시 기반
- **Prime+Probe**: 공유 캐시로 정보 전달

## 👶 어린이를 위한 3줄 비유 설명
1. 커버트 채널은 **"벽을 두드려서 메시지 전달"** 같아요.
2. 아무리 벽을 막아도 소리(부수 정보)는 넘어갈 수 있죠.
3. 컴퓨터도 완전히 차단해도 작은 틈새(캐시, 시간)으로 정보가 새어 나갈 수 있어요!

# TPM 2.0 (Trusted Platform Module 2.0)

## 핵심 인사이트 (3줄 요약)
> 1. **본질**: TPM 2.0은 **TPM 1.2의 단점을 개선**한 차세대 하드웨어 보안 칩으로, **ECC, SHA-256, 다양한 알고리즘**을 지원하고 **더 유연한 프로그래밍**을 제공한다.
> 2. **가치**: TPM 1.2는 RSA/SHA-1로 제한되었으나, TPM 2.0는 **更强的 암호화, 향상된 성능**을 제공하며, **Windows 11**의 최소 요구사항이다.
> 3. **융합**:**FIPS 140-2** 인증, **TCG(T Trusted Computing Group)** 표준으로, Intel PTT, AMD fTPM으로 CPU에 내장된다.

---

## Ⅰ. TPM 2.0의 개요

### 1. 정의
- TPM 2.0은 TCG가 개발한 차세대 TPM 표준으로, **향상된 암호화와 유연성**을 제공한다.

### 2. 등장 배경
- TPM 1.2의 한계: RSA만, 구조 경직

### 3. 💡 비유: '진화한 금고'
- TPM 2.0은 **"여러 잠금장치가 있는 스마트 금고"**와 같다.
- 더 강력한 잠금과 더 많은 기능을 제공한다.

---

## Ⅱ. TPM 2.0 개선 사항 (Deep Dive)

### 1. 암호화 강화
```text
    ┌─────────────────────────────────────────────────────────────────┐
    │               TPM 1.2 vs TPM 2.0 비교                          │
    ├─────────────────────────────────────────────────────────────────┤
    │                                                                 │
    │  [특징]        TPM 1.2              TPM 2.0                     │
    │  ───────────────────────────────────────────────────────────   │
    │   알고리즘      RSA-2048            RSA/ECC                   │
    │   해시          SHA-1               SHA-256                    │
    │   구조          고정적               모듈형/유연                │
    │   성능          느림                 빠름                       │
    │   표준          독자적              TCG 국제                   │
    │                                                                 │
    │  * ECC: 작은 키로 같은 보안, 더 빠름                          │
    └─────────────────────────────────────────────────────────────────┘
```

### 2. 계층형 인증
- **Endorsement Key**: 제조사 키
- **Attestation Identity Key**: 신원 증명
- **Storage Root Key**: 데이터 보호

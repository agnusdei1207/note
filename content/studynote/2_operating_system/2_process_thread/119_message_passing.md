# 119. 메시지 전달 (Message Passing)

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 커널을 통해 메시지를 교환하는 IPC 방식
> 2. **가치**: 안전한 통신, 동기화 자동 처리
> 3. **융합**: 직접/간접 통신, 블로킹/비블로킹과 연관

---

## Ⅰ. 개요

### 개념 정의

메시지 전달(Message Passing)은 **커널을 중개로 메시지를 교환하는 IPC 방식**이다. 공유 메모리보다 느리지만, 동기화가 자동으로 처리되어 안전하다.

### 💡 비유: 우편
메시지 전달은 **우편**과 같다. 보내는 사람이 우체국(커널)에 편지(메시지)를 맡기면, 우체국이 받는 사람에게 전달한다.

## Ⅱ. 상세 분석

### 상세 분석
```
┌─────────────────────────────────────────────────────────────────────┐
│                메시지 전달 상세                                       │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【직접 통신 vs 간접 통신】                                             │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │  구분              직접 통신             간접 통신              │ │
│  │  ────                    ────                  ────              │ │
│  │  수신자 지정        명시                   메일박스/포트         │ │
│  │  링크              자동 생성               메일박스 통해         │ │
│  │  예시              send(P, msg)            send(mailbox, msg)   │ │
│  │  구현              소켓                    메시지 큐             │ │
│  │                                                               │ │
│  └───────────────────────────────────────────────────────────────┘ │
│                                                                     │
│  【블로킹 vs 비블로킹】                                                  │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │  유형              Send                Receive                  │ │
│  │  ────                    ────                  ────               │ │
│  │  블로킹            메시지 전송 대기      메시지 수신 대기         │ │
│  │  비블로킹          즉시 반환            즉시 반환 (없으면 오류)   │ │
│  │                                                               │ │
│  │  블로킹: 동기식 통신 (rendezvous)                              │ │
│  │  비블로킹: 비동기식 통신                                       │ │
│  │                                                               │ │
│  └───────────────────────────────────────────────────────────────┘ │
│                                                                     │
│  【버퍼링】                                                             │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │  유형              설명                                       │ │
│  │  ────                    ────                                    │ │
│  │  Zero capacity     버퍼 없음, 블로킹 필수                       │ │
│  │  Bounded capacity  제한된 버퍼, 꽉 차면 블로킹                   │ │
│  │  Unbounded         무제한 버퍼, send는 블로킹 없음               │ │
│  │                                                               │ │
│  └───────────────────────────────────────────────────────────────┘ │
└─────────────────────────────────────────────────────────────────────┘
```

## Ⅲ. 실무 적용

### 코드 예시
```
┌─────────────────────────────────────────────────────────────────────┐
│                실무 적용                                            │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【System V 메시지 큐】                                                │
│  ──────────────────                                                  │
│  #include <sys/msg.h>                                               │
│                                                                     │
│  struct msgbuf {                                                    │
│      long mtype;                                                    │
│      char mtext[100];                                               │
│  };                                                                 │
│                                                                     │
│  key_t key = ftok("/tmp", 'A');                                     │
│  int msqid = msgget(key, IPC_CREAT | 0666);                        │
│                                                                     │
│  // 송신                                                             │
│  struct msgbuf msg = {1, "Hello"};                                  │
│  msgsnd(msqid, &msg, sizeof(msg.mtext), 0);                        │
│                                                                     │
│  // 수신                                                             │
│  msgrcv(msqid, &msg, sizeof(msg.mtext), 1, 0);                     │
│                                                                     │
│  【POSIX 메시지 큐】                                                   │
│  ──────────────────                                                  │
│  #include <mqueue.h>                                                │
│                                                                     │
│  mqd_t mq = mq_open("/myqueue", O_CREAT | O_RDWR, 0666, NULL);     │
│  mq_send(mq, "Hello", 5, 0);                                        │
│  mq_receive(mq, buffer, 100, NULL);                                 │
│  mq_close(mq);                                                      │
│  mq_unlink("/myqueue");                                             │
│                                                                     │
│  【메시지 큐 확인】                                                     │
│  ──────────────────                                                  │
│  $ ipcs -q                         # 메시지 큐 목록                  │
│  $ ipcrm -q <msqid>                # 제거                           │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

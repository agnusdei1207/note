# 089. 대기 큐 (Wait Queue)

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: I/O 완료나 이벤트를 기다리는 프로세스들이 모인 큐
> 2. **가치**: 이벤트 대기, 비동기 처리, 자원 공유
> 3. **융합**: 인터럽트, 이벤트, 세마포어와 연관

---

## Ⅰ. 개요

### 개념 정의

대기 큐(Wait Queue)는 **I/O 완료, 이벤트 발생, 자원 획득 등을 기다리는 프로세스들이 모인 큐**다. Waiting 상태 프로세스가 여기서 대기한다.

### 💡 비유: 병원 대기실
대기 큐는 **병원 대기실**과 같다. 검사 결과(I/O 완료)를 기다리는 환자(프로세스)들이 대기실에서 기다린다. 결과가 나오면(인터럽트) 호출된다.

## Ⅱ. 상세 분석

### 상세 분석
```
┌─────────────────────────────────────────────────────────────────────┐
│                대기 큐 상세                                          │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │

│  │  함수              설명                                        │ │
│  │  ────                ────                                        │ │
│  │  init_waitqueue_head()  대기 큐 초기화                         │ │
│  │  wait_event(q, cond)     조건까지 대기 (인터럽트 불가)         │ │
│  │  wait_event_interruptible()  조건까지 대기 (인터럽트 가능)      │ │
│  │  wake_up(q)             큐의 모든 프로세스 깨움                 │ │
│  │  wake_up_interruptible() 인터럽트 가능 프로세스만 깨움         │ │
│  │  sleep_on(q)            무조건 대기                            │ │
│  │  interruptible_sleep_on() 인터럽트 가능 대기                   │ │
│  │                                                               │ │
│  └───────────────────────────────────────────────────────────────┘ │
│                                                                     │
│  【대기 이유】                                                        │
│  ──────────────────                                                  │

│  │  이유              대기 큐                                      │ │
│  │  ────                ────                                        │ │
│  │  디스크 I/O        Block device wait queue                      │ │
│  │  네트워크 I/O      Socket wait queue                             │ │
│  │  자원 대기        Semaphore wait queue                          │ │
│  │  타이머           Timer wait queue                              │ │
│  │  사용자 입력       Terminal wait queue                          │ │
│  │                                                               │ │

```

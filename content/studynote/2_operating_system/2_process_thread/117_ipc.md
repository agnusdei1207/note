# 117. IPC (프로세스 간 통신, Inter-Process Communication)

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 프로세스 간 데이터 교환 및 동기화 메커니즘
> 2. **가치**: 협력적 프로세스 구현, 시스템 통합
> 3. **융합**: 공유 메모리, 메시지 전달, 파이프와 연관

---

## Ⅰ. 개요

### 개념 정의

IPC(Inter-Process Communication)는 **서로 다른 프로세스가 데이터를 교환하고 동기화하기 위한 메커니즘**이다. 협력적 프로세스 간 통신의 핵심 기술이다.

### 💡 비유: 우편/전화
IPC는 **사람들 간의 소통 수단**과 같다. 편지(메시지), 전화(소켓), 공유 게시판(공유 메모리) 등 다양한 방법이 있다.

## Ⅲ. 실무 적용

### 명령어 예시
```
┌─────────────────────────────────────────────────────────────────────┐
│                실무 적용                                            │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【IPC 리소스 확인】                                                   │
│  ──────────────────                                                  │
│  # System V IPC 확인                                                  │
│  $ ipcs -a                     # 모든 IPC                           │
│  $ ipcs -m                     # 공유 메모리                         │
│  $ ipcs -s                     # 세마포어                            │
│  $ ipcs -q                     # 메시지 큐                           │
│                                                                     │
│  # IPC 제거                                                          │
│  $ ipcrm -m <shmid>            # 공유 메모리 제거                    │
│  $ ipcrm -s <semid>            # 세마포어 제거                       │
│  $ ipcrm -q <msqid>            # 메시지 큐 제거                      │
│                                                                     │
│  【파이프 사용】                                                        │
│  ──────────────────                                                  │
│  # 익명 파이프                                                         │
│  $ ls | grep "txt"                                                   │
│                                                                     │
│  # 지명 파이프 (FIFO)                                                  │
│  $ mkfifo /tmp/mypipe                                                │
│  $ echo "hello" > /tmp/mypipe &                                      │
│  $ cat /tmp/mypipe                                                   │
│                                                                     │
│  【유닉스 도메인 소켓】                                                 │
│  ──────────────────                                                  │
│  $ nc -U /tmp/socket                                                 │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

# 092. 스레드 (Thread)

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 프로세스 내의 경량 실행 단위
> 2. **가치**: 자원 공유, 빠른 생성/전환, 병렬성
> 3. **융합**: 멀티스레딩, 동시성, LWP와 연관

---

## Ⅰ. 개요

### 개념 정의

스레드(Thread)는 **프로세스 내에서 실행되는 경량 단위(Lightweight Process)로, 같은 프로세스의 다른 스레드와 주소 공간을 공유**한다. 독립적인 실행 흐름을 가진다.

### 💡 비유: 한 집의 여러 방
스레드는 **한 집의 여러 방**과 같다. 집(프로세스)은 하나지만, 여러 방(스레드)에서 각자 활동한다. 거실, 주방(공유 자원)은 함께 쓴다.

## Ⅲ. 실무 적용

### 명령어 예시
```
┌─────────────────────────────────────────────────────────────────────┐
│                실무 적용                                            │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【스레드 확인】                                                      │
│  ──────────────────                                                  │
│  # 프로세스의 스레드 수                                               │
│  $ ps -o pid,nlwp,cmd -p <pid>                                     │
│  $ cat /proc/<pid>/status | grep Threads                          │
│                                                                     │
│  # 스레드 목록                                                        │
│  $ ps -T -p <pid>                                                   │
│  $ ls /proc/<pid>/task                                             │
│                                                                     │
│  # top에서 스레드 보기                                                │
│  $ top -H                                                          │
│  $ htop -t                                                          │
│                                                                     │
│  【pthread API】                                                      │
│  ──────────────────                                                  │
│  # 스레드 생성                                                        │
│  pthread_create(&tid, NULL, func, arg);                            │
│                                                                     │
│  # 스레드 종료 대기                                                   │
│  pthread_join(tid, NULL);                                          │
│                                                                     │
│  # 스레드 종료                                                        │
│  pthread_exit(retval);                                             │
│                                                                     │
│  # 스레드 ID                                                         │
│  pthread_self();                                                    │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

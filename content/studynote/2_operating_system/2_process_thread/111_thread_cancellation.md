# 111. 스레드 취소 (Thread Cancellation)

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 실행 중인 스레드를 종료시키는 메커니즘
> 2. **가치**: 작업 중단, 타임아웃, 리소스 정리
> 3. **융합**: 비동기/지연 취소, 취소점, 정리 핸들러와 연관

---

## Ⅰ. 개요

### 개념 정의

스레드 취소(Thread Cancellation)는 **한 스레드가 다른 스레드의 실행을 종료시키는 메커니즘**이다. 작업 중단, 타임아웃, 사용자 요청 등의 상황에서 활용된다.

### 💡 비유: 작업 중지
스레드 취소는 **관리자가 직원에게 작업 중지 명령**과 같다. 즉시 멈추거나(비동기식), 작업 단계에서 멈춘다(지연식).

## Ⅲ. 실무 적용

### 코드 예시
```
┌─────────────────────────────────────────────────────────────────────┐
│                실무 적용                                            │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【지연식 취소 예시】                                                   │
│  ──────────────────                                                  │
│  void* thread_func(void* arg) {                                     │
│      // 정리 핸들러 등록                                              │
│      pthread_cleanup_push(cleanup_handler, NULL);                  │
│                                                                     │
│      // 지연식 취소 설정                                              │
│      pthread_setcanceltype(PTHREAD_CANCEL_DEFERRED, NULL);         │
│      pthread_setcancelstate(PTHREAD_CANCEL_ENABLE, NULL);          │
│                                                                     │
│      while (running) {                                              │
│          do_work();                                                 │
│          pthread_testcancel();  // 취소점                           │
│      }                                                              │
│                                                                     │
│      pthread_cleanup_pop(0);  // 정리 핸들러 제거                    │
│      return NULL;                                                   │
│  }                                                                  │
│                                                                     │
│  【정리 핸들러】                                                        │
│  ──────────────────                                                  │
│  void cleanup_handler(void* arg) {                                  │
│      // 자원 해제                                                    │
│      free(buffer);                                                  │
│      close(fd);                                                     │
│      unlock_mutex();                                                │
│  }                                                                  │
│                                                                     │
│  【취소 요청】                                                          │
│  ──────────────────                                                  │
│  pthread_t tid;                                                     │
│  pthread_create(&tid, NULL, thread_func, NULL);                    │
│                                                                     │
│  // 취소 요청                                                        │
│  pthread_cancel(tid);                                               │
│                                                                     │
│  // 종료 대기                                                        │
│  void* result;                                                      │
│  pthread_join(tid, &result);                                        │
│  if (result == PTHREAD_CANCELED) {                                  │
│      printf("스레드가 취소됨\n");                                      │
│  }                                                                  │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

# 091. PCB 요소

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: PCB를 구성하는 개별 정보 필드들
> 2. **가치**: 프로세스 식별, 상태 추적, 자원 관리
> 3. **융합**: PID, 상태, 레지스터, 메모리, I/O와 연관

---

## Ⅰ. 개요

### 개념 정의

PCB 요소는 **PCB(Process Control Block)를 구성하는 개별 정보 필드들**로, PID, 상태, 프로그램 카운터, 레지스터, 스케줄링 정보, 메모리 정보, 회계 정보, I/O 상태 정보 등이 포함된다.

### 💡 비유: 신분증 정보
PCB 요소는 **신분증의 각 항목**과 같다. 이름, 생년월일, 주소, 사진(각 요소)이 모여 하나의 신분증(PCB)이 된다.

### PCB 요소 구조

```
┌─────────────────────────────────────────────────────────────────────┐
│                PCB 요소 구조                                         │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【PCB 요소 분류】                                                    │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │                                                               │ │
│  │  1. 식별자 (Identification)                                    │ │
│  │  ┌─────────────────────────────────────────────────────────┐ │ │
│  │  │  • PID (Process ID): 고유 프로세스 식별자                  │ │ │
│  │  │  • PPID (Parent PID): 부모 프로세스 ID                     │ │ │
│  │  │  • UID/GID: 사용자/그룹 ID                                 │ │ │
│  │  │  • SID: 세션 ID                                            │ │ │
│  │  └─────────────────────────────────────────────────────────┘ │ │
│  │                                                               │ │
│  │  2. 프로세스 상태 (Process State)                              │ │
│  │  ┌─────────────────────────────────────────────────────────┐ │ │
│  │  │  • State: New, Ready, Running, Waiting, Terminated       │ │ │
│  │  │  • Priority: 실행 우선순위                                │ │ │
│  │  │  • Exit Code: 종료 코드                                   │ │ │
│  │  └─────────────────────────────────────────────────────────┘ │ │
│  │                                                               │ │
│  │  3. CPU 상태 (CPU Context)                                    │ │
│  │  ┌─────────────────────────────────────────────────────────┐ │ │
│  │  │  • PC (Program Counter): 다음 명령어 주소                  │ │ │
│  │  │  • SP (Stack Pointer): 스택 포인터                        │ │ │
│  │  │  • PSW (Program Status Word): 상태 플래그                 │ │ │
│  │  │  • General Registers: 범용 레지스터                       │ │ │
│  │  └─────────────────────────────────────────────────────────┘ │ │
│  │                                                               │ │
│  │  4. 스케줄링 정보 (Scheduling Info)                            │ │
│  │  ┌─────────────────────────────────────────────────────────┐ │ │
│  │  │  • Priority: 정적/동적 우선순위                            │ │ │
│  │  │  • nice value: 사용자 조정 우선순위                        │ │ │
│  │  │  • vruntime: 가상 실행 시간 (CFS)                         │ │ │
│  │  │  • Time slice: 남은 타임 슬라이스                          │ │ │
│  │  └─────────────────────────────────────────────────────────┘ │ │
│  │                                                               │ │
│  │  5. 메모리 관리 정보 (Memory Management)                       │ │
│  │  ┌─────────────────────────────────────────────────────────┐ │ │
│  │  │  • Page Table Base: 페이지 테이블 기준 주소                 │ │ │
│  │  │  • Memory Limits: 메모리 한계                              │ │ │
│  │  │  • Stack Pointer: 스택 포인터                              │ │ │
│  │  │  • Heap Pointer: 힙 포인터                                 │ │ │
│  │  └─────────────────────────────────────────────────────────┘ │ │
│  │                                                               │ │
│  │  6. I/O 상태 정보 (I/O Status)                                │ │
│  │  ┌─────────────────────────────────────────────────────────┐ │ │
│  │  │  • Open File Table: 열린 파일 목록                        │ │ │
│  │  │  • I/O Request List: 대기 중인 I/O 요청                    │ │ │
│  │  │  • Device List: 할당된 장치                                │ │ │
│  │  └─────────────────────────────────────────────────────────┘ │ │
│  │                                                               │ │
│  │  7. 회계 정보 (Accounting)                                     │ │
│  │  ┌─────────────────────────────────────────────────────────┐ │ │
│  │  │  • CPU Time: 사용한 CPU 시간                               │ │ │
│  │  │  • Start Time: 시작 시간                                   │ │ │
│  │  │  • Job Number: 작업 번호                                   │ │ │
│  │  │  • Memory Used: 사용 메모리                                │ │ │
│  │  └─────────────────────────────────────────────────────────┘ │ │
│  │                                                               │ │
│  └───────────────────────────────────────────────────────────────┘ │
└─────────────────────────────────────────────────────────────────────┘
```

## Ⅲ. 실무 적용

### 명령어 예시
```
┌─────────────────────────────────────────────────────────────────────┐
│                실무 적용                                            │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【각 요소 확인】                                                     │
│  ──────────────────                                                  │
│  # 식별자                                                            │
│  $ cat /proc/<pid>/status | grep -E "Pid|PPid|Uid|Gid"            │
│                                                                     │
│  # 상태                                                              │
│  $ cat /proc/<pid>/status | grep -E "State"                        │
│  $ ps -p <pid> -o pid,state,cmd                                    │
│                                                                     │
│  # 스케줄링 정보                                                      │
│  $ cat /proc/<pid>/sched                                           │
│  $ chrt -p <pid>                                                    │
│                                                                     │
│  # 메모리 정보                                                        │
│  $ cat /proc/<pid>/statm                                           │
│  $ cat /proc/<pid>/status | grep -E "VmSize|VmRSS"                │
│                                                                     │
│  # 시간 정보                                                          │
│  $ cat /proc/<pid>/stat | awk '{print $14,$15}'                    │
│  $ ps -p <pid> -o pid,etime,cputime                                │
│                                                                     │
│  # 파일 정보                                                          │
│  $ ls /proc/<pid>/fd | wc -l                                        │
│  $ lsof -p <pid>                                                    │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

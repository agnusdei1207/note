# 083. BSS (Block Started by Symbol) 영역

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 초기화되지 않은 전역/정적 변수가 저장되는 메모리 영역
> 2. **가치**: 실행 파일 크기 감소, 0으로 자동 초기화
> 3. **융합**: Data 영역, ELF, 심볼 테이블과 연관

---

## Ⅰ. 개요

### 개념 정의

BSS(Block Started by Symbol) 영역은 **초기화되지 않은 전역 변수와 정적 변수가 저장되는 메모리 영역**이다. 프로그램 로드 시 0으로 자동 초기화된다.

### 💡 비유: 빈 창고
BSS는 **빈 창고**와 같다. 물건(변수)을 넣을 공간은 확보해두지만, 아직 아무것도 들어있지 않다(0). 나중에 물건을 채운다.

## Ⅲ. 기대효과 및 결론

### 핵심 요약

```
• BSS: 초기화되지 않은 전역/정적 변수 영역
• 특징: 실행 파일에 크기만 저장, 값은 저장 안 함
• 초기화: 프로그램 로드 시 0으로 자동 초기화
• 장점: 실행 파일 크기 감소, 로드 속도 향상
• 위치: Data 영역과 Heap 사이
```

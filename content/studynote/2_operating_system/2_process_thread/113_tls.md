# 113. 스레드 로컬 저장소 (TLS, Thread-Local Storage)

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 각 스레드마다 독립적인 변수 저장 공간
> 2. **가치**: 스레드 안전, 동기화 없는 독립 상태
> 3. **융합**: 스레드 독립 자원, errno, 재진입과 연관

---

## Ⅰ. 개요

### 개념 정의

스레드 로컬 저장소(Thread-Local Storage, TLS)는 **각 스레드가 자신만의 독립적인 변수 복사본을 갖는 저장 공간**이다. 전역 변수처럼 보이지만, 각 스레드마다 별도의 인스턴스가 존재한다.

### 💡 비유: 사물함
TLS는 **각 학생의 사물함**과 같다. 모두 같은 모양(변수)이지만, 각자(스레드)의 물건(값)이 따로 들어있다.

## Ⅲ. 실무 적용

### 코드 예시
```
┌─────────────────────────────────────────────────────────────────────┐
│                실무 적용                                            │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【C/C++ TLS 사용】                                                    │
│  ──────────────────                                                  │
│  // GCC 확장                                                         │
│  __thread int thread_counter = 0;                                   │
│                                                                     │
│  // C11 표준                                                         │
│  _Thread_local int thread_counter = 0;                              │
│                                                                     │
│  // C++11 표준                                                       │
│  thread_local int thread_counter = 0;                               │
│  thread_local std::string thread_name;                              │
│                                                                     │
│  void* thread_func(void* arg) {                                     │
│      thread_counter++;  // 각 스레드마다 독립                        │
│      printf("Counter: %d\n", thread_counter);                       │
│      return NULL;                                                   │
│  }                                                                  │
│                                                                     │
│  【Java ThreadLocal】                                                 │
│  ──────────────────                                                  │
│  ThreadLocal<Integer> threadId = new ThreadLocal<>() {              │
│      @Override protected Integer initialValue() {                   │
│          return Thread.currentThread().getId();                     │
│      }                                                              │
│  };                                                                 │
│                                                                     │
│  int id = threadId.get();  // 현재 스레드의 값                       │
│  threadId.set(100);       // 현재 스레드에 설정                      │
│  threadId.remove();       // 제거                                   │
│                                                                     │
│  【Python threading.local()】                                         │
│  ──────────────────                                                  │
│  import threading                                                   │
│                                                                     │
│  local_data = threading.local()                                     │
│  local_data.counter = 0  # 각 스레드마다 독립                        │
│                                                                     │
│  【pthread API】                                                       │
│  ──────────────────                                                  │
│  pthread_key_t key;                                                 │
│  pthread_key_create(&key, destructor);                              │
│  pthread_setspecific(key, value);                                   │
│  void* value = pthread_getspecific(key);                            │
│  pthread_key_delete(key);                                           │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

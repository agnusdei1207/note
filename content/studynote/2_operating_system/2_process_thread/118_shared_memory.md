# 118. 공유 메모리 (Shared Memory)

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 여러 프로세스가 동시에 접근 가능한 메모리 영역
> 2. **가치**: 가장 빠른 IPC, 대량 데이터 전송
> 3. **융합**: 동기화, 세마포어, 메모리 맵과 연관

---

## Ⅰ. 개요

### 개념 정의

공유 메모리(Shared Memory)는 **여러 프로세스가 동시에 접근할 수 있는 메모리 영역**이다. IPC 중 가장 빠른 성능을 제공하지만, 동기화가 필요하다.

### 💡 비유: 공유 게시판
공유 메모리는 **공유 게시판**과 같다. 여러 사람이(프로세스) 같은 게시판에(메모리) 글을 쓰고 읽을 수 있다.

## Ⅲ. 실무 적용

### 코드 예시
```
┌─────────────────────────────────────────────────────────────────────┐
│                실무 적용                                            │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【System V 공유 메모리】                                              │
│  ──────────────────                                                  │
│  #include <sys/shm.h>                                               │
│                                                                     │
│  key_t key = ftok("/tmp", 'A');                                     │
│  int shmid = shmget(key, 1024, IPC_CREAT | 0666);                  │
│  char* data = shmat(shmid, NULL, 0);                               │
│                                                                     │
│  // 데이터 쓰기                                                       │
│  strcpy(data, "Hello Shared Memory");                              │
│                                                                     │
│  // 분리 및 삭제                                                      │
│  shmdt(data);                                                       │
│  shmctl(shmid, IPC_RMID, NULL);                                    │
│                                                                     │
│  【POSIX 공유 메모리】                                                 │
│  ──────────────────                                                  │
│  #include <sys/mman.h>                                              │
│                                                                     │
│  int fd = shm_open("/myshm", O_CREAT | O_RDWR, 0666);              │
│  ftruncate(fd, 1024);                                               │
│  char* data = mmap(NULL, 1024, PROT_READ | PROT_WRITE,             │
│                    MAP_SHARED, fd, 0);                              │
│                                                                     │
│  // 사용 후 정리                                                      │
│  munmap(data, 1024);                                                │
│  close(fd);                                                         │
│  shm_unlink("/myshm");                                              │
│                                                                     │
│  【공유 메모리 확인】                                                   │
│  ──────────────────                                                  │
│  $ ipcs -m                         # 공유 메모리 목록                │
│  $ ipcs -m -i <shmid>              # 상세 정보                       │
│  $ ipcrm -m <shmid>                # 제거                           │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

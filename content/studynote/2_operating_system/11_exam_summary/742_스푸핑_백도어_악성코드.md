# 스푸핑 / 백도어 / 악성코드

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 스푸핑(신분 위장), 백도어(비인가 접근 경로), 악성코드(악의적 소프트웨어)
> 2. **가치**: 보안 위협 이해, 탐지 및 방어 체계 구축
> 3. **융합**: 네트워크 보안, 시스템 보안, 침입 탐지와 통합

---

### Ⅰ. 개요 (Context & Background)

**개념 정의**

스푸핑(Spoofing)은 신분을 위장하는 공격이고, 백도어(Backdoor)는 비인가 접근 경로이며, 악성코드(Malware)는 악의적 목적의 소프트웨어입니다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    보안 위협 분류                                    │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              스푸핑 (Spoofing)                                 │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │  IP Spoofing: IP 주소 위장                          │   │ │
│   │   │  MAC Spoofing: MAC 주소 위장                        │   │ │
│   │   │  DNS Spoofing: DNS 응답 위조                        │   │ │
│   │   │  Email Spoofing: 발신자 위장                        │   │ │
│   │   │                                                      │   │ │
│   │   │  예: 공격자가 신뢰된 IP로 위장하여 접속              │   │ │
│   │   │      ┌──────┐      ┌──────┐      ┌──────┐          │   │ │
│   │   │      │공격자│ ───→ │서버  │      │      │          │   │ │
│   │   │      │(위장)│      │(속음)│      │      │          │   │ │
│   │   │      └──────┘      └──────┘      │      │          │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              백도어 (Backdoor)                                │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │  하드웨어 백도어: 제조사 유지보수 포트              │   │ │
│   │   │  소프트웨어 백도어: 개발자 디버깅 계정              │   │ │
│   │   │  악성 백도어: 공격자가 설치한 숨겨진 경로           │   │ │
│   │   │                                                      │   │ │
│   │   │  예: 관리자 비밀번호 우회 계정                       │   │ │
│   │   │      admin / real_admin (숨겨진 계정)               │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
│   ┌──────────────────────────────────────────────────────────────┐ │
│   │              악성코드 (Malware)                               │ │
│   │                                                              │ │
│   │   ┌─────────────────────────────────────────────────────┐   │ │
│   │   │  바이러스: 다른 파일에 감염                          │   │ │
│   │   │  웜: 네트워크 통해 스스로 전파                       │   │ │
│   │   │  트로이목마: 정상 프로그램으로 위장                  │   │ │
│   │   │  랜섬웨어: 데이터 암호 후 금전 요구                  │   │ │
│   │   │  스파이웨어: 사용자 정보 수집                        │   │ │
│   │   │  루트킷: 시스템 깊이 숨어 탐지 회피                  │   │ │
│   │   └─────────────────────────────────────────────────────┘   │ │
│   │                                                              │ │
│   └──────────────────────────────────────────────────────────────┘ │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

> **해설**: 보안 위협은 다양한 형태로 존재합니다.

**💡 비유**: 스푸핑은 가짜 신분증, 백도어는 비밀 통로, 악성코드는 전염병과 같습니다.

**📢 섹션 요약 비유**: 위장, 비밀길, 바이러스!

### Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

**구성 요소 상세 분석**

| 요소명 | 역할 | 내부 동작 | 비유 |
|:---|:---|:---|:---|
| **IP Spoofing** | 주소 위장 | 패킷 헤더 조작 | 가짜 ID |
| **Backdoor** | 우회 경로 | 숨겨진 접근 | 비밀문 |
| **Rootkit** | 은닉 | 커널 수준 숨김 | 투명 망토 |
| **Ransomware** | 데이터 인질 | 암호화 | 강도 |

**악성코드 분류**

| 유형 | 전파 방식 | 피해 |
|:---|:---|:---|
| **Virus** | 파일 감염 | 데이터 파괴 |
| **Worm** | 네트워크 | 시스템 마비 |
| **Trojan** | 위장 설치 | 정보 탈취 |
| **Ransomware** | 이메일 등 | 금전 피해 |

**핵심 알고리즘**: 침입 → 은닉 → 확산 → 피해

**📢 섹션 요약 비유**: 랜섬웨어가 돈을 내라고 해요!

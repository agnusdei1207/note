# 494. 파일 시스템 계층 구조 (File System Hierarchy)

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 논리적 디렉토리 구조 표준
> 2. **가치**: 일관성, 호환성, 관리 용이성
> 3. **융합**: 마운트, 경로, 권한과 연관

---

## Ⅰ. 개요

### 개념 정의

파일 시스템 계층 구조(File System Hierarchy)는 **운영체제에서 파일과 디렉토리를 체계적으로 조직화하는 표준 구조**이다.

### 💡 비유: 도서관 분류 체계
파일 시스템 계층 구조는 **도서관 분류 체계**와 같다. 책들을 주제별로 정해진 위치에 배치한다.

### 파일 시스템 계층 구조

```
┌─────────────────────────────────────────────────────────────────────┐
│                파일 시스템 계층 구조 (FHS)                             │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【리눅스 디렉토리 구조】                                               │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │                                                             │ │
│  │                          / (root)                                       │
│  │                           │                                             │
│  │     ┌─────────┬─────────┼─────────┬─────────┬─────────┐            │ │
│  │     │         │         │         │         │         │              │ │
│  │   /bin      /etc      /home     /usr      /var      /tmp            │ │
│  │     │         │         │         │         │         │              │ │
│  │  기본     설정     사용자    프로그램   가변데이터  임시              │ │
│  │  명령     파일       홈                     파일                       │ │
│  │                                                             │ │
│  │  기타 주요 디렉토리:                                                     │ │
│  │  ┌────────────────────────────────────────────────────────┐           │ │
│  │  │  /boot   부팅 파일                                      │           │ │
│  │  │  /dev    장치 파일                                      │           │ │
│  │  │  /lib    공유 라이브러리                                 │           │ │
│  │  │  /mnt    마운트 포인트                                   │           │ │
│  │  │  /opt    추가 소프트웨어                                 │           │ │
│  │  │  /proc   프로세스 정보                                   │           │ │
│  │  │  /root   root 홈                                        │           │ │
│  │  │  /sbin   시스템 명령                                     │           │ │
│  │  │  /sys    시스템 정보                                     │           │ │
│  │  │  /run    런타임 데이터                                   │           │ │
│  │  └────────────────────────────────────────────────────────┘           │ │
│  │                                                             │ │
│  └───────────────────────────────────────────────────────────────┘ │
│                                                                     │
│  【공유 가능 vs 불가능】                                               │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │                                                             │ │
│  │  공유 가능: /usr, /opt, /home                                          │ │
│  │  공유 불가능: /etc, /boot, /var, /tmp                                  │ │
│  │  정적: /usr, /opt, /boot                                               │ │
│  │  가변: /var, /tmp, /home                                               │ │
│  │                                                             │ │
│  └───────────────────────────────────────────────────────────────┘ │
└─────────────────────────────────────────────────────────────────────┘
```

## Ⅱ. 상세 분석

### 상세 분석
```
┌─────────────────────────────────────────────────────────────────────┐
│                파일 시스템 계층 구조 상세                               │ |
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【주요 디렉토리 상세】                                                │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │                                                             │ │
│  │  디렉토리   내용                    중요도                          │ │
│  │  ────────   ────                    ────                          │ │
│  │  /bin      필수 사용자 명령            필수                           │ │
│  │  /sbin     필수 시스템 명령            필수                           │ │
│  │  /lib      필수 공유 라이브러리        필수                           │ │
│  │  /etc      시스템 설정 파일            필수                           │ │
│  │  /dev      장치 파일                   필수                           │ │
│  │  /proc     커널/프로세스 정보           가상                           │ │
│  │  /sys      시스템 하드웨어 정보         가상                           │ │
│  │  /boot     부트로더, 커널               필수                           │ │
│  │  /usr      사용자 프로그램/데이터       선택                           │ │
│  │  /home     사용자 홈 디렉토리           선택                           │ │
│  │  /var      가변 데이터 (로그, 캐시)     선택                           │ │
│  │  /tmp      임시 파일                    선택                           │ │
│  │  /opt      추가 소프트웨어              선택                           │ │
│  │  /mnt      임시 마운트                  선택                           │ │
│  │  /media    이동식 미디어                선택                           │ │
│  │  /srv      서비스 데이터                선택                           │ │
│  │                                                             │ │
│  └───────────────────────────────────────────────────────────────┘ │
│                                                                     │
│  【/usr 하위 구조】                                                     │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │                                                             │ │
│  │  /usr/bin      사용자 명령                                              │ │
│  │  /usr/sbin     시스템 데몬                                              │ │
│  │  /usr/lib      라이브러리                                               │ │
│  │  /usr/include  헤더 파일                                                │ │
│  │  /usr/share    아키텍처 독립 데이터                                      │ │
│  │  /usr/local    로컬 설치 소프트웨어                                      │ │
│  │  /usr/src      소스 코드                                                │ │
│  │                                                             │ │
│  └───────────────────────────────────────────────────────────────┘ │
│                                                                     │
│  【/var 하위 구조】                                                     │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │                                                             │ │
│  │  /var/log      로그 파일                                                │ │
│  │  /var/cache    캐시 데이터                                              │ │
│  │  /var/spool    스풀 데이터 (메일, 프린트)                                │ │
│  │  /var/lib      가변 상태 데이터                                         │ │
│  │  /var/tmp      임시 파일 (재부팅 보존)                                   │ │
│  │  /var/mail      사용자 메일                                             │ │
│  │  /var/www       웹 서버 데이터                                          │ │
│  │                                                             │ │
│  └───────────────────────────────────────────────────────────────┘ │
└─────────────────────────────────────────────────────────────────────┘
```

## Ⅲ. 실무 적용

### 구현 예시
```
┌─────────────────────────────────────────────────────────────────────┐
│                실무 적용                                            │ |
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【시스템 탐색】                                                        │
│  ──────────────────                                                  │
│  // 루트 디렉토리 확인                                                   │
│  $ ls -la /                                                          │
│  drwxr-xr-x  20 root root  4096 Jan 1 00:00 .                       │
│  drwxr-xr-x   2 root root  4096 Jan 1 00:00 bin                     │
│  drwxr-xr-x   4 root root  4096 Jan 1 00:00 boot                    │
│  ...                                                                 │
│                                                                     │
│  // 특정 파일 위치 찾기                                                  │
│  $ which ls                                                          │
│  /usr/bin/ls                                                         │
│  $ whereis sshd                                                      │
│  sshd: /usr/sbin/sshd /usr/share/man/man8/sshd.8.gz                 │
│                                                                     │
│  // 설정 파일 위치                                                      │
│  $ ls /etc/ssh/                                                      │
│  sshd_config  ssh_config                                             │
│                                                                     │
│  【로그 확인】                                                          │
│  ──────────────────                                                  │
│  // 시스템 로그                                                         │
│  $ ls /var/log/                                                      │
│  syslog  auth.log  kern.log  dmesg                                   │
│                                                                     │
│  // 실시간 로그 확인                                                     │
│  $ tail -f /var/log/syslog                                           │
│                                                                     │
│  // 저널 로그 (systemd)                                                │
│  $ journalctl -f                                                     │
│                                                                     │
│  【사용자 관리】                                                        │
│  ──────────────────                                                  │
│  // 사용자 홈 디렉토리                                                   │
│  $ ls /home/                                                         │
│  user1  user2  user3                                                 │
│                                                                     │
│  // 사용자 정보                                                         │
│  $ cat /etc/passwd                                                   │
│  $ cat /etc/group                                                    │
│  $ cat /etc/shadow                                                   │
│                                                                     │
│  【마운트 관리】                                                        │
│  ──────────────────                                                  │
│  // 마운트 포인트                                                        │
│  $ mount | grep "^/dev"                                             │
│  /dev/sda1 on / type ext4                                           │
│  /dev/sdb1 on /data type ext4                                       │
│                                                                     │
│  // 이동식 미디어                                                        │
│  $ ls /media/user/                                                   │
│  USB  CDROM                                                          │
│                                                                     │
│  【디스크 사용량】                                                      │
│  ──────────────────                                                  │
│  // 파일시스템별 사용량                                                   │
│  $ df -h                                                             │
│  Filesystem  Size  Used Avail Use% Mounted on                        │
│  /dev/sda1   100G   20G   80G  20% /                                 │
│  /dev/sdb1   500G  100G  400G  20% /data                             │
│                                                                     │
│  // 디렉토리별 사용량                                                    │
│  $ du -sh /*                                                         │
│  15M /bin                                                            │
│  200M /boot                                                          │
│  5G /home                                                            │
│  2G /var                                                             │
│                                                                     │
│  【FHS 표준 확인】                                                      │
│  ──────────────────                                                  │
│  // 배포판 가이드 확인                                                   │
│  $ man hier                                                          │
│  $ man file-hierarchy  // systemd                                   │
│                                                                     │
│  // 파일 시스템 표준 확인                                                 │
│  $ ls -ld /usr /var /tmp /home                                       │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

## Ⅳ. 기대효과 및 결론

### 핵심 요약

```
• 개념: 논리적 디렉토리 구조 표준
• 표준: FHS (Filesystem Hierarchy Standard)
• 루트: / 에서 시작하는 트리 구조
• 필수: /bin, /sbin, /lib, /etc, /dev
• 가상: /proc, /sys
• 사용자: /usr, /home, /opt
• 가변: /var, /tmp
• 마운트: /mnt, /media
• 목적: 일관성, 호환성, 관리 용이성
```

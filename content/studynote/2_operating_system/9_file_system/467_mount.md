# 467. 마운트 (Mount)

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 파일 시스템을 디렉토리 트리에 연결
> 2. **가치**: 통합된 파일 접근 인터페이스
> 3. **융합**: 파일 시스템, VFS와 연관

---

## Ⅰ. 개요

### 개념 정의

마운트(Mount)는 **저장 장치의 파일 시스템을 시스템의 디렉토리 트리에 연결하는 작업**이다.

### 💡 비유: 선반 설치
마운트는 **선반 설치**와 같다. 새 선반(파일 시스템)을 벽의 특정 위치(마운트 포인트)에 고정한다.

### 마운트 구조

```
┌─────────────────────────────────────────────────────────────────────┐
│                마운트 구조                                            │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【마운트 전/후】                                                     │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │                                                             │ │
│  │  마운트 전:                                                          │
│  │  ┌─────────────────┐     ┌─────────────────┐                    │ │
│  │  │   루트 (/)      │     │  /dev/sdb1      │                    │ │
│  │  │   ext4          │     │  (분리됨)        │                    │ │
│  │  │  ├── bin        │     │  ├── data       │                    │ │
│  │  │  ├── etc        │     │  └── files      │                    │ │
│  │  │  ├── mnt        │     │                 │                    │ │
│  │  │  │   └── usb    │     │                 │                    │ │
│  │  │  └── home       │     │                 │                    │ │
│  │  └─────────────────┘     └─────────────────┘                    │ │
│  │                                                             │ │
│  │  마운트 후: mount /dev/sdb1 /mnt/usb                              │
│  │  ┌─────────────────────────────────────────────────────────┐    │ │
│  │  │   루트 (/)                                               │    │ │
│  │  │   ext4                                                   │    │ │
│  │  │  ├── bin                                                 │    │ │
│  │  │  ├── etc                                                 │    │ │
│  │  │  ├── mnt                                                 │    │ │
│  │  │  │   └── usb  ←────── /dev/sdb1 (마운트됨)                 │    │ │
│  │  │  │       ├── data                                        │    │ │
│  │  │  │       └── files                                       │    │ │
│  │  │  └── home                                                │    │ │
│  │  └─────────────────────────────────────────────────────────┘    │ │
│  │                                                             │ │
│  │  /mnt/usb를 접근하면 /dev/sdb1의 내용이 보임                         │ │
│  │                                                             │ │
│  └───────────────────────────────────────────────────────────────┘ │
│                                                                     │
│  【마운트 포인트와 장치】                                            │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │                                                             │ │
│  │  장치 파일        마운트 포인트    파일 시스템                        │ │
│  │  ────            ───────────    ──────────                        │ │
│  │  /dev/sda1       /              ext4                              │ │
│  │  /dev/sda2       /home          ext4                              │ │
│  │  /dev/sdb1       /mnt/usb       vfat                              │ │
│  │  /dev/sr0        /mnt/cdrom     iso9660                           │ │
│  │  /dev/nvme0n1p1  /              ext4                              │ │
│  │                                                             │ │
│  └───────────────────────────────────────────────────────────────┘ │
└─────────────────────────────────────────────────────────────────────┘
```

## Ⅱ. 상세 분석

### 상세 분석
```
┌─────────────────────────────────────────────────────────────────────┐
│                마운트 상세                                            │ |
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【마운트 옵션】                                                      │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │                                                             │ │
│  │  일반 옵션:                                                           │
│  │  • ro: 읽기 전용 (read-only)                                         │ │
│  │  • rw: 읽기/쓰기 (기본)                                               │ │
│  │  • exec: 실행 파일 실행 허용                                          │ │
│  │  • noexec: 실행 파일 실행 불가                                        │ │
│  │  • suid: SUID 허용                                                   │ │
│  │  • nosuid: SUID 불허                                                 │ │
│  │  • dev: 장치 파일 허용                                                │ │
│  │  • nodev: 장치 파일 불허                                              │ │
│  │                                                             │ │
│  │  성능 옵션:                                                             │ │
│  │  • noatime: 접근 시간 갱신 안 함                                       │ │
│  │  • relatime: 접근 시간 선택적 갱신                                     │ │
│  │  • sync: 동기 쓰기                                                    │ │
│  │  • async: 비동기 쓰기 (기본)                                           │ │
│  │                                                             │ │
│  │  ext4 전용:                                                             │ │
│  │  • data=ordered: 메타데이터 저널링만                                   │ │
│  │  • data=writeback: 순서 보장 없음                                      │ │
│  │  • data=journal: 전체 저널링                                          │ │
│  │  • discard: TRIM 활성화 (SSD)                                         │ │
│  │                                                             │ │
│  └───────────────────────────────────────────────────────────────┘ │
│                                                                     │
│  【마운트 타입】                                                      │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │                                                             │ │
│  │  1. 수동 마운트: mount 명령으로 직접                                   │ │
│  │  2. 자동 마운트: /etc/fstab에 등록                                     │ │
│  │  3. 오토마운트: autofs, systemd-automount                             │ │
│  │  4. 바인드 마운트: 기존 마운트를 다른 위치에 연결                        │ │
│  │  5. 재마운트: 이미 마운트된 것의 옵션 변경                              │ │
│  │                                                             │ │
│  │  바인드 마운트 예시:                                                   │
│  │  $ mount --bind /home/user/data /mnt/backup                       │
│  │  // /home/user/data를 /mnt/backup에서도 접근 가능                     │ │
│  │                                                             │ │
│  └───────────────────────────────────────────────────────────────┘ │
│                                                                     │
│  【/etc/fstab 구조】                                                │
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │  # 장치        마운트포인트  FS타입  옵션         dump  pass           │ │
│  │  /dev/sda1    /          ext4   defaults      0     1              │ │
│  │  /dev/sda2    /home      ext4   defaults      0     2              │ │
│  │  /dev/sdb1    /mnt/usb   vfat   defaults,noauto 0   0              │ │
│  │  UUID=xxx...  /data      ext4   defaults      0     2              │ │
│  │  tmpfs        /tmp       tmpfs  defaults      0     0              │ │
│  │                                                             │ │
│  │  dump: dump 명령 백업 여부 (0=안 함)                                   │ │
│  │  pass: fsck 순서 (0=안 함, 1=루트 먼저, 2=그 다음)                      │ │
│  │                                                             │ │
│  └───────────────────────────────────────────────────────────────┘ │
└─────────────────────────────────────────────────────────────────────┘
```

## Ⅲ. 실무 적용

### 구현 예시
```
┌─────────────────────────────────────────────────────────────────────┐
│                실무 적용                                            │ |
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【마운트 명령어】                                                    │
│  ──────────────────                                                  │
│  // 기본 마운트                                                        │
│  $ sudo mount /dev/sdb1 /mnt/usb                                    │
│                                                                     │
│  // 파일 시스템 지정                                                    │
│  $ sudo mount -t ext4 /dev/sda1 /mnt/data                          │
│                                                                     │
│  // 옵션 지정                                                          │
│  $ sudo mount -o ro,noexec /dev/sdb1 /mnt/usb                      │
│                                                                     │
│  // UUID로 마운트                                                      │
│  $ sudo mount UUID="1234-5678" /mnt/data                           │
│                                                                     │
│  // 재마운트 (옵션 변경)                                               │
│  $ sudo mount -o remount,rw /                                       │
│                                                                     │
│  // 언마운트                                                           │
│  $ sudo umount /mnt/usb                                             │
│  $ sudo umount /dev/sdb1                                            │
│                                                                     │
│  【마운트 정보 확인】                                                  │
│  ──────────────────                                                  │
│  // 현재 마운트 목록                                                    │
│  $ mount                                                             │
│  /dev/sda1 on / type ext4 (rw,relatime)                             │
│  /dev/sda2 on /home type ext4 (rw,relatime)                         │
│                                                                     │
│  // /proc/mounts 확인                                                 │
│  $ cat /proc/mounts                                                  │
│                                                                     │
│  // findmnt (보기 좋은 출력)                                          │
│  $ findmnt                                                            │
│  TARGET    SOURCE     FSTYPE  OPTIONS                                │
│  /         /dev/sda1  ext4    rw,relatime                            │
│  └─/home   /dev/sda2  ext4    rw,relatime                            │
│                                                                     │
│  // 특정 마운트 확인                                                    │
│  $ findmnt /mnt/usb                                                  │
│  $ mountpoint /mnt/usb  // 마운트 포인트인지 확인                       │
│                                                                     │
│  【바인드 마운트】                                                      │
│  ──────────────────                                                  │
│  // 바인드 마운트                                                       │
│  $ sudo mount --bind /home/user/data /mnt/backup                   │
│                                                                     │
│  // 재귀적 바인드                                                       │
│  $ sudo mount -R /source /dest                                      │
│                                                                     │
│  // 바인드 마운트 읽기 전용                                              │
│  $ sudo mount --bind /source /dest                                   │
│  $ sudo mount -o remount,ro,bind /dest                              │
│                                                                     │
│  【프로그래밍에서 마운트】                                              │
│  ──────────────────                                                  │
│  #include <sys/mount.h>                                             │
│                                                                     │
│  // 마운트                                                            │
│  if (mount("/dev/sdb1", "/mnt/usb", "vfat",                        │
│            MS_NOEXEC | MS_NOSUID, NULL) == -1) {                    │
│      perror("mount");                                                │
│  }                                                                   │
│                                                                     │
│  // 언마운트                                                           │
│  if (umount("/mnt/usb") == -1) {                                    │
│      perror("umount");                                               │
│  }                                                                   │
│                                                                     │
│  // 지연 언마운트 (사용 중이어도)                                        │
│  umount2("/mnt/usb", MNT_DETACH);                                   │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

## Ⅳ. 기대효과 및 결론

### 핵심 요약

```
• 개념: 파일 시스템을 디렉토리 트리에 연결
• 구성요소: 장치, 마운트 포인트, 파일 시스템 타입
• 옵션: ro, rw, noatime, sync, async
• 영구 설정: /etc/fstab
• 명령: mount, umount, findmnt
• 타입: 수동, 자동, 오토, 바인드
• 시스템콜: mount(), umount(), umount2()
```

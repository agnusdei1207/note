# 트로이 목마 / 래퍼

## 핵심 인사이트

**트로이 목마(Trojan Horse)는 정상적인 프로그램으로 위장하여 악성 기능을 수행하는 프로그램이다.** 사용자가 무해한 소프트웨어라고 믿고 실행하면, 백그라운드에서 비밀번호 탈취, 백도어 생성, 시스템 파괴 등의 악성 행위를 수행한다. **래퍼(Wrapper)**는 시스템 관리자가 모니터링 목적으로 사용하는 정상 트로이 목마 기법이기도 하다.

---

## Ⅰ. 정의 및 특징

### 1. 트로이 목마의 특징

| 특징 | 설명 |
|:---|:---|
| **위장** | 정상 프로그램으로 가장 |
| **자기 복제 없음** | 바이러스와 달리 스스로 증식하지 않음 |
| **사용자 개입 필요** | 사용자가 직접 실행해야 활성화 |
| **숨겨진 기능** | 공개된 기능背后에 악성 코드 존재 |
| **권한 상향** | setuid를 이용해 루트 권한 획득 가능 |

### 2. 정상 vs 악성 트로이 목마

```

│              정상 래퍼 vs 악성 트로이 목마                   │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  정상 래퍼 (Wrapper)              악성 트로이 목마           │
│  ┌─────────────────────┐        ┌─────────────────────┐    │
│  │ 관리자가 설치       │        │ 공격자가 배포       │    │

│  │                     │        │                     │    │
```

## Ⅳ. 정상 래퍼 (Wrapper) 활용

### 1. 시스템 관리용 래퍼

```bash
#!/bin/bash
# finger 래퍼 - 사용자 정보 요청 로깅

# 원본 finger 실행
/usr/bin/finger "$@"

# 로깅
echo "$(date): finger $@" >> /var/log/finger.log

# 메일로 관리자에게 알림
mail -s "Finger usage alert" admin@company.com << EOF
User $(whoami) ran: finger $@
EOF
```

### 2. sudo 래퍼

```
┌─────────────────────────────────────────────────────────────┐
│            sudo 래퍼를 이용한 감시                           │
├─────────────────────────────────────────────────────────────┤

│  원본 sudo를 sudo.real로改名                                │
│  $ sudo mv /usr/bin/sudo /usr/bin/sudo.real                │
│                                                             │
│  새로운 sudo 스크립트 작성:                                 │
│  ┌─────────────────────────────────────────────────────┐   │
│  │ #!/bin/bash                                         │   │
```

## Ⅴ. 탐지 및 방어

### 1. 탐지 방법

```bash
# 1) 무결성 검증
$ sha256sum /bin/ls
abc123...  /bin/ls

# 2) 스트링 확인 (악성 문자열)
$ strings /bin/ls | grep -i password
/etc/passwd    # 의심스러운 참조

# 3) 네트워크 연결 확인
$ lsof -i -P
COMMAND   PID USER   FD TYPE DEVICE SIZE/OFF NODE NAME
unknown  1234 root   3u  IPv4  12345      0t0  TCP evil.com:443

# 4) 의심스러운 setuid 파일
$ find / -perm -4000 -user root -exec ls -l {} \;
-rwsr-xr-x 1 root root ... /tmp/.ls    # 의심!

# 5) AIDE (Advanced Intrusion Detection Environment)
$ aide --check
AIDE found differences between database and filesystem!!
```

## 어린이를 위한 비유

**트로이 목마는 "선물로 온 거대 목마"**와 같습니다.

그리스 신화에서 트로이 전쟁 때:
- 그리스군은 **거대한 목마**를 성 앞에 두고 "이건 선물이야!"라고 속였어요
- 트로이 사람들은 목마를 성 안으로 들여갔어요
- 밤이 되자 **목마 안에 숨어있던 병사들**이 나와서 성을 열었죠

컴퓨터의 트로이 목마도 똑같아요:
- **무료 게임**: "신나는 게임!"처럼 보이지만
- **실제로는**: 비밀번호를 훔치고, 백도어를 열고, 컴퓨터를 장악해요

**래퍼(Wrapper)**는 책 겉표지와 같아요:
- **정상 래퍼**: "책 겉은 바꿨지만 내용은 그대로야" - 도서관이 대출 기록용으로 쓰는 특별 표지
- **악성 트로이**: "겉은 유명한 책이지만 내용은 위험한 내용이야"

항상 **받는 사람을 확인**하고, **겉모습만 보지 말고 내용을 확인**하세요!

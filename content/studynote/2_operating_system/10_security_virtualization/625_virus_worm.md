# 바이러스 / 웜

## 핵심 인사이트

**바이러스(Virus)와 웜(Worm)은 자기 복제 악성코드지만, 본질적인 차이가 있다.** 바이러스는 **호스트 프로그램에 기생**하여 사용자가 실행해야 전파되지만, 웜은 **독립적으로 실행**되어 네트워크를 통해 자동으로 확산한다. OS 관점에서 바이러스는 파일 시스템 감염을, 웜은 네트워크 자원 소모와 확산을 다룬다.

## Ⅰ. 정의 및 차이점

### 1. 바이러스 vs 웜

```
┌─────────────────────────────────────────────────────────────┐
│               바이러스 vs 웜 비교                            │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  바이러스 (Virus)                  웜 (Worm)                 │
│  ┌─────────────────┐              ┌─────────────────┐      │
│  │ 호스트 필요     │              │ 독립 실행       │      │
│  │                 │              │                 │      │
│  │ .exe에 기생     │              │ 자가 복제       │      │
│  │ .doc 매크로     │              │                 │      │
│  │ 부트 섹터       │              │ 네트워크 전파   │      │
│  │                 │              │                 │      │
│  │ 사용자 실행     │              │ 자동 확산       │      │
│  │ 필요           │              │                 │      │
│  │                 │              │ 능동적 공격     │      │
│  │ 수동적 전파     │              │                 │      │
│  └─────────────────┘              └─────────────────┘      │
│                                                             │
│  복제 방식:                         복제 방식:              │
│  ┌─────────────────────────┐      ┌───────────────────┐   │
│  │  clean.exe + virus     │      │ worm.exe          │   │
│  │       │                │      │   │                │   │
│  │       ▼                │      │   ▼ 실행           │   │
│  │  infected.exe          │      │ worm.exe (copy 2)  │   │
│  │       │                │      │   │                │   │
│  │       ▼ 다운로드        │      │   ▼ 네트워크       │   │
│  │  다음 파일 감염        │      │ worm.exe (copy 3)  │   │
│  └─────────────────────────┘      └───────────────────┘   │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 2. 주요 차이점

| 특징 | 바이러스 | 웜 |
|:---|:---:|:---:|
| **호스트 필요** | ✓ | ✗ |
| **자가 복제** | ✓ | ✓ |
| **사용자 개입** | 필요 | 불필요 |
| **전파 속도** | 느림 | 빠름 |
| **네트워크 활용** | 제한적 | 적극적 |
| **감염 범위** | 로컬 시스템 | 네트워크 전체 |
| **숨기기** | 쉬움 | 어려움 |

### 2. 바이러스 감염 과정

```

│                                                             │
│  1. 침투 (Penetration)                                     │

│     ├── 파일 시스템 스캔                                   │

│     └── DDoS 봇 가입                                        │
│                                                             │
│  5. 전파 (Propagation)                                     │
│     ├── 이메일 발송                                         │
│     ├── 네트워크 공유                                      │
│     └── 이동식 드라이브                                     │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

## Ⅳ. 바이러스 유형

### 1. 감염 방식별 분류

```
┌─────────────────────────────────────────────────────────────┐
│                바이러스 유형별 분류                           │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  1. 파일 감염형 (File Infector)                             │
│     ┌───────────────────────────────────────────────────┐  │
│     │ .exe, .com, .sys 등 실행 파일 감염                │  │
│     │ 파일 실행 시 바이러스 활성화                      │  │
│     └───────────────────────────────────────────────────┘  │
│                                                             │
│  2. 부트 섹터 감염형 (Boot Sector Infector)                │
│     ┌───────────────────────────────────────────────────┐  │
│     │ 하드디스크 MBR, 부트 섹터 감염                   │  │
│     │ 시스템 부팅 시 로드                               │  │
│     │ 예: Brain, Michelangelo                           │  │
│     └───────────────────────────────────────────────────┘  │
│                                                             │
│  3. 매크로 바이러스 (Macro Virus)                          │
│     ┌───────────────────────────────────────────────────┐  │
│     │ Word, Excel 등 매크로 포함 문서 감염            │  │
│     │ 문서 열면 매크로 자동 실행                        │  │
│     │ 예: Melissa, Concept                              │  │
│     └───────────────────────────────────────────────────┘  │
│                                                             │
│  4. 스크립트 바이러스 (Script Virus)                        │
│     ┌───────────────────────────────────────────────────┐  │
│     │ VBScript, JavaScript, PowerShell 등             │  │
│     │ 이메일 첨부로 전파                                 │  │
│     │ 예: ILOVEYOU                                      │  │
│     └───────────────────────────────────────────────────┘  │
│                                                             │
│  5. 다형성 바이러스 (Polymorphic Virus)                     │
│     ┌───────────────────────────────────────────────────┐  │
│     │ 감염할 때마다 코드 변형                           │  │
│     │ 안티바이러스 탐지 회피                             │  │
│     │ 예: Storm Worm                                   │  │
│     └───────────────────────────────────────────────────┘  │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

## Ⅴ. OS 레벨 방어

### 1. 파일 시스템 보안

```
┌─────────────────────────────────────────────────────────────┐
│           바이러스/웜 방어를 위한 OS 기법                     │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  1. 실행 속성 제어                                          │
│     ┌───────────────────────────────────────────────────┐  │
│     │ # noexec 마운트 옵션                               │  │
│     │ /tmp  ext4 defaults,noexec                        │  │
│     │ /home ext4 defaults,noexec                        │  │
│     │                                                    │  │
│     │ → /tmp에서 실행 파일 방지                           │  │
│     └───────────────────────────────────────────────────┘  │
│                                                             │
│  2. 파일 무결성                                             │
│     ┌───────────────────────────────────────────────────┐  │
│     │ # AIDE로 무결성 검사                               │  │
│     │ aide --init                                       │  │
│     │ aide --check                                      │  │
│     │                                                    │  │
│     │ → 변경된 파일 탐지                                 │  │
│     └───────────────────────────────────────────────────┘  │
│                                                             │
│  3. 시스템 호출 격리                                         │
│     ┌───────────────────────────────────────────────────┐  │
│     │ seccomp-filter로 시스템 콜 제한                    │  │
│     │ → 루트킷 행위 제어                                 │  │
│     └───────────────────────────────────────────────────┘  │
│                                                             │
│  4. 네트워크 격리                                           │
│     ┌───────────────────────────────────────────────────┐  │
│     │ iptables로 불필요한 포트 차단                     │  │
│     │ → 웜의 전파 경로 차단                              │  │
│     └───────────────────────────────────────────────────┘  │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 2. SELinux/AppArmor 활용

```bash
# 웜이 시스템 파일 수정 방지
# SELinux 정책 예시
allow httpd_t httpd_sys_content_t:file { read };
deny httpd_t etc_t:file { write };  # 웜이 /etc 수정 방지

# AppArmor 프로필 예시
profile webbrowser /usr/bin/firefox {
    # 네트워크는 가능
    network inet stream,

    # 다운로드 디렉터리만 쓰기
    /home/user/Downloads/** rw,

    # 시스템 파일 쓰기 금지
    deny /etc/** rw,
    deny /bin/** rw,
    deny /usr/bin/** rw,
}
```

## 어린이를 위한 비유

**바이러스는 "기생하는 씨앗"**과 같고, **웜은 "스스로 이동하는 벌레"**와 같습니다.

**바이러스**:
- 책의 페이지 사이에 **씨앗**이 숨어 있어요
- 책을 펼치면(실행하면) 씨앗이 튀어나와요
- 다른 책에도 씨앗을 옮겨요
- 그래서 **사람이 열어야** 퍼져요

**웜**:

# 572. 디렉토리 (Directory)

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 파일을 계층적으로 조직화하는 컨테이너
> 2. **가치**: 논리적 구조, 탐색, 관리 편의성
> 3. **융합**: 경로, 트리 구조, 마운트와 연관

---

## Ⅰ. 개요

### 개념 정의
**디렉토리(Directory)**는 **파일과 하위 디렉토리를 조직화하는 컨테이너**입니다.

### 💡 비유: 서류함의 폴더
디렉토리는 **서류함의 폴더**와 같습니다. 관련 문서들을 모아둡니다.

### 디렉토리 구조
```
┌─────────────────────────────────────────────────────────────────────┐
│                디렉토리 구조                                         │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【계층적 디렉토리 구조】                                             │
│  ──────────────────                                                │
│  ┌─────────────────────────────────────────────────────────────┐   │
│  │                                                             │ │   │
│  │                          / (root)                             │ │   │
│  │                           │                                   │ │   │
│  │          ┌────────────────┼────────────────┐                  │ │   │
│  │          │                │                │                   │ │   │
│  │         home             etc              usr                 │ │   │
│  │          │                │                │                   │ │   │
│  │     ┌────┴────┐      ┌───┴───┐        ┌──┴──┐               │ │   │
│  │    user1    user2   passwd  hosts     bin  lib               │ │   │
│  │      │                                                  │      │ │   │
│  │   ┌──┴──┐                                            ┌──┴──┐   │ │   │
│  │  docs  pics                                         local  share│ │   │
│  │                                                             │ │   │
│  └─────────────────────────────────────────────────────────────┘   │
│                                                                     │
│  【디렉토리 내용】                                                   │
│  ──────────────────                                                │
│  ┌─────────────────────────────────────────────────────────────┐   │
│  │                                                             │ │   │
│  │  $ ls -la /home/user1                                          │ │   │
│  │  drwxr-xr-x  5 user user 4096 Jan 15 12:00 .               │ │   │
│  │  drwxr-xr-x  3 root root 4096 Jan 15 12:00 ..              │ │   │
│  │  drwxr-xr-x  2 user user 4096 Jan 15 12:00 docs            │ │   │
│  │  drwxr-xr-x  2 user user 4096 Jan 15 12:00 pics            │ │   │
│  │  -rw-r--r--  1 user user  512 Jan 15 12:00 file.txt        │ │   │
│  │                                                             │ │   │
│  │  .   : 현재 디렉토리                                           │ │   │
│  │  ..  : 상위 디렉토리                                           │ │   │
│  │                                                             │ │   │
│  └─────────────────────────────────────────────────────────────┘   │
│                                                                     │
│  【표준 디렉토리 (Linux FHS)】                                        │
│  ──────────────────                                                │
│  ┌─────────────────────────────────────────────────────────────┐   │
│  │                                                             │ │   │
│  │  디렉토리         용도                                          │ │   │
│  │  ────────         ────                                          │ │   │
│  │  /bin            필수 사용자 명령어                              │ │   │
│  │  /sbin           시스템 관리 명령어                              │ │   │
│  │  /etc            시스템 설정 파일                                │ │   │
│  │  /home           사용자 홈 디렉토리                              │ │   │
│  │  /root           root 홈 디렉토리                               │ │   │
│  │  /var            가변 데이터 (로그, 스풀)                         │ │   │
│  │  /tmp            임시 파일                                      │ │   │
│  │  /usr            사용자 프로그램                                 │ │   │
│  │  /lib            공유 라이브러리                                 │ │   │
│  │  /dev            장치 파일                                      │ │   │
│  │  /proc           프로세스 정보 (가상)                            │ │   │
│  │  /sys            시스템 정보 (가상)                              │ │   │
│  │  /opt            선택적 소프트웨어                               │ │   │
│  │  /mnt            마운트 포인트                                   │ │   │
│  │  /media          이동식 미디어                                   │ │   │
│  │                                                             │ │   │
│  └─────────────────────────────────────────────────────────────┘   │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

## Ⅱ. 상세 분석
### 상세 분석
```
┌─────────────────────────────────────────────────────────────────────┐
│                디렉토리 상세                                         │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【경로 표현】                                                        │
│  ──────────────────                                                │
│  ┌─────────────────────────────────────────────────────────────┐   │
│  │                                                             │ │   │
│  │  타입              예시                    설명                   │ │   │
│  │  ────              ────                    ────                   │ │   │
│  │  절대 경로          /home/user/docs        루트부터 시작            │ │   │
│  │  상대 경로          ./docs, ../pics        현재 위치 기준            │ │   │
│  │  홈 디렉토리        ~, ~/docs              사용자 홈                │ │   │
│  │                                                             │ │   │
│  │  특수 기호:                                                      │ │   │
│  │  .    현재 디렉토리                                              │ │   │
│  │  ..   상위 디렉토리                                              │ │   │
│  │  ~    홈 디렉토리                                                │ │   │
│  │  -    이전 디렉토리 (cd -)                                        │ │   │
│  │                                                             │ │   │
│  └─────────────────────────────────────────────────────────────┘   │
│                                                                     │
│  【디렉토리 엔트리】                                                   │
│  ──────────────────                                                │
│  ┌─────────────────────────────────────────────────────────────┐   │
│  │                                                             │ │   │
│  │  디렉토리 = 파일 이름 → inode 번호 매핑 테이블                    │ │   │
│  │                                                             │ │   │
│  │  ┌────────────────────────────────────────┐                   │ │   │
│  │  │  파일명          inode 번호              │                   │ │   │
│  │  │  ──────          ─────────              │                   │ │   │
│  │  │  .               1234567                │                   │ │   │
│  │  │  ..              1234560                │                   │ │   │
│  │  │  docs            1234570                │                   │ │   │
│  │  │  file.txt        1234580                │                   │ │   │
│  │  └────────────────────────────────────────┘                   │ │   │
│  │                                                             │ │   │
│  └─────────────────────────────────────────────────────────────┘   │
│                                                                     │
│  【링크 카운트】                                                       │
│  ──────────────────                                                │
│  ┌─────────────────────────────────────────────────────────────┐   │
│  │                                                             │ │   │
│  │  파일 링크 카운트:                                                │ │   │
│  │  • 일반 파일: 하드 링크 수                                        │ │   │
│  │  • 최소 1 (자신)                                                │ │   │
│  │                                                             │ │   │
│  │  디렉토리 링크 카운트:                                             │ │   │
│  │  • 최소 2 (.과 ..)                                              │ │   │
│  │  • 하위 디렉토리 수만큼 증가                                       │ │   │
│  │  • 2 + N (N = 하위 디렉토리 수)                                   │ │   │
│  │                                                             │ │   │
│  └─────────────────────────────────────────────────────────────┘   │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

## Ⅲ. 실무 적용
### 구현 예시
```
┌─────────────────────────────────────────────────────────────────────┐
│                실무 적용                                            │ |
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【디렉토리 조작】                                                    │
│  ──────────────────                                                │
│  // 디렉토리 생성                                                    │
│  $ mkdir newdir                                                     │
│  $ mkdir -p path/to/deep/dir   // 중간 디렉토리도 생성               │
│                                                                     │
│  // 디렉토리 삭제                                                    │
│  $ rmdir emptydir              // 빈 디렉토리만                       │
│  $ rm -r directory             // 내용 포함 삭제                      │
│  $ rm -rf directory            // 강제 삭제 (주의!)                   │
│                                                                     │
│  // 디렉토리 이동                                                    │
│  $ cd /path/to/dir                                                  │
│  $ cd ~                        // 홈으로                             │
│  $ cd -                        // 이전 디렉토리                       │
│  $ cd ..                       // 상위 디렉토리                       │
│                                                                     │
│  【디렉토리 조회】                                                    │
│  ──────────────────                                                │
│  // 내용 나열                                                        │
│  $ ls                          // 간단 목록                          │
│  $ ls -la                      // 상세 정보                          │
│  $ ls -laR                     // 재귀적 목록                        │
│  $ ls -lt                      // 수정 시간순                        │
│  $ ls -lS                      // 크기순                             │
│                                                                     │
│  // 트리 구조                                                        │
│  $ tree /path                                                       │
│  $ tree -L 2                   // 2단계까지만                        │
│  $ tree -d                     // 디렉토리만                         │
│                                                                     │
│  // 현재 위치                                                        │
│  $ pwd                         // 현재 경로 출력                      │
│                                                                     │
│  【디렉토리 검색】                                                    │
│  ──────────────────                                                │
│  // 디렉토리 찾기                                                     │
│  $ find /path -type d -name "docs"                                  │
│  $ find /path -type d -empty   // 빈 디렉토리                        │
│                                                                     │
│  // 위치 찾기                                                        │
│  $ locate directory                                                 │
│  $ whereis command                                                  │
│                                                                     │
│  【디렉토리 용량】                                                    │ |
│  ──────────────────                                                │
│  // 디스크 사용량                                                    │
│  $ du -sh /path                // 요약                               │
│  $ du -h --max-depth=1 /path   // 1단계까지만                        │
│  $ du -sh *                    // 현재 디렉토리 항목별                 │
│                                                                     │
│  // 파일 시스템 사용량                                                │
│  $ df -h                       // 전체 파일시스템                     │
│                                                                     │
│  【Python 디렉토리 처리】                                             │
│  ──────────────────                                                │
│  import os                                                           │
│  import shutil                                                       │
│                                                                     │
│  # 생성/삭제                                                         │
│  os.makedirs('path/to/dir', exist_ok=True)                          │
│  os.rmdir('emptydir')           # 빈 디렉토리만                       │
│  shutil.rmtree('directory')     # 내용 포함 삭제                      │
│                                                                     │
│  # 조회                                                             │
│  os.listdir('/path')            # 파일/디렉토리 목록                  │
│  os.getcwd()                    # 현재 경로                          │
│  os.chdir('/new/path')          # 이동                               │
│                                                                     │
│  # 순회                                                             │
│  for root, dirs, files in os.walk('/path'):                         │
│      print(f"Directory: {root}")                                     │
│      for f in files:                                                │
│          print(f"  File: {f}")                                      │
│                                                                     │
│  # 경로 조작                                                         │
│  import pathlib                                                      │
│  p = pathlib.Path('/home/user/docs')                               │
│  p.parent                       # 상위 디렉토리                       │
│  p.name                        # 마지막 부분                          │
│  p.exists()                    # 존재 여부                           │
│  p.is_dir()                    # 디렉토리 여부                        │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

## Ⅳ. 기대효과 및 결론
### 핵심 요약
```
• 개념: 파일을 계층적으로 조직화하는 컨테이너
• 구조: 트리 형태, 루트(/)에서 시작
• 내용: 파일명 → inode 매핑 테이블
• 특수 엔트리: . (현재), .. (상위)
• 경로: 절대(/로 시작), 상대(현재 기준)
• FHS: /bin, /etc, /home, /var, /tmp
• 링크 카운트: 파일(하드링크수), 디렉토리(2+N)
• 명령어: mkdir, rmdir, cd, ls, pwd, du
• Python: os, pathlib 모듈
• 순회: os.walk() 재귀적 탐색
```

# 545. VGA (Video Graphics Array)

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 비디오 출력을 위한 디스플레이 인터페이스
> 2. **가치**: 텍스트/그래픽 모드, 텍스텔 모드 지원
> 3. **융합**: 프레임버퍼, 해상도와 연관

---

## Ⅰ. 개요

### 개념 정의
**VGA(Video Graphics Array)**는 **비디오 출력을 위한 디스플레이 표준**입니다. 텍스트 모드(80x25)와 그래픽 모드(720p/1080p 등)을 지원합니다.

### 💡 비유: 디지털 액자프
VGA는 **디지털 액자프**와 같습니다. 각 층은 색상과 밝기 정보를 표시합니다.

## Ⅲ. 실무 적용
### 구현 예시
```
┌─────────────────────────────────────────────────────────────────────┐
│                실무 적용                                            │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【Linux VGA 설정】                                                  │
│  ──────────────────                                                │
│  // VGA 카드 정보                                                        │
│  $ lspci | grep -i vga                                              │
│  01:00.0 VGA compatible controller                                  │
│                                                                     │
│  // 현재 모드 확인                                                        │
│  $ cat /sys/class/graphics/fb0/modes                              │
│  U:1920x1080p-0                                           │
│                                                                     │
│  // 해상도 변경                                                        │
│  $ xrandr --output HDMI-0 --mode 1920x1080                          │
│                                                                     │
│  // 밝기 조정                                                          │
│  $ xrandr --output HDMI-0 --brightness 0.8                          │
│                                                                     │
│  // 프레임 버퍼 설정                                                      │
│  $ xrandr --output HDMI-0 --set fb_tiling off                          │
│                                                                     │
│  【VGA 메모리 할당 (커널 모듈)】                                       │
│  ──────────────────                                                │
│  #include <linux/fb.h>                                               │
│                                                                     │
│  struct fb_info *info;                                              │
│  struct fb_fix_screeninfo finfo;                                    │
│  struct fb_var_screeninfo vinfo;                                    │
│                                                                     │
│  // 프레임버퍼 디바이스 열기                                             │
│  int fd = open("/dev/fb0", O_RDWR);                                │
│                                                                     │
│  // 화면 정보 가져오기                                                   │
│  ioctl(fd, FBIOGET_FSCREENINFO, &finfo);                             │
│  ioctl(fd, FBIOGET_VSCREENINFO, &vinfo);                             │
│                                                                     │
│  // 프레임버퍼 매핑                                                      │
│  long screensize = vinfo.yres_virtual * vinfo.xres_virtual *          │
│                 vinfo.bits_per_pixel / 8;                             │
│  char *fbp = mmap(NULL, screensize, PROT_READ | PROT_WRITE,         │
│                 MAP_SHARED, fd, 0);                                  │
│                                                                     │
│  // 픽셀 쓰기 (32bpp)                                                   │
│  void put_pixel(int x, int y, uint32_t color) {                      │
│      long location = (x + vinfo.xoffset) *                            │
│                    (vinfo.bits_per_pixel / 8) +                      │
│                    (y + vinfo.yoffset) * finfo.line_length;            │
│      *((uint32_t *)(fbp + location)) = color;                          │
│  }                                                                   │
│                                                                     │
│  // 정리                                                              │
│  munmap(fbp, screensize);                                           │
│  close(fd);                                                          │
│                                                                     │
│  【DRM (Direct Rendering Manager)】                                    │
│  ──────────────────                                                │
│  // DRM 장치 확인                                                      │
│  $ ls /sys/class/drm/                                                │
│  card0                                                              │
│                                                                     │
│  // DRM 정보                                                          │
│  $ cat /sys/class/drm/card0/device/product                          │
│  $ drm_info                                                         │
│                                                                     │
│  // KMS (Kernel Mode Setting)                                        │
│  $ cat /sys/class/drm/card0/card0-DP-1/modes                        │
│                                                                     │
│  // 모드 설정                                                         │
│  $ xrandr --output HDMI-0 --mode 1920x1080 --rate 60.00             │
│                                                                     │
│  【가상 프레임버퍼 (virtio-gpu)】                                     │
│  ──────────────────                                                │
│  // QEMU에서 가상 GPU                                                  │
│  $ qemu-system-x86_64 -enable-kvm -m 2048 \                         │
│    -device virtio-gpu-pci                                           │
│                                                                     │
│  // virgl (가상 3D 가속)                                              │
│  $ qemu-system-x86_64 -enable-kvm -m 2048 \                         │
│    -device virtio-gpu-pci,virgl=on                                  │
│                                                                     │
│  【다중 모니터 설정】                                                  │
│  ──────────────────                                                │
│  // 현재 출력 확인                                                     │
│  $ xrandr --query                                                    │
│  Screen 0: minimum 320 x 200, current 1920 x 1080                  │
│  HDMI-0 connected 1920x1080+0+0 (normal left inverted) 530mm x 300mm│
│  HDMI-1 connected 1280x1024+1920+0 (normal left inverted) 376mm x 301mm│
│                                                                     │
│  // 프라이머리 설정                                                     │
│  $ xrandr --output HDMI-0 --primary                                 │
│                                                                     │
│  // 미러링                                                            │
│  $ xrandr --output HDMI-1 --same-as HDMI-0                          │
│                                                                     │
│  // 확장 데스크탑                                                      │
│  $ xrandr --output HDMI-1 --right-of HDMI-0                         │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

## Ⅳ. 기대효과 및 결론
### 핵심 요약
```
• 개념: 비디오 출력 표준 인터페이스
• 신호: HSYNC, VSYNC, RED/GREEN/BLUE
• 해상도: 640x480 ~ 7680x4320
• 커넥터: DE-15, DVI, HDMI, DisplayPort
• 모드: 텍스트 모드, 그래픽 모드
• 메모리: 프레임 버퍼, DAC
• Linux: /dev/fb0, xrandr, DRM
• 현대: HDMI, DisplayPort가 더 많이 사용
```

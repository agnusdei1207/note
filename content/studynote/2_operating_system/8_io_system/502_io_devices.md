# 502. I/O 장치 (I/O Devices)

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 컴퓨터와 외부 세계의 인터페이스
> 2. **가치**: 데이터 입력, 출력, 저장
> 3. **융합**: 컨트롤러, 드라이버, 버스와 연관

---

## Ⅰ. 개요

### 개념 정의

I/O 장치(Input/Output Devices)는 **컴퓨터 시스템과 외부 세계 간의 데이터 교환을 담당하는 하드웨어**이다.

### 💡 비유: 인체 감각 기관
I/O 장치는 **감각 기관**과 같다. 눈(입력)으로 보고, 입(출력)으로 말한다.

## Ⅲ. 실무 적용

### 구현 예시
```
┌─────────────────────────────────────────────────────────────────────┐
│                실무 적용                                            │ |
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【장치 정보 확인】                                                      │
│  ──────────────────                                                  │
│  // PCI 장치                                                           │
│  $ lspci -v                                                          │
│  $ lspci -nn | grep -i vga                                          │
│                                                                     │
│  // USB 장치                                                           │
│  $ lsusb -v                                                          │
│  $ lsusb -t                                                          │
│                                                                     │
│  // 블록 장치                                                           │
│  $ lsblk -o NAME,SIZE,TYPE,FSTYPE,MOUNTPOINT                        │
│  $ fdisk -l                                                          │
│                                                                     │
│  【장치 속도 테스트】                                                    │
│  ──────────────────                                                  │
│  // 디스크 속도                                                         │
│  $ sudo hdparm -tT /dev/sda                                         │
│                                                                     │
│  // 네트워크 속도                                                       │
│  $ ethtool eth0 | grep Speed                                        │
│  $ iperf3 -c server_ip                                              │
│                                                                     │
│  // USB 속도 확인                                                      │
│  $ lsusb -t | grep -i speed                                         │
│                                                                     │
│  【장치 사용량 모니터링】                                                │
│  ──────────────────                                                  │
│  // 디스크 I/O                                                          │
│  $ iostat -x 1                                                      │
│  $ iotop                                                            │
│                                                                     │
│  // 네트워크 I/O                                                        │
│  $ iftop                                                            │
│  $ nload                                                            │
│                                                                     │
│  // 장치 대기 시간                                                      │
│  $ iostat -x 1 | awk '{print $1, $10, $11}'                        │
│                                                                     │
│  【장치 드라이버 관리】                                                  │
│  ──────────────────                                                  │
│  // 로드된 모듈                                                        │
│  $ lsmod | grep -i usb                                              │
│  $ lsmod | grep -i net                                              │
│                                                                     │
│  // 모듈 정보                                                          │
│  $ modinfo e1000e                                                   │
│  $ modinfo usb_storage                                              │
│                                                                     │
│  // 모듈 로드/언로드                                                    │
│  $ sudo modprobe <module>                                           │
│  $ sudo rmmod <module>                                              │
│                                                                     │
│  【장치 펌웨어 업데이트】                                                │
│  ──────────────────                                                  │
│  // SSD 펌웨어                                                          │
│  $ sudo smartctl -a /dev/sda                                        │
│                                                                     │
│  // BIOS/UEFI 정보                                                    │
│  $ sudo dmidecode -t bios                                           │
│  $ sudo dmidecode -t system                                         │
│                                                                     │
│  【장치 에러 확인】                                                      │
│  ──────────────────                                                  │
│  // dmesg로 장치 에러 확인                                              │
│  $ dmesg | grep -i error                                            │
│  $ dmesg | grep -i "I/O error"                                      │
│  $ dmesg | grep -i fail                                             │
│                                                                     │
│  // 저널 로그                                                          │
│  $ journalctl -k | grep -i "error\|fail"                            │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

## Ⅳ. 기대효과 및 결론

### 핵심 요약

```
• 개념: 컴퓨터와 외부 세계의 인터페이스
• 분류: 입력, 출력, 입출력 겸용, 저장
• 유형: 블록 장치, 캐릭터 장치
• 속도: CPU > RAM > SSD > HDD > 네트워크
• 전송: 전이중/반이중, 동기/비동기, 직렬/병렬
• 인터페이스: PCIe, SATA, USB, Thunderbolt
• 확인: lspci, lsusb, lsblk, iostat
• 관리: lsmod, modprobe, rmmod
```

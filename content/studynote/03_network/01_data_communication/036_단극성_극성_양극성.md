> **핵심 인사이트**
> 1. 디지털 신호의 전압 패턴은 단극성(Unipolar)·극성(Polar)·양극성(Bipolar) 세 범주로 분류되며, 각 방식은 DC 성분·동기화·잡음 내성·대역폭 효율성에서 다른 트레이드오프를 가진다.
> 2. 양극성 방식(특히 AMI, B8ZS, HDB3)은 0 전압과 +/- 전압을 교대로 사용해 DC 성분을 제거하고 단순한 방식으로 오류 검출 기능까지 내포한다.
> 3. 현대 고속 통신은 단순한 3단계 분류를 넘어 다진 진폭 변조(PAM-4, PAM-8)와 라인 코딩의 조합으로 스펙트럼 효율을 극대화한다.

---

## I. 세 가지 신호 유형 비교

```
단극성 (Unipolar):
  1: +5V    0: 0V
  전압 범위: [0, +V]
  DC 성분: 많음 (1이 많으면 편향)

극성 (Polar / NRZ):
  NRZ-L:  1: +V    0: -V
  전압 범위: [-V, +V]
  DC 성분: 균형 시 0 (하지만 보장 안 됨)

양극성 (Bipolar / AMI):
  1: +V / -V 교대    0: 0V
  전압 범위: {-V, 0, +V}
  DC 성분: 없음 (1이 교대로 극성 반전)
```

| 방식      | 전압 레벨 | DC 성분 | 오류 검출 | 대역폭 효율 |
|---------|---------|--------|---------|-----------|
| 단극성   | 0, +V   | 있음    | 없음     | 낮음      |
| 극성 NRZ | -V, +V  | 있을 수  | 없음     | 높음      |
| AMI     | -V, 0, +V | 없음  | 가능     | 중간      |

> 📢 **섹션 요약 비유**: 단극성은 손전등(켜짐/꺼짐), 극성은 +/-건전지, 양극성은 +0- 교통신호 — 복잡할수록 더 많은 정보를 안전하게 담는다.

---

## II. AMI (Alternate Mark Inversion)

```
AMI 인코딩:
데이터: 1  0  1  1  0  0  1  0  1
신호:  +V  0 -V +V  0  0 -V  0 +V
       ↑ 1이 나올 때마다 +/-를 번갈아 사용

특성:
- DC 성분 = 0 (장거리 전화선에 적합)
- 0이 연속되면 동기화 문제 (B8ZS/HDB3로 해결)
- 교번 위반(AMI violation): 연속 같은 극성 -> 오류 탐지
```

> 📢 **섹션 요약 비유**: 홀수 번째 박수는 오른손, 짝수 번째는 왼손 — 규칙이 깨지면 실수가 발생했음을 바로 알 수 있다.

---

## III. B8ZS와 HDB3 (연속 0 문제 해결)

```
B8ZS (Bipolar 8-Zero Substitution):
  연속 8개의 0을 특정 패턴으로 교체
  000+-0-+ (현재 AMI 극성에 따라 선택)
  -> 동기화 유지 + 교번 위반 패턴으로 식별 가능
  사용처: 북미 T1/T3 라인

HDB3 (High Density Bipolar 3):
  연속 3개의 0을 특정 패턴으로 교체
  사용처: 유럽 E1/E3 라인
```

| 방식   | 치환 단위 | 사용 지역    | 라인     |
|------|---------|------------|---------|
| AMI  | 치환 없음 | 전 세계     | 기본     |
| B8ZS | 8개 0   | 북미        | T1/T3   |
| HDB3 | 3개 0   | 유럽        | E1/E3   |

> 📢 **섹션 요약 비유**: 긴 침묵(연속 0)이 지속되면 통화가 끊겼는지 알 수 없으니, 암호 신호(치환 패턴)를 넣어 "나 여기 있어"를 알린다.

---

## IV. 멀티레벨 신호 — PAM-4

```
PAM-4 (Pulse Amplitude Modulation):
  4개 전압 레벨: +3V, +1V, -1V, -3V
  2비트를 하나의 심볼로 표현
  
  심볼 -> 비트 매핑:
  +3V = 11
  +1V = 10
  -1V = 01
  -3V = 00

장점: 동일 대역폭에서 2배 데이터 전송
단점: 잡음 내성 감소 (레벨 간 간격이 좁아짐)
사용처: 400G/800G 이더넷, PCIe Gen 6
```

> 📢 **섹션 요약 비유**: 교통신호가 빨강/초록 2가지에서 빨강/주황/노랑/초록 4가지로 늘어남 — 한 번에 더 많은 정보를 전달하지만 구분이 더 어렵다.

---

## V. 실무 적용 — 전화망과 이더넷

| 기술          | 신호 방식      | 특징                      |
|-------------|--------------|--------------------------|
| T1 (1.544Mbps) | AMI + B8ZS | 북미 디지털 전화망          |
| E1 (2.048Mbps) | HDB3       | 유럽 디지털 전화망          |
| ISDN BRI     | 2B1Q        | 4레벨 극성 방식             |
| 100BASE-TX   | MLT-3       | 3레벨 (0, +V, -V) 순환    |
| 400G 이더넷   | PAM-4       | 4레벨, 스펙트럼 효율 2배    |

> 📢 **섹션 요약 비유**: T1은 1972년부터 쓴 기술, PAM-4는 최신 400G 기술 — 신호 방식의 진화가 대역폭 혁신의 핵심이다.

## 📌 관련 개념 맵

```
디지털 신호 유형
+-- 단극성 (Unipolar): {0, +V}, DC 성분 큼
+-- 극성 (Polar)
|   +-- NRZ-L: {-V, +V}
|   +-- NRZ-I: 변화=1, 유지=0
|   +-- Manchester: 클럭 내포
+-- 양극성 (Bipolar)
|   +-- AMI: 1은 +/-교대, 0은 0V
|   +-- B8ZS: 8연속 0 치환 (T1)
|   +-- HDB3: 3연속 0 치환 (E1)
+-- 멀티레벨
    +-- 2B1Q: 4레벨 (ISDN)
    +-- MLT-3: 3레벨 (100BASE-TX)
    +-- PAM-4: 4레벨 (400G 이더넷)
```

## 👶 어린이를 위한 3줄 비유 설명

1. 단극성은 전구 켜짐/꺼짐처럼 단순하지만, 오래 꺼져있으면 통신이 끊긴 건지 모르는 문제가 있어요.
2. 양극성은 +, 0, -를 번갈아 써서 신호가 균형 잡히고, 규칙이 깨지면 오류가 생긴 걸 알 수 있어요.
3. 현대 고속 인터넷은 4가지 전압 레벨을 써서 한 번에 더 많은 데이터를 보내요!

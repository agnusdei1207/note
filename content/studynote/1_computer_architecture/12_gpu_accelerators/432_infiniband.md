# 432. InfiniBand & RDMA

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 고성능 컴퓨팅을 위한 저지연 고대역폭 네트워크
> 2. **가치**: RDMA, CPU 오버헤드 없는 직접 메모리 전송
> 3. **융합**: HPC, AI 클러스터, 분산 학습, GPU Direct

---

## Ⅰ. 개요 (Context & Background)

### 개념 정의

InfiniBand는 **HPC와 데이터센터를 위한 고속 인터커넥트로, RDMA(Remote Direct Memory Access)를 통해 CPU 개입 없이 직접 메모리 전송을 수행한다.**

### 💡 비유: 특급 배송 서비스

InfiniBand는 **특급 배송 서비스**와 같다. 일반 배송이 있다. (Ethernet) 여러 경유. 느리다. 특급 배송이 있다. (InfiniBand) 직접 전달. 빠르다. 문 앞까지. (RDMA) 경유 없이. InfiniBand가 똑같다. 직접 전송, CPU 없음, 저지연.

### InfiniBand 개요

```
┌─────────────────────────────────────────────────────────────────────┐
│                InfiniBand & RDMA 개요                                     │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【InfiniBand 세대】                                                              │)
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │                                                               │ │
│  │  세대       속도            대역폭      레인       용도          │ │
│  │  ────       ────            ─────      ────       ────          │ │
│  │  SDR        2.5 Gbps       2 GB/s     1X         초기          │ │
│  │  DDR        5 Gbps         4 GB/s     1X         발전          │ │
│  │  QDR        10 Gbps        8 GB/s     1X         일반          │ │
│  │  FDR        14 Gbps        13 GB/s    1X         데이터센터      │ │
│  │  EDR        25 Gbps        25 GB/s    1X         AI/HPC        │ │
│  │  HDR        50 Gbps        50 GB/s    1X         AI 클러스터    │ │
│  │  NDR        100 Gbps       100 GB/s   1X         차세대        │ │
│  │  XDR        200 Gbps       200 GB/s   1X         미래          │ │
│  │                                                               │ │
│  │  4X 레인: 대역폭 × 4                                                       │ │
│  │  HDR 4X = 200 Gbps                                                        │ │
│  │                                                               │ │
│  └───────────────────────────────────────────────────────────────┘ │
│                                                                     │
│  【RDMA (Remote Direct Memory Access)】                                            │)
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │                                                               │ │
│  │  일반 네트워크 (Ethernet):                                                   │ │
│  │  App → Socket → TCP/IP → NIC → Network → NIC → TCP/IP → App             │ │
│  │  (CPU 개입 많음, 복사 많음)                                                 │ │
│  │                                                               │ │
│  │  RDMA (InfiniBand):                                                         │ │
│  │  App → NIC → Network → NIC → Memory (직접)                               │ │
│  │  (CPU 개입 없음, 제로 복사)                                                 │ │
│  │                                                               │ │
│  │  장점:                                                                    │ │
│  │  • CPU 오버헤드 없음                                                      │ │
│  │  • 낮은 지연 (~1μs)                                                       │ │
│  │  • 높은 처리량                                                            │ │
│  │                                                               │ │
│  └───────────────────────────────────────────────────────────────┘ │
└─────────────────────────────────────────────────────────────────────┘
```

## Ⅱ. InfiniBand 상세

### 상세 분석

```
┌─────────────────────────────────────────────────────────────────────┐
│                InfiniBand 상세                                             │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【RDMA 작업 유형】                                                               │)
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │                                                               │ │
│  │  1. SEND/RECV:                                                              │ │
│  │     • 양방향 통신                                                          │ │
│  │     • 수신 측 버퍼 필요                                                    │ │
│  │                                                               │ │
│  │  2. READ:                                                                   │ │
│  │     • 원격 메모리 읽기                                                     │ │
│  │     • 단방향                                                               │ │
│  │                                                               │ │
│  │  3. WRITE:                                                                  │ │
│  │     • 원격 메모리 쓰기                                                     │ │
│  │     • 단방향, 가장 일반적                                                  │ │
│  │                                                               │ │
│  │  4. ATOMIC:                                                                 │ │
│  │     • 원자적 연산                                                          │ │
│  │     • CAS, Fetch-and-Add                                                   │ │
│  │                                                               │ │
│  └───────────────────────────────────────────────────────────────┘ │
│                                                                     │
│  【GPUDirect RDMA】                                                               │)
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │                                                               │ │
│  │  개념: 네트워크 → GPU 메모리 직접 전송                                      │ │
│  │                                                               │ │
│  │  기존: NIC → CPU 메모리 → GPU 메모리                                       │ │
│  │  GPUDirect: NIC → GPU 메모리 (직접)                                        │ │
│  │                                                               │ │
│  │  장점:                                                                    │ │
│  │  • CPU 오버헤드 제거                                                      │ │
│  │  • PCIe 트래픽 감소                                                       │ │
│  │  • 지연 감소                                                              │ │
│  │                                                               │ │
│  └───────────────────────────────────────────────────────────────┘ │
│                                                                     │
│  【InfiniBand vs Ethernet】                                                       │)
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │                                                               │ │
│  │  특성           InfiniBand        Ethernet (RoCE)              │ │
│  │  ────           ──────────        ───────────────              │ │
│  │  지연           ~1 μs             ~10 μs                       │ │
│  │  대역폭         400 Gbps          400 Gbps                     │ │
│  │  RDMA           네이티브           RoCE                        │ │
│  │  비용           높음              낮음                         │ │
│  │  호환성         전용              IP 기반                      │ │
│  │                                                               │ │
│  └───────────────────────────────────────────────────────────────┘ │
│                                                                     │
│  【장단점】                                                                      │)
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │                                                               │ │
│  │  장점:                                                                  │ │
│  │  • 매우 낮은 지연                                                        │ │
│  │  • 높은 대역폭                                                            │ │
│  │  • RDMA 효율                                                             │ │
│  │                                                               │ │
│  │  단점:                                                                  │ │
│  │  • 높은 비용                                                            │ │
│  │  • 전용 장비                                                            │ │
│  │  • 복잡한 설정                                                          │ │
│  │                                                               │ │
│  └───────────────────────────────────────────────────────────────┘ │
└─────────────────────────────────────────────────────────────────────┘
```

## Ⅴ. 기대효과 및 결론

### 핵심 요약

```
• InfiniBand: 고속 인터커넥트
• RDMA: 원격 직접 메모리 접근
• HDR: 200 Gbps (4X)
• NDR: 400 Gbps (4X)
• 지연: ~1 μs
• GPUDirect: NIC → GPU 직접
• vs Ethernet: 지연↓ 비용↑
• 장점: 저지연, 고대역폭, RDMA
• 단점: 비용, 복잡성
• 활용: HPC, AI 클러스터, 분산 학습
```

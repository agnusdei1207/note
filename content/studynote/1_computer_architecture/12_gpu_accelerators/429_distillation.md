# 429. 지식 증류 (Knowledge Distillation)

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 큰 모델(Teacher)의 지식을 작은 모델(Student)로 전이
> 2. **가치**: 모델 축소, 추론 가속, 정확도 유지
> 3. **융합**: 소프트 레이블, 온도, 특징 증류, LLM

---

## Ⅰ. 개요 (Context & Background)

### 개념 정의

지식 증류(Knowledge Distillation)는 **대규모 Teacher 모델이 학습한 지식을 작은 Student 모델로 전이하여, 모델 크기를 줄이면서도 성능을 유지하는 모델 압축 기법**이다.

### 💡 비유: 스승이 제자에게 전수

지식 증류는 **스승이 제자에게 전수**와 같다. 스승이 있다. (Teacher) 오랜 경험. 깊은 통찰. 제자가 있다. (Student) 배우는 중. 스승이 가르친다. 단순한 답이 아니다. 노하우도 함께. 제자가 스승처럼. 지식 증류가 똑같다. Teacher → Student, 지식 전이, 성능 유지.

│  │    │             │                                                         │ │
│  │    │             ▼                                                         │ │
│  │    │    ┌─────────────────┐                                               │ │
│  │    │    │   Soft Labels   │  (소프트 레이블)                               │ │

│  │                  ▼                                                         │ │

│  │                                                               │ │

│                                                                     │
│  【지식 종류】                                                                  │)

│  ┌───────────────────────────────────────────────────────────────┐ │
│  │                                                               │ │
│  │  1. 응답 기반 (Response-based):                                               │ │
│  │     • 출력층 소프트맥스                                                     │ │

│  │     • 샘플 간 관계                                                          │ │
│  │     • 구조적 지식                                                          │ │
│  │                                                               │ │

## Ⅱ. 지식 증류 상세

### 상세 분석

```
┌─────────────────────────────────────────────────────────────────────┐
│                지식 증류 상세                                              │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【소프트 레이블과 온도】                                                          │)
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │                                                               │ │
│  │  소프트맥스 with 온도 T:                                                    │ │
│  │                                                               │ │
│  │  p_i = exp(z_i / T) / Σ_j exp(z_j / T)                                    │ │
│  │                                                               │ │
│  │  T = 1 (기본): 하드 레이블에 가까움                                         │ │
│  │  T > 1 (높음): 분포가 부드러워짐 → 어두운 지식 드러남                        │ │
│  │                                                               │ │
│  │  예시 (3-class):                                                            │ │
│  │  로짓: [10, 3, 1]                                                          │ │
│  │                                                               │ │
│  │  T=1:  [0.999, 0.001, 0.000]  ← 하드                                       │ │
│  │  T=4:  [0.70, 0.20, 0.10]     ← 소프트 (어두운 지식)                        │ │
│  │                                                               │ │
│  │  어두운 지식: 클래스 간 유사성 정보                                         │ │
│  │                                                               │ │
│  └───────────────────────────────────────────────────────────────┘ │
│                                                                     │
│  【손실 함수】                                                                  │)
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │                                                               │ │
│  │  L_total = α × L_KL(p_t || p_s) + (1-α) × L_CE(y, p_s)                     │ │
│  │                                                               │ │
│  │  L_KL: KL 발산 (Teacher vs Student 소프트 레이블)                            │ │
│  │  L_CE: 교차 엔트로피 (정답 vs Student)                                       │ │
│  │  α: 두 손실의 가중치                                                        │ │
│  │                                                               │ │
│  │  온도 T를 사용하여 KL 발산 계산                                             │ │
│  │                                                               │ │
│  └───────────────────────────────────────────────────────────────┘ │
│                                                                     │
│  【LLM 지식 증류】                                                               │)
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │                                                               │ │
│  │  DistilBERT:                                                               │ │
│  │  • BERT → 작은 BERT                                                        │ │
│  │  • 40% 작은, 60% 빠른, 97% 성능                                            │ │
│  │                                                               │ │
│  │  Alpaca/Vicuna:                                                             │ │
│  │  • GPT-4 → LLaMA                                                           │ │
│  │  • 지도 학습 데이터 생성                                                    │ │
│  │                                                               │ │
│  │  Orca:                                                                      │ │
│  │  • GPT-4 → 작은 모델                                                        │ │
│  │  • 사고 과정 증류                                                          │ │
│  │                                                               │ │
│  └───────────────────────────────────────────────────────────────┘ │
│                                                                     │
│  【장단점】                                                                      │)
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │                                                               │ │
│  │  장점:                                                                  │ │
│  │  • 모델 크기 감소                                                        │ │
│  │  • 추론 속도 향상                                                        │ │
│  │  • 정확도 유지                                                           │ │
│  │                                                               │ │
│  │  단점:                                                                  │ │
│  │  • Teacher 필요                                                         │ │
│  │  • 학습 시간                                                             │ │
│  │  • 하이퍼파라미터                                                        │ │
│  │                                                               │ │
│  └───────────────────────────────────────────────────────────────┘ │
└─────────────────────────────────────────────────────────────────────┘
```

## Ⅳ. 실무 적용

### 지식 증류 분석

```
┌─────────────────────────────────────────────────────────────────────┐
│                지식 증류 분석                                              │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【PyTorch 지식 증류】                                                            │)
│  ──────────────────                                                  │
│  import torch.nn.functional as F                                    │
│                                                                     │
│  def distillation_loss(student_logits, teacher_logits, labels,      │
│                        T=4.0, alpha=0.5):                           │
│      # KL 발산 (소프트 레이블)                                         │
│      soft_loss = F.kl_div(                                          │
│          F.log_softmax(student_logits / T, dim=1),                  │
│          F.softmax(teacher_logits / T, dim=1),                      │
│          reduction='batchmean'                                      │
│      ) * T * T                                                      │
│                                                                     │
│      # 교차 엔트로피 (하드 레이블)                                      │
│      hard_loss = F.cross_entropy(student_logits, labels)            │
│                                                                     │
│      return alpha * soft_loss + (1 - alpha) * hard_loss             │
│                                                                     │
│  【Hugging Face DistilBERT】                                                      │)
│  ──────────────────                                                  │
│  from transformers import DistilBertModel                           │
│                                                                     │
│  model = DistilBertModel.from_pretrained('distilbert-base-uncased') │
│                                                                     │
│  【LLM 지식 증류 (예시)】                                                           │)
│  ──────────────────                                                  │
│  # Teacher: GPT-4 API 호출로 응답 생성                                │
│  # Student: 작은 모델로 Teacher 응답 학습                              │
│                                                                     │
│  【지식 증류 최적화 팁】                                                          │)
│  ──────────────────                                                  │
│  1. 온도: T=4~10 권장                                                 │
│  2. α: 0.5~0.9 (증류 비중)                                           │
│  3. 특징 증류: 중간층 활용                                             │
│  4. 데이터: 충분한 양                                                  │
│  5. Teacher: 고성능 모델                                              │
│  6. 점진적: 여러 단계                                                 │
└─────────────────────────────────────────────────────────────────────┘
```

## Ⅴ. 기대효과 및 결론

### 핵심 요약

```
• 지식 증류: Teacher → Student 지식 전이
• 소프트 레이블: 온도로 부드럽게
• 어두운 지식: 클래스 간 유사성
• 손실: KL 발산 + 교차 엔트로피
• 온도: T↑ → 분포 부드러움
• DistilBERT: 40% 작은, 97% 성능
• LLM 증류: GPT-4 → 작은 모델
• 장점: 크기, 속도, 성능 유지
• 단점: Teacher 필요, 학습 시간
• 활용: 모델 압축, 모바일, 엣지
```

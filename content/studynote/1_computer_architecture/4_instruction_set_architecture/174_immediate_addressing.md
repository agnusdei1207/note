# 174. 즉시 주소 지정 (Immediate Addressing)

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 명령어 내에 피연산자 값을 직접 포함하는 방식
> 2. **가치**: 빠른 실행, 메모리 접근 없음, 상수 초기화 최적
> 3. **융합**: 루프 카운터, 플래그 설정, 마스크 연산

---

## Ⅰ. 개요 (Context & Background)

### 개념 정의

즉시 주소 지정(Immediate Addressing)은 **명령어 자체에 피연산자 값을 직접 포함**하는 방식이다. 별도의 메모리 접근 없이 명령어 fetch만으로 데이터를 얻을 수 있어 가장 빠르다.

### 💡 비유: 편지에 직접 적힌 답

즉시 주소 지정은 **편지에 답이 직접 적혀있는 것**과 같다. "값은 100입니다"라고 편지 본문에 적혀있어서, 다른 곳을 찾아볼 필요가 없다.

### 즉시 주소 지정 구조

```
┌─────────────────────────────────────────────────────────────────────┐
│                즉시 주소 지정 구조                                    │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【구조】                                                           │
│  ─────────                                                          │
│                                                                     │
│  ┌─────────────┬─────────────────────┐                              │
│  │   Opcode    │    Immediate Value  │                              │
│  │   연산코드   │     상수 값          │                              │
│  └─────────────┴─────────────────────┘                              │
│                                                                     │
│  【특징】                                                           │
│  ─────────                                                          │
│  • 메모리 접근 없음 (fetch만 필요)                                  │
│  • 실행 속도 가장 빠름                                               │
│  • 값이 고정됨 (실행 중 변경 불가)                                   │
│  • 크기 제한 있음 (명령어 길이 내)                                   │
│                                                                     │
│  【다양한 ISA 예시】                                                 │
│  ────────────────────                                               │
│                                                                     │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │  ISA      문법                  예시                           │ │
│  │  ───────  ────────────────────  ────────────────              │ │
│  │  x86      MOV reg, imm          MOV EAX, 100                   │ │
│  │  ARM      MOV Rd, #imm          MOV R0, #100                   │ │
│  │  MIPS     addi rt, rs, imm      addi $t0, $zero, 100           │ │
│  │  RISC-V   addi rd, rs1, imm     addi x5, x0, 100               │ │
│  └───────────────────────────────────────────────────────────────┘ │
│                                                                     │
│  【즉시값 크기 제한】                                                │
│  ────────────────────                                               │
│                                                                     │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │  ISA      명령어 길이   즉시값 크기    전략                      │ │
│  │  ───────  ───────────  ───────────  ─────────                 │ │
│  │  x86      가변          1/2/4바이트   자유로움                  │ │
│  │  ARM      32비트        12비트 회전    회전 인코딩              │ │
│  │  MIPS     32비트        16비트        lui+ori 조합              │ │
│  │  RISC-V   32비트        12비트        lui+addi 조합             │ │
│  └───────────────────────────────────────────────────────────────┘ │
│                                                                     │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │  즉시값 크기 제한 해결책:                                         │ │
│  │  ──────────────────────────                                      │ │
│  │  1. 여러 명령어 조합 (lui + ori)                                 │ │
│  │  2. 상수 풀 (literal pool, ARM)                                  │ │
│  │  3. 메모리에서 로드 (큰 상수)                                     │ │
│  │  4. 가변 길이 명령어 (x86)                                        │ │
│  └───────────────────────────────────────────────────────────────┘ │
└─────────────────────────────────────────────────────────────────────┘
```

## Ⅱ. 즉시 주소 지정 활용

### 활용 패턴

```
┌─────────────────────────────────────────────────────────────────────┐
│                즉시 주소 지정 활용                                    │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【1. 초기화】                                                       │
│  ─────────────                                                      │
│  MOV EAX, 0           ; 변수 초기화                                  │
│  MOV ECX, 100         ; 루프 카운터                                  │
│  MOV EDX, -1          ; 에러 플래그                                  │
│                                                                     │
│  【2. 산술 연산】                                                    │
│  ──────────────────                                                  │
│  ADD EAX, 10          ; EAX += 10                                    │
│  SUB ECX, 1           ; ECX -= 1                                     │
│  CMP EAX, 100         ; EAX와 100 비교                               │
│                                                                     │
│  【3. 비트 마스크】                                                  │
│  ──────────────────                                                  │
│  AND EAX, 0xFF        ; 하위 8비트 추출                              │
│  OR  EAX, 0x80        ; 비트 7 설정                                  │
│  XOR EAX, 0x0F        ; 하위 4비트 토글                              │
│                                                                     │
│  【4. 플래그 설정】                                                  │
│  ──────────────────                                                  │
│  MOV EAX, 1           ; TRUE                                        │
│  MOV EAX, 0           ; FALSE                                       │
│  MOV EAX, -1          ; ALL_BITS_SET                                │
│                                                                     │
│  【5. 주소 계산】                                                    │
│  ──────────────────                                                  │
│  ADD RSP, 8           ; 스택 조정                                    │
│  ADD RDI, 4           ; 포인터 증가                                  │
│  SUB RBP, 16          ; 로컬 변수 공간                               │
│                                                                     │
│  【ARM 특수 인코딩】                                                 │
│  ──────────────────                                                  │
│  #imm = imm8 순환 회전 (rot)                                        │
│                                                                     │
│  MOV R0, #256         ; 1 << 8 (회전)                               │
│  MOV R0, #0xFF00      ; 0xFF << 8                                   │
│  MOV R0, #0x0000FF00  ; 0xFF << 16                                  │
│                                                                     │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │  즉시 주소 지정 최적화:                                           │ │
│  │  ────────────────────────────                                     │ │
│  │  나쁨: MOV EAX, 0        (5바이트)                                │ │
│  │  좋음: XOR EAX, EAX      (2바이트)  ; 같은 효과                   │ │
│  │                                                               │ │
│  │  나쁨: CMP EAX, 0         (5바이트)                                │ │
│  │  좋음: TEST EAX, EAX      (2바이트)  ; ZF 설정                   │ │
│  │                                                               │ │
│  │  → 컴파일러는 자동으로 최적화                                     │ │
│  └───────────────────────────────────────────────────────────────┘ │
└─────────────────────────────────────────────────────────────────────┘
```

## Ⅳ. 실무 적용

### 즉시 주소 지정 분석

```
┌─────────────────────────────────────────────────────────────────────┐
│                즉시 주소 지정 분석                                    │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【어셈블리로 보는 즉시값】                                          │
│  ──────────────────────────                                         │
│  // C 코드                                                           │
│  int x = 42;                                                        │
│  int y = x + 10;                                                    │
│  if (y > 50) { ... }                                                │
│                                                                     │
│  // x86-64                                                          │
│  mov eax, 42           ; 즉시값 42                                   │
│  add eax, 10           ; 즉시값 10                                   │
│  cmp eax, 50           ; 즉시값 50                                   │
│  jg  .L2               ; 조건 분기                                   │
│                                                                     │
│  【ARM 즉시값 제약】                                                 │
│  ──────────────────────────                                         │
│  MOV R0, #42           ; OK (42 = 0x2A)                             │
│  MOV R0, #256          ; OK (1 << 8)                                │
│  MOV R0, #257          ; FAIL! (인코딩 불가)                         │
│                                                                     │
│  ; 해결책                                                            │
│  LDR R0, =257          ; 상수 풀에서 로드                            │
│  MOVW R0, #257         ; ARMv7 16비트 즉시값                         │
│                                                                     │
│  【MIPS 32비트 상수】                                                │
│  ──────────────────────────                                         │
│  # 32비트 상수 0x12345678                                            │
│  lui  $t0, 0x1234      ; 상위 16비트                                 │
│  ori  $t0, $t0, 0x5678  ; 하위 16비트                                │
│                                                                     │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │  즉시값 인코딩 확인:                                              │ │
│  │  ──────────────────────                                          │ │
│  │  $ objdump -d program                                            │ │
│  │  mov eax, 42    ; b8 2a 00 00 00  (5바이트)                      │ │
│  │  add eax, 10    ; 83 c0 0a        (3바이트, 작은 즉시값)          │ │
│  │  add eax, 1000  ; 05 e8 03 00 00  (5바이트, 큰 즉시값)            │ │
│  │                                                               │ │
│  │  → 작은 즉시값은 짧은 인코딩 사용                                 │ │
│  └───────────────────────────────────────────────────────────────┘ │
└─────────────────────────────────────────────────────────────────────┘
```

## Ⅴ. 기대효과 및 결론

### 핵심 요약

```
• 즉시 주소 지정: 명령어 내에 값 직접 포함
• 장점: 메모리 접근 없음, 가장 빠름
• 단점: 값 크기 제한, 고정된 상수만
• 활용: 초기화, 산술, 비트 마스크, 플래그
• 제한 해결: 명령어 조합, 상수 풀, 메모리 로드
```

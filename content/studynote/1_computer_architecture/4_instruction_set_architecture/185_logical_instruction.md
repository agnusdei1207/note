# 185. 논리 연산 명령어 (Logical Instruction)

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 비트 단위 논리 연산을 수행하는 명령어
> 2. **가치**: 비트 마스킹, 플래그 설정, 조건 검사, 암호화
> 3. **융합**: 권한 검사, 그래픽, 프로토콜, 해시

---

## Ⅰ. 개요 (Context & Background)

### 개념 정의

논리 연산 명령어(Logical Instruction)는 **비트 단위의 논리 연산(AND, OR, XOR, NOT)을 수행**하는 명령어다. 비트 마스킹, 플래그 검사, 부울 연산에 사용되며, 산술 연산과 함께 가장 기본적인 연산이다.

### 💡 비유: 스위치 조합

논리 연산 명령어는 **전기 스위치 조합**과 같다. AND는 두 스위치 모두 켜야 불이 켜지고, OR는 하나만 켜도 켜진다. XOR은 두 스위치가 다를 때만 켜진다. 이렇게 비트들을 조합해서 원하는 결과를 만든다.

### 논리 연산 명령어 분류

```
┌─────────────────────────────────────────────────────────────────────┐
│                논리 연산 명령어 분류                                  │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【분류】                                                           │
│  ─────────                                                          │
│                                                                     │
│  논리 연산                                                          │
│  ├── 기본: AND, OR, XOR, NOT                                       │
│  ├── 테스트: TEST, TST (결과 버림)                                  │
│  ├── 시프트: SHL/SAL, SHR, SAR                                     │
│  ├── 순환: ROL, ROR, RCL, RCR                                      │
│  └── 비트: BT, BTS, BTR, BTC (x86)                                  │
│                                                                     │
│  【주요 명령어】                                                     │
│  ──────────────                                                     │
│                                                                     │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │  명령어    연산               진리표(1비트)       플래그        │ │
│  │  ───────  ─────────────────  ─────────────────   ────────      │ │
│  │  AND      비트 논리곱        0&0=0, 0&1=0        ZNP (C=0,O=0) │ │
│  │                            1&0=0, 1&1=1                         │ │
│  │                                                               │ │
│  │  OR       비트 논리합        0|0=0, 0|1=1        ZNP (C=0,O=0) │ │
│  │                            1|0=1, 1|1=1                         │ │
│  │                                                               │ │
│  │  XOR      배타적 논리합      0^0=0, 0^1=1        ZNP (C=0,O=0) │ │
│  │            (서로 다를 때 1)   1^0=1, 1^1=0                       │ │
│  │                                                               │ │
│  │  NOT      비트 반전          ~0=1, ~1=0          없음          │ │
│  │                                                               │ │
│  │  TEST     AND (결과 버림)     -                   ZNP (C=0,O=0) │ │
│  └───────────────────────────────────────────────────────────────┘ │
│                                                                     │
│  【연산 예시 (8비트)】                                               │
│  ─────────────────────                                              │
│                                                                     │
│  AND 10110101        ; 값 A                                         │
│      11110000        ; 마스크                                       │
│  ────────────                                                       │
│  =   10110000        ; 상위 4비트 유지, 하위 4비트 0                 │
│                                                                     │
│  OR  10110101                                                      │
│      00001111        ; 마스크                                       │
│  ────────────                                                       │
│  =   10111111        ; 하위 4비트 1로 설정                          │
│                                                                     │
│  XOR 10110101                                                      │
│      11111111        ; 마스크                                       │
│  ────────────                                                       │
│  =   01001010        ; 모든 비트 토글                                │
│                                                                     │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │  플래그 설정:                                                      │ │
│  │  ──────────                                                       │ │
│  │  AND/OR/XOR/TEST:                                                 │ │
│  │  • CF = 0, OF = 0 (항상)                                          │ │
│  │  • ZF, SF, PF = 결과에 따라                                       │ │
│  │  • NOT: 플래그 변경 없음                                          │ │
│  └───────────────────────────────────────────────────────────────┘ │
└─────────────────────────────────────────────────────────────────────┘
```

## Ⅱ. 주요 논리 연산 명령어

### 명령어 상세

```
┌─────────────────────────────────────────────────────────────────────┐
│                주요 논리 연산 명령어                                  │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【1. AND (비트 논리곱)】                                            │
│  ────────────────────────                                           │
│  AND dest, src        ; dest = dest & src                           │
│                                                                     │
│  ; 비트 클리어 (0으로)                                               │
│  AND EAX, 0xFFFFFF00  ; 하위 8비트 클리어                           │
│  AND AL, 0x0F         ; 상위 4비트 클리어                           │
│                                                                     │
│  ; 특정 비트 검사                                                    │
│  TEST EAX, 0x80       ; 비트 7이 설정됨? (ZF로 판별)                │
│  JNZ bit_set                                                       │
│                                                                     │
│  【2. OR (비트 논리합)】                                             │
│  ──────────────────────────                                         │
│  OR dest, src         ; dest = dest | src                           │
│                                                                     │
│  ; 비트 설정 (1로)                                                   │
│  OR EAX, 0x000000FF   ; 하위 8비트 설정                             │
│  OR AL, 0xF0          ; 상위 4비트 설정                             │
│                                                                     │
│  ; 대소문자 변환                                                     │
│  OR AL, 0x20          ; 소문자로 ('A'→'a')                          │
│                                                                     │
│  【3. XOR (배타적 논리합)】                                          │
│  ──────────────────────────                                         │
│  XOR dest, src        ; dest = dest ^ src                           │
│                                                                     │
│  ; 0으로 초기화                                                      │
│  XOR EAX, EAX         ; EAX = 0 (MOV보다 짧음)                      │
│                                                                     │
│  ; 비트 토글                                                         │
│  XOR EAX, 0x80        ; 비트 7 토글                                  │
│                                                                     │
│  ; 암호화 (XOR 특성)                                                 │
│  XOR data, key        ; 암호화                                       │
│  XOR data, key        ; 복호화 (같은 키로 XOR)                       │
│                                                                     │
│  【4. NOT (비트 반전)】                                              │
│  ──────────────────────                                             │
│  NOT dest             ; dest = ~dest (1의 보수)                     │
│                                                                     │
│  NOT EAX              ; 모든 비트 반전                               │
│  ; 0x12345678 → 0xEDCBA987                                          │
│                                                                     │
│  【5. TEST (AND 후 결과 버림)】                                      │
│  ─────────────────────────────────                                  │
│  TEST dest, src       ; dest & src, 플래그만 설정                   │
│                                                                     │
│  TEST EAX, EAX        ; EAX == 0?                                   │
│  JZ is_zero                                                        │
│                                                                     │
│  TEST EAX, 0x80       ; 비트 7 설정?                                 │
│  JNZ bit_7_set                                                      │
│                                                                     │
│  【6. ARM TST/TEQ】                                                  │
│  ──────────────────                                                  │
│  TST R0, #0x80        ; R0 & 0x80, 플래그만                         │
│  TEQ R0, #0xFF        ; R0 ^ 0xFF, 같은지 비교                      │
│                                                                     │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │  XOR 활용 팁:                                                      │ │
│  │  ──────────────                                                    │ │
│  │  1. 0으로 초기화: XOR R0, R0 (MOV R0, #0보다 짧음)                │ │
│  │  2. 변수 교환: XOR을 3번 (임시 변수 없이)                          │ │
│  │     XOR A, B; XOR B, A; XOR A, B                                  │ │
│  │  3. 간단한 암호화: 스트림 암호                                     │ │
│  │  4. 패리티 계산: XOR로 짝수/홀수 검사                              │ │
│  └───────────────────────────────────────────────────────────────┘ │
└─────────────────────────────────────────────────────────────────────┘
```

## Ⅲ. 논리 연산 × 타 과목 융합

### 과목 융합 분석

```
┌─────────────────────────────────────────────────────────────────────┐
│                논리 연산 × 타 과목 융합                               │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【컴파일러】                                                        │
│  • if (flag & MASK) → TEST + JNZ                                    │
│  • bool → int: true=1, false=0                                      │
│  • 0 초기화: XOR vs MOV                                              │
│  • 최적화: AND/OR로 조건 단순화                                      │
│                                                                     │
│  【운영체제】                                                        │
│  • 권한 비트: AND로 검사                                             │
│  • 플래그 필드: OR로 설정, AND로 클리어                              │
│  • 페이지 테이블: 권한/상태 비트                                     │
│  • 락: XOR로 CAS 구현                                               │
│                                                                     │
│  【네트워크】                                                        │
│  • 서브넷 마스크: AND로 네트워크 주소                                │
│  • IP 플래그: 비트 조작                                              │
│  • 프로토콜 헤더: 필드 추출                                          │
│  • 체크섬: XOR 활용                                                  │
│                                                                     │
│  【그래픽】                                                          │
│  • 색상 마스킹: AND/OR                                               │
│  • 알파 블렌딩: 비트 연산                                            │
│  • 비트맵: 픽셀 조작                                                 │
│  • 클리핑: AND로 영역 제한                                           │
│                                                                     │
│  【보안】                                                            │
│  • 암호화: XOR 암호, 스트림 암호                                     │
│  • 해시: XOR로 혼합                                                  │
│  • 권한 검사: 비트 마스크                                            │
│  • 난수: XOR 시프트                                                  │
│                                                                     │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │  비트 마스크 활용 예시:                                           │ │
│  │  ────────────────────────────                                     │ │
│  │  // 권한 플래그                                                    │ │
│  │  #define PERM_READ    0x01                                       │ │
│  │  #define PERM_WRITE   0x02                                       │ │
│  │  #define PERM_EXEC    0x04                                       │ │
│  │                                                               │ │
│  │  // 권한 검사                                                      │ │
│  │  TEST perms, PERM_READ   ; 읽기 권한?                             │ │
│  │  JZ no_access                                                     │ │
│  │                                                               │ │
│  │  // 권한 추가                                                      │ │
│  │  OR perms, PERM_WRITE     ; 쓰기 권한 추가                        │ │
│  │                                                               │ │
│  │  // 권한 제거                                                      │ │
│  │  AND perms, ~PERM_EXEC    ; 실행 권한 제거                        │ │
│  └───────────────────────────────────────────────────────────────┘ │
└─────────────────────────────────────────────────────────────────────┘
```

## Ⅳ. 실무 적용

### 논리 연산 분석

```
┌─────────────────────────────────────────────────────────────────────┐
│                논리 연산 분석                                         │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【NULL 포인터 검사】                                                │
│  ──────────────────────                                             │
│  TEST RDI, RDI         ; ptr == NULL?                               │
│  JZ null_handler                                                    │
│  ; 또는                                                             │
│  CMP RDI, 0                                                         │
│  JE null_handler                                                    │
│                                                                     │
│  【서브넷 마스크 계산】                                              │
│  ──────────────────────                                             │
│  ; 네트워크 주소 = IP & 마스크                                       │
│  MOV EAX, [ip_addr]                                                │
│  AND EAX, [subnet_mask]                                            │
│  MOV [network_addr], EAX                                           │
│                                                                     │
│  【대소문자 변환】                                                   │
│  ──────────────────                                                  │
│  ; 소문자로: OR 0x20                                                │
│  OR AL, 0x20            ; 'A'(65) → 'a'(97)                        │
│                                                                     │
│  ; 대문자로: AND 0xDF                                               │
│  AND AL, 0xDF           ; 'a'(97) → 'A'(65)                        │
│                                                                     │
│  【비트 카운트 (population count)】                                  │
│  ────────────────────────────────────                               │
│  ; popcnt 명령어가 없을 때                                          │
│  MOV ECX, 32                                                       │
│  XOR EDX, EDX           ; count = 0                                 │
│  .loop:                                                             │
│  SHR EAX, 1             ; LSB → CF                                  │
│  ADC EDX, 0             ; count += CF                               │
│  LOOP .loop                                                         │
│                                                                     │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │  GDB 디버깅:                                                        │ │
│  │  ──────────                                                       │ │
│  │  (gdb) p/x $eax & 0xFF    ; 하위 8비트 확인                       │ │
│  │  (gdb) p/t $eflags        ; 플래그 이진 출력                      │ │
│  │  (gdb) p ($eflags & 0x40) ; ZF 확인                              │ │
│  │                                                               │ │
│  │  최적화:                                                           │ │
│  │  • TEST EAX, EAX > CMP EAX, 0                                   │ │
│  │  • XOR EAX, EAX > MOV EAX, 0                                    │ │
│  └───────────────────────────────────────────────────────────────┘ │
└─────────────────────────────────────────────────────────────────────┘
```

## Ⅴ. 기대효과 및 결론

### 핵심 요약

```
• 논리 연산 명령어: AND, OR, XOR, NOT
• AND: 비트 클리어, 마스킹
• OR: 비트 설정
• XOR: 토글, 0 초기화, 암호화
• TEST: 비트 검사 (AND 후 결과 버림)
• 플래그: CF=0, OF=0, ZF/SF/PF = 결과
```

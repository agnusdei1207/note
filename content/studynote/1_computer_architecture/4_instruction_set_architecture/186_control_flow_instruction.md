# 186. 제어 흐름 명령어 (Control Flow Instruction)

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 프로그램 실행 순서를 변경하는 명령어
> 2. **가치**: 조건 분기, 루프, 함수 호출의 기반
> 3. **융합**: 알고리즘, 상태 머신, 예외 처리

---

## Ⅰ. 개요 (Context & Background)

### 개념 정의

제어 흐름 명령어(Control Flow Instruction)는 **프로그램 카운터(PC)를 변경하여 실행 순서를 바꾸는 명령어**다. 분기(Jump), 호출(Call), 반환(Return), 인터럽트(Interrupt) 등이 포함되며, 프로그램의 논리적 흐름을 제어한다.

### 💡 비유: 지도에서 경로 변경

제어 흐름 명령어는 **지도에서 경로를 변경하는 것**과 같다. 보통은 도로를 따라 쭉 가지만(순차 실행), "신호등이 빨간 불이면 우회"(조건부 분기)하거나, "목적지에 도착하면 돌아오기"(함수 호출/반환) 같은 규칙이 있다.

### 제어 흐름 명령어 분류

```
┌─────────────────────────────────────────────────────────────────────┐
│                제어 흐름 명령어 분류                                  │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【분류】                                                           │
│  ─────────                                                          │
│                                                                     │
│  제어 흐름                                                          │
│  ├── 무조건 분기: JMP, BRA                                         │
│  ├── 조건부 분기: JE/JZ, JNE/JNZ, JG, JL, JGE, JLE, ...            │
│  ├── 함수 호출: CALL, BL                                            │
│  ├── 함수 반환: RET, BX LR                                          │
│  ├── 루프: LOOP, LOOPE, LOOPNE                                     │
│  └── 인터럽트: INT, IRET, SYSCALL                                   │
│                                                                     │
│  【주요 명령어】                                                     │
│  ──────────────                                                     │
│                                                                     │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │  명령어    기능                   예시                          │ │
│  │  ───────  ─────────────────────  ────────────────              │ │
│  │  JMP      무조건 분기             JMP label                     │ │
│  │  Jcc      조건부 분기             JZ, JNZ, JG, JL, ...          │ │
│  │  CALL     함수 호출               CALL func                     │ │
│  │  RET      함수 반환               RET                           │ │
│  │  LOOP     CX 감소, 0 아니면 분기   LOOP label                   │ │
│  │  INT      인터럽트 호출            INT 0x80                      │ │
│  │  IRET     인터럽트 반환            IRET                          │ │
│  │  SYSCALL  시스템 콜               SYSCALL                       │ │
│  └───────────────────────────────────────────────────────────────┘ │
│                                                                     │
│  【실행 흐름 변화】                                                  │
│  ──────────────────                                                  │
│                                                                     │
│  순차 실행:        PC += 명령어 길이                                │
│  분기:            PC = 목적지 주소                                   │
│  호출:            PUSH 다음주소; PC = 함수 주소                      │
│  반환:            PC = POP                                          │
│                                                                     │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │  분기 vs 호출:                                                      │ │
│  │  ──────────────                                                    │ │
│  │  분기 (JMP):                                                       │ │
│  │  • 돌아오지 않음                                                    │ │
│  │  • PC만 변경                                                        │ │
│  │  • goto, break, continue                                           │ │
│  │                                                               │ │
│  │  호출 (CALL):                                                       │ │
│  │  • 돌아옴 (RET)                                                     │ │
│  │  • 반환 주소 저장                                                    │ │
│  │  • 함수 호출                                                        │ │
│  └───────────────────────────────────────────────────────────────┘ │
└─────────────────────────────────────────────────────────────────────┘
```

## Ⅱ. 주요 제어 흐름 명령어

### 명령어 상세

```
┌─────────────────────────────────────────────────────────────────────┐
│                주요 제어 흐름 명령어                                  │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【1. JMP (무조건 분기)】                                            │
│  ──────────────────────────                                         │
│  JMP label            ; PC = label                                  │
│  JMP EAX              ; PC = EAX (간접 분기)                        │
│  JMP [addr]           ; PC = Mem[addr]                              │
│                                                                     │
│  ; C: goto                                                          │
│  goto label;            → JMP label                                 │
│                                                                     │
│  【2. 조건부 분기 (Conditional Branch)】                             │
│  ────────────────────────────────────                               │
│  ; CMP 후 플래그 기반                                                │
│  CMP EAX, EBX                                                       │
│  JE  equal            ; EAX == EBX                                  │
│  JNE not_equal        ; EAX != EBX                                  │
│  JG  greater          ; EAX > EBX (부호있음)                        │
│  JL  less            ; EAX < EBX (부호있음)                        │
│  JGE greater_equal    ; EAX >= EBX                                  │
│  JLE less_equal       ; EAX <= EBX                                  │
│                                                                     │
│  ; TEST 후 플래그 기반                                               │
│  TEST EAX, EAX                                                      │
│  JZ  is_zero          ; EAX == 0                                    │
│  JNZ not_zero         ; EAX != 0                                    │
│                                                                     │
│  【3. CALL/RET (함수 호출/반환)】                                    │
│  ───────────────────────────────                                    │
│  CALL func            ; PUSH next_addr; PC = func                   │
│  RET                 ; PC = POP                                     │
│  RET n               ; POP; SP += n (인자 정리)                      │
│                                                                     │
│  ; x86-64 예시                                                       │
│  CALL printf                                                        │
│  ; ...                                                              │
│  RET                                                               │
│                                                                     │
│  ; ARM                                                               │
│  BL func             ; LR = next_addr; PC = func                    │
│  BX LR               ; PC = LR                                      │
│                                                                     │
│  【4. LOOP (루프)】                                                  │
│  ──────────────────                                                  │
│  LOOP label           ; CX--; if (CX != 0) goto label               │
│  LOOPE label          ; CX-- && ZF                                  │
│  LOOPNE label         ; CX-- && !ZF                                 │
│                                                                     │
│  ; for 루프 구현                                                     │
│  MOV CX, 10                                                         │
│  loop_start:                                                        │
│      ; ... 루프 본문                                                 │
│  LOOP loop_start                                                    │
│                                                                     │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │  x86 조건부 분기 코드:                                            │ │
│  │  ────────────────────────                                         │ │
│  │  코드   조건           의미                                       │ │
│  │  ─────  ─────────────  ────────────────                          │ │
│  │  JO     OF=1           오버플로우                                 │ │
│  │  JNO    OF=0           오버플로우 아님                            │ │
│  │  JB/JC  CF=1           아래/캐리 (부호없음 <)                    │ │
│  │  JAE    CF=0           이상 (부호없음 >=)                         │ │
│  │  JE/JZ  ZF=1           같음/0                                     │ │
│  │  JNE    ZF=0           같지 않음                                  │ │
│  │  JS     SF=1           음수                                       │ │
│  │  JNS    SF=0           양수                                       │ │
│  │  JL     SF!=OF         작음 (부호있음 <)                          │ │
│  │  JGE    SF==OF         이상 (부호있음 >=)                         │ │
│  │  JLE    ZF=1 || SF!=OF 이하 (부호있음 <=)                         │ │
│  │  JG     ZF=0 && SF==OF 큼 (부호있음 >)                            │ │
│  └───────────────────────────────────────────────────────────────┘ │
└─────────────────────────────────────────────────────────────────────┘
```

## Ⅲ. 제어 흐름 × 타 과목 융합

### 과목 융합 분석

```
┌─────────────────────────────────────────────────────────────────────┐
│                제어 흐름 × 타 과목 융합                               │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【컴파일러】                                                        │
│  • if/else → CMP + Jcc                                              │
│  • while → CMP + Jcc + JMP                                          │
│  • for → 초기화 + CMP + Jcc + 증감 + JMP                            │
│  • switch → jump table or 비교 체인                                 │
│  • 함수 호출 → CALL/RET                                              │
│                                                                     │
│  【알고리즘】                                                        │
│  • 이진 검색: 중간 비교 후 분기                                      │
│  • 정렬: 비교 후 교환 분기                                           │
│  • 재귀: CALL로 자신 호출                                            │
│  • 그래프: 조건부 순회                                               │
│                                                                     │
│  【운영체제】                                                        │
│  • 스케줄러: 조건부 분기                                             │
│  • 시스템 콜: SYSCALL/INT                                            │
│  • 인터럽트: INT/IRET                                                │
│  • 예외 처리: 분기로 핸들러                                          │
│                                                                     │
│  【컴퓨터 구조】                                                     │
│  • 파이프라인: 분기 예측                                             │
│  • 슈퍼스칼라: 분기 디코딩                                           │
│  • 캐시: 분기 목표 버퍼 (BTB)                                        │
│  • 성능: 분기 비용                                                   │
│                                                                     │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │  C 코드 → 어셈블리 매핑:                                          │ │
│  │  ────────────────────────                                         │ │
│  │  // if (a > b) { ... }                                            │ │
│  │  CMP EAX, EBX                                                     │ │
│  │  JLE skip           ; a <= b면 건너뜀                             │ │
│  │  ; ... then 블록                                                  │ │
│  │  skip:                                                            │ │
│  │                                                               │ │
│  │  // while (i < n) { ... i++; }                                    │ │
│  │  loop:                                                            │ │
│  │  CMP ECX, EDX                                                     │ │
│  │  JGE done           ; i >= n이면 종료                             │ │
│  │  ; ... 루프 본문                                                  │ │
│  │  INC ECX                                                          │ │
│  │  JMP loop                                                         │
│  │  done:                                                            │ │
│  └───────────────────────────────────────────────────────────────┘ │
└─────────────────────────────────────────────────────────────────────┘
```

## Ⅳ. 실무 적용

### 제어 흐름 분석

```
┌─────────────────────────────────────────────────────────────────────┐
│                제어 흐름 분석                                         │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【if-else 컴파일】                                                  │
│  ──────────────────────                                             │
│  // if (x > 0) y = 1; else y = -1;                                  │
│  CMP EDI, 0                                                         │
│  JLE .else                                                          │
│  MOV ESI, 1             ; y = 1                                     │
│  JMP .done                                                          │
│  .else:                                                             │
│  MOV ESI, -1            ; y = -1                                    │
│  .done:                                                             │
│                                                                     │
│  【switch-case (jump table)】                                        │
│  ────────────────────────────────                                   │
│  ; switch (eax) { case 0: ... case 1: ... }                         │
│  CMP EAX, max_case                                                  │
│  JA  default                                                        │
│  JMP [jump_table + EAX*4]                                           │
│                                                                     │
│  .case0:                                                            │
│  ; ...                                                              │
│  .case1:                                                            │
│  ; ...                                                              │
│  default:                                                           │
│  ; ...                                                              │
│                                                                     │
│  【함수 호출 분석】                                                  │
│  ──────────────────                                                  │
│  $ objdump -d program                                               │
│  1000: e8 fb 0f 00 00    call 2000 <func>                           │
│       └──────────────────  PC 상대 호출                              │
│  1005: ...               ; 반환 주소                                 │
│                                                                     │
│  【GDB 디버깅】                                                      │
│  ──────────────────                                                  │
│  (gdb) si                 ; CALL 단계 실행                           │
│  (gdb) p/x $pc            ; 현재 PC                                  │
│  (gdb) x/i $pc            ; 현재 명령어                              │
│  (gdb) break *0x1000      ; 주소에 브레이크포인트                    │
│                                                                     │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │  분기 최적화 팁:                                                    │ │
│  │  ──────────────                                                    │ │
│  │  1. 자주 실행되는 경로를 fall-through으로                          │ │
│  │  2. 분기 예측 힌트 (__builtin_expect)                             │ │
│  │  3. 루프 언롤링으로 분기 감소                                       │ │
│  │  4. 조건부 이동 (CMOV)으로 분기 제거                               │ │
│  └───────────────────────────────────────────────────────────────┘ │
└─────────────────────────────────────────────────────────────────────┘
```

## Ⅴ. 기대효과 및 결론

### 핵심 요약

```
• 제어 흐름 명령어: 실행 순서 변경
• 분류: 무조건 분기, 조건부 분기, 호출/반환
• JMP: 무조건 분기
• Jcc: 조건부 분기 (플래그 기반)
• CALL/RET: 함수 호출/반환
• 활용: if/else, while/for, 함수
```

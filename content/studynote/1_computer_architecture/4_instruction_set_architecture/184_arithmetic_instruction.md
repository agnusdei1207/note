# 184. 산술 연산 명령어 (Arithmetic Instruction)

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 수치 계산(덧셈, 뺄셈, 곱셈, 나눗셈)을 수행하는 명령어
> 2. **가치**: 모든 계산의 기반, 플래그 설정, 주소 계산
> 3. **융합**: 알고리즘, 수치 해석, 컴파일러 최적화

---

## Ⅰ. 개요 (Context & Background)

### 개념 정의

산술 연산 명령어(Arithmetic Instruction)는 **정수 및 부동소수점 수치 연산을 수행**하는 명령어다. ADD, SUB, MUL, DIV, INC, DEC 등이 포함되며, 연산 결과는 플래그 레지스터에 영향을 준다.

### 💡 비유: 계산기 기능

산술 연산 명령어는 **계산기의 기본 기능**과 같다. 더하기(ADD), 빼기(SUB), 곱하기(MUL), 나누기(DIV) 버튼이 있다. 계산 결과에 따라 "0이야"(ZF), "음수야"(NF) 같은 표시등(플래그)이 켜진다.

### 산술 연산 명령어 분류

```
┌─────────────────────────────────────────────────────────────────────┐
│                산술 연산 명령어 분류                                  │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【분류】                                                           │
│  ─────────                                                          │
│                                                                     │
│  산술 연산                                                          │
│  ├── 기본: ADD, SUB                                                │
│  ├── 증감: INC, DEC                                                │
│  ├── 곱셈: MUL, IMUL                                               │
│  ├── 나눗셈: DIV, IDIV                                             │
│  ├── 부호: NEG (2의 보수)                                          │
│  └── 확장: ADC (Add with Carry), SBB (Sub with Borrow)             │
│                                                                     │
│  【주요 명령어】                                                     │
│  ──────────────                                                     │
│                                                                     │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │  명령어    기능              예시                  플래그       │ │
│  │  ───────  ────────────────  ─────────────────     ────────     │ │
│  │  ADD      덧셈              ADD EAX, EBX          ZNCVS        │ │
│  │  SUB      뺄셈              SUB EAX, EBX          ZNCVS        │ │
│  │  INC      1 증가            INC EAX               ZNV (C 제외) │ │
│  │  DEC      1 감소            DEC EAX               ZNV (C 제외) │ │
│  │  NEG      부호 반전         NEG EAX               ZNCVS        │ │
│  │  MUL      부호없는 곱셈     MUL EBX               CV (ZF=?)    │ │
│  │  IMUL     부호있는 곱셈     IMUL EAX, EBX         CV           │ │
│  │  DIV      부호없는 나눗셈   DIV EBX               없음         │ │
│  │  IDIV     부호있는 나눗셈   IDIV EBX              없음         │ │
│  │  ADC      캐리 포함 덧셈    ADC EAX, EBX          ZNCVS        │ │
│  │  SBB      빌림 포함 뺄셈    SBB EAX, EBX          ZNCVS        │ │
│  └───────────────────────────────────────────────────────────────┘ │
│                                                                     │
│  【플래그 영향】                                                     │
│  ──────────────                                                     │
│  Z (Zero):     결과 == 0                                            │
│  N/S (Sign):   결과 < 0 (부호 비트)                                 │
│  C (Carry):    부호없는 오버플로우                                  │
│  V/O (Overflow): 부호있는 오버플로우                                │
│                                                                     │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │  RISC vs CISC:                                                    │ │
│  │  ──────────────                                                   │ │
│  │  RISC (ARM, MIPS):                                                │ │
│  │  • ADD Rd, Rn, Rm    ; Rd = Rn + Rm                               │ │
│  │  • SUB Rd, Rn, Rm    ; Rd = Rn - Rm                               │
│  │  • 3-피연산자, 목적지 별도                                        │ │
│  │                                                               │ │
│  │  CISC (x86):                                                      │ │
│  │  • ADD EAX, EBX     ; EAX = EAX + EBX                             │ │
│  │  • SUB EAX, EBX     ; EAX = EAX - EBX                             │ │
│  │  • 2-피연산자, 목적지 = 원본                                      │ │
│  └───────────────────────────────────────────────────────────────┘ │
└─────────────────────────────────────────────────────────────────────┘
```

## Ⅱ. 주요 산술 연산 명령어

### 명령어 상세

```
┌─────────────────────────────────────────────────────────────────────┐
│                주요 산술 연산 명령어                                  │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【1. ADD (Addition)】                                              │
│  ──────────────────────                                             │
│  ADD dest, src        ; dest = dest + src                           │
│                                                                     │
│  ADD EAX, EBX         ; EAX = EAX + EBX                             │
│  ADD EAX, 10          ; EAX = EAX + 10                              │
│  ADD EAX, [mem]       ; EAX = EAX + Mem[mem]                        │
│                                                                     │
│  ; ARM (3-피연산자)                                                  │
│  ADD R0, R1, R2       ; R0 = R1 + R2                                │
│  ADD R0, R1, #10      ; R0 = R1 + 10                                │
│                                                                     │
│  【2. SUB (Subtraction)】                                           │
│  ──────────────────────────                                         │
│  SUB dest, src        ; dest = dest - src                           │
│                                                                     │
│  SUB EAX, EBX         ; EAX = EAX - EBX                             │
│  SUB EAX, 1           ; EAX = EAX - 1                               │
│                                                                     │
│  ; 비교: SUB와 같으나 결과 저장 안함                                │
│  CMP EAX, EBX         ; EAX - EBX (플래그만 설정)                   │
│                                                                     │
│  【3. INC/DEC (Increment/Decrement)】                               │
│  ──────────────────────────────────────                             │
│  INC EAX              ; EAX++                                       │
│  DEC EAX              ; EAX--                                       │
│                                                                     │
│  ; 특징: CF 플래그 변경 안함                                         │
│  ; INC 0xFF → 0x00, CF=0 (원래 CF 유지)                             │
│  ; ADD 0xFF, 1 → CF=1                                               │
│                                                                     │
│  【4. MUL/IMUL (Multiplication)】                                    │
│  ────────────────────────────────────                               │
│  ; x86 - 1-피연산자 형식                                             │
│  MUL EBX              ; EDX:EAX = EAX * EBX (부호없음)              │
│  IMUL EBX             ; EDX:EAX = EAX * EBX (부호있음)              │
│                                                                     │
│  ; x86 - 2/3-피연산자 형식                                           │
│  IMUL EAX, EBX        ; EAX = EAX * EBX                             │
│  IMUL EAX, EBX, 10    ; EAX = EBX * 10                              │
│                                                                     │
│  ; ARM                                                               │
│  MUL R0, R1, R2       ; R0 = R1 * R2                                │
│                                                                     │
│  【5. DIV/IDIV (Division)】                                         │
│  ──────────────────────────                                         │
│  ; x86: 피제수는 EDX:EAX (64비트) 또는 EAX (32비트)                  │
│  DIV EBX              ; EAX = EDX:EAX / EBX (몫)                    │
│                       ; EDX = EDX:EAX % EBX (나머지)                │
│                                                                     │
│  ; 8비트 나눗셈                                                      │
│  DIV BL               ; AL = AX / BL, AH = AX % BL                  │
│                                                                     │
│  【6. ADC/SBB (Add/Sub with Carry)】                                │
│  ────────────────────────────────────                               │
│  ; 다중 정밀도 연산                                                  │
│  ADD EAX, EBX         ; 하위 32비트                                  │
│  ADC EDX, ECX         ; 상위 32비트 + 캐리                          │
│                                                                     │
│  SUB EAX, EBX         ; 하위 32비트                                  │
│  SBB EDX, ECX         ; 상위 32비트 - 빌림                          │
│                                                                     │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │  오버플로우 감지:                                                  │ │
│  │  ──────────────────                                               │ │
│  │  부호없음: CF = 1                                                 │ │
│  │  부호있음: OF = 1                                                 │ │
│  │                                                               │ │
│  │  예: 8비트 연산                                                    │ │
│  │  MOV AL, 255                                                     │ │
│  │  ADD AL, 1           ; AL=0, CF=1, OF=0 (부호없음 오버플로우)     │ │
│  │                                                               │ │
│  │  MOV AL, 127                                                     │ │
│  │  ADD AL, 1           ; AL=128, CF=0, OF=1 (부호있음 오버플로우)   │ │
│  └───────────────────────────────────────────────────────────────┘ │
└─────────────────────────────────────────────────────────────────────┘
```

## Ⅲ. 산술 연산 × 타 과목 융합

### 과목 융합 분석

```
┌─────────────────────────────────────────────────────────────────────┐
│                산술 연산 × 타 과목 융합                               │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【컴파일러】                                                        │
│  • 상수 폴딩: 컴파일 타임 계산                                       │
│  • 강도 절감: 곱셈→시프트, 나눗셈→시프트                            │
│  • 루프 최적화: 카운터 감소                                          │
│  • LEA 활용: 복잡한 주소 계산                                        │
│                                                                     │
│  【알고리즘】                                                        │
│  • 누적합: ADD 반복                                                  │
│  • 카운팅: INC/DEC                                                   │
│  • 해시: 곱셈+나눗셈                                                 │
│  • 모듈러: DIV 후 나머지                                             │
│                                                                     │
│  【자료구조】                                                        │
│  • 포인터 연산: ADD/SUB                                             │
│  • 배열 인덱스: 곱셈+덧셈                                           │
│  • 스택: SP 증감                                                    │
│  • 힙: 블록 크기 계산                                                │
│                                                                     │
│  【수치 해석】                                                       │
│  • 부동소수점: FADD, FSUB, FMUL, FDIV                               │
│  • 고정소수점: 정수 연산으로 구현                                    │
│  • 다중 정밀도: ADC/SBB                                              │
│  • 포화 연산: SIMD                                                   │
│                                                                     │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │  컴파일러 최적화 예시:                                            │ │
│  │  ────────────────────────────                                     │ │
│  │  // C: x * 2                                                      │ │
│  │  SHL EAX, 1            ; 곱셈 대신 시프트                         │ │
│  │                                                               │ │
│  │  // C: x / 4                                                      │ │
│  │  SAR EAX, 2            ; 나눗셈 대신 산술 시프트                  │ │
│  │                                                               │ │
│  │  // C: x % 8                                                      │ │
│  │  AND EAX, 7            ; 나눗셈 대신 AND                          │ │
│  │                                                               │ │
│  │  // C: a + b + c                                                  │ │
│  │  LEA EAX, [EBX + ECX + EDX]  ; 1명령어로 3개 더하기               │ │
│  └───────────────────────────────────────────────────────────────┘ │
└─────────────────────────────────────────────────────────────────────┘
```

## Ⅳ. 실무 적용

### 산술 연산 분석

```
┌─────────────────────────────────────────────────────────────────────┐
│                산술 연산 분석                                         │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【abs 함수】                                                        │
│  ──────────────────                                                  │
│  ; int abs(int x)                                                   │
│  abs:                                                               │
│      MOV EAX, EDI         ; x                                       │
│      CDQ                 ; EAX 부호 확장 → EDX                      │
│      XOR EAX, EDX         ; 조건부 보수                             │
│      SUB EAX, EDX         ; 결과                                    │
│      RET                                                          │
│                                                                     │
│  【64비트 덧셈 (32비트 CPU)】                                        │
│  ──────────────────────────────                                     │
│  ; struct uint64 { uint32 lo, hi; };                                │
│  ; a + b                                                            │
│  MOV EAX, [a_lo]                                                   │
│  ADD EAX, [b_lo]         ; 하위 32비트                              │
│  MOV [result_lo], EAX                                              │
│  MOV EAX, [a_hi]                                                   │
│  ADC EAX, [b_hi]         ; 상위 32비트 + 캐리                       │
│  MOV [result_hi], EAX                                              │
│                                                                     │
│  【GDB 디버깅】                                                      │
│  ──────────────────                                                  │
│  (gdb) p/x $eflags        ; 플래그 확인                             │
│  (gdb) p ($eflags & 0x801)  ; OF(11), SF(7), ZF(6), CF(0)          │
│                                                                     │
│  ; 조건 코드 확인                                                    │
│  (gdb) p/t (($eflags >> 6) & 1)  ; ZF                               │
│  (gdb) p/t (($eflags >> 0) & 1)  ; CF                               │
│  (gdb) p/t (($eflags >> 11) & 1) ; OF                               │
│                                                                     │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │  산술 연산 체크리스트:                                            │ │
│  │  ────────────────────────                                        │ │
│  │  □ 오버플로우 가능성 확인                                         │ │
│  │  □ 부호있음/없음 구분                                             │ │
│  │  □ 0으로 나누기 처리                                              │ │
│  │  □ 정밀도 손실 확인                                               │ │
│  │  □ DIV 전 EDX 초기화 (CDQ)                                        │ │
│  └───────────────────────────────────────────────────────────────┘ │
└─────────────────────────────────────────────────────────────────────┘
```

## Ⅴ. 기대효과 및 결론

### 핵심 요약

```
• 산술 연산 명령어: ADD, SUB, MUL, DIV, INC, DEC
• 플래그: Z, N, C, V (오버플로우 감지)
• RISC: 3-피연산자 (목적지 별도)
• CISC: 2-피연산자 (목적지 = 원본)
• 최적화: 시프트, LEA 활용
```

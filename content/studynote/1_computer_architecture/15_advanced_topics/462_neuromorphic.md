# 462. 뉴로모픽 컴퓨팅 (Neuromorphic Computing)

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 뇌의 신경망 구조를 모방한 컴퓨팅 아키텍처
> 2. **가치**: 에너지 효율, 실시간 학습, 이벤트 기반
> 3. **융합**: 스파이킹 NN, NPU, Intel Loihi, BrainChip

---

## Ⅰ. 개요

### 개념 정의

뉴로모픽 컴퓨팅(Neuromorphic Computing)은 **인간 뇌의 신경망 구조와 동작 원리를 하드웨어로 구현하여, 에너지 효율적이고 실시간 학습이 가능한 컴퓨팅 시스템**이다.

### 💡 비유: 인공 뇌

뉴로모픽 컴퓨팅은 **인공 뇌**와 같다. 뇌가 있다. (신경망) 뉴런이 있다. (인공 뉴런) 시냅스가 있다. (연결) 전기 신호. (스파이크) 학습한다. (가중치 변경) 기억한다. (연결 강화) 뉴로모픽이 똑같다. 신경망 구조, 스파이크, 학습.

### 뉴로모픽 컴퓨팅 개요

```
┌─────────────────────────────────────────────────────────────────────┐
│                뉴로모픽 컴퓨팅 개요                                       │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【기존 NN vs 스파이킹 NN】                                                       │)
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │                                                               │ │
│  │  특성           기존 NN               스파이킹 NN               │ │
│  │  ────           ────────             ──────────                │ │
│  │  신호           연속값                이진 스파이크               │ │
│  │  시간           프레임 기반            이벤트 기반                │ │
│  │  전력           항상 소비              활성 시만                  │ │
│  │  학습           역전파                 STDP                      │ │
│  │  구현           GPU/NPU              뉴로모픽 칩                 │ │
│  │                                                               │ │
│  └───────────────────────────────────────────────────────────────┘ │
│                                                                     │
│  【스파이킹 뉴런 모델】                                                           │)
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │                                                               │ │
│  │  모델              설명                    특징                   │ │
│  │  ────              ────                    ────                   │ │
│  │  LIF             Leaky Integrate-Fire    단순, 효율             │ │
│  │  Izhikevich      2변수 모델              생체 모사               │ │
│  │  Hodgkin-Huxley  생체 물리 모델           정확, 복잡              │ │
│  │                                                               │ │
│  │  LIF (Leaky Integrate-and-Fire):                                         │ │
│  │  • 입력 전위 누적                                                          │ │
│  │  • 임계값 도달 시 스파이크 발생                                              │ │
│  │  • 발생 후 리셋                                                            │ │
│  │                                                               │ │
│  └───────────────────────────────────────────────────────────────┘ │
│                                                                     │
│  【STDP 학습】                                                                   │)
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │                                                               │ │
│  │  Spike-Timing-Dependent Plasticity:                                       │ │
│  │                                                               │ │
│  │  • pre-spike → post-spike: 연결 강화 (LTP)                                 │ │
│  │  • post-spike → pre-spike: 연결 약화 (LTD)                                 │ │
│  │  • 시간 차이에 따른 가중치 변화                                               │ │
│  │                                                               │ │
│  │  Δw = f(Δt)  where Δt = t_post - t_pre                                    │ │
│  │                                                               │ │
│  └───────────────────────────────────────────────────────────────┘ │
└─────────────────────────────────────────────────────────────────────┘
```

## Ⅳ. 실무 적용

### 뉴로모픽 컴퓨팅 분석

```
┌─────────────────────────────────────────────────────────────────────┐
│                뉴로모픽 컴퓨팅 분석                                       │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【Intel Lava 예시】                                                              │)
│  ──────────────────                                                  │
│  from lava.magma.core.process.process import AbstractProcess         │
│  from lava.magma.core.process.ports.ports import InPort, OutPort     │
│                                                                     │
│  class LIFProcess(AbstractProcess):                                  │
│      def __init__(self, **kwargs):                                   │
│          super().__init__(**kwargs)                                  │
│          self.a_in = InPort(shape=(10,))                             │
│          self.s_out = OutPort(shape=(10,))                           │
│                                                                     │
│  【snnTorch 예시】                                                                │)
│  ──────────────────                                                  │
│  import snntorch as snn                                              │
│  import torch                                                        │
│                                                                     │
│  # 스파이킹 뉴런                                                        │
│  lif = snn.Leaky(beta=0.9)                                           │
│                                                                     │
│  # 순전파                                                              │
│  spk, mem = lif(cur_in, mem)                                         │
│                                                                     │
│  【BrainChip Akida】                                                              │)
│  ──────────────────                                                  │
│  from akida import Model, FullyConnected                             │
│                                                                     │
│  model = Model()                                                     │
│  model.add(FullyConnected(num_neurons=100))                          │
│  model.fit(x_train, y_train)                                         │
│                                                                     │
│  【뉴로모픽 팁】                                                                    │)
│  ──────────────────                                                  │
│  1. 데이터: 이벤트 기반 변환 (스파이크)                                      │
│  2. 모델: SNN 아키텍처 설계                                               │
│  3. 학습: STDP 또는 ANN→SNN 변환                                         │
│  4. 하드웨어: Loihi, Akida, Mythic                                      │
│  5. 프레임워크: Lava, snnTorch                                          │
│  6. 최적화: 에너지 vs 정확도                                               │
└─────────────────────────────────────────────────────────────────────┘
```

## Ⅴ. 기대효과 및 결론

### 핵심 요약

```
• 뉴로모픽 컴퓨팅: 뇌 구조 모방 하드웨어
• 스파이킹 NN: 이진 스파이크 기반 신경망
• LIF: Leaky Integrate-and-Fire
• STDP: 시간 기반 학습 규칙
• Intel Loihi: 대표 뉴로모픽 칩
• 이벤트 기반: 활성 시만 전력 소비
• 장점: 에너지 효율, 실시간 학습
• 단점: 초기 단계, 호환성
• 활용: 엣지 AI, 로보틱스, BCI
```

# 466. 3D XPoint / Optane

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: Intel/Micron 공동 개발한 차세대 비휘발성 메모리
> 2. **가치**: NVMe 속도, 비휘발성, 용량
> 3. **융합**: SCM, PCM, Intel Optane, Persistent Memory

---

## Ⅰ. 개요

### 개념 정의

3D XPoint(Intel 브랜드명: Optane)은 **Intel과 Micron이 공동 개발한 교차점(Cross-Point) 아키텍처 기반 비휘발성 메모리 기술**이다.

### 💡 비유: 듀얼 레이어 스토리지
3D XPoint는 **듀얼 레이어 스토리지**와 같다. SSD가 있다. (빠름, 비쌂) HDD가 있다. (느림, 저렴) 3D XPoint가 있다. (중간, 중간) 두 세계를 잇는다. 3D XPoint가 똑같다. DRAM 속도, NAND 용량, 비휘발성.

```
┌─────────────────────────────────────────────────────────────────────┐
│                3D XPoint 분석                                            │

│                                                                     │

│  ──────────────────                                                  │
│  # NVMe 장치 목록                                                           │
│  $ sudo nvme list                                                         │
│                                                                     │
│  # 장치 상세 정보                                                            │
│  $ sudo nvme id-ctrl /dev/nvme0                                           │
│                                                                     │

│  ──────────────────                                                  │
│  # DAX 모드로 마운트                                                         │
│  $ sudo mount -o dax /dev/pmem0 /mnt/pmem                                │
│                                                                     │
│  # ext4 DAX                                                                │
│  $ sudo mkfs.ext4 -F /dev/pmem0                                          │
│  $ sudo mount -o dax /dev/pmem0 /mnt/pmem                                │
│                                                                     │
│  【PMDK 예시】                                                                    │)
│  ──────────────────                                                  │
│  # libpmemobj 예시                                                          │
│  #include <libpmemobj.h>                                                   │
│  TOID(struct my_data) data;                                                │
│  PMEMobjpool *pop = pmemobj_open("/mnt/pmem/pool", "layout");             │
│  data = POBJ_ROOT(pop, root);                                              │
│  D_RW(data)->value = 42;  // 직접 쓰기                                     │
│                                                                     │
│  【Optane 팁】                                                                    │)
│  ──────────────────                                                  │
│  1. 마운트: DAX 모드 필수                                                  │
│  2. 애플리케이션: PMEM 인식 코드 작성                                        │
│  3. 백업: 정기적 백업 필요                                                  │
│  4. 내구성: 쓰기 내구성 모니터링                                             │
│  5. BIOS: PMEM 지원 확인                                                   │
│  6. NUMA: NUMA 인식 애플리케이션                                            │
└─────────────────────────────────────────────────────────────────────┘
```

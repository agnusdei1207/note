## Ⅰ. 개요

### 개념 정의

양자 컴퓨팅(Quantum Computing)은 **양자역학 원리(중첩, 얽힘)를 활용하여 특정 연산을 고전 컴퓨터보다 획기적으로 빠르게 수행하는 컴퓨팅 패러다임**이다.

### 💡 비유: 동전 회전

양자 컴퓨팅은 **동전 회전**과 같다. 동전이 있다. (큐비트) 앞뒤가 있다. (0, 1) 회전시킨다. (중첩) 앞뒤 동시에. (중첩 상태) 멈춘다. (측정) 앞 또는 뒤. (결과) 양자 컴퓨팅이 똑같다. 중첩, 얽힘, 병렬 연산.

│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │                                                               │ │
│  │  특성           고전 비트              큐비트                    │ │

│  │  병렬성          없음                  중첩으로 병렬               │ │
│  │  복사           가능                  불가능 (복제 불가)           │ │
│  │                                                               │ │
│  │  큐비트 상태: |ψ⟩ = α|0⟩ + β|1⟩  (|α|² + |β|² = 1)                          │ │
│  │                                                               │ │

│  │  ────              ────                    ────                   │ │

│  │  얽힘(Entanglement)   두 큐비트 상관관계      |00⟩ + |11⟩          │ │
│  │  간섭(Interference)   확률 진폭 합/차        |α+β|², |α-β|²        │ │
│  │  측정(Measurement)    상태 결정             파동 함수 붕괴          │ │
│  │                                                               │ │
│  └───────────────────────────────────────────────────────────────┘ │

│  【n 큐비트의 상태 공간】                                                         │)

│  ┌───────────────────────────────────────────────────────────────┐ │

│  │                                                               │ │
│  └───────────────────────────────────────────────────────────────┘ │

## Ⅳ. 실무 적용

### 양자 컴퓨팅 분석

```
┌─────────────────────────────────────────────────────────────────────┐
│                양자 컴퓨팅 분석                                           │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【Qiskit 기본 (IBM Quantum)】                                                    │)
│  ──────────────────                                                  │
│  # 설치                                                                   │
│  $ pip install qiskit                                                     │
│                                                                     │
│  # 기본 회로                                                              │
│  from qiskit import QuantumCircuit, execute, Aer                        │
│                                                                     │
│  qc = QuantumCircuit(2, 2)                                              │
│  qc.h(0)          # Hadamard 게이트                                     │
│  qc.cx(0, 1)      # CNOT 게이트 (얽힘)                                   │
│  qc.measure([0,1], [0,1])  # 측정                                        │
│                                                                     │
│  # 실행                                                                  │
│  backend = Aer.get_backend('qasm_simulator')                            │
│  result = execute(qc, backend, shots=1000).result()                     │
│  counts = result.get_counts()                                            │
│                                                                     │
│  【IBM Quantum Lab】                                                             │)
│  ──────────────────                                                  │
│  # https://quantum-computing.ibm.com/                                     │

│  # 시뮬레이터 + 실제 하드웨어                                              │
│                                                                     │
│  【양자 컴퓨팅 팁】                                                              │)
│  ──────────────────                                                  │
│  1. 기초: 선형대수, 양자역학 기본                                          │
│  2. SDK: Qiskit, Cirq, Pennylane                                         │
│  3. 실습: IBM Quantum Lab                                                 │
│  4. 알고리즘: Shor, Grover, VQE                                           │
│  5. 응용: 화학, 최적화, 머신러닝                                           │
│  6. 동향: 양자 우위, 양자 내성 암호                                         │

```

## Ⅴ. 기대효과 및 결론

### 핵심 요약

```
• 양자 컴퓨팅: 양자역학 기반 연산
• 큐비트: 중첩 상태의 양자 비트
• 중첩: 여러 상태 동시 존재
• 얽힘: 두 큐비트 간 상관관계
• 게이트: Hadamard, CNOT, Pauli
• Shor: 소인수분해 (RSA 위협)
• Grover: 검색 가속
• 장점: 특정 문제 기하급수적 가속
• 단점: 오류율, 극저온, 큐비트 수
• 활용: 화학, 최적화, 암호, ML
```

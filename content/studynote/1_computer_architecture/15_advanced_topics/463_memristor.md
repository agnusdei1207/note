# 463. 멤리스터 (Memristor)

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 전류 이력에 따라 저항이 변하는 비휘발성 메모리
> 2. **가치**: DRAM+플래시 장점, 아날로그 연산
> 3. **융합**: ReRAM, 뉴로모픽, 인메모리 컴퓨팅

---

## Ⅰ. 개요

### 개념 정의

멤리스터(Memristor)는 **전류가 흐른 이력(history)에 따라 저항값이 변하고, 전원 차단 후에도 상태가 유지되는 비휘발성 메모리 소자**다.

### 💡 비유: 파이프 물높이

멤리스터는 **파이프 물높이**와 같다. 파이프가 있다. (멤리스터) 물이 흐른다. (전류) 물높이 변한다. (저항) 물 끊어도 높이 유지. (비휘발성) 다시 물 흐르면 이어서. (상태 유지) 멤리스터가 똑같다. 전류 이력, 저항 변화, 비휘발성.

```
│    --ioengine=libaio --direct=1 --rw=randread --bs=4k \                  │
│    --numjobs=1 --size=1G --runtime=60 --group_reporting                   │
│                                                                     │
│  【MRAM 장치】                                                                  │)
│  ──────────────────                                                  │
│  # MRAM 메모리 정보 (Everspin)                                              │
│  $ cat /proc/mtd                                                          │
│                                                                     │
│  【ReRAM 시뮬레이터】                                                            │)
│  ──────────────────                                                  │
│  # Memristor 시뮬레이션 (Python)                                            │
│  import numpy as np                                                        │
│  # 간단한 멤리스터 모델                                                      │
│  def memristor(v, w):                                                      │
│      # w: 상태 변수 (0-1)                                                   │
│      dw = v * (1 - w) if v > 0 else v * w                                  │
│      return w + dw, w * v  # 새 상태, 전류                                   │
│                                                                     │
│  【멤리스터 팁】                                                                │)
│  ──────────────────                                                  │
│  1. 내구성: 쓰기 사이클 제한 고려                                             │
│  2. 변동성: 소자 간 편차 보정                                                │
│  3. 열: 발열 관리                                                           │
│  4. 스케일링: 나노미터 공정                                                   │
│  5. 신뢰성: 데이터 유지 시간                                                  │
│  6. 비용: DRAM/NAND 대비 가격                                               │
└─────────────────────────────────────────────────────────────────────┘
```

## Ⅴ. 기대효과 및 결론

### 핵심 요약

```
• 멤리스터: 전류 이력에 따라 저항 변화
• 비휘발성: 전원 차단 후 상태 유지
• ReRAM: 저항 변화 메모리
• PCM: 상변화 메모리
• MRAM: 자기 저항 메모리
• 인메모리 컴퓨팅: 메모리에서 직접 연산
• 폰 노이만 병목: 메모리-프로세서 간격
• 장점: 속도+비휘발성+밀도
• 단점: 내구성, 변동성, 상용화
• 활용: 저장, AI, 뉴로모픽
```

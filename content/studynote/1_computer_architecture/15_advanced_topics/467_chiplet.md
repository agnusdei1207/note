# 467. 칩렛 아키텍처 (Chiplet Architecture)

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 여러 작은 칩을 패키지에 통합한 모듈형 CPU/GPU
> 2. **가치**: 수율 향상, 비용 절감, 유연한 설계
> 3. **융합**: EMIB, Foveros, Infinity Fabric, UCIe

---

## Ⅰ. 개요

### 개념 정의

칩렛 아키텍처(Chiplet Architecture)는 **하나의 큰 단일 칩(Monolithic) 대신 여러 작은 칩(Chiplet)을 패키지에 통합하여 성능과 수율을 최적화하는 설계 방식**이다.

### 💡 비유: LEGO 블록
칩렛 아키텍처는 **LEGO 블록**과 같다. 큰 블록이 있다. (단일 칩) 만들기 어렵다. (수율 낮음) 비싸다. 작은 블록들이 있다. (칩렛) 조립한다. (패키징) 쉽다. (수율 높음) 유연하다. 칩렛이 똑같다. 모듈형 설계, 수율 향상, 유연성.

## Ⅱ. 칩렛 아키텍처 상세

### 상세 분석
```
┌─────────────────────────────────────────────────────────────────────┐
│                칩렛 아키텍처 상세                                           │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  【Intel 패키징 기술】                                                            │)
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │                                                               │ │
│  │  EMIB (Embedded Multi-die Interconnect Bridge):                           │ │
│  │  ┌─────────────────────────────────────────────────────────┐ │ │
│  │  │  칩렛 A     │←── EMIB ──→│     칩렛 B                     │ │ │
│  │  │             │   (실리콘 브리지)  │                             │ │ │
│  │  └─────────────────────────────────────────────────────────┘ │ │
│  │  • 실리콘 브리지로 칩렛 연결                                              │ │
│  │  • 2D 패키징                                                             │ │
│  │  • 고속 인터커넥트 (224 GT/s)                                             │ │
│  │                                                               │ │
│  │  Foveros:                                                                  │ │
│  │  ┌─────────────────────────────────────────────────────────┐ │ │
│  │  │                 ┌─────┐                                 │ │ │
│  │  │  칩렛 A         │ TSV │         칩렛 B                   │ │ │
│  │  │                 │     │                                 │ │ │
│  │  │                 └─────┘                                 │ │ │
│  │  └─────────────────────────────────────────────────────────┘ │ │
│  │  • 3D 스태킹                                                             │ │
│  │  • TSV (Through-Silicon Via)                                             │ │
│  │  • 높은 밀도                                                              │ │
│  │                                                               │ │
│  └───────────────────────────────────────────────────────────────┘ │
│                                                                     │
│  【UCIe (Universal Chiplet Interconnect Express)】                               │)
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │                                                               │ │
│  │  특징:                                                                  │ │
│  │  • 개방형 표준 (Intel, AMD, ARM, TSMC 등)                                   │ │
│  │  • 물리 계층, 프로토콜 계층, 소프트웨어 스택                                 │ │
│  │  • 224 GT/s, 446 GT/s 속도                                               │ │
│  │  • 전력 효율적                                                            │ │
│  │                                                               │ │
│  │  목표:                                                                  │ │
│  │  • 칩렛 생태계 표준화                                                      │ │
│  │  • 벤더 간 호환성                                                         │ │
│  │  • Plug-and-Play 칩렛                                                   │ │
│  │                                                               │ │
│  └───────────────────────────────────────────────────────────────┘ │
│                                                                     │
│  【장단점】                                                                      │)
│  ──────────────────                                                  │
│  ┌───────────────────────────────────────────────────────────────┐ │
│  │                                                               │ │
│  │  장점:                                                                  │ │
│  │  • 수율 향상 (작은 칩 = 높은 수율)                                          │ │
│  │  • 비용 절감                                                             │ │
│  │  • 공정 혼합 (CPU 5nm + I/O 14nm)                                         │ │
│  │  • 설계 유연성                                                            │ │
│  │  • 모듈형 업그레이드                                                        │ │
│  │                                                               │ │
│  │  단점:                                                                  │ │
│  │  • 인터커넥트 복잡성                                                       │ │
│  │  • 지연 증가 (칩렛 간 통신)                                                 │ │
│  │  • 패키징 비용                                                            │ │
│  │  • 전력 소모                                                              │ │
│  │                                                               │ │
│  └───────────────────────────────────────────────────────────────┘ │
└─────────────────────────────────────────────────────────────────────┘
```

## Ⅴ. 기대효과 및 결론

### 핵심 요약

```
• 칩렛: 작은 칩을 패키지에 통합
• Monolithic: 단일 대형 칩
• EMIB: Intel 실리콘 브리지
• Foveros: Intel 3D 스태킹
• Infinity Fabric: AMD 인터커넥트
• UCIe: 범용 칩렛 인터커넥트 표준
• CCD: Core Complex Die (AMD)
• IOD: I/O Die
• 장점: 수율, 비용, 유연성
• 단점: 지연, 복잡성, 패키징 비용
• 활용: CPU, GPU, AI 가속기
```

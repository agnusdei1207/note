# 16. FET (Field-Effect Transistor, 전계효과 트랜지스터)

## 핵심 인사이트 (3줄 요약)
> 1. **본질**: FET은 게이트에 인가되는 전압(전기장, 전계)을 이용하여 소스와 드레인 사이의 전류 흐름을 제어하는 단극성(Unipolar) 전압 제어형 반도체 소자다.
> 2. **가치**: 입력 임피던스가 사실상 무한대에 가까워 입력 전류(전력 소모)가 없으며, 고집적화가 용이하여 현대 VLSI(초고밀도 집적회로) 혁명의 물리적 기반을 제공했다.
> 3. **융합**: JFET에서 출발하여 산화막을 더한 MOSFET으로 발전했으며, 이는 CPU, 메모리(DRAM, NAND), 아날로그 스위치 등 모든 컴퓨팅 시스템의 근원적 빌딩 블록이 되었다.

---

## Ⅰ. 개요 (Context & Background)

- **개념**: FET(전계효과 트랜지스터)은 채널을 형성하는 한 가지 종류의 전하 캐리어(전자 또는 정공)만을 사용하여 전류를 전도하는 소자다. 게이트(Gate) 전극에 걸린 전압이 생성하는 전기장(전계, Field-Effect)으로 채널의 폭이나 캐리어 농도를 변화시켜 소스(Source)에서 드레인(Drain)으로 흐르는 전류를 통제한다.
- **💡 비유**: FET은 수도관(채널) 밖에서 호스를 꽉 쥐거나 놓아서 물(전류)의 흐름을 조절하는 것과 같다. 호스 안으로 물이 직접 새어 들어가지 않듯(입력 전류 0), 외부의 압력(전압)만으로 제어한다.
- **등장 배경**:
  1. **BJT의 한계 극복**: BJT는 제어를 위해 베이스 전류가 계속 흘러야 하므로 전력 소모와 발열이 크고, 대규모 집적(LSI) 시 칩이 녹아버리는 "열 장벽(Thermal Wall)"에 부딪혔다.
  2. **전압 제어 패러다임**: 1950~60년대, 입력 전류 없이 전압만으로 스위칭을 구현하고자 하는 연구가 결실을 맺어 JFET과 금속-산화물 반도체(MOS) 구조가 실용화되었다.
  3. **디지털 시대로의 전환**: 저전력, 고집적 요구사항이 폭발적으로 증가하면서 반도체 산업의 메인스트림은 BJT에서 완벽하게 FET(특히 MOSFET)으로 이동했다.

## Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

### 구성 요소 (기본 JFET / MOSFET 공통 개념)

| 요소명 | 역할 | 내부 동작 | 특성 | 비유 |
|:---|:---|:---|:---|:---|
| **소스 (Source)** | 전하 캐리어 공급 | 채널로 전자(또는 정공)를 쏟아냄 | 전류의 기점 | 수원지 (호스 입구) |
| **드레인 (Drain)** | 전하 캐리어 수집 | 채널을 통과한 전자(또는 정공)를 받아냄 | 전류의 종점 | 배수구 (호스 출구) |
| **게이트 (Gate)** | 전류 제어 (전기장 형성) | 인가된 전압이 공핍층을 확장/축소시키거나(JFET) 반전층을 형성(MOSFET) | 채널과 절연됨 | 호스를 쥐는 손 |
| **채널 (Channel)** | 전하 이동 통로 | 소스와 드레인 사이의 물리적 혹은 유도된 전도 경로 | 폭/두께에 따라 저항 변화 | 물 호스 |
| **기판 (Body/Substrate)** | 소자 형성의 물리적 기반 | 문턱 전압(Threshold Voltage)에 영향을 주는 바디 효과 유발 | 4번째 단자로 작용하기도 함 | 대지(Ground) |

### 전기장 제어 메커니즘 (N-Channel 기준)

게이트 전압이 채널의 전도도를 어떻게 바꾸는지 이해하는 것이 핵심이다. 기본적으로 소스와 드레인 사이에 전압(V_DS)을 걸어 물이 흐를 준비를 한 뒤, 게이트 전압(V_GS)으로 밸브를 조작한다.

```text
 ┌───────────────────────────────────────────────────────────────┐
 │               FET(전계효과)의 채널 제어 원리 도해                │
 ├───────────────────────────────────────────────────────────────┤
 │                                                               │
 │   [ V_GS = 0V : 채널 열림 (JFET 기준) ]                        │
 │                                                               │
 │       Source (N)                        Drain (N)             │
 │        ┌───┐                              ┌───┐               │
 │        │ e-│ ◀────── 넓은 채널 ────────▶ │ e-│               │
 │        │   │ ──────────────────────────── │   │               │
 │        └───┘                              └───┘               │
 │               ↑ 게이트 전기장 없음 ↑                            │
 │                                                               │
 │   [ V_GS = -V (음전압) : 핀치오프(Pinch-off) 발생 ]              │
 │                                                               │
 │       Source (N)                        Drain (N)             │
 │        ┌───┐    공핍층(Depletion Region)   ┌───┐               │
 │        │ e-│ ◀──██████████████████████──▶ │ e-│               │
 │        │   │ ───채널이 막힘(전류 차단)──── │   │               │
 │        └───┘                              └───┘               │
 │               ↑ 게이트의 강한 전기장 ↑                           │
 │               (전자를 밀어내어 길을 없앰)                        │
 └───────────────────────────────────────────────────────────────┘
```

**[다이어그램 해설]** JFET(Junction FET)을 예로 들면, 게이트 전압이 0일 때는 N형 채널을 통해 전자(e-)가 소스에서 드레인으로 자유롭게 흐른다(저항이 작음). 게이트(P형)에 역방향 바이어스인 음전압(-V)을 인가하면, PN 접합 부위에 전하 캐리어가 없는 "공핍층(Depletion Region)"이 두껍게 형성된다. 게이트 전압을 계속 내리면 공핍층이 확장되어 마침내 채널 전체를 막아버리는데, 이를 **핀치오프(Pinch-off)**라고 한다. 이때는 전류가 완전히 차단되어 스위치가 OFF된 것과 같다. 중요한 것은 게이트는 역방향 바이어스 상태이므로 전류가 기판으로 새어나가지 않으며, 오직 "전기장의 힘"만으로 채널의 저항을 무한대로 만든다는 점이다.

### 핵심 영역 3단계
1. **차단 영역 (Cut-off)**: 채널이 형성되지 않거나 막혀 전류(I_D)가 0인 상태 (스위치 OFF).
2. **오믹/선형 영역 (Ohmic/Linear)**: V_DS가 작을 때, I_D가 V_DS에 비례하여 증가하는 저항처럼 동작하는 구간.
3. **포화 영역 (Saturation)**: V_DS가 일정 수준을 넘으면 핀치오프가 드레인 쪽에 발생하여, I_D가 더 이상 증가하지 않고 일정하게 유지되는 구간 (아날로그 증폭 또는 디지털 스위치 ON 상태).

## Ⅲ. 융합 비교 및 다각도 분석

### 심층 기술 비교: JFET vs MOSFET

| 비교 항목 | JFET (Junction FET) | MOSFET (Metal-Oxide-Semiconductor FET) |
|:---|:---|:---|
| **게이트 절연 구조** | PN 접합의 역방향 바이어스 이용 (절연막 없음) | 산화막(SiO2)으로 게이트와 채널을 완벽 절연 |
| **입력 저항** | 높음 (약 10^9 Ω) | 극도로 높음 (약 10^14 Ω) |
| **동작 모드** | 공핍형(Depletion mode)만 존재 (기본 ON) | 공핍형 및 증가형(Enhancement, 기본 OFF) 모두 가능 |
| **상업성 / 집적도** | 노이즈 특성이 좋아 특수 오디오, 계측기에 사용 | VLSI 제작에 완벽히 부합, 현대 디지털 반도체의 99% |
| **취약점** | 정전기에 비교적 강함 | 산화막이 매우 얇아 정전기(ESD)에 뚫리기 쉬움 |

### 과목 융합 관점

- **전자회로 / 센서**: 무선 마이크나 고정밀 의료기기의 아날로그 프론트엔드에서는 JFET이 입력단의 버퍼 증폭기로 사용된다. 입력 저항이 커서 미세한 생체 신호를 왜곡 없이 그대로 읽어올 수 있기 때문이다.
- **컴퓨터 구조**: 폰 노이만 아키텍처의 레지스터, ALU, 컨트롤 유닛은 모두 FET(특히 MOSFET)의 스위칭 특성을 활용한 논리 게이트의 조합으로 이루어진다.

## Ⅳ. 실무 적용 및 기술사적 판단

### 실무 시나리오

1. **시나리오 — 정밀 계측기의 입력단 설계**: pH 센서나 피코암페어(pA) 단위의 미세 전류를 측정하는 장비. BJT를 입력단에 쓰면 베이스 전류가 센서 신호를 갉아먹는다(로딩 효과). 아키텍트는 입력 임피던스가 극도로 높은 FET 기반의 연산 증폭기(Op-Amp)를 전위계(Electrometer)로 사용하여 신호 손실을 제로에 가깝게 만든다.
2. **시나리오 — 전원 공급 장치(SMPS) 스위칭**: 수백 kHz로 동작하는 전원 회로에서 스위칭 손실을 줄이는 것이 핵심. 전력용 파워 MOSFET은 On-저항(R_DS(on))이 밀리옴(mΩ) 단위로 매우 작고 스위칭 속도가 빨라 열 발생을 최소화하며 효율적인 전력 변환을 가능하게 한다.
3. **시나리오 — 정전기(ESD) 방지 설계**: MOSFET 소자는 손으로 만지는 것만으로도 게이트 산화막이 파괴될 수 있다. 회로 설계 시 입출력 핀에 클램핑 다이오드(TVS) 회로를 추가하여 외부 서지(Surge) 전압이 칩 내부의 FET 게이트를 파괴하지 못하게 바이패스하는 ESD 보호 전략이 필수적이다.

### 도입 체크리스트
- **기술적**: 고속 스위칭 시 게이트 커패시턴스(C_iss) 충방전을 위한 게이트 드라이버 회로의 전류 공급 능력이 충분한가?
- **운영적**: 조립 라인에서 제전복, 제전 밴드 착용 등 정전기 방지(ESD) 프로세스가 완벽히 준수되고 있는가?

### 안티패턴
- **플로팅 게이트 방치**: 회로 설계 시 FET의 게이트 단자를 연결하지 않고 공중에 띄워두면(Floating), 주변의 미세한 노이즈나 정전기에도 스위치가 무작위로 ON/OFF 되며 회로가 타버릴 수 있다. 반드시 풀업(Pull-up)이나 풀다운(Pull-down) 저항으로 상태를 확정해야 한다.

## Ⅴ. 기대효과 및 결론

### 정량/정성 기대효과

| 구분 | BJT 기반 로직 설계 | FET(CMOS) 기반 설계 | 기대 효과 |
|:---|:---|:---|:---|
| **정량** | 수만 개 집적 시 발열 감당 불가 | 수십 억 개 집적 가능 | 단일 칩 트랜지스터 집적도 **10만 배 향상** |
| **정성** | 입력 전류로 인한 신호 감쇠 발생 | 무한대 입력 저항으로 로딩 효과 없음 | 다단 논리 회로 설계의 간소화 |

### 미래 전망
- 구조적 혁신을 거친 최신 트랜지스터(FinFET, GAA) 역시 본질적인 동작 원리는 "전기장(Field-Effect)으로 채널을 제어한다"는 FET의 근간을 그대로 계승하고 단채널 효과를 억제하는 방향으로 끝없이 진화할 것이다.

### 참고 표준
- **AEC-Q101**: 자동차 전장용 개별 반도체(Discrete FET 포함) 신뢰성 표준. 자율주행차의 전력 제어 모듈 필수 규격.

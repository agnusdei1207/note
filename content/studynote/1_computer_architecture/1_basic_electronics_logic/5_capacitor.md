> **핵심 인사이트**
> 1. 커패시터(Capacitor, 축전기)는 두 도체 판 사이에 절연체(유전체)를 두고 **전하를 일시적으로 저장**하는 전자 부품이다.
> 2. 컴퓨터 아키텍처의 핵심인 **DRAM의 기본 저장 단위(1비트)**로 사용되며, 전하가 있으면 1, 없으면 0으로 인식한다.
> 3. 회로 전원부에서는 급격한 전압 변화를 흡수하여 전원을 안정화시키는 **디커플링 커패시터(Decoupling Capacitor)** 역할을 수행한다.

---

## Ⅰ. 커패시터의 구조와 원리
커패시터는 금속과 같은 두 개의 도체 판(Plate)을 마주 보게 하고, 그 사이에 전기가 통하지 않는 유전체(Dielectric)를 채워 넣은 구조입니다. 
전압을 가하면 한쪽 판에는 양전하(+), 다른 판에는 음전하(-)가 모이게 되며, 유전체가 이 둘이 섞이지 않도록 막아주어 전하가 팽팽하게 대치하는 상태로 저장됩니다. 이 저장 용량을 정전용량(Capacitance, $C$)이라 부릅니다.

```ascii
[커패시터의 기본 구조]

       + 전압 (V)
       |
  +----+----+  도체 극판 (Plate) -> 양전하(+) 축적
  |         |
  | 유전체  |  (Dielectric) -> 전기가 통하지 않음
  |         |
  +----+----+  도체 극판 (Plate) -> 음전하(-) 축적
       |
      GND
```

📢 **섹션 요약 비유**: 커패시터는 '물탱크'나 '댐'과 같습니다. 평소에 전기를 물처럼 가득 채워두었다가, 회로가 필요할 때 밸브를 열어 전기를 한꺼번에 흘려보내거나 천천히 내보냅니다.

---

## Ⅱ. DRAM 아키텍처의 핵심: 1T1C 구조
메인 메모리인 DRAM(Dynamic RAM)은 수십억 개의 셀로 이루어져 있으며, 각 셀은 **1개의 트랜지스터(1T)와 1개의 커패시터(1C)**로 구성됩니다.
커패시터에 전하가 충전되어 있으면 논리 '1', 방전되어 있으면 논리 '0'으로 데이터를 기억합니다. 하지만 커패시터는 시간이 지나면 유전체를 통해 미세하게 전하가 새어나가는(방전되는) 특징이 있습니다.

따라서 데이터가 지워지는 것을 막기 위해 주기적으로 커패시터를 다시 충전해 주어야 하며, 이를 **리프레시(Refresh)**라고 부릅니다. (이 때문에 Dynamic이라는 이름이 붙었습니다.)

```ascii
[DRAM 셀 (1T1C) 구조]

 워드라인(Wordline) ----[ 게이트 ]
                       (트랜지스터)
 비트라인(Bitline)  ----[ 소스/드레인 ]----+
                                       |
                                     -----
                                     ----- (커패시터: 0 또는 1 저장)
                                       |
                                      GND
```

📢 **섹션 요약 비유**: DRAM의 커패시터는 '밑빠진 독'입니다. 물(데이터)을 부어놓아도 조금씩 새어나가기 때문에, 물이 다 빠지기 전에 CPU가 계속해서 다시 물을 가득 채워줘야(리프레시) 데이터를 잃어버리지 않습니다.

---

## Ⅲ. 전원 안정화: 디커플링 커패시터 (Bypass Capacitor)
현대 CPU는 1초에 수십억 번(수 GHz) 스위칭하며 0과 1을 만들어냅니다. 이때 순간적으로 엄청난 양의 전류를 끌어 쓰기 때문에 전압이 뚝 떨어지는 현상(전압 강하)이 발생할 수 있습니다. 

이를 방지하기 위해 CPU 칩 주변이나 메인보드 뒷면에 **디커플링 커패시터(Decoupling Capacitor)**를 무수히 많이 배치합니다. 이들은 평소에 전기를 머금고 있다가 CPU가 갑자기 전기를 요구할 때 마치 '미니 배터리'처럼 즉각적으로 전력을 공급하여 전압을 일정하게 유지(노이즈 필터링)해 줍니다.

```ascii
[디커플링 커패시터의 역할]

 전원(PSU) ------/\/\/\/-----(긴 배선, 저항/인덕터 성분)------+---> CPU (수십억번 스위칭!)
                                                            |
                                                          ----- (디커플링 커패시터)
                                                          ----- : CPU 바로 옆에서
                                                            |     부족한 전력을 즉각 쏴줌
                                                           GND
```

📢 **섹션 요약 비유**: 디커플링 커패시터는 동네의 '물탱크'입니다. 동네 사람들이 동시에 샤워기를 틀면 수도사업소(파워서플라이)에서 물이 오기까지 시간이 걸려 수압이 떨어지지만, 집집마다 있는 물탱크가 즉시 물을 뿜어주어 수압(전압)을 유지해 줍니다.

## 개념 맵 (Concept Map)
```ascii
[커패시터 (Capacitor)]
   │
   ├── 물리적 특성: 전하 저장 (유전체를 사이 둔 두 도체)
   │
   ├── 데이터 저장 (메모리)
   │     └── DRAM 셀 (1T1C 구조): 충전(1) / 방전(0) -> 리프레시 필수
   │
   └── 전원 안정화 (시스템)
         └── 디커플링 커패시터: 고주파 노이즈 제거 및 순간적 전력 공급
```

## 어린이 비유
커패시터는 컴퓨터 마을의 **요술 저금통**이에요! 전기가 남을 때는 뱃속에 꿀꺽꿀꺽 모아두었다가, CPU 친구가 갑자기 밥(전기)이 모자라서 헉헉댈 때 뱉어내서 도와줘요. 또, 기억력을 잃어버리지 않게 비밀번호(1과 0)를 뱃속에 품고 있는 아주 똑똑한 저금통이랍니다.

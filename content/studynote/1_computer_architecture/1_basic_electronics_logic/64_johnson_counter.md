# 존슨 카운터 (Johnson Counter)

## 핵심 인사이트 (3줄 요약)
> 1. **본질**: 존슨 카운터(Johnson Counter)는 링 카운터의 평범한 꼬리 물기 연결을 뒤틀어, 마지막 출력의 **'반전 신호($Q'$)'**를 맨 앞 입력으로 꼬아 넣은 **뫼비우스의 띠형 기형 순환 시프트 레지스터(Twisted Ring Counter)**다.
> 2. **가치**: 링 카운터가 $N$개의 칩 공간을 써서 고작 $N$개의 상태를 내뱉는 극악의 비효율을 타파하여, **정확히 2배인 $2N$개**의 순환 상태 파동을 뽑아내면서도, 디코더 오버헤드를 병아리 눈물만큼만 요구하는 경제적 하드웨어 타협안의 정석이다.
> 3. **융합**: 비트열이 $000 \to 100 \to 110 \to 111 \to 011$ 처럼 구렁이 담 넘어가듯 인접 비트가 단 1개씩만 변동하는 글리치 프리(Glitch-free) 위상을 창조해 내어, 고주파 안테나 위상 천이기(Phase Shifter)나 디지털 모터 정밀 제어의 영구 엔진으로 추앙받는다.

---

## Ⅰ. 개요 (Context & Background)

- **개념**: 존슨 카운터는 4개의 플립플롭을 직렬 연쇄 구조로 세워놓고, 마지막 4번째 놈이 뱉어내는 맑은 출력(Q) 대신 '거꾸로 뒤집힌 출력(Q')' 구리선을 끌고 와 1번째 놈의 입에 쑤셔 넣는 파괴적 결선이다. 그 결과, 처음에 $0000$으로 시작한 칩에 `1`이라는 파도가 스멀스멀 차올라 기어이 $1111$을 꽉 채웠다가, 곧이어 뒤집힌 꼬리의 저주를 받아 다시 `0`의 파도가 스멀스멀 파고들어 $0000$으로 되돌아가는 물결 카운팅을 창조한다. 

- **💡 비유**: 존슨 카운터는 **"기차가 한 바퀴 돌고 나면 색깔이 반대로 뒤집히는 뫼비우스의 롤러코스터"**와 같다. 흰색 기차 칸 4개가 터널을 한 바퀴 통과하면 앞창문부터 검은색 페인트를 뒤집어쓰고 나와 검은색 4칸 기차로 변하고, 다음 트랙에서 터널을 또 돌면 다시 흰색 페인트를 순서대로 뒤집어쓰며 흰색 기차로 백업 복원되는, 2배로 길어지는 기묘한 순환 여정이다.

- **등장 배경 및 기술적 도약**:
  1. **링 카운터 면적 파산의 분노**: 엔지니어들은 링 카운터의 0 클럭 디코딩 파워에 환호했지만, "상태 10번째"를 만들려면 진짜 플립플롭이 물리적으로 10개나 칩에 도배되어 면적($\text{Cost}$)을 박살 낸다는 사실에 경악했다.
  2. **반전 꼬리의 트위스트 기믹**: 누군가가 "맨 끝단에서 전선을 뽑을 때, Q 말고 그 밑에 있는 NOT 게이트 구멍(Q')에서 뽑아서 앞에 꽂으면 어떻게 될까?" 칩을 뒤틀어 연결하자, 플립플롭 5개만 써도 상태가 밀고 당겨지며 무려 10 틱(Tick)짜리 $2N$ 카운팅 시퀀스가 튀어나왔다. 공간을 잡아먹지 않고도 카운팅 용량이 2배로 폭발하는 기형적 증폭이 탄생했다.

- **📢 섹션 요약 비유**: 4명이 앉을 수 있는 식탁(플립플롭)에서, 한 바퀴는 사람들이 차례차례 와서 빈 의자를 꽉 채우고(4틱), 다음 한 바퀴는 앉아있는 사람들이 차례차례 일어나서 식탁을 다시 싹 비우는 동작(4틱)을 구사함으로써, 의자 4개만으로 총 '8장의 연속된 다른 사진(상태)'을 찍어내는 공간 창출 마술과 같습니다.

## Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

### 1. 트위스티드 결속 아키텍처 (Twisted Ring Architecture)

단순히 선 하나를 다른 구멍에 꽂았을 때 일어나는 위대한 변형을 추적해 보자.

```text
  ┌──────────────────────────────────────────────────────────────────┐
  │        존슨 카운터의 뒤틀린 꼬리: 4-bit Johnson Architecture                │
  ├──────────────────────────────────────────────────────────────────┤
  │                                                                  │
  │     (오리지널 클럭선이 4명의 목줄을 관통)                               │
  │    CLK ───────┬──────────────┬──────────────┬─────────────┐      │
  │               │              │              │             │      │
  │            ┌──▼──┐         ┌──▼──┐         ┌──▼──┐        ┌──▼──┐ │
  │        ┌─▶│ D  Q0│── 1 ──▶│ D  Q1│── 2 ──▶│ D  Q2│── 3 ─▶│ D  Q3│ │
  │        │   │ FF0 │         │ FF1 │         │ FF2 │        │ FF3 │ │
  │        │   │   Q0'         │   Q1'         │   Q2'        │   Q3'│ │
  │        │   └─────┘         └─────┘         └─────┘        └──┬──┘ │
  │        │                                                      │   │
  │        └──────────────────── (Twisted Feedback 결속) ─────────┘   │
  │            * Q3가 아닌 반전된 녀석(Q3')의 뒤집힌 파도를 0번 입에 먹인다!   │
  │                                                                  │
  │   [존슨의 2N 물결 상전이 테이블 (8-State Wave)]                       │
  │    CLK   Q0 Q1 Q2 Q3     거동                                     │
  │     0    0  0  0  0    <초기 상태, 텅 빈 파이프>                     │
  │     1    1  0  0  0    (Q3'의 역전파동 '1'이 맨앞으로 기어들어옴)         │
  │     2    1  1  0  0    (1의 파도가 뒤로 밀려감)                      │
  │     3    1  1  1  0    (밀려감)                                    │
  │     4    1  1  1  1    <1로 꽉 찬 팽창의 극점>                       │
  │     5    0  1  1  1    (Q3가 1이 되어, 이번엔 Q3'의 파동 '0'이 맨잎 기어옴)  │
  │     6    0  0  1  1    (0의 빈 파도가 뒤로 까임)                      │
  │     ...                                                           │
  │     8    0  0  0  0    <태초의 진공 상태로 롤백. 무한 반복>             │
  └──────────────────────────────────────────────────────────────────┘
```

**[다이어그램 해설]** 일반적인 이진 카운터가 `000 -> 001 -> 010` 처럼 비트 수들이 마구잡이로 뛰며 글리치 노이즈 소나기를 퍼부을 때, 존슨 카운터의 테이블을 보면 징그럽게도 매 클럭 찰각거릴 때마다 **단 1개의 비트 숫자만 앞뒤로 스무스하게 바뀐다(Gray Code 특성).** $0000 \to 1000$ (1개 파동) $\to 1100$ (1개 파동). 스위칭 노이즈가 최소화되며, 4개의 플립플롭이 빚어낸 8단계의 파형이 겹치면서, 모터나 통신기파의 위상을 섬세하게 밀어주는 가장 예쁜 구형파 계단이 완성된다.

### 2. 가처분 디코딩 (2-Input AND 소거법)

링 카운터는 선만 빼면 출력 해석(디코딩)이 끝났지만, 존슨 카운터는 4개의 칩으로 8개를 쏟아내니 약간의 해석이 필요하다.
- **기적적인 가벼움**: 이진 카운터에서 $110(6)$을 알아내려면 변수 3개를 전부 처박는 3입력 AND 게이트 구조망을 깔아야 한다. 하지만 존슨 카운터의 파도는 비트 1개가 연속으로 이어지는 특성 때문에, **"무조건 주변에 겹치는 변수 2개짜리 AND 게이트 하나"**만 꽂으면 해당 타이밍의 상태 번호를 칼같이 짚어낸다.
- 즉, 타이밍 $T_5$ (`0111`)를 알아내고 싶으면 4핀짜리 뚱뚱한 디코더를 안 쓰고, 그냥 $Q_0'$ 핀과 $Q_1$ 핀 2개만 AND 게이트로 딱 묶어버려도 고유값 해석이 오결점 없이 종결된다.

- **📢 섹션 요약 비유**: 복잡한 금고의 비밀번호 8자리를 눌러야 문이 열리는 게 아니라, "첫 번째 버튼 파란색 켜졌고, 두 번째 버튼 안 켜졌네? 아, 지금 다섯 번째 차례구나!" 하고 단 두 개의 불빛만 주변에서 슬쩍 곁눈질로 조합해도 지금이 몇 번째 턴인지 0.1초 만에 논리적으로 유추 가능한 가벼운 논리 추리 게임과 같습니다.

## Ⅲ. 융합 비교 및 다각도 분석

### 심층 비교 1: 카운터 제국 삼국지 (Binary vs Ring vs Johnson)

3개의 플립플롭(Flip-Flop) 칩을 주었을 때, 아키텍처 결선만으로 갈라지는 극단의 생태계.

| 생태계 지표 | 이진 카운터 (Binary) | 링 카운터 (Ring) | 존슨 카운터 (Johnson / Twisted) |
|:---|:---|:---|:---|
| **창출해 내는 타이밍 틱 수** | $2^3 = 8$ 개의 무지막지한 수용 공간 | 칩 개수 그대로 $= 3$ 개의 상태 국한 | 칩 개수의 2배, $2 \times 3 = 6$ 개의 타협적 틱 |
| **디코더 게이트 사이즈/지연** | 엄청난 지연. 3-to-8 그물망 로직 필수 | 나노초 지연 0. 선만 빼다 직렬 사용 | 2입력 AND 게이트 하나짜리 초경량 해독의 축복 |
| **글리치 (Glitch) 방사선** | 최악. 상태 바뀔 때 비트 여러 개가 한 번에 튀어 스위칭 노이즈 파티 | 없음 (원-핫 특성) | **가장 우수함**. 스무스하게 물결 타듯 그레이 편향 |

### 심층 비교 2: 금지된 이탈 구역 (Illegal State) 폭주 위협

존슨 카운터는 4비트를 썼을 때, 인간이 만들어 둔 정상 궤도 표 8개를 빼고 나면, $2^4 - 8 = 8$ 가지의 **'버려진 이진 쓰레기 진법(Illegal State)'** 구역이 남는다.
우주 방사선 노이즈나 부팅 결함으로 플립플롭이 미쳐서 `0101` 같은 기괴한 징검다리 숫자를 삼키게 되면, 이 비정상 상태들이 지들끼리 뫼비우스 루프를 돌며 칩에 폭주 데스루프를 건다. (이를 Livelock 이라 부름). 존슨 카운터 설계 시, 칩을 합성할 때 비정상 상태들을 감시하다가 발견 즉시 $0000$ 정상 코스로 목덜미를 낚아채 던져버리는 **셀프 스타팅 복구 로직(Self-Correcting NAND Network)** 한 줌을 끼워 넣는 백도어 보호가 필수적이다.

- **📢 섹션 요약 비유**: 존슨 카운터는 효율이 좋아서 작은 배에 많은 짐을 실을 수 있는 절충형 돛단배지만, 설계된 '정상적인 바닷길 8곳'을 벗어나 폭풍우(노이즈)에 휩싸여 '버려진 암초 구역 8곳'으로 튕겨 나가면 배 궤도가 영원히 무한 맴돌기에 빠져버리는, 정밀한 GPS 복구 복구가 생명인 하이리스크 항해와 같습니다.

## Ⅳ. 실무 적용 및 기술사적 판단

### 실무 시나리오: 다상 클럭 통신 믹서 (Multi-Phase Clock Generator)

1. **상황 (Incidence)**: 5G 초고주파 통신 PHY 보드에서 직교 진폭 변조(QAM)를 위해, 메인으로 들어오는 1GHz 클럭 파장을 4등분 하여 0도, 90도, 180도, 270도 위상 간격으로 살짝 밀려있는 4개의 정밀 클럭 파형 무지개가 필요하다.
2. **단순 접근 (Bad Tool Use)**: 위상을 미루는 아날로그 RC 딜레이 지연 회로를 저항으로 덕지덕지 납땜한다. 열받으면 저항값이 변해 90도가 110도로 틀어져 통신이 작살 난다.
3. **기술적 우위 (Johnson 분주 코어)**: 코어 안에 딸랑 플립플롭 2개짜리 `2비트 존슨 카운터`를 하드와이어드 결속한다. 존슨의 물결 파도 $Q_0, Q_1, Q_0', Q_1'$ 4개의 핀에서 뿜어져 나오는 출력은 클럭이 칠 때마다 물리 법칙에 의해 영원불멸 고정된 $90^\circ$ 위상 간격으로 지연된 예쁜 위상을 오버랩(Overlap)하며 뱉어낸다. 
4. **결언 (Outcome)**: 덩치 큰 안테나 위상 천이기 부품 원가를 100% 삭제하고, 고가의 아날로그 도메인을 실리콘 속 가장 차갑고 견고한 디지털 트위스티드 논리로 대체하여 RF-SoC 칩의 극단적 소형화를 이룩해 냈다.

### 도입 체크리스트 (Johnson Sign-off)
- **그레이 코드 (Gray-code) 연속성 결함**: 레이아웃 배선 상에서 클럭 스큐(Clock Skew)가 틀어져, 존슨 카운터 플립플롭이 토글되는 순서 역전 현상이 발생하여 단 하나의 핀만이 스무스하게 0-1로 넘어가는 그레이 위상 불문율이 깨지지 않게 클럭 H-트리 라우팅 검사를 돌렸는가?

- **📢 섹션 요약 비유**: 오케스트라 4대의 바이올린이 동시에 활을 켜기 시작하는 게 아니라, 1번이 켜고(0도), 1초 뒤 2번이 켜고(90도), 1초 뒤 3번이 켜는(180도) 식으로 순서대로 박자를 딱딱 맞춰 환상적인 계단식 반주 화음을 만들어내는 절대음감의 로직 톱니바퀴입니다.

| **융합 (위상 제어 혼합)** | 위상 밀림 현상을 풀기 위해 DSP 계산 로직 S/W 가동 | **구조적 회전 그 자체만으로 4분원 위상 천이($90^\circ$) 즉결** | 5G 빔포밍 안테나 및 SDR 다상 통신 장치의 물리 제어 매크로 장악 |

### 미래 전망
- **양자-존슨 비트열 생성체**: 극저온 양자 환경에서는 노이즈 하나가 큐빗을 죽이기 때문에, 한 번의 클럭 타격에 1개의 비트 전압만 미끌어지듯 뒤집히는 이 존슨 카운터의 '단일 비트 플립 파형(Gray Code Transition)' 철학이 양자 오류 정정(QEC) 표면 코드 제어 회로의 냉동 극저저항 인터페이스 설계 철학으로 통째 계승되고 있다.

### 참고 표준
- **JEDEC 시리얼 버스 랭크**: DDR 메모리 컨트롤러나 고속 SerDes 칩 인터페이스 내에서 채널을 시분할 다중화(TDM)하여 고속으로 번갈아 스위칭할 때, 그 스위치를 다다다닥 통제하기 위한 픽업 타이밍 발생기로 존슨 카운터 회로 IP 코드가 필수적으로 레퍼런스 가이드에 매핑된다.

- **📢 섹션 요약 비유**: 존슨 카운터는 링 카운터의 과소비(플립플롭 낭비)와 이진 카운터의 무거움(디코더 지연) 사이에서, "선을 반대로 뒤틀어 엮기만 하면 2배의 차원이 열린다"는 기적적인 발상을 이뤄낸, 디지털 회로계의 코페르니쿠스적 우주율 왜곡 발명품이라 결론짓습니다.

## 📌 관련 개념 맵 (Knowledge Graph)

| 개념 명칭 | 정교 시너지 매핑 |
|:---|:---|
| **링 카운터 (Standard Ring)** | 존슨의 아버지. 이 녀석이 단순하고 직관적이지만 자리 차지(면적 낭비)가 너무 심해서, 그 뒷머리를 뒤틀어버린 돌연변이가 바로 존슨 마술 카운터다. |
| **그레이 코드 (Gray Code / 1-bit Transition)** | 이진법 파동이 한 칸 넘어갈 때 미쳐 날뛰며 여러 개 비트가 바뀌지 않고, 단!! 한 개의 전구 불만 예쁘게 달칵거리는 고귀한 귀족의 신호 체계. 존슨은 태생부터 이 코드의 저주 혜택을 입고 태어난다. |
| **불법 상태 추락 (Illegal State)** | 플립플롭의 수용량($2^n$)에 비해 효율이 많이 남는($2n$) 존슨 카운터의 유일한 그림자. 시스템이 노이즈를 쳐맞고 우주 미아 상태 루프에 빠지는 데스존. |
| **위상 천이기 (Phase Shifter / Multi-phase)** | 메인 클럭 심장 박동 하나를 칼로 4토막을 예쁘게 스쳐내어, 서로 엇박자로 찰칵찰칵 꽂히는 무지개 클럭 위상을 뱉어내는 통신/모터 제어의 파형 생성 안식처. 존슨 카운터가 가장 사랑받는 응용처. |

## 👶 어린이를 위한 3줄 비유 설명
1. 존슨 카운터는 의자 4개를 놓고 친구들이 **"검은 모자 차례차례 쓰기, 하얀 모자 차례차례 쓰기" 게임**을 하는 거예요.
2. 4명이 빈 의자부터 한 칸씩 차례로 채워 앉으며 다 앉고 나면, 이번엔 맨 앞사람부터 차례차례 일어나서 결국 의자가 다시 텅 비는 길~고 재미있는 순서 규칙이죠.
3. 링 카운터는 4명이 고작 4번밖에 동작을 못 하지만, 이 재밌는 꼼수를 부리면 의자 4개만 가지고도 무려 8가지의 서로 다른 사진(상태) 셔터를 아주 조용하고 스무스하게 찰칵찰칵 찍어낼 수 있는 천재적인 방법이랍니다!

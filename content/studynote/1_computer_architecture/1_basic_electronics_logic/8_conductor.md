## 핵심 인사이트 (3줄 요약)
> 1. **본질**: 도체 (Conductor)는 원자 최외곽의 자유 전자 (Free Electron)가 핵의 구속을 벗어나 자유롭게 이동할 수 있는 에너지 대역 (Conduction Band)을 가진 물리적 소재로, 전기적 신호와 전력을 전달하는 매질이다.
> 2. **가치**: 컴퓨터 아키텍처에서 칩 내부의 트랜지스터들을 연결하는 인터커넥트 (Interconnect) 배선망과 외부 시스템 버스 (System Bus)를 형성하며, 클럭 동기화와 데이터 처리량 (Throughput)을 결정하는 핵심 인프라로 작용한다.
> 3. **융합**: 미세 공정이 3nm 이하로 진입함에 따라 전통적인 Cu (Copper, 구리) 도선의 표면 산란과 EM (Electromigration, 전자 이동) 현상이 한계에 다다르면서, Ru (Ruthenium, 루테늄) 등 신소재 도체 전환과 BSPDN (Backside Power Delivery Network, 후면 전력 공급망) 아키텍처 융합을 강제하고 있다.

## Ⅰ. 개요 및 필요성 (Context & Necessity)

- **개념**: 도체 (Conductor)는 외부에서 전압이 인가되었을 때 자유 전자들이 한 방향으로 일제히 이동하며 전류를 형성할 수 있는, 전기 저항이 극히 낮은 금속성 물질을 의미한다.
- **필요성**: 트랜지스터가 논리 연산을 수행하는 뇌의 역할을 담당한다면, 도체는 그 결과를 실시간으로 전달하는 '신경망'의 역할을 수행하므로, 전기 저항을 극복하여 신호 지연 없이 데이터를 초고속으로 운반할 수 있는 물리적 경로가 필수적이다. 과거 Al (Aluminum, 알루미늄) 배선의 고유 저항으로 인한 RC (Resistor-Capacitor) 지연을 해결하기 위해 전도도가 높은 구리 다마신 공정이 도입되어 초고주파 클럭 동작을 보장한다.
- **💡 비유**: 도체는 자동차가 막힘없이 달릴 수 있게 잘 포장된 넓은 '고속도로'와 같다. 전압이라는 가속 페달을 밟으면 전자라는 자동차들이 텅 빈 도로를 타고 목적지까지 폭발적인 속도로 도달한다.
- **등장 배경**: 반도체 기판 위에 무수히 많은 트랜지스터를 새겨 넣어도, 이들을 전기적으로 이어주지 않으면 무용지물이다. 초기 집적회로에서는 알루미늄을 증착해 도체 배선을 형성했으나, 클럭이 GHz 대역으로 높아지면서 알루미늄 고유의 높은 저항이 한계에 부딪혔다. 이에 따라 저항이 낮은 구리 다마신 공정이 도입되어 수십 년간 반도체 칩 배선의 표준으로 군림해왔다.

절연체와 도체의 전기적 차이를 양자역학적 에너지 밴드 관점에서 시각화하면 다음과 같다. 이 도식은 도체가 왜 전기를 저항 없이 잘 흘려보내는지를 근본적으로 설명한다.

```text
  ┌────────────────────────────────────────────────────────────────────────────────────────────────┐
  │              도체 내부 에너지 밴드와 자유 전자의 이동                                          │
  ├────────────────────────────────────────────────────────────────────────────────────────────────┤
  │                                                                                                │
  │  [에너지 밴드 모델]                                                                            │
  │    에너지(E)                                                                                   │
  │       ▲   ┌──────────────────┐                                                                 │
  │       │   │ 전도대 (Conduction Band) - 전자가 자유롭게 질주 │                                  │
  │       │   ├──────────────────┤ ◀ 중첩됨 (Band Gap 없음)                                        │
  │       │   │ 가전자대 (Valence Band) - 전자가 원자핵에 묶여있음│                                │
  │       │   └──────────────────┘                                                                 │
  │                                                                                                │
  │  [전압 인가 시 전자 이동 (Drift Velocity)]                                                     │
  │    (-) ──────▶ e- ──▶ e- ──▶ e- ───────▶ (+)                                                   │
  │    전압 V         자유 전자 무리(전류 I)                                                       │
  └────────────────────────────────────────────────────────────────────────────────────────────────┘
```

**[다이어그램 해설]** 이 도식의 핵심은 도체가 유독 전기를 잘 통하는 이유를 양자역학적 에너지 밴드 모델로 설명한 것이다. 절연체나 반도체는 전자가 묶여 있는 가전자대와 자유롭게 움직일 수 있는 전도대 사이에 넘기 힘든 거대한 장벽(Band Gap)이 존재하여 전기가 흐르지 않는다. 반면 금속 같은 도체는 이 두 밴드가 물리적으로 겹쳐 있어서, 상온에서도 전자가 아무런 외부 에너지를 들이지 않고 전도대로 뛰어올라 자유 전자 (Free Electron)가 된다. 따라서 외부에서 약간의 전압만 걸어주면, 대기하고 있던 이 수많은 자유 전자들이 양극(+)을 향해 마치 쏟아지는 강물처럼 쏠려가며 시스템의 데이터를 운반하게 된다. 실무에서는 이 자유 전자의 이동도(Mobility)를 저해하는 모든 요소를 병목으로 간주하고 제거한다.

- **📢 섹션 요약 비유**: 도체는 평소에도 놀고 있는 알바생(자유 전자)이 꽉 차 있는 넓은 대기실과 같아서, 점장(전압)이 소리쳐 부르기만 하면 곧바로 우르르 뛰어나와 배달(전류)을 쏟아내는 완벽한 일터입니다.

## Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

### 구성 요소 (도체 인터커넥트 시스템)

반도체 칩 내부에서 도체를 활용해 데이터망을 구성하는 핵심 요소는 다음과 같다.

| 요소명 | 물리적 역할 | 성능/아키텍처 영향 | 관련 소재 | 비유 |
|:---|:---|:---|:---|:---|
| **자유 전자 (Free Electron)** | 전하를 운반하는 실제 매개체 | 전자의 이동도가 클럭 속도의 물리적 상한을 결정 | Au, Ag, Cu | 달리는 택배 차량 |
| **인터커넥트 (Interconnect)** | 칩 내부 트랜지스터 간 데이터 링크 | 상위 층일수록 도선을 굵게 하여 저항 통제 | 구리(Cu) 배선 | 좁은 골목길부터 넓은 8차선 도로 |
| **비아 (Via)** | 서로 다른 층(Layer)의 도체를 수직 연결 | 기생 인덕턴스 유발 병목 지점 | 텅스텐(W), Cu 기둥 | 층간 화물 엘리베이터 |
| **장벽 금속 (Barrier Metal)** | 구리 입자의 실리콘 층 침투 방지 | 두껍게 바를수록 실제 구리 면적이 줄어 저항 극증 | 탄탈럼(Ta) | 터널 붕괴를 막는 콘크리트 코팅 벽 |
| **솔더 범프 (Solder Bump)** | 칩(Die)과 패키지 기판을 외부로 연결 | I/O 대역폭 확장 및 패키지 발열(TDP) 해소 경로 | Sn-Ag 무연 합금 | 칩 도시와 외부를 잇는 톨게이트 |

### 반도체 다층 배선망 아키텍처

수십억 개의 트랜지스터를 연결하기 위해 현대 마이크로프로세서는 최대 15~20층에 달하는 입체적 도체망을 구축한다. 이 단면도는 각 계층이 담당하는 네트워크 대역폭의 차이를 시각화한다.

```text
  ┌────────────────────────────────────────────────────────────────────────┐
  │                 반도체 다층 배선망(Interconnect) 아키텍처              │
  ├────────────────────────────────────────────────────────────────────────┤
  │                                                                        │
  │ [최상위] 글로벌 배선 (Global Interconnect)                             │
  │   - 두껍고 넓은 구리 도선 (저항 R 최소화 최우선)                       │
  │   - 코어 간 메인 데이터 버스, 전원(VDD), 클럭(Clock) 트리 고속 분배    │
  │  ======================================================                │
  │          || 비아(Via) 수직 연결                                        │
  │ [중간층] 세미 글로벌 배선 (Semi-Global Interconnect)                   │
  │   - 중간 굵기, L2/L3 캐시 간 데이터 블록 이동                          │
  │  ------------------------------------------------------                │
  │          ||                                                            │
  │ [최하위] 로컬 배선 (Local Interconnect)                                │
  │   - 극도로 얇고 촘촘한 도선 (단면적 작음 → 저항 극심한 병목)           │
  │   - 인접한 트랜지스터들을 묶어 기본 논리 게이트 구성                   │
  │  ......................................................                │
  │ ^^^^^^^^^^^^^^^^^ 실리콘 기판 (트랜지스터 층) ^^^^^^^^^^^^^^^^^^       │
  └────────────────────────────────────────────────────────────────────────┘
```

**[다이어그램 해설]** 이 단면도는 현대 CPU가 왜 단순한 평면이 아니라 거대한 3D 입체 빌딩인지 명확히 보여준다. 트랜지스터가 밀집한 최하위 로컬 배선은 수많은 게이트를 촘촘하게 엮어야 하므로 도선이 극도로 얇다. 단면적이 얇으므로 저항이 시스템 전체에서 가장 크다. 반면 위로 올라갈수록(글로벌 배선) 도선의 폭을 대폭 넓히고 두껍게 만들어 저항을 줄인다. 글로벌 층은 전원과 클럭을 뿌려주는 초고속도로 역할을 담당한다. 각 층은 비아(Via)라는 수직 기둥으로 연결되는데, 신호가 위아래로 오르내릴 때 이 다층 구조의 배선 저항에 의한 RC 지연이 최대 병목으로 작용한다. 따라서 실무 설계에서는 지연에 민감한 글로벌 신호는 무조건 최상위 층에 배치하는 라우팅 최적화를 거친다.

- **📢 섹션 요약 비유**: 비좁은 큐비클(로컬 배선)에서 서류를 모아 엘리베이터(비아)를 타고 꼭대기 층으로 올라간 뒤, 컨베이어 벨트(글로벌 배선)에 태워 빛의 속도로 목적지에 배송하는 거대한 입체적 우편집중국 시스템과 같습니다.

## Ⅲ. 융합 비교 및 다각도 분석

### 심층 기술 비교: 반도체 배선 도체 재료의 세대교체

| 비교 항목 | 알루미늄 (Al) 시대 | 구리 (Cu) 시대 (표준) | 코발트 (Co) / 루테늄 (Ru) (미래) |
|:---|:---|:---|:---|
| **전기 전도도** | 보통 수준 | **매우 높음** (저항 낮음) | 구리보단 낮지만 극미세 선폭에선 유리 |
| **제조 공정** | 깎아내는 식각 (Etching) | 파내고 채우는 다마신 | 상향식 도금 등 혁신 공정 |
| **EM 내성** | 전류가 강하면 쉽게 단선됨 | 알루미늄 대비 100배 견고 | **극강의 물리적 내구성** (단선 거의 없음) |
| **장벽 금속** | 거의 불필요함 | 필수 (실리콘 오염 방지) | 불필요 (도체 단면적 극대화 가능) |
| **도입 가치** | 초기 집적회로 배선망 | 클럭 저항(R) 한계 돌파 | 표면 산란 및 배선 폭 병목 완전 극복 |

### 과목 융합 관점

- **네트워크 (NW)**: PCIe (Peripheral Component Interconnect Express) 채널이나 이더넷 UTP (Unshielded Twisted Pair) 케이블 내부의 도체는 고주파 신호 전송 시 표면 효과 (Skin Effect)에 의해 저항이 급증한다. 이로 인해 물리 계층(L1)에서 케이블 길이가 길어질수록 신호 감쇠가 심각하게 발생한다.
- **운영체제/보안 (OS/Security)**: 하드웨어 배선의 한계로 발생한 통신 오류를 커버하기 위해, OS 네트워크 스택은 막대한 양의 FEC (Forward Error Correction, 전진 오류 수정) 연산이나 TCP 패킷 재전송 오버헤드를 필연적으로 떠안게 된다.

구리 배선의 근본적 한계를 나타내는 다음 표면 산란 다이어그램은 왜 신소재 도입이 불가피한지 보여준다.

```text
  ┌───────────────────────────────────────────────────────────────────────────────────┐
  │         구리 배선 미세화의 역설: 표면 산란(Surface Scattering)                    │
  ├───────────────────────────────────────────────────────────────────────────────────┤
  │                                                                                   │
  │  [선폭 10nm 이상: 정상]             [선폭 3nm 이하: 병목 현상]                    │
  │   Barrier 두께 비중이 적음            Barrier가 면적의 대부분을 차지              │
  │  ┌────────────────────┐          ┌────────┐                                       │
  │  │ ■■■■■■■■■■■■■■■■■■ │          │■■■■■■■■│ ← 전자가 좁은 벽에 충돌               │
  │  │ ■■ 구리(Cu) 중심부 ■■ │          │■ Cu ■│    (표면 산란 발생)                  │
  │  │ ■ e- ───▶ e- ■■■■■ │          │■ e- ↘│                                         │
  │  │ ■■■■■■■■■■■■■■■■■■ │          │■■■■■■■■│ ← 물리적 저항 비정상 폭증             │
  │  └────────────────────┘          └────────┘                                       │
  │  │← 장벽(Barrier) 1nm →│          │←1nm→│                                         │
  │                                                                                   │
  │ * 공식 R = ρL/A 에서 단면적(A)가 극한으로 줄어드는데, 배선 내부 독성 차단용       │
  │   장벽 금속 두께는 줄일 수 없어, 실제 구리가 차지하는 면적이 급감해 저항이 폭주함.│
  └───────────────────────────────────────────────────────────────────────────────────┘
```

**[다이어그램 해설]** 이 그림은 현대 반도체 공정이 직면한 가장 치명적인 구리의 딜레마를 보여준다. 구리는 전기를 잘 통하지만, 실리콘과 닿으면 칩을 오염시키므로 도선 테두리에 반드시 절연 장벽을 발라야만 한다. 반면 공정이 고도화되어 배선 폭 자체가 10nm 이하로 극단적으로 줄어들었음에도 이 장벽은 원자 크기의 한계상 더 이상 줄일 수 없다. 결국 좁은 배선 내부에서 정작 진짜 전기가 통하는 구리가 차지할 수 있는 중심부 단면적이 쪼그라든다. 이 때문에 좁아진 길에서 전자들이 일직선으로 달리지 못하고 배선 표면에 부딪히는 '표면 산란'이 극심해져 저항이 폭증한다. 실무에서는 이 한계 때문에 장벽을 바를 필요가 없는 루테늄(Ru) 같은 대체 도체로 하위 레이어를 교체하는 사활을 건 전환을 진행 중이다.

- **📢 섹션 요약 비유**: 넓은 터널에서는 벽에 시멘트를 두껍게 발라도 차가 다니는 데 문제가 없었지만, 좁은 골목길에서는 똑같은 두께로 시멘트를 바르고 나니 정작 차(전자)가 지나갈 공간 자체가 막혀버리는(저항 폭증) 상황과 같습니다.

## Ⅳ. 실무 적용 및 기술사적 판단

### 실무 시나리오

1. **시나리오 — 데이터센터 칩의 EM (Electromigration) 단선 장애**: 
  **판단**: 엄청난 전류 밀도와 열이 결합되면 구리 원자 자체가 물살에 쓸려가듯 이동하여 배선이 단선(Void)된다. 실무 아키텍트는 칩 레이아웃 단계에서 전류가 집중적으로 꺾이는 병목 배선 폭을 두 배 넓히고, 구리 배선 위에 Co(코발트)나 Ru(루테늄)을 캡핑 (Capping)하여 원자 이동을 억누르도록 EDA (Electronic Design Automation) 툴 규칙을 세팅해야 한다.
2. **시나리오 — 극미세 공정의 BSPDN (Backside Power Delivery Network) 도입**:
  **판단**: 칩 상단에서 쏟아져 내려오는 전원선과 데이터 선들이 서로 엉켜 라우팅 한계에 직면하면, 웨이퍼를 통째로 뒤집어 실리콘 뒷면을 관통하는 굵직한 도체 기둥 (Nano-TSV)으로 트랜지스터에 직결하는 BSPDN 아키텍처로 대전환한다. 이를 통해 저항으로 인한 IR Drop (전압 강하) 손실과 배선 체증을 동시에 혁명적으로 해결한다.

차세대 전력망 도입을 결정하는 거시적 아키텍처 의사결정 흐름을 시각화하면 다음과 같다.

```text
  ┌────────────────────────────────────────────────────────────────────────────┐
  │        도선 병목 해결을 위한 차세대 전력망 의사결정 (BSPDN)                │
  ├────────────────────────────────────────────────────────────────────────────┤
  │                                                                            │
  │   [칩 배선 밀집도 포화 및 전압 강하(IR Drop) 한계 초과 경고]               │
  │                │                                                           │
  │                ▼                                                           │
  │    배선 층수(Layer)를 무작정 20층 30층 위로 계속 올릴 수 있는가?           │
  │          ├─ 예 ─────▶ [기존 전면 구리 듀얼 다마신 공정 고도화 지속]        │
  │          │                                                                 │
  │          └─ 아니오                                                         │
  │                │                                                           │
  │                ▼                                                           │
  │    굵은 전원선(VDD)이 미세 데이터 신호선의 라우팅 길을 막고 있는가?        │
  │          ├─ 예 ─────▶ [웨이퍼 뒷면 가공: BSPDN 도입 전환]                  │
  │          │                     │                                           │
  │          │                     └─▶ [Nano-TSV 도체 관통 기술 필수]          │
  │          └─ 아니오 ──▶ [신소재 도체(Co, Ru) 적용 층 확대 적용]             │
  │                                                                            │
  │  최종 조치: 데이터 전선과 전력 전선이 같은 평면에서 싸우지 못하게 입체 분리│
  └────────────────────────────────────────────────────────────────────────────┘
```

**[다이어그램 해설]** 이 의사결정 트리의 핵심은 기존의 "어떻게든 전선을 얇게 만들자"는 재료공학적 한계를 넘어, 아예 전선의 배치 구조를 뒤집어버리는 시스템 아키텍처 관점의 혁명을 지시한다. 기존에는 데이터 전선과 전원 전선이 칩 위쪽 공간에서 서로 엉켜 싸웠다면, BSPDN 구조에서는 웨이퍼 바닥을 뚫어 전원 도체를 트랜지스터에 직결한다. 따라서 15층을 내려오며 발생하던 고질적인 IR Drop을 근절하고 칩 성능을 극적으로 끌어올릴 수 있다. 실무에서는 이 패러다임 전환이 파운드리의 기술 패권을 결정짓는 가장 거대한 척도로 평가된다.

### 도입 체크리스트
- **기술적**: 레이아웃 설계에서 각 배선 층별 저항(R)과 정전용량(C)을 엄격히 분석해, 클럭 주파수 병목을 일으키는 특정 메탈 레이어를 식별하고 코발트 등 신소재를 선별 할당했는가?
- **운영적**: 서버 장기 가동 시 온도 핫스팟 내 배선의 EM 침식 수명이 엔터프라이즈 환경의 보증 기준(최소 5년)을 초과하는지 검증했는가?

### 안티패턴
- **구리(Cu) 만능주의의 관성적 적용**: 10nm 이상 구형 공정의 여유로운 설계 규칙을 3nm 이하 공정에 관성적으로 적용해 최하위 배선에 구리를 무지성 고집하는 경우다. 장벽 금속 두께에 정작 진짜 구리 단면적이 파먹혀 배선 저항이 수십 배 폭증하고 타이밍 에러가 쏟아진다.

- **📢 섹션 요약 비유**: 대도시에 차가 꽉 막히면 자동차(전자) 크기를 줄이는 것을 포기하고, 거대한 지하 터널(BSPDN)을 뚫어서 큰 트럭(전력)은 지하로만 다니고 승용차(데이터)는 지상으로만 달리게 입체를 분리하는 셈입니다.

## Ⅴ. 기대효과 및 결론

### 정량/정성 기대효과

| 구분 | 최적화 전 | 최적화 후 | 개선 효과 |
|:---|:---|:---|:---|
| **정량** | 최하위 레이어 10nm 구리 배선 | 무장벽 Ru (루테늄) 신소재 | 배선 저항 폭증 차단, 신호 지연 **20% 감소** |
| **정량** | 기존 전면 전력 공급망 | 후면 전력 공급망 (BSPDN) 분리 | 전압 강하 근절, 칩 물리적 면적 **10% 대폭 축소** |
| **정성** | EM 현상 단선으로 인한 칩 돌연사 | 초고강도 배선망 캡핑 채택 | 고전류 단선 위험 해소 (MTBF 극상승) |

### 미래 전망
- **광 인터커넥트 (Optical Interconnect)**: 도체를 통해 전자를 밀어 보내는 대신 레이저 빛(광자)을 쏴서 전송하는 실리콘 포토닉스(Silicon Photonics) 기술이 적용되어, 저항과 발열 병목이 0에 수렴하는 통신이 실현될 것이다.
- **탄소 나노 소재**: 저항과 열이 거의 없는 꿈의 신소재인 그래핀(Graphene)이나 탄소 나노 튜브 (Carbon Nanotube) 배선이 금속 도체의 궁극적 대체재로 연구되고 있다.

### 참고 표준
- **IEEE IRDS (국제 반도체 소자 기술 로드맵)**: 미세 공정 노드별 배선 금속의 저항 상한선 및 신소재 도입 가이드라인.

차세대 도체 기술의 진화 방향은 물리적 한계를 우회하는 거대한 패러다임 시프트를 보여준다.

```text
  ┌────────────────────────────────────────────────────────────────────────────┐
  │             반도체 배선(Interconnect) 도체 진화 로드맵                     │
  ├────────────────────────────────────────────────────────────────────────────┤
  │                                                                            │
  │  [과거] Al 도선  ──▶   [현재 표준] Cu 다층 다마신 ──▶ [과도기] Ru / Co     │
  │    (RC 지연 한계)          (초미세화 표면 산란 한계)       (미세층 신소재) │
  │                                                          │                 │
  │                                                          ▼                 │
  │  [궁극] 광 인터커넥트 ◀── [미래] 탄소나노튜브(CNT) ◀── [아키텍처 혁신]     │
  │    (실리콘 포토닉스 빛)        (저항 0에 도전하는 신물질)      BSPDN 후면망│
  └────────────────────────────────────────────────────────────────────────────┘
```

**[다이어그램 해설]** 이 로드맵은 도체를 얇게 다듬으려는 인류의 한계 돌파 투쟁 역사다. 알루미늄의 저항을 구리로 극복했지만, 5nm 이하로 극단적으로 얇아져 길이 막히는 문제는 신소재 발굴과 전선을 아예 뒤로 빼는 BSPDN으로 힘겹게 우회하고 있다. 궁극적으로 금속 도체는 전자와 원자의 충돌(저항)이라는 숙명을 안고 있으므로, 발열과 저항이 아예 없는 탄소 소재 배선이나 빛으로 데이터를 쏘아 보내는 실리콘 포토닉스 기반으로의 전환이 반드시 도래할 것이다. 따라서 실무 기술자들은 더 이상 금속 재료공학에만 매몰되지 않고 광학과 3D 아키텍처를 융합해야만 생존할 수 있다.

- **📢 섹션 요약 비유**: 마차(알루미늄)가 느려 고속열차 철도(구리)를 깔았지만 막히자, 이제는 선로 위를 떠서 달리는 자기부상열차(신소재)나 아예 비행기(광통신)를 띄워 바닥 마찰력을 소멸시켜 버리는 진화가 다가와 있습니다.

## 📌 관련 개념 맵 (Knowledge Graph)

| 개념 명칭 | 관계 및 시너지 설명 |
|:---|:---|
| **자유 전자 (Free Electron)** | 도체의 전도대를 돌아다니며 실제 전하 에너지와 데이터를 빛의 속도로 운반하는 주체다. |
| **인터커넥트 (Interconnect)** | 칩 내부 트랜지스터들을 유기적으로 묶어 동작하게 만드는 다층 금속 배선망으로 연산 성능의 최종 병목이다. |
| **RC 지연 (RC Delay)** | 도선이 극도로 가늘어질수록 기하급수적으로 커지는 저항(R)과 정전용량(C)의 곱으로 시스템 클럭 상한선을 깎아내린다. |
| **전자 이동 (Electromigration)** | 좁은 도체에 과도한 전류가 쏠려 흐를 때 금속 원자가 쓸려 밀려나가 배선이 물리적으로 끊어지는 치명적 파괴 현상이다. |
| **BSPDN (Backside Power Delivery Network)** | 칩 전면의 복잡한 배선을 피해 웨이퍼 뒷면을 뚫고 도체로 전기를 꽂아 넣는 최첨단 3D 라우팅 혁신이다. |

## 👶 어린이를 위한 3줄 비유 설명
1. 도체는 장난감 자동차(전기)가 돌부리 없이 쌩쌩 막힘없이 달릴 수 있게 잘 포장된 **넓고 미끄러운 고속도로**예요!
2. 컴퓨터 두뇌 안에는 수십억 개의 작은 집(트랜지스터)들이 일하고 있는데, 이 집들을 이어주는 엄청나게 복잡한 입체 고속도로망이 구리(Cu)라는 도체로 만들어져 있어요.
3. 요즘은 컴퓨터 마을이 너무 좁아져서 구리 고속도로를 머리카락보다 얇게 깔았더니 자동차가 벽에 부딪혀 느려져서, 훨씬 단단하고 미끄러운 새로운 강철(루테늄)로 좁은 도로들을 다시 공사하고 있답니다!

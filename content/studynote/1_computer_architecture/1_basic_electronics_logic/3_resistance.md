> **핵심 인사이트**
> 1. 저항(Resistance)은 도체 내에서 전류의 흐름을 방해하는 물리적 성질로, 옴의 법칙($V=IR$)에 의해 전압과 전류의 관계를 정의한다.
> 2. 컴퓨터 회로에서 의도적인 저항(Resistor) 소자는 전압을 분배하거나 전류량을 제한하여 민감한 반도체 부품을 보호하는 데 사용된다.
> 3. 미세 공정 칩 내부의 배선(Interconnect) 저항은 신호 지연(RC Delay)과 발열(Joule Heating)을 유발하는 주요 성능 병목 요인이다.

---

## Ⅰ. 저항의 원리와 옴의 법칙 (Ohm's Law)
저항은 전하가 이동할 때 원자핵이나 다른 전자들과 충돌하여 흐름이 방해받는 정도를 나타냅니다. 단위는 옴($\Omega$)을 사용합니다. 전압($V$), 전류($I$), 저항($R$) 간의 관계는 **옴의 법칙($V = I \times R$)**으로 설명되며, 저항이 클수록 동일한 전압에서 흐르는 전류의 양이 줄어듭니다.

저항의 크기는 도체의 길이($L$)에 비례하고 단면적($A$)에 반비례하며, 물질 고유의 비저항($\rho$)에 의해 결정됩니다. ($R = \rho \frac{L}{A}$)

```ascii
[옴의 법칙 시각화 (V=IR)]

   전압(V)이 민다 -------->    저항(R)이 조인다
                              |   |
          전류(I) ----------->|   |-------> 흐름이 줄어듦
                              |   |
```

📢 **섹션 요약 비유**: 저항은 도로의 '과속 방지턱'이나 '병목 구간'과 같습니다. 도로가 좁거나 험할수록(저항이 클수록) 차(전류)가 빠르게 지나가기 힘들어집니다.

---

## Ⅱ. 컴퓨터 하드웨어에서의 저항의 역할
논리 회로나 메인보드 설계 시, 저항기는 필수적인 수동 소자로 사용됩니다.

* **풀업/풀다운 저항 (Pull-up / Pull-down Resistor)**: 디지털 핀이 플로팅(Floating, 0도 1도 아닌 상태)되는 것을 방지하기 위해 사용됩니다. 전원(Vdd)에 연결하면 풀업, 접지(GND)에 연결하면 풀다운입니다.
* **전압 분배 (Voltage Divider)**: 여러 저항을 직렬로 연결하여 특정 부품에 필요한 정확한 기준 전압을 만들어냅니다.
* **전류 제한 (Current Limiting)**: LED나 특정 IC로 너무 많은 전류가 흘러가 부품이 타버리는 것을 막습니다.

```ascii
[풀업 저항(Pull-up Resistor)의 원리]

  Vdd (5V)
   |
   |
  [R] (풀업 저항)
   |
   +-------------> 마이크로컨트롤러 입력 핀 (스위치 개방시 안정적인 1 유지)
   |
  /  (스위치)
 /
 |
 GND (0V)
```

📢 **섹션 요약 비유**: 풀업/풀다운 저항은 배의 '닻'과 같습니다. 파도(노이즈)에 배(신호)가 이리저리 떠내려가지(플로팅) 않도록, 특정 위치(1 또는 0)에 꽉 묶어두는 역할을 합니다.

---

## Ⅲ. 아키텍처 성능 병목: RC 지연 (RC Delay)
최신 CPU 및 GPU는 수백억 개의 트랜지스터를 구리(Cu)나 코발트(Co) 배선으로 연결합니다. 공정이 미세화될수록 도선의 단면적($A$)이 좁아져 **배선 저항($R$)이 기하급수적으로 증가**합니다.

이 배선 저항($R$)과 주변 구조물 간의 정전용량($C$, Capacitance)이 결합하면 **RC 지연(RC Delay)**이 발생합니다. 이는 신호가 목적지에 도달하는 시간을 지연시켜 프로세서의 클럭 속도(Clock Speed)를 높이는 데 치명적인 한계로 작용합니다.

```ascii
[RC 지연 모델]
               저항 (R)
 입력 신호 ----/\/\/\/\/\--------+---- 출력 신호 (지연됨)
                                 |
                                ===  정전용량 (C)
                                 |
                                GND
```

📢 **섹션 요약 비유**: RC 지연은 좁은 복도(저항 R)를 많은 사람(신호)이 지나가려 할 때, 옆방의 문틈(정전용량 C)으로 사람들이 자꾸 새어나가 결국 목적지 도착 시간이 늦어지는 현상입니다.

## 개념 맵 (Concept Map)
```ascii
[저항 (Resistance)]
   │
   ├── 물리적 특성: 전류 흐름의 방해 (V = IR)
   │
   ├── 회로 설계 활용
   │     ├── 신호 안정화: 풀업/풀다운 저항
   │     └── 부품 보호: 전류 제한, 전압 분배
   │
   └── 시스템 아키텍처 한계
         └── 배선 미세화 -> 저항(R) 증가 -> RC 지연(Delay) 발생 -> 클럭 상승 억제
```

## 어린이 비유
저항은 슬라이드 미끄럼틀의 **마찰력(거칠거칠한 표면)**과 같아요! 미끄럼틀이 반질반질하면 슝~ 하고 빨리 내려오지만, 표면이 거칠거칠(저항이 큼)하면 옷이 쓸려서 천천히 내려오게 되죠. 컴퓨터 안에서도 전기가 너무 빨리 흘러서 고장 나지 않도록, 부품들 앞에 일부러 거칠거칠한 길(저항기)을 만들어 준답니다.

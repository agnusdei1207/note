## 핵심 인사이트 (3줄 요약)
> 1. **본질**: 인덕터 (Inductor)는 도선을 코일 형태로 감아 전류가 흐를 때 자기장 (Magnetic Field) 형태로 에너지를 저장하며, 렌츠의 법칙에 따라 전류의 급격한 변화인 AC (Alternating Current, 교류)에 저항하고 직류인 DC (Direct Current)는 통과시키는 수동 소자이다.
> 2. **가치**: 메인보드의 VRM (Voltage Regulator Module, 전압 조정 모듈)에서 벅 컨버터 (Buck Converter)의 핵심 소자로 작용하여, 스위칭된 거친 고압 전류를 칩이 요구하는 평탄한 저전압·고전류로 변환 및 평활화(Filtering)하는 심장 역할을 한다.
> 3. **융합**: 고속 디지털 시스템에서 배선과 패키지 핀이 갖는 기생 인덕턴스 (Parasitic Inductance)는 트랜지스터의 고속 스위칭 시 치명적인 전압 변동 노이즈($V=L \cdot di/dt$)인 SSN (Simultaneous Switching Noise, 동시 스위칭 노이즈) 또는 그라운드 바운스 (Ground Bounce)를 유발하여 프로세서 최대 속도를 제한하는 근본적인 아키텍처 장애물이다.

## Ⅰ. 개요 및 필요성 (Context & Necessity)

- **개념**: 인덕터 (Inductor)는 전자기 유도 현상을 이용해 전류의 변화량에 비례하여 역기전력(전압)을 발생시키는 수동 소자다. 전압의 변화에 저항하는 커패시터와 달리, 인덕터는 전류 흐름의 변화를 거부하는 "전류의 물리적 관성"을 회로에 부여한다.
- **필요성**: CPU (Central Processing Unit)의 고성능화로 인해 전원부에서 수백 kHz 대역의 초고속 스위칭을 통해 고전압을 극저전압으로 강하시키는 과정이 필수적이면서도, 이때 발생하는 급격한 전류 변동과 스위칭 충격이 반도체 소자를 파괴할 위험이 있기 때문에 필요하다. 인덕터는 이러한 거친 PWM (Pulse Width Modulation, 펄스 폭 변조) 파형을 자기장 에너지의 충전과 방출 과정을 통해 완만한 삼각파로 평활화하여, 전압 강하 기술의 핵심 소자로서 회로의 안정성을 확보하는 역할을 수행한다.
- **💡 비유**: 무거운 '물레방아'와 같다. 물(전류)을 처음 흘려보내어 물레방아를 돌리려고 할 때는 무거워서 잘 안 돌아가지만(전류 증가 방해), 한 번 빙글빙글 돌기 시작하면 물을 끊어도 무거운 관성 때문에 당분간 계속 돌아가며 하류로 물을 밀어내는(전류 감소 방해) 성질과 완벽히 일치한다.
- **등장 배경**: 초기에는 라디오 주파수 튜닝 용도로 주로 쓰였으나, 프로세서가 수백 와트(W) 단위의 막대한 전력을 소모하면서도 발열 제어를 위해 1.0V 이하의 극저전압을 요구하게 되면서 위상이 변했다. PSU (Power Supply Unit)에서 공급되는 12V 전압을 칩 바로 옆에서 초고속 스위칭을 통해 강하시키는 다상 전원부 아키텍처가 필수적이게 되었고, 스위칭의 충격을 부드럽게 뿜어내는 거대한 인덕터가 메인보드의 필수 인프라로 자리 잡았다.

인덕터가 전원 회로에서 거친 스위칭 파형을 부드럽게 깎아내는 평활화 과정을 도식화하면 다음과 같다. 이 다이어그램은 벅 컨버터 회로의 심장부에서 일어나는 전압과 전류의 파형 변환 과정을 보여준다.

```text
  ┌────────────────────────────────────────────────────────────────────┐
  │         스위칭 레귤레이터(벅 컨버터) 내 인덕터의 평활 효과         │
  ├────────────────────────────────────────────────────────────────────┤
  │                                                                    │
  │  [입력 측: MOSFET 스위칭 노드]                                     │
  │  전압(V):  12V ─┐    ┌─┐    ┌─┐   (거친 펄스 폭 변조, PWM)         │
  │              0V └────┘ └────┘ └────                                │
  │                   │                                                │
  │                   ▼                                                │
  │              [ 인덕터 (L) ] UUUUUUU (전류의 관성 제공)             │
  │                   │                                                │
  │                   ▼                                                │
  │  [출력 측: CPU 입력 노드]                                          │
  │  전류(I):       / \  / \  / \     (연속적인 삼각파 전류)           │
  │                /   \/   \/   \    (이후 커패시터가 DC로 평탄화)    │
  │                                                                    │
  │   * 인덕터가 없으면 12V가 직접 CPU를 타격하여 칩이 즉시 소손됨     │
  └────────────────────────────────────────────────────────────────────┘
```

**[다이어그램 해설]** 이 그림의 핵심은 메인보드 전원부인 VRM (Voltage Regulator Module)에서 고전압 펄스를 저전압 직류로 바꾸는 벅 컨버터의 동작 메커니즘이다. 트랜지스터인 MOSFET (Metal-Oxide-Semiconductor Field-Effect Transistor) 스위치가 12V 전원을 엄청난 속도로 켰다 끄며 네모난 직사각형 펄스를 만들어낸다. 만약 인덕터가 없다면 이 12V의 펄스가 그대로 실리콘에 직격하여 칩이 즉시 타버린다. 반면 인덕터는 이 12V 펄스가 들어올 때 전류가 급증하려는 성질을 자기장 에너지로 흡수하며 억제하고, 스위치가 꺼져 0V가 되면 저장했던 자기장 에너지를 전기로 방출하며 전류를 계속 밀어준다. 따라서 치명적인 네모 펄스가 완만한 삼각파 전류로 다듬어지며 시스템이 파괴되는 것을 물리적으로 막아낸다. 실무에서는 이 리플 전류(Ripple Current)의 허용치를 정밀하게 계산하여 인덕터의 스펙을 결정한다.

- **📢 섹션 요약 비유**: 마치 댐의 수문(스위치)을 열었다 닫았다 하며 쏟아지는 거친 폭포수를, 커다란 물레방아(인덕터)가 묵직하게 돌아가며 충격을 다 흡수한 뒤 하류로 잔잔하고 일정한 물살을 흘려보내는 것과 같습니다.

## Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

### 구성 요소 및 인덕턴스 결정 인자

인덕터의 물리적 성능인 인덕턴스 (L)를 결정짓는 핵심 구조 요소는 다음과 같다.

| 요소명 | 물리적 역할 | 성능 영향 | 관련 기술/재질 | 비유 |
|:---|:---|:---|:---|:---|
| **권선 (Coil / Winding)** | 전류가 흐르는 물리적 통로 | 턴(Turn) 수의 제곱에 비례하여 인덕턴스 대폭 증가 | 구리 에나멜선, 평각선 | 물레방아의 물받이 날개 |
| **코어 (Core)** | 자기력선을 집중시키는 매질 | 투자율($\mu$)이 높을수록 인덕턴스 배수 증가 | 페라이트 (Ferrite) 코어 | 물레방아의 무거운 쇳덩이 중심축 |
| **에어 갭 (Air Gap)** | 코어 내부를 미세하게 자른 빈 공간 | 자기 포화 (Saturation) 방지, 대전류 허용 | 코어 절단 및 비자성체 접착 | 댐이 넘치는 걸 막는 안전 여수로 |
| **직류 저항 (DCR)** | 도선 자체가 갖는 불가피한 저항 | 전원부 발열 및 전력 손실의 주범 | 굵고 짧은 권선 사용으로 통제 | 물레방아 축에 낀 녹(마찰력) |

### 기생 인덕턴스와 그라운드 바운스 (SSN) 아키텍처

의도적으로 만든 소자뿐 아니라, 일자 형태의 미세한 배선과 칩 패키지의 핀 구조물 자체도 어쩔 수 없는 '기생 인덕턴스'를 가진다. 아래 타이밍 도식은 기생 인덕턴스가 어떻게 칩 내부의 기준 전압을 무너뜨리는지 그 파괴적 영향을 시각화한다.

```text
  ┌──────────────────────────────────────────────────────────────────┐
  │                 기생 인덕턴스에 의한 V = L(di/dt) 노이즈         │
  ├──────────────────────────────────────────────────────────────────┤
  │                                                                  │
  │   [칩 내부] (수십억 개 트랜지스터 동시 스위칭)                   │
  │      │                                                           │
  │      │  I (수십 암페어의 급격한 수요, 높은 di/dt)                │
  │      ▼                                                           │
  │    ┌┴┴┴┴┐ 패키지 핀 (구조적 한계로 기생 인덕턴스 L 존재)         │
  │    │UUUU│                                                        │
  │    └┬┬┬┬┘                                                        │
  │      │                                                           │
  │      ▼ 메인보드 GND (Ground) 기준면                              │
  │                                                                  │
  │   [칩 내부에서 관측되는 Ground 전압의 요동]                      │
  │   V_gnd                                                          │
  │    ▲           (Ground Bounce / SSN 현상)                        │
  │    │   /\        /\                                              │
  │  0V┼──/  \      /  \──────────────▶ 시간(t)                      │
  │    │       \    /                                                │
  │    │        \  /     * V_bounce = L_pkg * (di/dt)                │
  │    │         \/                                                  │
  └──────────────────────────────────────────────────────────────────┘
```

**[다이어그램 해설]** 이 그림은 칩 설계에서 의도치 않게 발생하는 기생 인덕턴스 (Parasitic Inductance)의 치명적 부작용을 경고한다. 칩을 메인보드에 물리적으로 연결하는 핀은 전류가 흐르는 경로이므로 반드시 인덕턴스(L)를 갖는다. 최신 프로세서 내부에서 수십억 개의 트랜지스터가 동시에 상태를 바꿀 때(SSN: Simultaneous Switching Noise), 순식간에 엄청난 전류를 폭발적으로 끌어당긴다. 이때 핀의 기생 인덕턴스가 관성으로 작용해 역전압($V=L \cdot di/dt$)을 만들어낸다. 이 때문에 완벽한 0V로 고정되어야 할 칩 내부의 GND (Ground) 전압 자체가 미친 듯이 출렁이게 되는 그라운드 바운스 (Ground Bounce) 현상이 발생한다. 따라서 논리 회로가 0을 1로 오해하는 치명적인 소프트 에러를 유발하게 되며, 실무 아키텍트들은 전원 핀을 기하급수적으로 늘려 병렬로 L 값을 줄이는 우회 전략을 사용한다.

- **📢 섹션 요약 비유**: 수돗물을 콸콸 틀어놓고 밸브를 순식간에 확 잠그면, 파이프 속을 빠르게 흐르던 물의 거대한 질량(관성) 때문에 파이프 전체가 '쾅!' 하고 요동치며 터질 듯 흔들리는 '수격 현상'과 완전히 똑같은 물리 법칙입니다.

## Ⅲ. 융합 비교 및 다각도 분석

### 심층 기술 비교: 필터 관점에서의 L과 C 특성

전원부를 안정화시키는 필터에서 인덕터(L)와 커패시터(C)는 완벽한 대칭적 역할을 수행한다. 이 그래프는 두 소자의 주파수 특성 차이를 보여준다.

```text
  ┌──────────────────────────────────────────────────────────────────────┐
  │         디커플링 필터 관점에서의 L과 C의 주파수 응답 특성 비교       │
  ├──────────────────────────────────────────────────────────────────────┤
  │                                                                      │
  │  임피던스(Z)                                                         │
  │   ▲                   [ 인덕터 (Inductor, Z = 2πfL) ]                │
  │   │                   ↗ 고주파 노이즈 완벽 차단                      │
  │   │                 /  (High Impedance)                              │
  │   │               /                                                  │
  │   │             /                                                    │
  │   │           /   [ 커패시터 (Capacitor, Z = 1/2πfC) ]               │
  │   │         /   ↘ 고주파 노이즈 통과시켜 버림 (그라운드로 방출)      │
  │   │       /        \ (Low Impedance)                                 │
  │   │     /            \                                               │
  │   └──────────────────────────────────────────────────────▶ 주파수(f) │
  │                                                                      │
  │   * 회로 적용: L은 신호 경로에 직렬(Series)로 연결해 방파제를 치고,  │
  │                C는 병렬(Parallel)로 연결해 하수구로 버린다.          │
  └──────────────────────────────────────────────────────────────────────┘
```

**[다이어그램 해설]** 이 그래프는 주파수 변화에 따른 인덕터와 커패시터의 상반된 임피던스(교류 저항) 반응을 명확히 보여준다. 인덕터는 주파수가 높아질수록 저항 성분이 선형적으로 커져서 고주파 스파이크가 통과하지 못하게 튕겨내는 '직렬 장벽' 역할을 한다. 반면 커패시터는 주파수가 높을수록 저항이 기하급수적으로 작아지므로, 이를 칩 앞단에 병렬로 달아두면 노이즈만 골라서 땅으로 흘려보내는 '하수구' 역할을 한다. 따라서 이 두 소자의 물리적 교차 대칭성을 완벽히 결합한 것이 메인보드 전원부의 핵심 철학이다. 실무에서는 이 두 소자의 필터 대역폭을 절묘하게 겹치게 설계하여 모든 대역의 노이즈를 완벽하게 말살시킨다.

### 과목 융합 관점

- **운영체제 (OS)**: 전력 관리 기술인 DVFS (Dynamic Voltage and Frequency Scaling)와 직접 맞물린다. OS (Operating System) 스케줄러가 CPU 부하를 낮추기 위해 클럭과 전압을 급격히 떨어뜨릴 때, 인덕터의 물리적 반응 지연 때문에 전압이 너무 느리게 변하거나 과도하게 튀어 오르는 오버슈트가 발생할 수 있다. 이 때문에 OS 스케줄러의 전환 속도 파라미터는 하드웨어 인덕터의 한계를 반드시 반영해야 한다.
- **컴퓨터 아키텍처 (CA)**: FIVR (Fully Integrated Voltage Regulator) 아키텍처 도입의 최고 난제였다. 인텔은 칩의 패키지 기판 내부에 극소형 인덕터를 매립하여 메인보드 의존 없이 코어마다 완벽히 독립적인 전압 제어를 가능하게 했다.

- **📢 섹션 요약 비유**: 불순물이 섞인 물이 파이프를 타고 들어올 때, 직렬로 막아선 인덕터는 굵은 자갈을 튕겨내는 '거름망 방파제' 역할을 하고, 병렬로 뚫려 있는 커패시터는 찌꺼기를 흘려보내는 '침전조' 역할을 하여 완벽하게 정수된 물을 공급합니다.

## Ⅳ. 실무 적용 및 기술사적 판단

### 실무 시나리오

1. **시나리오 — GPU의 거슬리는 고주파음 (Coil Whine) 발생**: 
  **판단**: 이 소음은 불량이나 전기적 에러가 아니라, 인덕터 코어 내부의 구리 권선이 고주파의 강력한 스위칭 자기장 변동에 의해 물리적으로 진동하며 부딪히는 마찰음이다. 하드웨어 엔지니어는 초크 덮개 내부에 에폭시 수지를 몰딩(Molding)하여 권선이 떨리지 않게 단단히 굳혀버린 부품으로 스펙을 변경하여 소음을 원천 차단한다.
2. **시나리오 — 서버 메인보드의 저부하 전력 낭비**:
  **판단**: CPU의 요구 전류량이 적은데도 병렬로 구성된 수십 개의 전원부 인덕터 라인(페이즈)이 전부 다 켜져 있으면, 인덕터의 코어 자기 손실(Core Loss)이 효율을 다 깎아먹는다. 서버 BMC (Baseboard Management Controller) 펌웨어에서 시스템 부하율을 감지하여, 필요 없는 전원 페이즈를 동적으로 꺼버리는 위상 차단(Phase Shedding) 로직을 활성화한다.

인덕터 노이즈 방어를 위한 실무 의사결정 흐름을 시각화하면 다음과 같다.

```text
  ┌───────────────────────────────────────────────────────────────────────┐
  │           인덕터 노이즈 방어를 위한 실무 의사결정 플로우              │
  ├───────────────────────────────────────────────────────────────────────┤
  │                                                                       │
  │   [칩/메인보드 레벨 노이즈 장애 및 효율 저하 보고 접수]               │
  │                │                                                      │
  │                ▼                                                      │
  │    원인이 고부하 스위칭 시 발생하는 메인 전압 출렁임(Droop)인가?      │
  │          ├─ 예 ─────▶ [VRM 전원부 페이즈 수 증설 (전류 분산)]         │
  │          │                     │                                      │
  │          │                     └─▶ [저항성(DCR)이 더 낮은 인덕터 채택]│
  │          └─ 아니오                                                    │
  │                │                                                      │
  │                ▼                                                      │
  │    원인이 칩 내부에 국한된 그라운드 바운스(SSN) 현상인가?             │
  │          ├─ 예 ─────▶ [패키지 핀(Pin) 맵 전면 재설계 (병렬 L 감소)]   │
  │          │                     │                                      │
  │          │                     └─▶ [VDD/GND 핀 할당 비율 대폭 상향]   │
  │          └─ 아니오 ──▶ [Coil Whine 등 물리적 진동 불량 우선 검토]     │
  │                                                                       │
  │  최종 판단: 전류의 급변량(di/dt) 자체를 쪼개어 인덕턴스 충격을 완화   │
  └───────────────────────────────────────────────────────────────────────┘
```

**[다이어그램 해설]** 물리적 법칙상 시스템에 존재하는 기생 인덕턴스를 완전히 없앨 수 없다면, 실무적인 아키텍처 해결책은 언제나 $di/dt$를 분산시켜 충격을 희석하는 것이다. 프로세서가 너무 많은 전기를 원할 때 거대한 인덕터 하나를 다는 대신 작은 인덕터 16개를 병렬로 달아 한 채널이 감당해야 할 순간 전류 변화 폭을 쪼갠다. 반면 칩 내부 핀의 기생 장벽이 문제라면, 데이터 핀 주변의 VDD (전원) 핀과 GND (접지) 핀의 개수를 기하급수적으로 늘린다. 따라서 병렬 연결 시 총합이 감소하는 물리 공식($L_{total} = L/N$)을 활용해 인덕턴스를 무력화시키는 것이 서버 칩 설계의 절대적 불문율이다. 실무에서는 이 방어 논리를 통해 수백 와트의 전력을 소비하는 서버의 크래시를 방지한다.

### 도입 체크리스트
- **기술적**: 코어가 최대 부하로 전력을 당길 때, VRM (Voltage Regulator Module) 초크 코일이 자기 포화 (Magnetic Saturation) 구간에 진입하여 과전류를 허용하는 사태가 배제되었는가?
- **물리적**: 발열이 심한 파워 인덕터 주변에 액체 상태인 전해 커패시터가 밀착 배치되어 전체 MTBF (Mean Time Between Failures, 평균 무고장 시간) 수명을 깎아먹고 있지 않은가?

### 안티패턴
- **인덕턴스 맹신주의**: "인덕턴스 수치가 클수록 방어력이 강력해진다"는 착각이다. 용량이 너무 큰 인덕터는 전류 변화에 대한 저항(관성)이 너무 무거워져, CPU가 급격히 전류를 요구할 때 전원부가 이를 빠르게 쫓아가지 못해 전압이 바닥을 치는 블랙아웃 대참사가 발생한다.

- **📢 섹션 요약 비유**: 자동차의 서스펜션을 튜닝할 때 무조건 뻣뻣한 스프링을 달면 돌발 상황에서 핸들을 꺾을 때 차체가 둔하게 반응하여 가드레일을 들이받는 부작용과 같습니다.

## Ⅴ. 기대효과 및 결론

### 정량/정성 기대효과

| 구분 | 최적화 전 | 최적화 후 | 개선 효과 |
|:---|:---|:---|:---|
| **정량** | 싱글 페이즈 모듈 사용 | 다상 (Multi-Phase) 병렬 인덕터 | 리플 (Ripple) 전압 **50% 이상 감소** 및 발열 억제 |
| **정량** | 긴 와이어 본딩 방식 | 솔더 범프 (Solder Bump) 기반 | 패키지 기생 인덕턴스 **수십 pH 수준 극소화** |
| **정성** | 고주파 찌르름 소음 발생 | 몰딩형 금속 복합체 인덕터 채택 | 고부하 환경에서도 음향 소음 완벽 차단 |

### 미래 전망
- **실리콘 웨이퍼 내장형 IVR (Integrated Voltage Regulator)**: 차세대 공정에서는 인덕터를 실리콘 웨이퍼 기판 내부에 코일로 에칭하여 내장하는 기술이 상용화되어 코어 내부에서 극정밀 전압 제어가 가능해진다.
- **3D 전력망**: 패키징 내부 구조에서 인터포저 밑에 초소형 인덕터 어레이가 적층된 완전한 3D 수직 전력망인 PDN (Power Delivery Network) 시대로 진입하고 있다.

### 참고 표준
- **Intel VRM/IMVP 규격**: 인텔 프로세서 세대별로 전압 레귤레이터의 스위칭 주파수 상한선과 허용 리플 수치를 엄격히 통제하는 하드웨어 가이드라인.

전원 공급망에서 인덕터의 물리적 배치 변화 로드맵을 시각화하면 칩 외부에서 내부로 파고드는 치열한 진화 과정을 볼 수 있다.

```text
  ┌─────────────────────────────────────────────────────────────────────┐
  │              프로세서 전원부(VRM) 내 인덕터 배치 진화 로드맵        │
  ├─────────────────────────────────────────────────────────────────────┤
  │                                                                     │
  │   2010s               2020s                2030s+                   │
  │    │                   │                    │                       │
  │    ▼                   ▼                    ▼                       │
  │ [보드 탑재형 MBVR] → [패키지 탑재형 FIVR] → [다이 내장형 IVR]       │
  │    │                   │                    │                       │
  │    ├─ 메인보드에 납땜  ├─ 패키지 기판 매립  ├─ 실리콘 웨이퍼 집적   │
  │    ├─ 수백 nH 큰 용량  ├─ 수 nH 초소형화    ├─ 수십 pH 극소화       │
  │    └─ 기생 L 방어 실패 └─ 코어별 개별 제어  └─ 완전한 수직 전력망   │
  │                                                                     │
  │   진화 철학: "물리적 거리를 좁혀 기생 노이즈를 없애고 응답성 극대화"│
  └─────────────────────────────────────────────────────────────────────┘
```

**[다이어그램 해설]** 컴퓨터 메인보드 상의 굵은 인덕터 덩어리들이 수십 년에 걸쳐 점차 프로세서 안으로 파고드는 극한의 미세화 과정이다. 과거에는 CPU 밖 메인보드(MBVR)에 거대한 인덕터를 배치했지만 핀의 기생 인덕턴스 때문에 전압 파형이 훼손되었다. 반면 이를 극복하기 위해 패키지 기판 바로 밑 공간에 미세 인덕터를 촘촘히 박아 넣는 FIVR (Fully Integrated Voltage Regulator) 아키텍처가 등장했고, 미래에는 실리콘 다이 안에 직접 그려 넣는 IVR 기술로 진화 중이다. 따라서 부품 간 물리적 거리를 나노 단위로 줄여 기생 성분을 말살시키는 것이 컴퓨터 전력망 진화의 궁극적 해답이다.

- **📢 섹션 요약 비유**: 대형 발전소에서 전기를 보내던 방식에서, 아파트 지하실을 거쳐 이제는 각 세대 방 안마다 초소형 발전기를 두어 송전 과정의 손실을 완전히 없애버리는 것과 같습니다.

## 📌 관련 개념 맵 (Knowledge Graph)

| 개념 명칭 | 관계 및 시너지 설명 |
|:---|:---|
| **VRM (Voltage Regulator Module)** | 12V 전압을 CPU용 1V 전후로 강하할 때 핵심인 인덕터와 커패시터를 조합하여 전압을 평탄화하는 모듈이다. |
| **렌츠의 법칙 (Lenz's Law)** | 인덕터 소자가 외부 자기장 변화를 상쇄하려는 방향으로 스스로 역기전력을 생성해내는 물리적 원리다. |
| **SSN (Simultaneous Switching Noise)** | 기생 인덕턴스에 트랜지스터의 급격한 스위칭 전류 변화가 부딪혀 내부 기준 전압이 요동치는 장애 현상이다. |
| **FIVR (Fully Integrated Voltage Regulator)** | 메인보드 인덕터를 잘게 쪼개어 패키징 내부에 집어넣음으로써 코어별 초고속 전력 제어를 가능하게 한 혁신이다. |
| **디커플링 커패시터 (Decoupling Capacitor)** | 인덕터가 직렬 방파제 역할을 한다면, 커패시터는 병렬 하수구 역할을 하여 서로의 약점을 완벽히 보완한다. |

## 👶 어린이를 위한 3줄 비유 설명
1. 인덕터는 쌩쌩 달리는 썰매(전류)를 뒤에서 단단하게 잡아당기고 있는 **커다랗고 무거운 고무줄** 같아요! 
2. 썰매가 갑자기 튀어 나가려고 하면 고무줄이 팽팽해지며 충격을 흡수해주고, 멈추려고 하면 고무줄이 수축하며 뒤에서 살짝 밀어준답니다.
3. 그래서 컴퓨터 안에서 무서운 고압 전기가 심하게 널뛰기를 할 때, 이 듬직한 고무줄이 충격을 다 흡수해서 컴퓨터 두뇌가 편안하게 일할 수 있도록 지켜줘요.

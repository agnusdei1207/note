> **핵심 인사이트**
> 1. 정류 회로(Rectifier)는 다이오드(Diode)의 단방향 전도 특성을 이용하여 **교류(AC) 전원을 직류(DC) 전원으로 변환**하는 전자 회로이다.
> 2. 컴퓨터의 심장인 **전원 공급 장치(PSU, Power Supply Unit)**의 가장 첫 번째 관문으로, 벽면 콘센트의 220V 교류를 시스템이 사용할 수 있는 안정적인 직류의 기초 형태로 깎아낸다.
> 3. 단순한 반파 정류(Half-wave) 대신 다이오드 4개를 결합한 **브리지 전파 정류(Bridge Full-wave Rectifier)** 구조를 사용하여 전력 효율을 극대화한다.

---

## Ⅰ. 정류 회로의 필요성
컴퓨터 내부의 CPU, 메모리, 메인보드는 매우 민감한 논리 회로(0과 1)로 구성되어 있어, 전압이 조금만 출렁거려도 데이터가 깨집니다. 따라서 항상 일정한 압력으로 흐르는 **직류(DC)**가 필수적입니다.
하지만 발전소에서 집까지 전기를 보낼 때는 송전 효율이 좋은 **교류(AC)** 파형(플러스와 마이너스가 1초에 60번씩 교차함)으로 보냅니다. 따라서 컴퓨터 파워서플라이(PSU)는 이 출렁이는 교류를 평탄한 직류로 강제로 다림질해야 하는데, 이 작업의 1단계를 담당하는 것이 정류 회로입니다.

```ascii
[전원 변환의 전체 흐름]

 콘센트(AC) --> [ 1. 정류 (Rectifier) ] --> [ 2. 평활 (Filter) ] --> [ 3. 레귤레이팅 ] --> CPU(DC)
  /\  /\        (마이너스 파형 뒤집기)       (울퉁불퉁한 선 펴기)     (정확히 1.2V로 맞춤)
 /  \/  \       /\/\/\/\/\/\/\/\           ====================   ================
```

📢 **섹션 요약 비유**: 교류는 파도가 심하게 치는 '바다'이고, 직류는 잔잔한 '수영장'입니다. 정류 회로는 출렁이는 파도를 칼로 싹둑 잘라내거나 위로 접어 올려, 잔잔한 수영장을 만들기 위한 첫 번째 파도막이 방파제 역할을 합니다.

---

## Ⅱ. 정류 회로의 종류: 반파 vs 전파
정류 회로는 다이오드를 몇 개 사용하느냐에 따라 효율이 극명하게 갈립니다.

1. **반파 정류 (Half-wave Rectifier)**
   - 다이오드 1개를 사용합니다. 플러스(+) 파형은 통과시키고, 마이너스(-) 파형은 버립니다. 회로가 간단하지만, 버려지는 전기가 딱 절반이어서 **효율이 50% 이하**로 매우 나쁩니다.
   
2. **브리지 전파 정류 (Bridge Full-wave Rectifier)**
   - **다이오드 4개**를 마름모꼴(브리지 형태)로 배치합니다. 교류의 플러스(+) 파형일 때는 다이오드 2개가 열려 통과하고, 마이너스(-) 파형일 때는 나머지 다이오드 2개가 열려 파형의 방향을 180도 뒤집어서 위로 올려보냅니다.
   - 버려지는 전기 없이 촘촘한 산봉우리 모양의 파형을 만들어 내어 **컴퓨터 PSU의 표준 구조**로 사용됩니다.

```ascii
[브리지 전파 정류기의 동작 (다이오드 4개)]

 입력: AC 파형 (~ / ~ / ~)

 (+) 사이클: D1, D2 도통, D3, D4 차단  -->  출력: /\
 (-) 사이클: D3, D4 도통, D1, D2 차단  -->  출력: /\ (방향이 뒤집혀 양수로 나옴)

 최종 출력: /\/\/\/\ (버려지는 빈틈이 없음)
```

📢 **섹션 요약 비유**: 반파 정류가 삽으로 흙을 퍼서 절반은 담고 절반은 바닥에 흘리는 어설픈 작업꾼이라면, 브리지 전파 정류는 흘린 흙마저 다시 뜰채로 쓸어 담아 수레에 꽉꽉 채워 넣는 완벽한 일꾼입니다.

---

## Ⅲ. 아키텍처 관점: 리플(Ripple)과 평활 회로(Filter)
브리지 정류 회로를 통과해도 전기는 여전히 산봉우리처럼 출렁거리는 상태(맥류, Pulsating DC)입니다. 이 출렁거림의 정도를 **리플(Ripple)**이라고 합니다. 컴퓨터 부품에 리플이 섞인 전기가 들어가면 노이즈(오작동)가 발생합니다.

따라서 정류 회로 바로 뒤에는 반드시 커패시터(Capacitor)와 인덕터(Inductor)를 거대한 저수지처럼 배치하는 **평활 회로(Smoothing Filter)**가 따라붙습니다.
* 산봉우리(전압 상승)일 때: 커패시터가 물(전하)을 가득 저장함.
* 골짜기(전압 하락)일 때: 커패시터가 저장한 물을 뿜어내어 빈 공간을 채워 평평하게 만듦.

```ascii
[평활 회로를 통한 리플 제거]

 정류 직후 파형 (맥류)     =>     커패시터(C) 통과 후 파형 (평활된 직류)
   /\    /\    /\                   ----------_---------_--------
  /  \  /  \  /  \                 /           \         \
 /    \/    \/    \               /             \         \
 (골짜기가 깊음)                  (골짜기를 커패시터가 메워줌 = 리플 감소)
```

📢 **섹션 요약 비유**: 리플이 있는 전기는 계단식으로 물이 콸콸 쏟아졌다가 멈추는 '양동이 물 붓기'입니다. 커패시터(평활 회로)는 그 밑에 커다란 '물탱크'를 두어, 양동이로 물을 붓든 말든 수도꼭지에서는 물이 일정한 굵기로 쪼르르 나오게 만들어 줍니다.

## 개념 맵 (Concept Map)
```ascii
[정류 회로 (Rectifier)]
   │
   ├── 핵심 소자: 다이오드 (단방향 통과)
   │
   ├── 주요 역할: 교류(AC) -> 맥동 직류(Pulsating DC) 변환
   │
   ├── 종류
   │     ├── 반파 정류 (다이오드 1개): 효율 50%, 낭비 심함
   │     └── 전파 정류 (다이오드 4개 브리지): 효율 100%, PSU 표준
   │
   └── 후속 처리: 평활 회로 (Filter)
         └── 커패시터/인덕터 사용 -> 리플(출렁임) 제거 -> 안정적인 순수 DC 생성
```

## 어린이 비유
정류 회로는 요리사가 **오이 껍질을 예쁘게 까는 도구(감자칼)** 같아요! 콘센트에서 오는 전기는 껍질이 울퉁불퉁 튀어나와 있어서(교류) 컴퓨터가 바로 먹을 수 없어요. 정류 회로라는 감자칼로 슥슥 밀어내서 위로 뾰족한 부분들만 예쁘게 다듬어주면, 컴퓨터가 체하지 않고 맛있게 냠냠(직류) 먹을 수 있답니다.

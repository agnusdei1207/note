> **핵심 인사이트**
> 1. 「데이터 산업진흥 및 이용촉진에 관한 기본법」(데이터기본법, 2022년 시행)은 데이터를 국가 전략 자산으로 선언하고, 데이터 생산·거래·활용의 기본 원칙과 데이터 산업 생태계 육성 체계를 규정한 한국 최초의 데이터 기본법이다.
> 2. 데이터기본법의 핵심은 공정한 데이터 이용 보장(데이터 독점 방지) + 데이터 기술·서비스 산업 육성 + 데이터 거래소·데이터 바우처 법적 근거 마련의 3축이며, 기존 개인정보보호법(PIPA)과 상호 보완 관계다.
> 3. GDPR(유럽)·CCPA(미국 캘리포니아)·데이터기본법(한국)으로 대표되는 글로벌 데이터 규제 삼각 구도를 이해하는 것이 글로벌 데이터 서비스 설계의 필수 전제가 됐다.

---

## I. 데이터기본법 개요

```
법령 정보:
  명칭: 데이터 산업진흥 및 이용촉진에 관한 기본법
  약칭: 데이터기본법
  시행: 2022년 4월 20일
  소관: 과학기술정보통신부

주요 내용:
  1. 데이터 정의 확장
     전자적 형태의 모든 자료·정보 포함
     
  2. 데이터 생산·거래·활용 기본 원칙
     공정 이용, 독점 방지
     
  3. 국가 데이터 거버넌스 체계
     국가데이터정책위원회 설치
     
  4. 데이터 산업 생태계 지원
     데이터 전문기업 인증
     데이터 거래사 자격
     
  5. 분쟁 조정 제도
     데이터 분쟁조정위원회
```

> 📢 **섹션 요약 비유**: 부동산 거래에 부동산 기본법이 있듯이, 이제 데이터 거래에도 기본법이 생겨 공정한 거래와 산업 육성 기반이 마련됐다.

---

## II. 데이터기본법 핵심 조항

```
제3조 (데이터 이용 기본 원칙):
  1. 공정 이용 원칙
  2. 데이터 독점 배제
  3. 개인정보 보호 조화

제11조 (데이터 거래):
  데이터 거래 기준 계약 고시
  데이터 거래소 지정 운영

제15조 (데이터 전문기업 인증):
  데이터 수집·가공·분석·판매 전문기업 인증
  인증 기업 우선 입찰, 지원 사업 참여 혜택

제17조 (데이터 바우처):
  중소기업·스타트업의 데이터 구매·가공 비용 지원
  정부가 바우처 지급 -> 데이터 공급기업에 사용

부정경쟁방지법 연계:
  데이터 부정취득·사용행위를 부정경쟁행위로 규정
  (2021년 개정으로 데이터 자산 법적 보호)
```

| 조항         | 내용                        | 의미                 |
|------------|---------------------------|---------------------|
| 제3조       | 데이터 이용 기본 원칙          | 독점 방지, 공정 거래  |
| 제11조      | 데이터 거래 제도화             | 거래소, 표준 계약     |
| 제15조      | 전문기업 인증제도              | 산업 육성             |
| 제17조      | 데이터 바우처                 | 수요 기업 지원        |

> 📢 **섹션 요약 비유**: 데이터기본법은 데이터 경제의 헌법 — 공정 거래 규칙(독점 방지)과 인프라 지원(바우처, 거래소)을 동시에 제공.

## III. 관련 법령 체계

```
한국 데이터 관련 법령 체계:

                 [헌법: 개인정보 보호 기본권]
                        |
        +---------------+---------------+
        |                               |
[개인정보 보호법]              [데이터기본법]
(PIPA - 개인 데이터 보호)      (산업 데이터 활용)
        |                               |
[정보통신망법]              [데이터 산업 지원 정책]
[신용정보법]               [데이터 거래소 운영 지침]
[의료법]                   [공공데이터 제공 및 이용 활성화법]
[금융거래법]
```

| 법률             | 주요 대상          | 목적            |
|---------------|-----------------|----------------|
| 개인정보보호법  | 개인 데이터        | 프라이버시 보호  |
| 데이터기본법    | 산업 데이터        | 산업 육성        |
| 공공데이터법    | 공공기관 데이터    | 개방 촉진        |
| 신용정보법      | 금융 데이터       | 마이데이터 근거  |

> 📢 **섹션 요약 비유**: 개인정보보호법이 개인의 데이터 권리 지킴이라면, 데이터기본법은 데이터 경제 성장 촉진제 — 둘은 충돌하지 않고 상호 보완한다.

## IV. 글로벌 데이터 규제 비교

```
주요국 데이터 규제 비교:

한국 (데이터기본법, 2022):
  + 산업 데이터 활용 중심
  + 데이터 바우처, 전문기업 인증
  + 개인정보보호법(2020 개정) 병행

EU (GDPR, 2018):
  + 개인정보 보호 초강력
  + 정보 주체 권리 (삭제권, 이동권)
  + Data Act(2024): 산업 데이터 공유

미국 (연방 단위 없음):
  + 주(State)별 규제
  + CCPA (캘리포니아, 2020)
  + HIPAA (의료), COPPA (아동) 분야별

중국 (데이터보안법/PIPL, 2021):
  + 데이터 현지화 의무
  + 국가 주도 데이터 안보 관리
```

> 📢 **섹션 요약 비유**: EU는 "개인 프라이버시 최우선", 한국은 "프라이버시 + 산업 활용 균형", 중국은 "국가 통제 + 보안 최우선", 미국은 "자유 시장 + 일부 규제".

## V. 실무 시나리오 — 데이터 바우처 활용

```
지원 내용 (2024 기준):
  공급기업: 데이터 보유 기업 (네이버, KCB 등 인증 기업)
  수요기업: 중소기업, 스타트업
  지원 규모: 기업당 최대 7,000만원

활용 절차:
  1. 수요기업 신청 (K-Data 포털)
  2. 지원 대상 선정 (심사)
  3. 바우처 발급
  4. 공급기업으로부터 데이터 구매/가공 서비스 이용
  5. 대금 청구 시 바우처로 결제

성과 사례:
  의료 스타트업:
    병원 임상 데이터 바우처로 구매
    AI 진단 모델 학습 -> 제품 출시 가속

  물류 스타트업:
    배송 데이터 구매 -> 경로 최적화 AI 개발
    투자 유치 성공
```

> 📢 **섹션 요약 비유**: 정부가 데이터 구매 상품권을 주는 것 — 돈이 없는 스타트업도 양질의 데이터를 이용해 AI 제품을 만들 수 있게 해준다.

## 👶 어린이를 위한 3줄 비유 설명

1. 데이터기본법은 데이터를 사고 팔 때 지켜야 할 규칙을 정한 법이에요 — 누군가가 데이터를 독점하거나 허락 없이 사용하면 안 된다고 규정해요.
2. 정부가 작은 회사들에게 데이터 구매 상품권(바우처)을 줘서 값비싼 데이터도 살 수 있게 해줘요.
3. 한국, 유럽(GDPR), 미국이 각각 다른 데이터 규칙을 가지고 있어서 글로벌 서비스를 만들려면 세 나라 규칙을 모두 알아야 해요!

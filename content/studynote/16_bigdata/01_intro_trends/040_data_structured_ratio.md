> **핵심 인사이트**
> 1. 전 세계 데이터의 80% 이상이 비정형(Unstructured) 데이터이며, 기업이 실제로 분석에 활용하는 데이터는 20% 이하의 정형 데이터에 불과하다 — 이 "80/20 역설"이 빅데이터 분석의 핵심 도전이자 기회다.
> 2. 정형 데이터(Structured) → 반정형 데이터(Semi-structured) → 비정형 데이터(Unstructured)의 스펙트럼에서, 기업 정보화의 성숙도는 비정형 데이터를 얼마나 활용 가능한 형태로 전환하느냐로 측정된다.
> 3. 비정형 데이터 활용의 폭발적 성장은 NLP(자연어 처리)·컴퓨터 비전·멀티모달 AI의 발전과 직결되며, 2025년 IDC 전망에 따르면 전 세계 연간 생성 데이터의 93%가 비정형 데이터가 된다.

---

## Ⅰ. 데이터 유형 분류

```
데이터 구조화 스펙트럼:

정형 데이터 (Structured Data, ~20%):
  정의: 사전 정의된 스키마(Schema)를 따르는 데이터
  형태: 테이블(행/열), 고정 형식
  예: RDBMS 테이블, 엑셀 스프레드시트
  처리: SQL로 직접 쿼리 가능
  분석 용이성: ★★★★★

반정형 데이터 (Semi-structured, ~10%):
  정의: 완전한 스키마는 없지만 구분자/태그로 구조 있음
  형태: 계층적, 자기 기술적(Self-describing)
  예: JSON, XML, HTML, CSV, 이메일
  처리: 파싱 후 분석, NoSQL DB
  분석 용이성: ★★★☆☆

비정형 데이터 (Unstructured, ~80%):
  정의: 고정된 구조 없는 데이터
  형태: 자유 형식
  예: 텍스트, 이미지, 동영상, 음성, SNS
  처리: AI/ML, NLP, CV 필요
  분석 용이성: ★☆☆☆☆ (AI 전에는)
```

> 📢 **섹션 요약 비유**: 데이터 유형은 서류 정리 방식 — 정형은 표로 정리된 영수증, 반정형은 태그 붙은 폴더, 비정형은 무분별하게 쌓인 사진/메모.

## Ⅱ. 비정형 데이터 현황

```
비정형 데이터 폭증 현황 (2024):

생성 원천:
  SNS:        게시글, 댓글, 이미지, 동영상
  IoT:        센서 로그, 스트림 데이터
  기업 내부:  이메일, 계약서, 회의록, 채팅
  의료:       의무기록(EMR), MRI, CT 이미지
  금융:       뉴스, 공시, 영업 메모
  웹:         HTML, 검색 쿼리, 클릭스트림

규모 (IDC 2025 전망):
  전 세계 연간 생성: ~175 제타바이트(ZB)
  비정형 비율: 93%
  기업이 분석에 활용: 전체의 32%만

활용 못 하는 "다크 데이터":
  기업 데이터 55~80%가 수집만 하고 미활용
  비용만 발생 (스토리지)
  가치 창출 기회 손실
```

> 📢 **섹션 요약 비유**: 다크 데이터는 창고에 쌓아둔 상자 — 뭐가 들었는지 몰라서 열어보지도 않고 임대비(스토리지)만 내는 상황.

## Ⅲ. 비정형 데이터 분석 기술

```
비정형 데이터 처리 기술:

텍스트 (NLP):
  형태소 분석, 토큰화
  감성 분석 (Sentiment Analysis)
  개체명 인식 (NER)
  LLM (GPT, BERT): 문서 요약, Q&A
  
이미지 (Computer Vision):
  CNN 기반 분류, 탐지
  OCR (Optical Character Recognition)
  의료 영상 진단 AI (영상의학 보조)
  
음성 (Speech):
  ASR (Automatic Speech Recognition)
  화자 인식 (Speaker Identification)
  감정 인식
  
동영상:
  시계열 이미지 분석 + 음성 분석
  CCTV 이상 탐지
  스포츠 영상 분석

멀티모달 AI (현재 최전선):
  텍스트 + 이미지 + 음성 통합 처리
  GPT-4V, Gemini Ultra
```

> 📢 **섹션 요약 비유**: 비정형 데이터 분석은 사진첩 자동 분류 — AI가 수천 장 사진에서 사람 얼굴, 장소, 날짜를 자동으로 인식하고 정리.

## Ⅳ. 비정형 데이터 → 정형화 파이프라인

```
비정형 데이터 정형화 프로세스:

[원천 데이터]
    비정형 (이메일, PDF, 이미지)
         |
         v
[전처리 (Preprocessing)]
    OCR, 음성 인식, 크롤링
    텍스트 정제 (노이즈 제거)
         |
         v
[특징 추출 (Feature Extraction)]
    NLP: 키워드, 감성, 개체명
    CV: 특징 벡터, 객체 좌표
         |
         v
[구조화 (Structuring)]
    추출된 특징 -> 테이블/JSON 변환
    메타데이터 생성
         |
         v
[저장 및 분석]
    데이터 웨어하우스 / 데이터 레이크
    BI 도구, ML 모델 입력

예시:
  보험 청구서(PDF) -> OCR -> 텍스트 추출
  -> NLP 정보 추출 (사고 유형, 금액, 날짜)
  -> 정형 데이터 -> 청구 분류 모델 입력
```

> 📢 **섹션 요약 비유**: 비정형 정형화는 손으로 쓴 일기를 구조화된 엑셀로 변환하기 — AI가 자동으로 날짜, 감정, 장소를 추출해 표로 만들어준다.

## Ⅴ. 실무 시나리오 — 금융 비정형 데이터 활용

```
금융사 A의 비정형 데이터 분석 전략:

현황:
  정형 데이터: 거래 내역, 고객 정보 (20%)
  비정형 데이터: 상담 메모, 민원 이메일, 
                계약서, 뉴스, SNS (80%)
  
활용 사례:

1. 고객 상담 메모 분석:
   콜센터 상담 내용 -> STT(음성→텍스트)
   NLP 감성 분석 -> 이탈 위험 고객 식별
   -> 선제적 고객 유지 캠페인

2. 계약서 자동 검토:
   PDF 계약서 -> OCR + NLP
   이상 조항 탐지, 리스크 키워드 추출
   법무팀 검토 시간 70% 단축

3. 뉴스/SNS 리스크 모니터링:
   기업 고객 관련 뉴스 실시간 수집
   부정적 기사 감지 -> 신용 위험 조기 경보

4. 공시 문서 분석:
   DART 공시 자동 파싱
   재무 지표 추출 + 경영진 발언 감성 분석

ROI:
  다크 데이터 활용 전환 -> 신규 수익원
  운영 비용 절감 (자동화)
  리스크 관리 고도화
```

> 📢 **섹션 요약 비유**: 금융 비정형 데이터 활용은 창고 정리 후 숨은 보물 발견 — 쌓아둔 상담 메모와 계약서에서 AI로 가치 있는 인사이트를 발굴.

## 📌 관련 개념 맵

```
데이터 정형화 비율
+-- 유형
|   +-- 정형 (~20%): RDBMS, 스프레드시트
|   +-- 반정형 (~10%): JSON, XML, CSV
|   +-- 비정형 (~80%): 텍스트, 이미지, 영상
+-- 비정형 활용 기술
|   +-- NLP (텍스트)
|   +-- Computer Vision (이미지/영상)
|   +-- ASR (음성)
|   +-- 멀티모달 AI
+-- 파이프라인
|   +-- 전처리 -> 특징 추출 -> 구조화
+-- 과제
    +-- 다크 데이터 (미활용 데이터)
    +-- 데이터 활용 성숙도
```

```
      v

      |
      v

비정형 데이터 분석 가능성 폭발
      |
      v

GPT, BERT: 텍스트 비정형 완전 정복

      |
      v
[현재: 비정형 = 가장 큰 데이터 자산]

RAG, 기업 LLM 도입 붐
```

## 👶 어린이를 위한 3줄 비유 설명

1. 데이터에는 표처럼 깔끔한 정형 데이터(약 20%)와 사진·영상·글처럼 뒤죽박죽인 비정형 데이터(약 80%)가 있어요.
2. 예전엔 비정형 데이터는 컴퓨터가 이해 못 해서 창고에 쌓아두기만 했는데, AI 덕분에 이 80%도 분석할 수 있게 됐어요.
3. 고객 상담 녹음, 계약서 PDF, 소셜미디어 게시글 같은 비정형 데이터를 분석하면 숨겨진 중요한 정보를 찾을 수 있어요!

## 핵심 인사이트 (3줄 요약)
> 1. **본질**: 출력의 오차를 출발점으로 연쇄 법칙(Chain Rule)을 역방향 적용해 각 가중치의 기여도(기울기)를 계산한다.
> 2. **가치**: 다층 신경망의 학습을 가능하게 한 알고리즘으로, 딥러닝 부흥의 수학적 핵심이다.
> 3. **판단 포인트**: 기울기 소실(Vanishing Gradient) 문제는 ReLU와 배치 정규화(Batch Normalization)로 완화하고, 깊은 망은 잔차 연결(Residual Connection)로 보완한다.

## Ⅰ. 개요 및 필요성

신경망은 수천~수억 개의 가중치(Weight)를 갖는다. 올바른 예측을 위해서는 이 가중치들을 조정해야 하는데, "어떤 가중치가 오류에 얼마나 기여했는가?"를 측정하는 것이 역전파(Backpropagation)이다.

순전파(Forward Pass)에서 입력이 층을 거쳐 출력으로 나오면, 손실 함수(Loss Function)가 예측값과 실제값의 차이를 계산한다. 역전파는 이 오차를 출력층에서 입력층 방향으로 전달하며, 각 가중치에 대한 편미분(∂L/∂w)을 구한다.

**연산 흐름**
1. Forward Pass: x → h → ŷ → L(ŷ, y)
2. Loss 계산: L = MSE 또는 Cross-Entropy
3. Backward Pass: ∂L/∂w = 연쇄 법칙 적용
4. 가중치 업데이트: w ← w − η · ∂L/∂w

📢 **섹션 요약 비유**: 오답 답안지를 받아 "어느 문제에서 틀렸는지" 역순으로 채점하는 것과 같다.

## Ⅱ. 아키텍처 및 핵심 원리

### 연쇄 법칙 (Chain Rule)

합성 함수 L = f(g(h(w)))에서 ∂L/∂w = (∂L/∂f)·(∂f/∂g)·(∂g/∂h)·(∂h/∂w)

| 개념 | 수식 | 역할 |
|:---|:---|:---|
| 편미분 | ∂L/∂w | 손실에 대한 가중치 기울기 |
| 연쇄 법칙 | ∂L/∂w₁ = ∂L/∂y · ∂y/∂h · ∂h/∂w₁ | 다층 기울기 전파 |
| 학습률 (Learning Rate) | η | 업데이트 보폭 조절 |
| 기울기 소실 | sigmoid 반복 시 기울기 → 0 | 깊은 망 학습 불가 |
| 기울기 폭발 | 기울기 > 1 반복 누적 | 클리핑(Gradient Clipping)으로 해결 |

```
순전파 (Forward Pass)
┌─────┐   w₁   ┌─────┐   w₂   ┌─────┐
│  x  │──────▶│  h  │──────▶│  ŷ  │──▶ Loss L
└─────┘        └─────┘        └─────┘

역전파 (Backward Pass)
┌─────┐   ∂L/∂w₁  ┌─────┐  ∂L/∂w₂  ┌─────┐
│  x  │◀──────────│  h  │◀──────────│  ŷ  │◀── ∂L/∂ŷ
└─────┘            └─────┘            └─────┘
         연쇄 법칙으로 기울기 역방향 전달
```

### 기울기 소실 vs 폭발

```
기울기 소실 (Vanishing)        기울기 폭발 (Exploding)
σ'(x) ≤ 0.25                  기울기 > 1 반복 시
깊어질수록 0에 수렴             기하급수 증가
→ ReLU, 배치 정규화로 해결      → Gradient Clipping
```

📢 **섹션 요약 비유**: 귓속말 전달 게임에서 목소리가 점점 작아지는 것이 기울기 소실이고, 소리를 키우다 귀청이 터지는 것이 기울기 폭발이다.

## Ⅲ. 비교 및 연결

| 항목 | 역전파 (Backprop) | BPTT (Backpropagation Through Time) |
|:---|:---|:---|
| 적용 대상 | 일반 신경망 (MLP, CNN) | 순환 신경망 (RNN) |
| 시간 축 전개 | 없음 | 시퀀스 길이만큼 전개 |
| 기울기 소실 | 깊이에 비례 | 시퀀스 길이에 비례 |
| 해결책 | ResNet 잔차 연결 | LSTM (Long Short-Term Memory) |

역전파는 경사 하강법(Gradient Descent)과 결합해 최적화기(Optimizer)로 발전했다. SGD (Stochastic Gradient Descent) → Momentum → Adam의 진화 과정 모두 역전파로 계산된 기울기를 활용한다.

📢 **섹션 요약 비유**: 기울기는 산에서 가장 가파른 경사 방향이고, 역전파는 어느 쪽이 더 가파른지 측정하는 도구이다.

## Ⅳ. 실무 적용 및 기술사 판단

**설계 체크리스트**
- 활성화 함수: 은닉층은 ReLU (Rectified Linear Unit) 계열 사용 → 기울기 소실 방지
- 가중치 초기화: Xavier(Glorot) 또는 He 초기화로 초기 기울기 분산 조정
- 배치 정규화(Batch Normalization): 각 층의 출력을 정규화해 기울기 흐름 안정화
- 잔차 연결(Residual Connection): 기울기를 직접 전달해 100층 이상 깊은 망 훈련 가능
- 그래디언트 클리핑(Gradient Clipping): RNN 학습 시 기울기 폭발 방지, max_norm 설정

**기술사 출제 포인트**
- "역전파의 수학적 원리와 기울기 소실 문제 해결 방안을 설명하시오"
- 연쇄 법칙 수식과 순전파/역전파 흐름 구분이 핵심

📢 **섹션 요약 비유**: 역전파는 공장 불량 제품의 원인을 제조 라인 역순으로 추적하는 품질 관리 과정이다.

## Ⅴ. 기대효과 및 결론

역전파 알고리즘은 1986년 Rumelhart 등이 발표한 이래 딥러닝 전체의 근간이 되었다. GPU 병렬 계산과 결합해 수십억 파라미터의 대규모 모델도 효율적으로 학습할 수 있게 되었다.

현대 딥러닝 프레임워크(PyTorch, TensorFlow)는 자동 미분(Autograd) 기능으로 역전파를 자동화한다. 개발자는 순전파 로직만 작성하면 프레임워크가 연쇄 법칙을 자동 적용한다.

📢 **섹션 요약 비유**: 역전파는 신경망이 스스로 틀린 이유를 파악하는 '자기 성찰 능력'이다. 이 능력이 없으면 아무리 깊어도 아무것도 배울 수 없다.

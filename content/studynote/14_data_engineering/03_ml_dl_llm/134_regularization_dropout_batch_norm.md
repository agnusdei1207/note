## 핵심 인사이트 (3줄 요약)
> 1. **본질**: L1/L2 패널티, 드롭아웃(Dropout), 배치 정규화(Batch Normalization)는 모두 모델이 훈련 데이터에만 과도하게 맞춰지는 과적합(Overfitting)을 방지한다.
> 2. **가치**: 훈련 정확도와 검증 정확도의 격차를 줄여 실제 배포 환경에서도 일반화(Generalization) 성능을 보장한다.
> 3. **판단 포인트**: L1은 희소(Sparse) 모델이 필요할 때, L2는 전반적 가중치 억제에, Dropout은 앙상블 효과, BN은 학습 속도와 안정성을 동시에 높이고 싶을 때 선택한다.

## Ⅰ. 개요 및 필요성

과적합(Overfitting)은 모델이 훈련 데이터의 노이즈까지 암기해 새 데이터에 일반화하지 못하는 현상이다. 파라미터가 많은 딥러닝 모델일수록 과적합 위험이 높다.

규제(Regularization)는 모델 복잡도에 페널티를 부여하거나 학습 과정에 무작위성을 도입해 과적합을 억제한다. 주요 기법으로 L1/L2 정규화, Dropout, Batch Normalization이 있다.

**과적합 vs 과소적합**
- 과적합(Overfitting): Train Loss 낮음, Validation Loss 높음 → 복잡도 축소 필요
- 과소적합(Underfitting): Train Loss 높음 → 모델 표현력 부족

📢 **섹션 요약 비유**: 시험 직전 답안을 통째 암기한 학생은 시험장에서 응용 문제를 못 푼다. 규제는 "이해"하도록 강제하는 교육 방침이다.

## Ⅱ. 아키텍처 및 핵심 원리

| 기법 | 손실 함수 변형 | 효과 | 단점 |
|:---|:---|:---|:---|
| L1 규제 (Lasso) | L + λΣ|wᵢ| | 희소(Sparse) 가중치, 특징 선택 | 미분 불연속 |
| L2 규제 (Ridge/Weight Decay) | L + λΣwᵢ² | 작은 가중치 유지, 부드러운 모델 | 중요 특징도 축소 |
| Dropout | 학습 시 뉴런 임의 비활성화 (p=0.5) | 앙상블 효과, 공동 적응 방지 | 추론 시 비활성화 해제 |
| Batch Normalization (BN) | 미니배치 단위 정규화 | 학습 속도 향상, Internal Covariate Shift 감소 | 배치 크기 의존 |
| Layer Normalization | 샘플 단위 정규화 | 배치 크기 무관, Transformer 적합 | |

```
[Dropout 적용 예시]

학습 시 (p=0.5 드롭아웃)          추론 시 (전체 활성)
  ┌───┐   ┌───┐   ┌───┐            ┌───┐   ┌───┐   ┌───┐
  │ h₁│   │ ✕ │   │ h₃│            │ h₁│   │ h₂│   │ h₃│
  └─┬─┘   └───┘   └─┬─┘            └─┬─┘   └─┬─┘   └─┬─┘
    │   (비활성화)   │                │         │         │
    └───────┬────────┘                └────┬────┘         │
            ▼                              ▼              ▼
         출력층                          출력층 (×(1-p) 스케일)

[Batch Normalization]
  미니배치 B = {x₁, x₂, ..., xₘ}
  μ_B = (1/m)Σxᵢ         ← 평균
  σ²_B = (1/m)Σ(xᵢ-μ)²  ← 분산
  x̂ᵢ = (xᵢ - μ_B) / √(σ²_B + ε)
  yᵢ = γ·x̂ᵢ + β          ← 스케일/시프트 파라미터 학습
```

📢 **섹션 요약 비유**: Dropout은 팀 연습 때 무작위로 선수를 빼 특정 선수 의존도를 낮추는 훈련이다. BN은 매 연습 전 모든 선수의 컨디션을 균일하게 조정하는 준비 운동이다.

## Ⅲ. 비교 및 연결

| 기준 | L1 | L2 | Dropout | BN |
|:---|:---|:---|:---|:---|
| 파라미터 수 변화 | 일부 0으로 수렴 | 작게 유지 | 변화 없음 | γ, β 추가 |
| 특징 선택 효과 | ✅ 있음 | ❌ 없음 | ❌ | ❌ |
| 추론 시 동작 | 동일 | 동일 | 비활성화 해제 | 이동 평균 사용 |
| 주요 사용처 | 선형 모델, 희소 | 딥러닝 기본 | FC 레이어 | Conv/FC 레이어 사이 |

BN 이후 Transformer에서는 Layer Normalization (LN)이 주류가 되었으며, 배치 크기에 무관해 소형 배치나 언어 모델에 적합하다.

📢 **섹션 요약 비유**: L1은 불필요한 물건을 과감히 버리는 미니멀리스트, L2는 모든 물건을 작게 줄이는 절약형, Dropout은 팀원을 랜덤으로 빼는 훈련법이다.

## Ⅳ. 실무 적용 및 기술사 판단

**설계 가이드라인**
- 소규모 데이터 + 고차원 입력: L1 규제로 불필요한 특징 제거
- 일반 딥러닝: L2 (Weight Decay, λ=1e-4 수준)를 기본으로
- FC 레이어 과적합 방지: Dropout(p=0.3~0.5) 삽입
- Conv 레이어 후: BN 삽입으로 학습 속도 및 안정성 향상
- Transformer 계열: BN 대신 Layer Normalization 사용

**기술사 출제 포인트**
- "Dropout이 앙상블 효과를 갖는 이유를 설명하시오": 매 학습 스텝마다 다른 서브네트워크를 학습하는 효과
- "Batch Normalization의 Internal Covariate Shift 감소 원리를 설명하시오"

📢 **섹션 요약 비유**: 운동선수가 체중 관리(L2), 약점 노출(L1), 조건 불리한 훈련(Dropout), 일정한 컨디션 유지(BN)를 통해 경기력을 극대화하는 것과 같다.

## Ⅴ. 기대효과 및 결론

규제 기법의 올바른 조합은 모델 성능의 일반화를 보장한다. 현대 딥러닝 모델은 대부분 L2 + BN + Dropout을 함께 사용하며, 특히 BN의 도입으로 학습률을 10배 이상 높일 수 있게 됐다.

과적합 방지는 데이터 증강(Data Augmentation), 조기 종료(Early Stopping), 앙상블(Ensemble) 등과 함께 종합적으로 적용해야 한다.

📢 **섹션 요약 비유**: 규제는 모델이 "시험 답안 암기"가 아닌 "원리 이해"를 하도록 훈련하는 교육 철학이다.

### 👶 어린이를 위한 3줄 비유 설명
1. 시험 답안을 통째 외우지 않고 개념을 이해하게 만드는 규칙이에요 — 그게 규제예요.
2. 드롭아웃은 연습 때 팀원을 랜덤으로 쉬게 해서 누구 하나에만 의존하지 않게 훈련하는 방법이에요.
3. 배치 정규화는 매번 수업 전 학생들 집중력을 같은 수준으로 맞춰주는 준비 체조예요.

## 핵심 인사이트 (3줄 요약)
> 1. **본질**: 커널(Kernel/Filter)이 입력 이미지 위를 슬라이딩하며 공간적 패턴(엣지, 질감, 형태)을 계층적으로 추출한다.
> 2. **가치**: 파라미터 공유(Parameter Sharing)와 지역 수용 영역(Local Receptive Field)으로 완전 연결망 대비 파라미터를 수십 배 줄이면서 이미지 인식 성능을 압도한다.
> 3. **판단 포인트**: 깊이(Depth)는 복잡한 특징 추출에, 풀링(Pooling)은 공간 불변성(Spatial Invariance) 확보에, Stride/Padding은 특징 맵(Feature Map) 크기 조절에 사용한다.

## Ⅰ. 개요 및 필요성

이미지를 단순 완전 연결 신경망(MLP)으로 처리하면 224×224×3 이미지 하나에 150,528개의 입력 노드가 필요하다. CNN (Convolutional Neural Network)은 공간 구조를 활용해 이 문제를 해결한다.

커널은 작은 가중치 행렬(3×3, 5×5)로, 이미지 전체를 슬라이딩하며 특징 맵(Feature Map)을 생성한다. 같은 커널을 모든 위치에 공유하므로 파라미터가 극적으로 줄어든다.

**핵심 특성**
- 지역 연결성 (Local Connectivity): 인근 픽셀만 연결
- 가중치 공유 (Parameter Sharing): 동일 커널 전체 사용
- 계층적 특징 추출: 저수준(엣지) → 중수준(질감) → 고수준(객체) 패턴

📢 **섹션 요약 비유**: CNN은 그림에서 윤곽선, 색상, 형태를 차례로 파악하는 인간 시각 피질과 같다.

## Ⅱ. 아키텍처 및 핵심 원리

| 레이어 | 역할 | 주요 파라미터 |
|:---|:---|:---|
| Conv Layer (합성곱 층) | 커널로 특징 맵 생성 | 필터 수, 커널 크기, Stride, Padding |
| ReLU Activation | 비선형성 추가, 기울기 소실 방지 | - |
| Pooling Layer (풀링 층) | 공간 크기 축소, 불변성 확보 | Max Pooling, Average Pooling |
| Flatten | 특징 맵 → 1D 벡터 변환 | - |
| FC Layer (완전 연결 층) | 분류/회귀 출력 | 뉴런 수 |
| Softmax | 다중 클래스 확률 출력 | 클래스 수 |

     │               │                │                 │               │

📢 **섹션 요약 비유**: 커널은 사진을 확대경으로 부분부분 훑으며 특징을 기록하는 '패턴 감지 도장'이다.

## Ⅲ. 비교 및 연결

| 항목 | CNN | MLP (Multi-Layer Perceptron) |
|:---|:---|:---|
| 파라미터 수 | 적음 (공유) | 매우 많음 |
| 공간 구조 활용 | ✅ 위치 관계 반영 | ❌ 1D 벡터 처리 |
| 이미지 성능 | 압도적 우위 | 열등 |
| 시퀀스 처리 | 1D CNN 사용 가능 | 가능 |

**대표 CNN 모델 발전사**
- AlexNet (2012): ReLU, Dropout 도입, ImageNet 우승
- VGGNet (2014): 3×3 커널만 사용, 깊이 16~19층
- ResNet (2015): 잔차 연결(Skip Connection)로 152층 학습
- EfficientNet (2019): 너비/깊이/해상도 균형 스케일링

📢 **섹션 요약 비유**: MLP가 사진 전체를 한 번에 보는 것이라면, CNN은 사진을 부분부분 확대경으로 차근차근 훑는 방식이다.

## Ⅳ. 실무 적용 및 기술사 판단

**설계 가이드라인**
- 커널 크기: 3×3을 여러 번 쌓는 것이 5×5 한 번보다 파라미터 효율적 (VGG 방식)
- Padding='same': 입출력 크기 동일 유지 → 특징 맵 크기 보존
- Stride=2: Pooling 대체해 공간 축소, 계산 효율 향상
- Depthwise Separable Convolution: MobileNet에서 사용, 연산량 1/8 수준
- Global Average Pooling (GAP): Flatten 대신 사용해 FC 파라미터 절감

**기술사 출제 포인트**
- "CNN의 파라미터 공유 원리와 완전 연결망 대비 장점을 설명하시오"
- "Max Pooling과 Average Pooling의 차이와 용도를 비교하시오"

📢 **섹션 요약 비유**: 스마트폰 카메라가 얼굴 부위를 인식할 때 전체 사진이 아닌 눈·코·입 부분을 따로 분석하는 것이 CNN의 동작 방식이다.

## Ⅴ. 기대효과 및 결론

CNN은 컴퓨터 비전 분야를 완전히 바꿨다. ImageNet 대회에서 2012년 AlexNet이 기존 방법 대비 오류율을 10%p 이상 낮추며 딥러닝 시대를 열었다. 이후 의료 영상 분석, 자율주행 물체 인식, 제조업 불량 검출 등 실무에 광범위하게 적용된다.

최근에는 Vision Transformer (ViT)가 등장해 순수 CNN의 독주에 도전하고 있으나, 소규모 데이터·엣지 디바이스 환경에서는 CNN이 여전히 우위를 점한다.

📢 **섹션 요약 비유**: CNN은 인간 눈의 '시각 피질'을 수학으로 구현한 것이다. 작은 부분에서 큰 그림을 이해하는 계층적 시각 처리를 모방한다.

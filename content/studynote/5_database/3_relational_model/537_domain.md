# 도메인 (Domain)

## 핵심 인사이트 (3줄 요약)
> 1. **본질**: 도메인 (Domain)은 속성이 가질 수 있는 값의 집합으로, 데이터 타입과 제약조건으로 정의된다.
> 2. **가치**: 도메인을 통해 속성의 유효 값 범위를 제한하고, 데이터 무결성을 보장할 수 있다.
> 3. **융합**: 사용자 정의 도메인을 생성하여 업무에 특화된 데이터 타입을 만들 수 있다.

---

## Ⅰ. 개요 및 필요성

### 개념 정의

도메인 (Domain)은 **하나의 속성이 가질 수 있는 값의 범위**로, 데이터베이스에서 속성의 타입과 유효성을 정의한다.

```
┌─────────────────────────────────────────────────────────────────────┐
│                    도메인 예시                                             │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   [도메인 정의]                                                     │
│                                                                     │
│   CREATE DOMAIN GenderType AS VARCHAR(1)                            │
│       CHECK (VALUE IN ('M', 'F'));                                 │
│                                                                     │
│   CREATE DOMAIN PositiveInt AS INTEGER                              │
│       CHECK (VALUE > 0);                                           │
│                                                                     │
│   [도메인 사용]                                                     │
│                                                                     │
│   CREATE TABLE employees (                                           │
│       emp_id INTEGER,                                               │
│       name VARCHAR(50),                                            │
│       gender GenderType,  -- 사용자 정의 도메인 사용                │
│       age PositiveInt       -- 사용자 정의 도메인 사용                │
│   );                                                               │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

### 비유

도메인은 **항목卡的許容範囲**와 같다. 예를 들어, "나이" 항목卡에는 0세부터 150세까지만 입력 가능하다면, 도메인은 0~150의 정수이다.

## 👶 어린이를 위한 3줄 비유 설명

1. 도메인은 **말의 영역**과 같아요. "포유류" 도메인에는 개, 고양이 등이 포함되고, "조류" 도메인에는 독수리, 참새 등이 포함돼요.
2. 컴퓨터에서도 "나이" 도메인은 0살부터 150살까지의 숫자인 것처럼, 데이터의 허용 범위를定めておく必要があります.
3. 이렇게 정해두면 그 범위를 벗어난 데이터는 들어올 수 없게 돼요!

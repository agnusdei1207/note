> **💡 핵심 인사이트**
> 시계열 보간(Interpolation)은 **"시계열 데이터에서 결측치(Missing Values)나 불규칙한 샘플을, 주변 값들의 통계적 특성이나 시간적趨勢를 利用하여推定填充하는 기법"**입니다.
> IoT 센서, 금융 데이터, 모니터링 시스템에서는 네트워크 문제, 센서 오작동, 샘플링 실패 등으로 인해 데이터에间隙(Gap)이 발생합니다. 이间隙을そのまま放置하면 분석 결과가歪められ、分析不能になります。 보간은 **"ないデータを推定により埋める"**して、分析基盤を完整に保つ技術です。

---

## Ⅰ. 시계열 보간의 필요성과 원인

```
[시계열 데이터에서 결측이 발생하는 주요 원인]

  1. 센서 오작동 / 일시적 장애
     - 온도 센서가 每분 측정のはずが, 突然 5분 동안 값이 안 옴
     - 이유: 센서 내부 버퍼 오버플로우, 전원 문제

  2. 네트워크 지연 / 패킷 손실
     - IoT 센서가 데이터를 UDP로 전송
     - 네트워크 혼잡 시 일부 패킷 유실

  3. 샘플링 주기의 불규칙성
     - "1초마다 측정"이 목표였지만, 부하 상황에 따라 1.2초~2초로 변동

  4. 계획된维护 / 데이터 소스 일시 중단
     - 서버 재시작,定期 점검으로 인한 데이터空白
```

**보간 없이는:**
- 시계열 예측 모델(ARIMA, LSTM 등) 학습 불가
- 이동 평균, 차분 등 時系列 연산 결과 왜곡
- 대시보드에서 그래프가 끊겨 보임

---

## II. 주요 보간 방법

```
[보간 방법 분류]

  1. 선형 보간 (Linear Interpolation)
     - 결측 앞뒤 값들을 직선으로 연결
     - 단점:趨勢의转折点が 있으면 왜곡

  2. 다항식 보간 (Polynomial Interpolation)
     - Lagrange, Newton 등 고차원 다항식 적용
     - 다수 결측 구간에 사용

  3. 스플라인 보간 (Spline Interpolation)
     - 3차 스플라인(Cubic Spline): 구간별 3차 다항식
     - 매끄러운 곡선 보간에 적합

  4. 시간 加权 平均 보간 (Time-Weighted)
     - 시간적으로 가까운 값에 더 큰 가중치

  5.Forward/Backward Fill (이전/이후 값 채우기)
     - 마지막 값으로 채우기 (ffill)
     - 다음 값으로 채우기 (bfill)
     - 단점: 큰 결측 구간에서現実味 없는 值

  6. 평균값 / 中央값 보간
     - 주변값들의 평균 또는中央値로 채우기
     - 극단적 값에 강건

  7.ARIMA / ML 模型 보간
     - 시계열 모델로 결측 구간을 예측
     - 가장 정확하지만 계산 비용 높음
```

### 실제 코드 예시 (Python / Pandas)

```python
import pandas as pd
import numpy as np

# 샘플 시계열 데이터 (결측치 포함)
ts = pd.Series(
    [20.5, 21.2, np.nan, 22.8, np.nan, np.nan, 25.1, 26.3],
    index=pd.date_range('2024-01-01', periods=8, freq='h')
)
print("원본:", ts)

# 1. 선형 보간
ts_linear = ts.interpolate(method='linear')
print("선형 보간:", ts_linear)
# 결과: 20.5, 21.2, 22.0, 22.8, 23.65, 24.5, 25.1, 26.3
#        (21.2와 22.8 사이 2개 결측 → 균등 배분)

# 2. 시간 加权 보간 (시간에 비례)
ts_time = ts.interpolate(method='time')
# 시간 간격에 따라 가중치 부여

# 3. 스플라인 보간 (3차)
ts_spline = ts.interpolate(method='spline', order=3)
# 매끄러운 곡선으로 보간

# 4.Forward/Backward Fill
ts_ffill = ts.ffill()  # 이전 값으로 채우기
ts_bfill = ts.bfill()  # 이후 값으로 채우기

# 5. Panda久化均值 보간
ts_mean = ts.fillna(ts.mean())
```

## III. 시계열 보간과 보間の区别

```
[보간(Interpolation) vs 외挿(Extrapolation)]

  보간 (Interpolation):
  ┌─────────────────────────────────────────┐
  │  기존 데이터 범위 내의 결측을 채우는 것    │
  │                                          │
  │  시간: t=1 ── t=2 ── t=3 ── t=4 ── t=5 │
  │  값:   10    ?    30    ?    50        │
  │                    ↑                      │
  │              이 구간의 ? 들을 채움          │
  └─────────────────────────────────────────┘

  외挿 (Extrapolation):
  ┌─────────────────────────────────────────┐
  │  기존 데이터 범위를 넘어서는 값을 예측      │
  │                                          │
  │  시간: t=1 ── t=2 ── t=3 ── t=4 ── t=5 │
  │  值:   10    20    30    40    50        │
  │                                      ↑   │
  │                               t=6, t=7 예측 │
  └─────────────────────────────────────────┘
```

**보간 vs 외挿의 중요성:**
- 보간은 "있는 데이터를 바탕으로추정" → 비교적 정확
- 외挿은 "없는 미래를 예측" → 불확실성 높음 → 과도한 외삽은 위험

## Ⅴ. 보간 선택 가이드と 📢 비유

**보간 방법 선택 기준:**

```
[결측 구간의 크기에 따른 방법 선택]

  결측 구간 < 5% of 전체:
  → 선형 보간 / 스플라인 보간으로 충분
  → ffill/bfill도 가능 (값이 크게 변하지 않음)

  결측 구간 5~20%:
  → 스플라인 보간 또는 時系列 模型 (ARIMA)
  → 주변값의趨勢를 고려하므로 왜곡 적음

  결측 구간 > 20%:
  → 주의 필요! 과도한 보간은 데이터 왜곡
  → 해당 구간을 제거하고 분석하는 것이 나을 수 있음
  → 또는 다중 대체 (Multiple Imputation) 방법 고려
```

**시계열 특성에 따른 선택:**

| 시계열 특성 | 권장 보간 방법 |
|------------|-------------|
| 랜덤 변동이 큰 금융 데이터 | 평균값, 중앙값 보간 |
| 매끄러운 趨勢 (온도, 습도) | 스플라인, 선형 보간 |
| 계절성 있는 데이터 (매출) | 계절 조정 + 스플라인 |
| 불규칙한 샘플링 | 시간 加权 보간 (method='time') |

> 📢 **섹션 요약 비유:** 시계열 보간은 **" 끊어진影片을修復하는 것"**과 같습니다.影片에서 5초가 누락되면 그 부분을 어떻게 채울지가 문제입니다. 전후 이미지가 "해가 뜨는 장면"이었다면, **"그 사이에도 해가 뜨고 있었을 것"**이라는 정상적 추론으로 보간할 수 있습니다. 하지만 갑자기 바다 장면으로 바뀌면? 전후가 아무리 해가 뜨는 장면이라도 바다 장면을 집어넣으면 이상하죠. 보간도 같습니다: **"전후 데이터가 정상적이라면 그 사이도 정상적일 것이다"**라는仮説에 기반합니다. 하지만 데이터 자체가 급격히 변하는 구간에서는 보간이 오히려 **"보기에는 매끄럽지만 잘못된 값"**을生む 위험이 있습니다. **보간 방법選択 также 중요하지만, 보간한 결과를 신용하지 않는姿勢도 중요합니다.**

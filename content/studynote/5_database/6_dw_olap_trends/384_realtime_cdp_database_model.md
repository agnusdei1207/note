# 384. 실시간 커스터머 데이터 플랫폼 (CDP) 아키텍처

> ⚠️ 이 문서는 한 고객의 행동 데이터가 쇼핑몰 앱, 오프라인 매장 결제, 카카오톡 상담 등 여러 시스템에 흩어져 있어 '이 사람이 진짜 원하는 게 뭔지' 알 수 없는 파편화 문제를 해결하기 위해, **모든 고객 데이터를 1초 만에 한곳에 모아 실시간으로 마케팅 액션을 쏘아주는 'CDP' 데이터베이스 아키텍처**를 다룹니다.

## 핵심 인사이트 (3줄 요약)
> 1. **본질**: CDP(Customer Data Platform)는 단순히 데이터를 모아두는 창고(Data Lake)가 아니다. 마케터가 개발자의 도움 없이도 수천만 명의 고객 데이터를 실시간으로 조합해서 타겟팅 문자를 쏠 수 있게 해주는 **'행동(Action) 중심의 데이터 엔진'**이다.
> 2. **가치**: "방금 장바구니에 나이키 신발을 담았지만 5분 동안 결제하지 않고 나간, 과거에 아디다스 신발을 산 적이 있는 20대 남자"를 실시간으로 찾아내어 10% 할인 쿠폰을 즉시 쏘게 해준다.
> 3. **기술 체계**: 여러 DB에 흩어진 파편화된 ID(이메일, 전화번호, 쿠키 등)를 하나의 사람으로 묶어주는 **신원 확인(Identity Resolution)** 알고리즘이 CDP 기술의 핵심이다.

---

### Ⅰ. 개요: 흩어진 퍼즐 맞추기 (Context & Necessity)
우리가 흔히 쓰는 CRM(고객 관계 관리)은 수동적이다.
- 마케터: "어제 회원 가입한 사람한테 환영 메일 좀 보내볼까?" (이미 늦음)

현대의 커머스는 0.1초의 타이밍 싸움이다. 고객이 앱을 끄기 직전에 쿠폰을 먹여야 한다.
하지만 데이터는 찢어져 있다.
- 모바일 앱 클릭 로그 $\rightarrow$ Amplitude (A 데이터베이스)
- 오프라인 매장 결제 내역 $\rightarrow$ POS 시스템 (B 데이터베이스)
- 카카오톡 상담 내역 $\rightarrow$ CRM (C 데이터베이스)

**CDP (커스터머 데이터 플랫폼)**는 이 세 곳에서 쏟아지는 데이터를 실시간(Kafka 등)으로 빨아들여, **"이 3개의 데이터가 모두 김철수 1명의 행동이다!"**라고 조립한 뒤, 즉각적인 마케팅 툴(알림톡, 푸시)과 연동시켜 주는 허브다.

> 📢 **섹션 요약 비유**: 기존 방식이 형사들이 각자 수첩에 범인의 단서를 따로따로 적어두는 것이라면, CDP는 경찰서 한가운데 있는 **'거대한 수사 보드판'**입니다. 지문, CCTV, 통화 기록을 실로 모두 연결해서 "아, 이 모든 흔적이 범인 한 사람을 가리키고 있구나!" 하고 실시간으로 체포 명령을 내리는 곳이죠.

---

### Ⅱ. CDP의 3대 핵심 아키텍처 (파이프라인) ★

#### 1. 데이터 수집 (Data Ingestion)
- 앱, 웹, 서버, 오프라인 등 모든 채널에서 데이터를 긁어온다.
- 실시간 이벤트는 **Kafka / Kinesis**, 과거 배치 데이터는 **Airflow**를 통해 수집한다.

#### 2. 신원 확인 및 통합 (Identity Resolution)
- **가장 중요하고 어려운 단계다.**
- 앱에서는 `Cookie_ID: 123`으로 활동하고, 오프라인에서는 `Phone: 010-1111-2222`로 적립하고, 로그인할 때는 `Email: abc@gmail.com`을 썼다.
- CDP는 이 세 가지 흩어진 흔적을 결정적(Deterministic) 또는 확률적(Probabilistic) 매칭 알고리즘을 돌려 **'고객 1명(Unified Profile)'**으로 묶어낸다.

#### 3. 세그먼테이션과 활성화 (Segmentation & Activation)
- 마케터가 엑셀을 다루듯 직관적인 화면에서 조건을 건다.
- "최근 3일 내 접속자 + VIP 등급 + 장바구니 유기자" $\rightarrow$ 버튼 클릭 $\rightarrow$ 실시간 카카오톡 발송.

---

### Ⅲ. 일반 DB(Data Warehouse)와의 차이점

"어차피 Snowflake나 BigQuery 같은 DW에 다 모아두는데 굳이 CDP가 필요한가요?"

- **DW (데이터 웨어하우스)**:
  - 목적: "이번 달 매출 분석해 줘" (과거 분석)
  - 주 사용자: 데이터 분석가 (SQL 필수)
  - 속도: 한 시간 뒤에 결과가 나와도 됨.
- **CDP (커스터머 데이터 플랫폼)**:
  - 목적: "방금 들어온 이 사람한테 뭐 팔까?" (미래 예측 및 행동)
  - 주 사용자: 마케터 (SQL 몰라도 됨)
  - 속도: **0.1초 안에 결과가 나와야 함 (실시간).**

```text
┌──────────────────────────────────────────────────────────────┐
│           실시간 CDP (Customer Data Platform) 작동 흐름 시각화         │
├──────────────────────────────────────────────────────────────┤
│                                                              │
│ [ 🛍️ 앱 행동 ] ─(Kafka)─▶ ┌─────────────────┐                 │
│ [ 💳 결제 내역 ] ─(Batch)─▶ │   CDP 엔진      │                 │
│ [ 💬 상담 내역 ] ─(API)───▶ │(신원 통합 & 분석)│                 │
│                           └────────┬────────┘                 │
│                                    │ (실시간 세그먼트 생성)         │
│                                    ▼                          │
│                    [ 🎯 마케터의 룰: 장바구니 유기 고객 ]             │
│                                    │                          │
│             ┌──────────────────────┴──────────────────────┐   │
│             ▼                      ▼                      ▼   │
│       [ 💌 카카오톡 ]          [ 📱 앱 푸시 ]         [ 📧 이메일 ] │
│     "깜빡하신 물건이..."      "지금 당장 결제 시..."     "VIP 쿠폰 도착!"│
└──────────────────────────────────────────────────────────────┘
```

### Ⅳ. 결론

"데이터의 가치는 쌓아둘 때가 아니라, 행동으로 이어질 때 발생한다."
최근 서드파티 쿠키(Third-party Cookie)가 구글/애플 정책으로 박살 나면서, 다른 회사가 수집한 데이터에 의존하는 페이스북 타겟팅 광고는 생명력을 잃어가고 있다. 이제 기업들은 스스로 수집한 '퍼스트 파티(1st-party) 데이터'만으로 승부를 봐야 한다. 이 거친 전장에서 CDP는 내 고객의 진짜 마음을 읽고 0.1초 만에 타격하는 기업의 가장 예리하고 강력한 저격총이 될 것이다.

### 📌 관련 개념 맵
* **관련 기술**: Identity Resolution, Data Warehouse, Kafka
* **대표 상용 솔루션**: Segment(Twilio), mParticle, Treasure Data
* **연관 데이터 패턴**: Event Sourcing (555번 문서), DataOps (372번 문서)
* **마케팅 용어**: CRM, DMP (Data Management Platform - CDP의 구버전 격)

### 👶 어린이를 위한 3줄 비유 설명
1. 어떤 손님이 가게에서 초콜릿을 구경만 하고 그냥 나갔어요 (앱 방문).
2. 내일 그 손님이 마스크를 쓰고 와서 우유를 샀죠 (오프라인 결제).
3. 일반 기계는 두 사람이 다른 사람인 줄 알지만, CDP라는 똑똑한 탐정 기계는 "아! 어제 그 초콜릿 구경한 손님이구나!" 하고 바로 알아채고 "어제 보신 초콜릿 오늘 할인해요!"라고 쿠폰을 딱! 챙겨준답니다.

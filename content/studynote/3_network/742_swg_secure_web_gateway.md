# 742. SWG (Secure Web Gateway) - 프록시 보안 패키지 모델 구조적 설계

> **핵심 인사이트**: 1990년대의 프록시 서버는 직원들의 인터넷 속도를 높여주기 위해 웹페이지를 임시 저장(캐시)해두는 착한 도서관 사서였다. 하지만 인터넷에 악성코드가 넘쳐나자 이 사서가 무기를 들었다. 직원이 도박 사이트나 바이러스 링크를 클릭하면, 접속 자체를 막아버리고 "이 사이트는 회사 규정 위반입니다"라는 경고창을 띄워버리는 무자비한 사내 웹 경찰관, 그것이 SWG다.

## Ⅰ. SWG (Secure Web Gateway)의 개념
- **개념**: 사내 직원의 PC와 외부 인터넷 웹 사이트(HTTP/HTTPS) 사이에 위치하여, 직원이 접속하려는 **웹 트래픽의 유해성을 실시간으로 검사하고, 기업의 보안 정책에 어긋나는 유해 사이트 접속이나 악성 파일 다운로드를 원천 차단하는 특화된 보안 프레임워크 장비**입니다.
- **포지션**: 방화벽(FW)이 '어떤 통신 포트를 막을까'를 고민한다면, SWG는 '이 웹페이지 URL과 다운받는 파일이 악성인가?'를 심층 분석(L7)하는 웹 전용 보디가드입니다. (SASE 아키텍처의 필수 구성 요소)

## Ⅱ. 핵심 아키텍처: 투명 프록시 (Transparent Proxy) 🌟
- **구조적 설계**: SWG의 근본 뼈대는 **'포워드 프록시(Forward Proxy)'**입니다. 직원이 네이버에 접속하려고 브라우저를 켜면, 패킷은 네이버로 직행하지 않고 무조건 SWG 장비 안으로 들어갑니다.
- **투명 프록시(Transparent)**: 직원들은 자기 PC에 따로 프록시 IP 설정을 하지 않아도, 라우터가 웹 트래픽(80, 443번 포트)을 강제로 SWG 쪽으로 꺾어버립니다(Redirect). 직원은 투명 인간이 중간에 서 있는 줄도 모르고 자연스럽게 인터넷을 쓰게 됩니다.

## Ⅲ. SWG의 3대 핵심 보안 기능 패키지 🌟
SWG는 단순히 사이트를 차단하는 것을 넘어 종합 보안 툴의 역할을 합니다.

1. **URL 필터링 (URL Filtering)**
   - 가장 기본적인 기능입니다. 도박, 음란물, 악성코드 유포지, P2P 사이트 등 카테고리별로 분류된 거대한 전 세계 웹사이트 DB를 바탕으로, 직원이 금지된 URL을 클릭하면 접속을 즉각 끊고 차단 안내 페이지를 띄웁니다.
2. **안티 바이러스 & 악성코드 샌드박싱 연동**
   - 직원이 인터넷에서 `update.exe`라는 파일을 다운받는 순간, SWG가 그 파일을 낚아채어 백신 스캔을 돌리고, 미심쩍으면 내부 샌드박스 가상 환경(699번)에서 터뜨려봅니다. 악성 파일로 판명되면 직원 PC로 내려가는 다운로드를 취소시켜 버립니다.
3. **HTTPS(SSL/TLS) 가시성 확보 (복호화 검사)** 🌟
   - 오늘날 웹의 95%는 HTTPS로 암호화되어 있어 패킷을 까볼 수 없습니다. SWG는 직원의 PC에 미리 '사설 루트 인증서'를 깔아둔 뒤, **중간자 공격(MitM) 원리를 합법적으로 역이용하여 HTTPS 암호를 강제로 풀고(복호화), 내용물에 바이러스나 기밀문서가 있는지 100% 엑스레이 검사**를 마친 뒤 다시 암호화해서 내보냅니다.

## Ⅳ. CASB와의 차이점 (헷갈림 주의)
- **SWG (경찰관)**: 직원이 "어느 웹사이트(URL)"를 서핑하고 "무슨 파일"을 다운받는지, 즉 **'웹 브라우징(Outbound)의 안전성'**을 통제합니다.
- **CASB (감사관)**: 직원이 정상적인 구글 드라이브(클라우드)에 들어가서, "회사 극비 엑셀 파일"을 몰래 올리지 못하게 **'클라우드 데이터(DLP/API)의 무결성'**을 통제합니다. (현대 트렌드는 SWG와 CASB가 하나의 클라우드 장비로 통폐합되는 중입니다.)

> 📢 **섹션 요약 비유**: SWG는 회사 출입구에 서 있는 '위생 및 마약 탐지 검색대'입니다. 직원이 회사 밖(인터넷)으로 나가서 점심(웹 페이지)을 사 올 때, SWG 검색대 직원이 "어? 저 골목 식당(유해 URL)은 식중독 걸리니까 가지 마!"라고 막아섭니다. 직원이 식당에서 빵(다운로드 파일)을 사 오면, SWG는 그 빵을 쪼개서 안에 독(악성코드)이 들었는지 샅샅이 검사한 뒤에야 빵을 직원의 책상으로 보내주는 철통 같은 사내 위생 방역 시스템입니다.

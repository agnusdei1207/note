## 핵심 인사이트 (3줄 요약)
> 1. **본질**: 유선 매체(Guided Media)는 전기 신호 또는 광 신호가 물리적导体(동축 케이블, 꼬임 쌍 케이블, 광섬유 등)을 통해 전달되는 전송 매체로, 신호가 매체 내부에 "guided"되어 전달된다.
> 2. **가치**: 무선 매체 대비 대역폭이 높고, 외부 간섭(EMI)에 강하며, 신호 누출이 적어 보안성이 우수하여, 기업 데이터 센터, 통신 백본망 등 핵심 인프라에 필수적이다.
> 3. **융합**: 구리 기반 매체(동축, 꼬임 쌍)와 광섬유는 각기 다른 특성을 가지며, 장거리 통신에는 광섬유가, 단거리 내부 배선에는 구리 케이블이 효율적이며, 현대 네트워크에서는 둘이 적절히 혼용된다.

---

## Ⅰ. 개요 및 필요성 (Context & Necessity)

- **개념**: 유선 매체(Guided Media, Wired Media)는Electrical 신호 또는 Optical 신호가 물리적 매체를 따라 이동하도록誘導되는 전송 매체를 의미한다. 대표적인 유선 매체로는 꼬임 쌍 케이블(Twisted Pair Cable), 동축 케이블(Coaxial Cable), 광섬유(Fiber Optic Cable)가 있다. 꼬임 쌍 케이블은 두 개의 구리 도체를 나선처럼 꼬아 만든 것으로, UTP (Unshielded Twisted Pair)와 STP (Shielded Twisted Pair)가 있다. 동축 케이블은 중심 도체를 절연체, 외부 도체, 시스 장갑이 둘러싸는 구조로, 높은 주파수에서 낮은 감쇠를 보인다. 광섬유는 Glass 또는 Plastic 섬유를 통해 Light를伝送하는 것으로, 매우 높은 대역폭과 극低的 신호 감쇠, EMI 면역력을 가진다.

- **필요성**: 무선 통신이方便的이지만, 대역폭, 보안, 안정성 측면에서 유선 매체가 여전히 요구된다. 유선 매체는 외부 전자기파 간섭(EMI, Electromagnetic Interference)에 강하며, 물리적으로 케이블이 존재하므로 도청(detection)이 어렵고 보안성이 높다. 또한, 장거리传输에서도 신호 품질이 유지되어, 국가間 해저 케이블부터 건물 내 LAN布线까지 전 세계 네트워크 인프라의 기반이 된다.

- **💡 비유**: 유선 매체는 도시에 설치된 상하수도와 같다. 수도관(매체)을 통해 물(데이터)이 특정 경로로 guided되어 전달되므로,蒸発(무선 신호의 감쇠)이나 다른 물과 섞임(EMI 간섭)이 거의 없이 목적지에 도달한다. 반대로 무선은 하늘에서 비가 내리는 것과 같아서, 바람(간섭)에 의해 경로가 흐트러지고,沐당(신호 감쇠)될 수 있다.

- **등장 배경 및 발전 과정**:
  1. **1880년대:전신/전화선의 등장**: 전기 신호를 Wire를 통해 전송하는 개념이 처음 등장했다.
  2. **1970년대:동축 케이블과 꼬임 쌍의 표준화**: Ethernet (10BASE5, 10BASE2)에서 동축 케이블이 사용되었고, 이후 스타 토폴로지普及으로 꼬임 쌍 케이블이主流이 되었다.
  3. **1980년대 이후:광섬유의 확산**: 광섬유 기술의 발전과 함께, 통신 백본망, 데이터 센터 인터커넥션, 장거리 통신에서 광섬유가 표준이 되었다.

## Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

### 유선 매체 유형 비교

| 매체 유형 | 대역폭 | 최대 거리 | 감쇠 (Attenuation) | EMI 저항 | 비용 |
|:---|:---|:---|:---|:---|:---|
| **UTP (Category 5e/6)** | 100MHz~250MHz | 100m | 중간 | 낮음 | 매우 낮음 |
| **STP (Shielded)** | 100MHz~600MHz | 100m | 중간 | 높음 | 중간 |
| **동축 케이블 (RG-6)** | ~1GHz | ~500m (증폭기 필요) | 낮음 | 높음 | 중간 |
| **광섬유 ( Multimode)** | 수 GHz | ~500m (OM3/OM4) | 매우 낮음 | 무 EMI | 중간~높음 |
| **광섬유 (Single Mode)** | 수 THz | 수십 km | 극히 낮음 | 무 EMI | 높음 |

### 꼬임 쌍 케이블 (Twisted Pair Cable)

꼬임 쌍 케이블은 두 개의 구리 도체를 나선 꼬아서 간섭(크로스토크, crosstalk)을 줄이도록 설계되었다. UTP는 차폐 없이 가장 일반적이고, STP는 금속 호일드(foil)로 차폐하여 EMI에 더 강하다.

```text
  ┌─────────────────────────────────────────────────────────────────────┐
  │                    꼬임 쌍 케이블 구조와 종류                             │
  ├─────────────────────────────────────────────────────────────────────┤
  │
  │  [꼬임 쌍 케이블 구조]                                              │
  │
  │  ┌──────────────────────────────────────────────────────┐          │
  │  │  외부 시스 장갑 (PVC/LSP)                              │          │
  │  │  ┌──────────────────────────────────────────────┐  │          │
  │  │  │  차폐 (STP의 경우) - 금속 호일드 또는 브레이드   │  │          │
  │  │  │  ┌────────────────────────────────────────┐  │  │          │
  │  │  │  │ 4쌍의 꼬임 쌍 (각 쌍 꼬인 구조)         │  │  │          │
  │  │  │  │  ┌─────┐ ┌─────┐ ┌─────┐ ┌─────┐   │  │  │          │
  │  │  │  │  │ 쌍1 │ │ 쌍2 │ │ 쌍3 │ │ 쌍4 │   │  │  │          │
  │  │  │  │  └─────┘ └─────┘ └─────┘ └─────┘   │  │  │          │
  │  │  │  └────────────────────────────────────────┘  │  │          │
  │  │  └──────────────────────────────────────────────┘  │          │
  │  └──────────────────────────────────────────────────────┘          │
  │
  │  ※ 꼬임(crossover)의 역할: 인접 쌍 간 크로토크(crosstalk) 상쇄        │
  │  ※ 꼬임 비율이 높을수록(crossover/길이) EMI 성능 향상                   │
  │
  │  [UTP vs STP 비교]                                                   │
  │  ┌────────────────┬─────────────────────┬─────────────────────┐    │
  │  │     항목        │        UTP           │        STP           │    │
  │  ├────────────────┼─────────────────────┼─────────────────────┤    │
  │  │ 차폐           │ 없음                 │ 금속 호일드/브레이드  │    │
  │  │ EMI 저항       │ 낮음                 │ 높음                 │    │
  │  │ 비용           │ 가장 저렴            │ 중간                │    │
  │  │ 무게/질량      │ 가벼움              │ 무거운 편           │    │
  │  │ 설치 용이성     │ 용이                 │ 어려움 (차폐 접지)   │    │
  │  │ 사용처         │ 일반 사무실 LAN      │ 공장, 의료,EMI 환경 │    │
  │  └────────────────┴─────────────────────┴─────────────────────┘    │
  │
  └─────────────────────────────────────────────────────────────────────┘
```

**[다이어그램 해설]** 꼬임 쌍 케이블의 핵심은"꼬임(crossover)" 구조다. 두 개의 도체를 나란히 두지 않고 나선으로 꼬아두면, 외부 EMI가 두 도체에 동일한 수준으로耦合된다. 수신 측은 두 도체 간의 차이(차동 신호)만을 amplification하므로, 공통으로 결합된 EMI는 상쇄된다. 이것을_common mode rejection"이라 한다. 또한, 한 쌍에서 발생하는EMI도 인접 쌍에 영향을 주지만, 인접 쌍의 꼬임 패턴이 다르므로 상쇄 효과를 보인다. UTP는 이러한 원리로 동작하며, 차폐(Shielded)가 없으므로 비용이 낮지만 EMI 환경에脆弱하다. STP는 금속 호일드로 추가 차폐를 제공하여, 공장, 병원, 군사 시설처럼 EMI가 심한 환경에서 사용된다.

### 광섬유 (Fiber Optic Cable)

광섬유는 Light를 통한数据传输 매체로, 유선 매체 중 가장 높은 대역폭과 최저 신호 감쇠, EMI 면역력을 가진다. 구조적으로는Core (광신호 전퍽), Cladding ( 전반사), Coating (보호), Jacket (외부 시스)로 구성된다.

```text
  ┌─────────────────────────────────────────────────────────────────────┐
  │                    광섬유 구조와 동작 원리                               │
  ├─────────────────────────────────────────────────────────────────────┤
  │
  │  [광섬유 구조]                                                      │
  │
  │  ┌──────────────────────────────────────────────────────┐          │
  │  │  외부 재킷 (PVC/LSZH)                                 │          │
  │  │  ┌──────────────────────────────────────────────┐  │          │
  │  │  │  코팅 (丙烯酸酯 / silicone) - 핵심 보호          │  │          │
  │  │  │  ┌────────────────────────────────────────┐  │  │          │
  │  │  │  │  클래딩 (屈折率 ↓)                      │  │  │          │
  │  │  │  │  ┌──────────────────────────────┐   │  │  │          │
  │  │  │  │  │  코어 (屈折率 ↑)              │   │  │  │          │
  │  │  │  │  │  ← 광신호 전파                 │   │  │  │          │
  │  │  │  │  └──────────────────────────────┘   │  │  │          │
  │  │  │  └────────────────────────────────────────┘  │  │          │
  │  │  └──────────────────────────────────────────────┘  │          │
  │  └──────────────────────────────────────────────────────┘          │
  │
  │  [광신호 전파 원리: 전반사]                                          │
  │
  │           코어 (n1)              클래딩 (n2, n1 > n2)               │
  │       ╔═══════════════╗          ╔═══════════════════╗             │
  │       ║   ≮ 경사 입사   ║          ║  ■ 반사          ║             │
  │       ║  ≮  入射   ≮  ║          ║ ■   ↗           ║             │
  │       ║    ╲        ╱  ║          ║ ■  ╱ ■          ║             │
  │       ╚════╗════════╝          ╚═══════════════════╝             │
  │             ╲ 임계각 이상                                         │
  │              ╲   임계각 미만 ( 전반사 실패)                           │
  │
  │  ※ 임계각 (Critical Angle) = arcsin(n2/n1)                        │
  │  ※ 임계각 이상의 입사각 → 전반사 → 코어 내에서 광신호 전파          │
  │  ※ 임계각 미달 → 신호 누출 → 감쇠 증가                              │
  │
  └─────────────────────────────────────────────────────────────────────┘
```

**[다이어그램 해설]** 광섬유의 동작 원리는"전반사(Total Internal Reflection)"다. 코어의 굴절률(n1)이 클래딩의 굴절률(n2)보다 높으므로, 빛이 코어-클래딩 경계면에서 임계각 이상으로 입사하면 전반사가 발생하여 빛이 코어 내부에 가둔 채 전파된다. Multimode 광섬유는 코어 지름이较大(50~62.5 µm)하여 여러 모드(광선 경로)로光が 전파되며, Single Mode 광섬유는 코어 지름이 작아(9 µm 이하) 하나의 모드만 전파한다. Single Mode는 모드 분산(Mode Dispersion)이 없어 장거리 고속 전송에 적합하며, Multimode는 짧은 거리에서 저비용 전송에 사용된다.

## Ⅲ. 융합 비교 및 다각도 분석

### 꼬임 쌍 vs 동축 케이블 vs 광섬유 비교

| 비교 항목 | 꼬임 쌍 (UTP) | 동축 케이블 | 광섬유 ( Multimode) | 광섬유 (Single Mode) |
|:---|:---|:---|:---|:---|
| **대역폭** | ~10 Gbps (Cat6a) | ~10 Gbps | ~100 Gbps | ~100 Tbps |
| **거리** | 100m | ~500m | ~500m | ~80km+ (증폭기 없이) |
| **감쇠** | 중간 (~20 dB/100m) | 낮음 (~10 dB/100m) | 매우 낮음 (~3 dB/km) | 극히 낮음 (~0.2 dB/km) |
| **EMI** | 중간 (꼬임으로 일부 회피) | 높음 | 완전 면역 | 완전 면역 |
| **보안** | 중간 (전자기 방출 있음) | 높음 | 매우 높음 (도핑 어려움) | 매우 높음 |
| **비용** | 매우 낮음 | 중간 | 중간 | 높음 |

### 과목 융합 관점

- **컴퓨터구조 (CA)**: 광섬유는 컴퓨터 간 고속 인터커넥션(InfiniBand, 100GbE 등)의 매체로 사용되며, 데이터 센터 내 서버 간通信を高速化한다.
- **반도체**: 광섬유 트랜시버(광电 변환 모듈)의 개발은 반도체 공정과 光デバイス 기술의 진보에 의해 발전했다.

- **📢 섹션 요약 비유:** 유선 매체는 물건配送의 포장 방법と似ています. 꼬임 쌍은 두 줄기 사탕을 튜브에 넣은 것(안便宜하지만基本的), 동축 케이블은同心圆状のパイプライン(더强的だが больще), 광섬유는 빛의之道을ガラス관에 가둔 것(가장高效이지만 깨지기 쉬운)과 같습니다.

## Ⅳ. 실무 적용 및 기술사적 판단

### 실무 시나리오

1. **시나리오 — 데이터 센터 내케이블 선택**: 10Gbps 이더넷 연결이 필요한 서버 rack을 위해서는 SFP+ DAC (Direct Attach Copper) 또는光收发器+光纤가 사용된다. 5m 이하 짧은 거리에서는 DAC (구리)가 비용 효율적이며, 100m 이상 거리에서는光섬유가 대역폭과 거리 측면에서優れる.

2. **시나리오 — 장거리 통신 백본**: 국가間 해저 광섬유 케이블은 수천 km에 걸쳐 대륙을 연결한다. Single Mode 광섬유에 DWDM (Dense Wavelength Division Multiplexing)을 적용하여, 1本の光纤에 수백 개의 파장을 superimpose하여 이론적 용량 100 Tbps 이상을 달성한다.

```text
  ┌─────────────────────────────────────────────────────────────────────┐
  │              데이터 센터 내 케이블 선택 가이드                              │
  ├─────────────────────────────────────────────────────────────────────┤
  │
  │  거리         권장 매체          대역폭      비용 효율성              │
  │  ────────────────────────────────────────────────────────          │
  │  1~5m       DAC (Direct Attach Copper)   10~25 Gbps    最速       │
  │  5~30m      SFP+ Twinax (구리)           10~25 Gbps    높음        │
  │  30~100m    UTP Cat6a/7                 10 Gbps       중간        │
  │  100m~2km   OM3/OM4 Multimode Fiber    10~100 Gbps   중간        │
  │  2km+       Single Mode Fiber (OS2)     10~400+ Gbps  높음 (장거리) │
  │
  │  ※ 데이터 센터 내에서는 전력 소모, 발열,rack 공간 절약도 중요          │
  │  ※ DAC는 광 transceiver보다 전력 소모가 적음                          │
  │  ※ 장래 확장을 위해 광섬유를 미리敷設하는 것이 일반적                   │
  │
  └─────────────────────────────────────────────────────────────────────┘
```

**[다이어그램 해설]** 데이터 센터에서 케이블 선택은거리, 대역폭, 비용, 전력 소모의trade-off다. DAC (Direct Attach Copper)는 별도의 光 transceiver 없이 구리 케이블에 직접 SFP+ 모듈을 연결하므로 전력 소모가 적고 비용이 낮지만, 5m 이상에서는 신호 품질 문제가 발생할 수 있다. 장거리에서는光섬유가 필수이며, OM3/OM4 Multimode는 100m~500m에서 10~100Gbps를 지원하여 서버-스위치 연결에 적합하고, Single Mode는 장거리 백본과 DCI (Data Center Interconnect)에使用된다. 또한, Cat6a UTP는 건물 내수평 배선(100m)에서 가장一般的な選擇이다.

### 도입 체크리스트
- **기술적**: 전송 거리와 대역폭 요구사항에 따라 적절한 매체를 선택했는가?
- **운영·보안적**: EMI 환경(공장, 병원)에서는 차폐 케이블(STP) 또는 광섬유 사용을 검토했는가?

### 안티패턴
- **장거리 UTP 사용**: UTP 케이블을 100m 이상 사용하면 신호 감쇠로 인해 프레임 오류율이 증가하고 네트워크 성능이 저하된다.
- **광섬유 커넥터 오염**: 광섬유 커넥터가 오염되면 신호 감쇠가 급격히 증가하여 링크가 불안정해진다.

- **📢 섹션 요약 비유:** 케이블 선택은 물건送货의 Vehicles 선택과 같습니다. 가까운距离에는中小型トラック(UTP)를 쓰고, 긴距离에는高速 Interstate(광섬유)를 쓰며, 超长距离에는船舶(해저 케이블)을 씁니다. Vehicles 선택을 잘못하면、时间と비용이 초과하고、客户 만족도이 떨어집니다.

## Ⅴ. 기대효과 및 결론

### 정량/정성 기대효과

| 구분 | 구리 (UTP) | 광섬유 (Single Mode) | 개선 효과 |
|:---|:---|:---|:---|
| **정량** | 최대 거리: 100m | 최대 거리: 80km+ (증폭기 없이) | 거리 **800배 확대** |
| **정량** | 대역폭: 10 Gbps | 대역폭: 400 Gbps+ | 대역폭 **40배+** |
| **정성** | EMI 영향 受침 | EMI 완전 면역 | 신뢰성 향상 |

### 미래 전망
- **단일-fiber 400G+**: DWDM과 SCDMA 기술을 결합하여, 단일 光 fiber에서 400Gbps 이상의 전송을 달성하는 연구가 진행 중이다.
- ** Plastic Optical Fiber (POF)**: automotive 내부 네트워크와 Kurzstecke에서 저비용 광通信解决方案として期待されている。

- **📢 섹션 요약 비유:** 유선 매체의 발전은送货 Vehicles의 발전と似ています. 처음에는普通的道路(동축) 위의 트럭(전기 신호)이었지만, 高規格道路(광섬유) 위의고속 택배(광신호)로 진화하여,より 많은 화물을より 빠른 속도로配送できるようになりました.

## 📌 관련 개념 맵 (Knowledge Graph)

| 개념 명칭 | 관계 및 시너지 설명 |
|:---|:---|
| **UTP (Unshielded Twisted Pair)** | 차폐 없는 꼬임 쌍 케이블로, 일반 사무실 LAN에서 가장 널리 사용된다. |
| **STP (Shielded Twisted Pair)** | 금속 호일드로 차폐된 꼬임 쌍으로, EMI 환경에 적합하다. |
| **동축 케이블 (Coaxial Cable)** | 중심 도체와 외부 도체로 구성된 고주파 전송 매체로, 케이블 TV, 레ーダー 등에 사용된다. |
| **Single Mode Fiber (SMF)** | 단일 모드만 전파하는 광섬유로, 장거리 고속 전송에 필수적이다. |
| ** Multimode Fiber (MMF)** | 여러 모드로光が 전파하는 광섬유로, 단거리 전송에 경제적이다. |
| **DWDM (Dense WDM)** | 단일 광섬유에 여러 파장의光を多重화하여 전송 용량을 크게 늘리는 기술이다. |

## 👶 어린이를 위한 3줄 비유 설명
1. 유선 매체는 종단접힌 빨대と似ています. 빨대 안에서 음료(데이터)가漏れ없이 전달되듯이, 케이블 안의 신호도 매체에 가둔 채目的地까지 전달됩니다.
2. 꼬임 쌍 케이블은 두 가닥의 빨대를 함께 접어卷いたもので,外部压力(EMI)에 좀 더強い입니다. 동축 케이블은 빨대周りにうき輪を巻いたもので, 더 강한保护를 제공합니다.
3. 광섬유는 빛の之道을유리管에 가둔 것으로, 어떻게 보면 가장高性能하지만 깨지기 쉽습니다. 그래도 먼 곳까지 빛の скорость로 данных을 보낼 수 있어서, 국제 인터넷의 기반이죠!

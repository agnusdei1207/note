## 핵심 인사이트 (3줄 요약)
> 1. **본질**: 광섬유 통신은 Glass 또는 Plastic 섬유를 통해 Light 신호를 전파하여 데이터를 전송하는 기술로, 유선 매체 중 가장 높은 대역폭,最低 신호 감쇠, 완전한 EMI 면역력을 가진다.
> 2. **가치**: 현재 광섬유는 장거리 통신 백본, 데이터 센터 인터커넥션, 해저 케이블 등 핵심 인프라의 표준이며, DWDM (Dense Wavelength Division Multiplexing) 기술과 결합하여 수 THz의 전송 용량을 달성한다.
> 3. **융합**: 5G移动通信백홀(Backhaul), FTTH (Fiber To The Home), 데이터 센터 간 DCI (Data Center Interconnect) 등 현대 네트워크 인프라의 거의 모든 핵심 구간에서 광섬유가 사용된다.

---

## Ⅰ. 개요 및 필요성 (Context & Necessity)

- **개념**: 광섬유 통신은 Hair-like한 Glass 또는 Plastic 섬유를 통해光 신호를 전송하여 데이터를 교환하는 방식이다. 기본 원리는 전반사(Total Internal Reflection)다. 코어(core)의 굴절률이 클래딩(cladding)보다 높으므로,光が 코어-클래딩 경계면에서 임계각 이상으로 입사하면 전반사가 발생하여光が 코어 내부에 가둔 채 전파된다. 광섬유는 Multimode와 Single Mode로 구분된다. Multimode 광섬유는 코어 지름이較大(50~62.5 µm)하여 여러 모드로光が 전파되며, 짧은~중간 거리(수백 미터~수 km)에서 사용된다. Single Mode 광섬유는 코어 지름이 작아(9 µm 이하) 오직 단일 모드만 전파되어, 장거리 전송(수십~수백 km)에 사용된다.

- **필요성**: 데이터 트래픽은 매년 지수적으로 증가하고 있으며(특히 video, cloud, AI), 구리 케이블의 대역폭-거리 한계가 드러나고 있다. 광섬유는 대역폭 면에서 구리를 크게 웃돌며, 장거리에서도 신호 품질이 유지되어 중계기 없이 수십 km 전송이 가능하다. 또한, 금속을 사용하지 않아 EMI(电磁干扰) 문제에서 자유롭고, 도청이 어렵아 보안성도 우수하다.

- **💡 비유**: 광섬유는 놀이터의 미끄럼막과 같습니다. 광신호는 코어(미끄럼막)를 타고 이동하며, 클래딩(미끄럼벽)은 전반사로光を反射하여膜壁 밖으로 나가지 못하게 합니다. 물리적으로 완전히 갇힌 빛은外部 간섭(EMI) 없이 빠르게 이동하며, 미끄럼 끝(수신단)에서만 빠져나옵니다.

- **등장 배경 및 발전 과정**:
  1. **1960년대:광섬유 기술의 기초**: Charles K. Kao (고든 Кao)가 "광섬유를 통한 정보 전송 가능성"을 이론적으로 제안했다.
  2. **1970년대:초저손실 광섬유 개발**: Corning이 실험실에서 손실률 20 dB/km 이하의溶融 실리카 광섬유를 개발했다.
  3. **1980년대 이후:상용화와 확산**: 1988년 최초의商用 광섬유 전화 해저 케이블(TAT-8)이 대서양에敷設되었다.
  4. **2000년대 이후:DWDM의 보급**: DWDM (Dense Wavelength Division Multiplexing) 기술로 1本の光纤에 수십~수백 채널을多重화하여 전송 용량이爆発적으로 증가했다.

## Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

### 광섬유의 종류와 특성

광섬유는 코어 지름과 굴절률 분포에 따라 Multimode, Single Mode로 나뉘며, 각기 다른 특성과 용도를 가진다.

```text
  ┌─────────────────────────────────────────────────────────────────────┐
  │                    광섬유 종류와 특성 비교                               │
  ├─────────────────────────────────────────────────────────────────────┤
  │
  │  [Multimode Fiber (MMF)]                                          │
  │
  │       코어: 50/62.5 µm            클래딩: 125 µm                     │
  │       │                                                                  │
  │       │  ┌──────────────────────────────────────────────────┐   │
  │       │  │                    코어                           │   │
  │       │  │         ●───────────────▶                        │   │
  │       │  │        ╱        ╲        ╱                        │   │
  │       │  │       ●          ●       ●  ← 다중 모드 전파        │   │
  │       │  │        ╲        ╱        ╱                         │   │
  │       │  │         ●───────────────▶                        │   │
  │       │  └──────────────────────────────────────────────────┘   │
  │       │                                                                  │
  │       ※ 단계형 굴절률 (Step-Index): 굴절률 차이 명확                  │
  │       ※ 등급형 굴절률 (Graded-Index): 중심 빠르게, 외곽 느리게          │
  │          → 모드 분산 보상, 대역폭 확대                                   │
  │
  │  [Single Mode Fiber (SMF)]                                          │
  │
  │       코어: 9 µm                 클래딩: 125 µm                     │
  │       │                                                                  │
  │       │  ┌──────────────────────────────────────────────────┐   │
  │       │  │                    코어 (극히 가늘음)              │   │
  │       │  │                                                    │   │
  │       │  │              ●──────────────────▶                │   │
  │       │  │                     단일 모드 직선 전파            │   │
  │       │  │                                                    │   │
  │       │  └──────────────────────────────────────────────────┘   │
  │       │                                                                  │
  │       ※ 모드 분산(Mode Dispersion) 없음 → 초고속 장거리 전송 가능       │
  │       ※ 코어가 매우 가늘어 커넥터 연결, 융착 난이도 높음                 │
  │
  │  ┌──────────────────────────────────────────────────────────────┐   │
  │  │  종류        코어     대역폭-거리곱      주요 용도              │   │
  │  │  OM1 (62.5)   62.5µm   ~500 MHz·km    레거시 (100BASE-FX)  │   │
  │  │  OM2 (50)     50µm     ~700 MHz·km    짧은거리 (1-10GbE)   │   │
  │  │  OM3 (50)     50µm     ~1500 MHz·km   데이터 센터 (10GbE+)  │   │
  │  │  OM4 (50)     50µm     ~3500 MHz·km   데이터 센터 (16GbE+)  │   │
  │  │  OS1/OS2      9µm      ~100+ THz·km   장거리/해저 백본     │   │
  │  └──────────────────────────────────────────────────────────────┘   │
  │
  └─────────────────────────────────────────────────────────────────────┘
```

**[다이어그램 해설]** Multimode 광섬유에서"모드(mode)"란光が 전파하는 경로를 의미한다. 코어 지름이 클수록 다양한 각도로 입사한光が各自 다른 경로(모드)를 따라 전파된다. 그러나 경로가 다르면 도착 시간도 다르므로(모드 분산), 고속 전송 시パルスが広がって隣接パルスと重なる (ISI, Inter-Symbol Interference) 문제가 발생한다. 이를 완화하기 위해 GI (Graded-Index) MMF는 코어 중심부 굴절률을 높게, 외곽으로 갈수록 낮게 하여, 가장자리를 가는 모드의 속도를 높여 중심부 모드와 도착 시간이 비슷해지도록 한다. Single Mode 광섬유는 코어 지름이빛의 파장(약 1.3~1.5 µm)와 비슷한 수준까지 가늘어져, Hanya satu 모드만 전파할 수 있으므로 모드 분산이 없어 초고속 장거리 전송에 적합하다.

---

### DWDM (Dense Wavelength Division Multiplexing)

DWDM은 단일 광섬유에 서로 다른 파장의빛을多重화하여 전송하는 기술로, 수십~수백 개의 채널을 동시에 전송할 수 있다.

```text
  ┌─────────────────────────────────────────────────────────────────────┐
  │              DWDM (Dense Wavelength Division Multiplexing) 동작       │
  ├─────────────────────────────────────────────────────────────────────┤
  │
  │  송신 측                                                            │
  │
  │  Ch1 (λ1) ──▶ │                                          │
  │  Ch2 (λ2) ──▶ │        ┌──────────────┐    ┌─────────┐ │
  │  Ch3 (λ3) ──▶ ├─ MUX ─┤   Optical    ├────┤Fiber    │ │
  │  ...         ──▶ │        │   Filter    │    │(single) │ │
  │  ChN (λN) ──▶ │        └──────────────┘    └────┬────┘ │
  │                                             │         │
  │  ※ 각 파장: C-Band (1530~1565nm) 또는 L-Band (1565~1625nm)   │
  │
  │  수신 측                                                            │
  │
  │                    ┌──────────────┐    ┌─────────┐ │
  │  Ch1 (λ1) ◀───────│  DEMUX       ├────┤Fiber    │ │
  │  Ch2 (λ2) ◀───────┤  (격자/필터)  ├────┤(single) │ │
  │  Ch3 (λ3) ◀───────┤              ├────┤         │ │
  │  ...         ◀──────┤              ├────┤         │ │
  │  ChN (λN) ◀───────└──────────────┘    └─────────┘ │
  │
  │  ※ 채널 간격: 50 GHz 또는 100 GHz (ITU-T G.694.1 표준)            │
  │  ※ 이론적 용량: 채널 80~160개 × 채널당 100 Gbps+ = 8~16 Tbps     │
  │
  └─────────────────────────────────────────────────────────────────────┘
```

**[다이어그램 해설]** DWDM의 핵심은"파장多重화"다. 각 송신기는 서로 다른 파장의 Laser光源을 사용하며(보통 C-Band 1530~1565 nm), 이것들이 MUX (Multiplexer)를 통해 하나의 광섬유에 통합된다. 각 파장은 독립적인 데이터 채널처럼 동작하므로, 기존 광섬유 인프라를 활용하면서도 전송 용량을배가할 수 있다. 수신 측에서는 DEMUX (Demultiplexer)가 각 파장을 분리하여 각 수신기에 전달한다. 채널 간격은 ITU-T G.694.1 표준으로 50 GHz 또는 100 GHz로 규정되어 있어, 상호干渉없이 다수의 채널을 수용한다. 현대 DWDM 시스템은채널당 100 Gbps, 200 Gbps,甚至는 400 Gbps를 지원하며, 채널 80~160개를 통합하면 8~64 Tbps의 용량이 가능하다.

## Ⅲ. 융합 비교 및 다각도 분석

### 광섬유 vs 구리 케이블: 종합 비교

| 비교 항목 | 구리 (UTP/동축) | 광섬유 (SMF) |
|:---|:---|:---|
| **대역폭** | ~10 Gbps (Cat6a/7) | ~400 Gbps (채널당) |
| **거리** | ~100m (UTP) | ~80km (증폭기 없이) |
| **감쇠** | 20~30 dB/km | 0.2~0.5 dB/km |
| **EMI** | 受침 | 완전 면역 |
| **보안** | 전자기 방출 → 도청 가능 | 도청 어려움 (光漏れ 감지) |
| **전력 공급** | 가능 (PoE) | 불가 (별도 전원 필요) |
| **비용 (per meter)** | 낮음 | 높음 (but 전체 비용은距離 고려) |
| **설치 용이성** | 용이 (구부리기 쉬움) | 어려움 (깨지기 쉬움) |

### 과목 융합 관점

- **5G移动通信**: 5G 기지국(基站)からのバック홀(基地局→核心網) 연결에 광섬유가 필수적이며, 5G의 고속 저지연 서비스 제공을 위한 기반이 된다.
- ** 클라우드 컴퓨팅**: AWS, Google Cloud, Azure 등 주요 클라우드 プロバイダ는전 세계 데이터 센터를 광섬유 백본으로 상호 연결하여, 대규모 데이터転送と低지연을 실현한다.

- **📢 섹션 요약 비유:** DWDM은 고속도로의 차선 확장방법과 같습니다. 차선이 하나뿐인 도시에Expressway를 새로 만들지 않고, Expressway 위에 차선을 더 추가하는(파장 추가) 것은 불가능합니다. DWDM은 빛의 파장을追加하여同じ道路(광섬유)에 차선을 加算한 것이며,これにより新規道路を敷設する보다 비용 효율적으로容量을 늘릴 수 있습니다.

## Ⅳ. 실무 적용 및 기술사적 판단

### 실무 시나리오

1. **시나리오 — 해저 광섬유 케이블**: 대서양을 가로지르는 해저 광섬유 케이블(TAT-8, 이후 TAT-14 등)은 미국과 유럽, 그리고 아시아를 연결하는 국제 통신의 artery다. 현재 하나의 해저 케이블 시스템은 수십 Tbps의 용량을 가지며, 전 세계 데이터 통신의 90%以上가 해저 케이블을 통해 전송된다. 수 km 간격으로 도달력 증폭기(erbium-doped fiber amplifier, EDFA)가 설치되며, 케이블이切断될 경우 수백만 사용자에게 영향이 미친다.

2. **시나리오 — FTTH (Fiber To The Home)**: 광케이블을 가정까지 가져오는 FTTH는 초고속 인터넷 액세스의 표준이 되었다. GPON (Gigabit PON) 기술은下行 2.5 Gbps, 상행 1.25 Gbps를 지원하며, 하나의 광 분기기를 통해 32~64가구에 동시에 서비스를 제공한다.

```text
  ┌─────────────────────────────────────────────────────────────────────┐
  │              해저 광섬유 케이블 시스템 구조                              │
  ├─────────────────────────────────────────────────────────────────────┤
  │
  │  [해저 케이블 단면]                                                  │
  │
  │  ┌──────────────────────────────────────────────────────────┐     │
  │  │  외부 재킷 ( polyethylene )                              │     │
  │  │  ┌──────────────────────────────────────────────────┐   │     │
  │  │  │  금속 장갑 (탠탈럼/스테인리스) - 내압/보호           │   │     │
  │  │  │  ┌────────────────────────────────────────────┐  │   │     │
  │  │  │  │  광섬유 (2~8심) + jellying 화합물 충진     │  │   │     │
  │  │  │  └────────────────────────────────────────────┘  │   │     │
  │  │  └──────────────────────────────────────────────────┘   │     │
  │  └──────────────────────────────────────────────────────────┘     │
  │
  │  [해저 시스템 구성]                                                │
  │
  │  [육상]── 케이블 ──┤                                              │
  │                 │                                                 │
  │           [SLTE] │ (Submarine Line Terminating Equipment)          │
  │                 │                                                 │
  │  [럽양]── 케이블 ──┼─── [증폭기]─── [증폭기] ──── [럽양]           │
  │                     │       ~60km 간격        │                    │
  │                [증폭기]                                              │
  │                ~60km                                                │
  │
  │  ※ 증폭기: EDFA (Erbium-Doped Fiber Amplifier)                   │
  │    - 광-광 증폭으로 신호 재생 없이 증폭                              │
  │    - 수 km 간격으로海底에 설치                                       │
  │
  └─────────────────────────────────────────────────────────────────────┘
```

**[다이어그램 해설]** 해저 광섬유 케이블은 해저 환경의 가혹한 조건(수압, 해류, 닷물림,漁撈)에耐える내구 구조가 필요하다. 광섬유는抗張力 members(케블러)로保護되며, 외부는 polyethylene 재킷과 금속 장갑(탠탈럼/스테인리스)으로覆된다. 수 km 간격으로 설치되는 EDFA는 광신호를 직접 증폭하여 재생 없이 장거리 전송을 가능하게 한다. 또한 해저 케이블은 의도치 않은切断에 취약하여,船舶 예인망(닻)에 의해切断되는事故가频発한다. 2022년tonsberg에서는프랑스와 아시아를 연결하는 해저 케이블이船舶 닻에의해切断되어, 해당 구간의通信이瞬时的に影响了었다.

### 도입 체크리스트
- **기술적**: 장거리 전송에는 Single Mode 광섬유, 단거리에는 Multimode 광섬유를 선택했는가? 파장 다중화(DWDM) 적용이 필요한가?
- **운영·보안적**: 광섬유 보안(도핑 방지, 커넥터 접근 통제)을 고려했는가?

### 안티패턴
- ** Multimode 광섬유로 장거리 전송 시도**: OM3 광섬유를 수 km 이상 전송하면 모드 분산으로 인해 신호 품질이 급격히 저하된다.
- **광 커넥터 오염 방치**: 광 커넥터가 오염되면 신호 감쇠가 增加하고, 수신 에러율이 높아진다.

- **📢 섹션 요약 비유:** 광섬유通信は книжный шкафと似ています. 책장(광섬유) 안에서 책(빛)이미끄럼러를 타고 이동하며, 책장벽(클래딩)이 전반사로 책을 안에서 가둬둡니다. 외부干扰(EMI)가 들어올 수 없는封印된移動장치이기 때문에, 멀리 떨어진 곳까지 빠르게 깨끗한 데이터르 전달할 수 있습니다.

## 👶 어린이를 위한 3줄 비유 설명
1. 광섬유는 놀이터 미끄럼막과 같아요. 빛이 밝은 색공(광신호)이고, 미끄럼벽(클래딩)이 금속으로両面을覆여 있어 공이 튜어나가지 못하게 합니다.
2. 그래서 미끄럼벽 안에 갇힌 공이아무干扰 없이 미끄럼 끝까지 빠르게 굴러갈 수 있어요. 멀리 있는 집(수신자)까지 빛의速度로 데이터를 전달할 수 있죠.
3. 그리고 한 가닥 미끄럼에 여러 색깔 공을 동시에 보내면(파장 다중화),より 많은 데이터를 한번에 보낼 수 있습니다!

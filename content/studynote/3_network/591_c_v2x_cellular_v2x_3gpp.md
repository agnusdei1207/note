# 591. C-V2X (Cellular V2X) - 이동통신 연계 차량 통신

> **핵심 인사이트**: 전 세계 고속도로마다 차량 전용 통신탑(WAVE 기지국)을 1km 간격으로 새로 박으려면 천문학적인 돈이 든다. 그래서 생각했다. "어차피 전국에 스마트폰 터지게 하려고 LTE 기지국 다 깔아놨잖아? 자동차 통신도 그냥 그 LTE/5G 망을 같이 쓰게 만들면 안 돼?" 이 경제적이고 강력한 아이디어가 바로 현재 글로벌 자동차 통신의 표준을 휩쓸고 있는 C-V2X다.

## Ⅰ. C-V2X의 개념과 등장 배경
- **개념**: 국제 이동통신 표준화 기구인 **3GPP**가 주도하여 만든 기술로, 기존 우리가 쓰는 스마트폰 통신망인 **LTE(4G)와 5G 통신 기술을 차량의 V2X 통신용으로 개조하여 사용하는 차세대 차량 무선 통신 표준**입니다.
- **배경**: 기존 WAVE(와이파이 기반) 기술은 거리가 짧아 인프라를 너무 촘촘히 깔아야 하는 한계가 명확했습니다. 이를 극복하기 위해 통신사들이 주도하여 제안한 것이 C-V2X입니다.

## Ⅱ. C-V2X의 두 가지 통신 모드 (핵심 아키텍처) 🌟
C-V2X는 두 가지 엔진을 번갈아 쓰며 완벽함을 추구합니다.

### 1. 직접 통신 모드 (Direct Communication, PC5 인터페이스)
- **목적**: 초저지연이 생명인 차량 간(V2V) 충돌 방지용.
- **특징**: 산속이나 터널에서 LTE 기지국 전파가 안 터져도, 자동차들끼리 직접 LTE 전파를 쏴서 주변 1km 내외의 차들과 대화합니다. (기존 WAVE와 동일한 역할을 하되 거리가 2배 이상 깁니다.)

### 2. 네트워크 통신 모드 (Network Communication, Uu 인터페이스)
- **목적**: 장거리, 대용량 정보(내비게이션, 날씨, 넷플릭스) 수신용.
- **특징**: 자동차가 스마트폰처럼 일반 통신사의 대형 LTE/5G 기지국과 통신합니다. 전방 50km 앞의 사고 소식을 클라우드 센터를 거쳐 실시간으로 받아보는 장거리 통신의 핵심입니다.

## Ⅲ. WAVE vs C-V2X 양대 표준의 최종 승자
| 구분 | WAVE (DSRC 기반) | C-V2X (Cellular 기반) |
| :--- | :--- | :--- |
| **기반 기술** | Wi-Fi (IEEE 802.11p) | 이동통신 (3GPP LTE / 5G) |
| **통달 거리** | 짧음 (최대 1km 미만) | **매우 긺 (셀룰러 기지국 활용)** |
| **상용망(기지국) 의존**| 없음 (독자 인프라 구축 필요) | **기존 통신사 LTE/5G 망 재활용 가능** |
| **최대 단점** | 천문학적인 전용 인프라 구축 비용 | 통신사에 매달 통신 요금을 내야 할 수도 있음 |
| **시장 상황** | 과거 미국 중심 표준, 현재 퇴조 | **현재 미국, 중국, 5GAA 지지로 글로벌 대세 확정** |

> 📢 **섹션 요약 비유**: WAVE는 차들끼리 쓰는 '특수 무전기'입니다. 무전기라 산속에서도 즉시 쓸 수 있지만, 아주 멀리 떨어진 본부와는 연락할 수 없습니다. C-V2X는 1km 내에서는 무전기처럼 차들끼리 직접 대화하고(직접 모드), 50km 밖의 정보가 필요할 때는 폰으로 전환해 기지국(네트워크 모드)과 전화하는 '전천후 스마트폰'입니다. 기지국 인프라를 재활용하는 경제성 덕분에 전 세계 V2X의 승리자가 되었습니다.

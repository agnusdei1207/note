# 554. 주파수 재사용 (Frequency Reuse) - 셀룰러 클러스터 디자인, 용량 확장 기법

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: 주파수 재사용(Frequency Reuse)은 한정된 라디오 주파수 대역을 서로 전파가 닿지 않을 만큼 멀리 떨어진 다른 셀(Cell)에서 다시 반복하여 사용함으로써, 무선망 전체의 트래픽 수용 용량을 무한대로 늘리는 셀룰러 네트워크의 가장 위대한 공학적 아이디어다.
> 2. **가치**: 특정 개수의 셀을 묶어 하나의 '클러스터(Cluster)'를 만들고, 이 클러스터 단위로 전체 주파수를 쪼개어 할당(K=3, 4, 7 등)함으로써, 기지국이 뿜어내는 동일 채널 간섭(Co-Channel Interference)을 물리적 거리로 차단하는 절대적 기준을 제공한다.
> 3. **융합**: 가입자가 폭증하여 할당된 주파수로 감당이 안 될 때, 기지국 출력을 줄여 셀을 쪼개는 **셀 분할(Cell Splitting)**과 안테나 지향성을 좁혀 간섭을 피하는 **섹터링(Sectoring)** 기법으로 융합되어 공간의 경제학을 극대화한다.

---

## Ⅰ. 개요 및 필요성 (Context & Necessity)

- **개념**: 이동통신 사업자가 국가로부터 경매를 통해 할당받는 주파수 대역폭은 매우 좁고 비싸다. 이 주파수를 N개의 그룹으로 나눈 뒤 인접한 셀들이 서로 다른 그룹을 쓰게 하고, 멀리 떨어진 셀에서 다시 같은 그룹의 주파수를 재활용하는 공간적 다중화 기법이 주파수 재사용이다.
- **필요성**: 만약 서울 전역을 덮는 거대한 안테나가 있다면, 서울에 할당된 100개의 주파수 채널로 동시에 100명밖에 통화할 수 없다. 하지만 서울을 1만 개의 작은 셀로 쪼개고 "A동네 1번 주파수, 옆 동네 2번 주파수, 저 멀리 C동네 다시 1번 주파수"를 배정하면, 똑같은 100개의 주파수로 수십만 명이 통화하는 용량 증폭의 마법이 성립한다.
- **등장 배경**: ① FM 라디오/아날로그 TV 시절의 단일 고출력 송신탑 방식의 끔찍한 비효율 봉착 → ② 1979년 벨 연구소(Bell Labs)에서 지형을 육각형 벌집(Hexagon)으로 쪼개는 기하학적 셀룰러 모델 제안 → ③ K계수(재사용률) 기반의 클러스터 아키텍처가 전 세계 모든 1G~5G 통신망의 물리적 뼈대로 정착.

```text
┌─────────────────────────────────────────────────────────────┐
│             주파수 재사용(Frequency Reuse)의 마법 기하학 시각화    │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│   [클러스터 K = 3 인 경우] (3개의 셀이 한 세트로 묶임)                │
│                                                             │
│             / \     / \     / \                            │
│           / f1  \ / f2  \ / f1  \  ◀─ 저 멀리 f1이 다시 등장!  │
│          |       |       |       |                         │
│         / \     / \     / \     / \                        │
│       / f2  \ / f3  \ / f3  \ / f2  \                      │
│      |       |       |       |       |                     │
│       \     / \     / \     / \     /                      │
│         \ / f1  \ / f2  \ / f1  \ /                        │
│                                                             │
│   * 핵심: f1을 쓰는 셀끼리 일정 거리(재사용 거리, D) 이상 떨어져 있어, │
│           상대방의 전파가 도달하지 않아 서로 혼선(간섭)되지 않음!      │
└─────────────────────────────────────────────────────────────┘
```

**[다이어그램 해설]** 이 그림은 이동통신 공학의 상징과도 같은 벌집 모양 토폴로지다. 통신사가 총 300개의 통화 채널을 가지고 있다면, `K=3` 클러스터에서는 f1, f2, f3 셀이 각각 100개씩의 채널을 나눠 갖는다. 이 f1~f3 묶음(클러스터)이 서울 땅덩어리 위에 도장 찍듯 계속 반복(복사+붙여넣기) 설치된다. 결과적으로 100개의 클러스터가 깔리면, 원래 300명밖에 못 쓰던 망이 "300 x 100 = 30,000명"을 수용하는 거대한 용량 확장(Capacity Expansion)을 이뤄낸다.

- 📢 **섹션 요약 비유**: 넓은 도서관에서 크게 말하면 1명밖에 말할 수 없지만, 도서관을 여러 개의 작은 방(셀)으로 쪼개고 목소리를 낮추면, 저쪽 끝방(f1)에서 속닥이는 주제와 이쪽 끝방(f1)에서 속닥이는 주제가 같아도 서로 방해되지 않아 수천 명이 동시에 떠들 수 있는 공간 마법입니다.

## Ⅱ. 아키텍처 및 핵심 원리 (Deep Dive)

### 재사용 계수 (Frequency Reuse Factor, K)

클러스터를 몇 개의 셀로 묶을 것인가를 결정하는 숫자 `K`가 셀룰러 설계의 모든 것을 결정한다. 육각형 모델의 기하학적 특성상 K는 아무 숫자나 될 수 없고, `K = i^2 + ij + j^2` (i, j는 0 이상의 정수) 공식에 따라 **1, 3, 4, 7, 9, 12...** 등의 정해진 값만 가질 수 있다.

| K 값 (클러스터 크기) | 주파수 효율 (Capacity) | 간섭 억제력 (Interference) | 주로 쓰이는 세대 (Generation) |
|:---|:---|:---|:---|
| **K = 1** | **최고 (모든 셀이 모든 주파수 사용)** | 최악 (바로 옆 셀과 같은 주파수를 써서 전파 폭발) | 3G (CDMA), 4G (LTE), 5G (OFDMA) |
| **K = 3 또는 4** | 중간 | 중간 (1칸 띄우고 재사용) | 2G (GSM), 초기 아날로그 방식 |
| **K = 7** | **최저 (전체 채널을 7등분해서 쪼개 씀)** | 최상 (간섭이 일어날 물리적 거리가 아주 멂) | FM 라디오, 과거 아날로그 통신망 |

---

### "왜 4G LTE/5G는 재사용 계수 K=1을 쓰는가?"

위 표를 보면 이상한 점이 있다. 옆 셀과 거리를 두어야 간섭이 없다고 했는데, 최신 4G LTE와 5G망은 기지국마다 다른 주파수를 배정하지 않고 **K=1 (전국 모든 기지국이 100% 동일한 주파수를 씀)** 방식을 채택했다. 어떻게 끔찍한 전파 간섭을 뚫고 통신이 가능할까?

```text
┌───────────────────────────────────────────────────────────────┐
│               최신 망(LTE/5G)이 K=1 방식을 채택한 수학적 마법       │
├───────────────────────────────────────────────────────────────┤
│                                                               │
│   [과거: 주파수(차선)를 아예 나눔]                                 │
│   기지국 A (100MHz 사용)  ──▶ (벽) ◀── 기지국 B (101MHz 사용)      │
│   => 간섭 0. 하지만 기지국당 쓸 수 있는 대역폭이 너무 좁아 속도가 느림.    │
│                                                               │
│   [현재 4G/5G: K=1 주파수 재사용 + 직교성(OFDMA) 및 암호코드(CDMA)]  │
│   기지국 A (100MHz 통째로 씀) ──▶ (꽝!) ◀── 기지국 B (100MHz 통째로 씀)│
│                                                               │
│   "근데 안 부서짐! 왜?"                                          │
│   1. CDMA 방식: 주파수가 충돌해도 단말기에 부여된 '암호 코드'가 달라서 뚫어냄. │
│   2. OFDMA 방식(LTE/5G): 간섭이 가장 심한 셀 가장자리 단말기들에게는, 기지국끼리 │
│      서로 0.1밀리초 단위로 통신하여 빈 시간/빈 블록(Resource Block)을 피해서 쏴줌.│
│                                                               │
│   => 결과: 간섭을 피하기 위해 대역폭을 7토막 내는 바보짓을 멈추고,          │
│            모든 기지국이 풀(Full) 대역폭을 써서 기가비트 속도를 달성!       │
└───────────────────────────────────────────────────────────────┘
```

**[다이어그램 해설]** 초창기 아키텍트들은 간섭을 두려워하여 주파수를 쪼갰다(K=7). 하지만 통신사 입장에서 주파수를 7토막 내면 고객 1명이 누릴 수 있는 최대 속도도 1/7로 줄어든다. 스마트폰 시대에 속도 경쟁이 붙으면서, 3G(CDMA)는 고유한 확산 코드를 통해, 4G/5G(OFDMA)는 ICIC(Inter-Cell Interference Coordination)라는 기지국 간 실시간 찰나의 눈치싸움 스케줄링을 통해 간섭을 극복해 냈다. 이로써 전국의 모든 기지국이 통신사가 가진 모든 주파수 대역폭을 100% 한 톨도 안 남기고(K=1) 쓸 수 있는 궁극의 효율 진화가 이루어졌다.

- 📢 **섹션 요약 비유**: 옛날엔 옆 사람 말소리가 섞일까 봐 방을 7개로 쪼개고 한 명씩 들어가서 말했습니다. 지금(K=1)은 1만 명이 큰 강당 한곳에 모여 동시에 떠들지만, 각자 서로만 알아듣는 자기들만의 '비밀 언어'를 쓰거나, 아주 미세하게 서로 박자를 엇갈려 말하기 때문에 절대로 헷갈리지 않는 엄청난 스마트 파티가 되었습니다.

## Ⅲ. 융합 비교 및 다각도 분석

### 트래픽 폭증 시 용량 확장 3대 기법 (Capacity Expansion Techniques)

"우리 동네에 스마트폰 사용자가 갑자기 10배 늘어났는데 주파수를 더 살 돈이 없다." 아키텍트가 꺼낼 수 있는 3가지 물리적 무기가 있다.

| 확장 기법 | 영문 명칭 | 원리 및 물리적 변경 사항 | 부작용 및 트레이드오프 |
|:---|:---|:---|:---|
| **1. 셀 분할** | **Cell Splitting** | 반경 1km의 큰 셀 하나를 쪼개서 반경 500m의 작은 셀 4개로 만듦 (기지국 출력 강제 다운) | 기지국을 4개 새로 세워야 하므로 **비용 막대함**, 핸드오버 폭증 |
| **2. 섹터링** | **Sectoring** | 막대기 모양 무지향성(360도) 안테나를 버리고, 120도 지향성 안테나 3개를 달아 빔의 각도를 조각냄 | 셀 가장자리에서 안테나 빔 간의 **간섭(사각지대) 증가**, 트렁킹 효율 감소 |
| **3. 마이크로셀 할당** | **Microcell Zone** | 큰 셀 안에 핫스팟(정류장 등)에만 쪼끄만 기지국(Small Cell)을 박아 넣어 데이터만 빨아들임 | 매크로 셀과 스몰 셀 간의 수직적 간섭(고막 테러) 방어 기술(eICIC) 필수 |

```text
┌───────────────────────────────────────────────────────────────┐
│               섹터링(Sectoring) 기법을 통한 간섭 감소 마법          │
├───────────────────────────────────────────────────────────────┤
│                                                               │
│   [과거: 360도 무지향성 안테나 (Omni-directional)]                │
│       (((( 📡 )))) ──▶ 내가 전파를 쏘면 동서남북 6개의 이웃 셀 전체에  │
│                           원치 않는 전파 폭력(간섭)을 가하게 됨.        │
│                                                               │
│   [혁신: 120도 3-섹터 안테나 분할 (Directional)]                  │
│             /                                                 │
│   (섹터 A) 📡 -- (섹터 B) ──▶ 전파를 120도 부채꼴로 제한해서 쏨!       │
│             \               내가 쏘는 전파가 특정 방향 이웃 2곳에만 맞음.│
│          (섹터 C)                                              │
│                                                               │
│   => 결과: 안테나 모양만 부채꼴로 바꿨을 뿐인데, 기지국 간 간섭(CCI)이 │
│            1/3 로 줄어들어, 남는 공간에 더 많은 사람을 구겨 넣을 수 있음!│
└───────────────────────────────────────────────────────────────┘
```

**[다이어그램 해설]** 셀 분할은 땅을 사서 철탑을 새로 세워야 하니 수억 원이 든다. 가장 가성비가 좋은 기법이 섹터링(Sectoring)이다. 기지국 철탑 꼭대기를 보면 원통형 기둥 하나만 있는 것이 아니라, 보통 3개의 길쭉한 안테나가 삼각형 모양으로 120도씩 등을 지고 붙어있다. 이렇게 쏘는 각도를 제한하면, 재사용 거리 밖으로 날아가 다른 셀을 때리는 간섭(Interference)의 개수가 획기적으로 줄어든다. 간섭이 줄어들면 통신사는 주파수 클러스터 K값을 7에서 4나 3으로 낮출 수 있고, 이는 곧 가입자 용량 폭발로 이어진다.

- 📢 **섹션 요약 비유**: 사방으로 빛이 퍼지는 백열전구(무지향성)를 쓰면 옆 사람 눈을 너무 부시게 해서 전구를 많이 달 수 없습니다. 전구에 검은색 갓을 씌워 앞만 비추는 손전등(120도 섹터 안테나) 3개로 바꾸면, 다른 사람 눈을 부시게 하지 않으면서 한 방에 훨씬 더 많은 조명을 달 수 있습니다.

## Ⅳ. 실무 적용 및 기술사적 판단

### 실무 시나리오: 도심지 빌딩 숲 동일 채널 간섭(Co-Channel Interference) 해결

1. **상황**: 강남 테헤란로의 고층 빌딩 숲에서 통화 끊김(Drop) 현상이 급증했다. 분석 결과, 멀리 떨어진 송파구에서 동일한 주파수를 재사용(K=1)하는 기지국의 전파가, 한강 변을 타고 혹은 높은 빌딩 유리에 반사되어 강남역까지 날아와 고객의 스마트폰을 정통으로 때리는 심각한 **동일 채널 간섭(CCI)**이 발생하고 있었다.
2. **원인**: 아키텍트가 초기 셀 플래닝(Cell Planning)을 할 때, 전파가 평지로만 퍼진다고 가정하고 송신 출력과 안테나 틸팅(기울기)을 과도하게 높게 세팅하여 **전파 오버슈팅(Overshooting)**이 발생한 것이다.
3. **의사결정 및 물리적 최적화 조치**:
   - **안테나 다운 틸팅 (Down-Tilting)**: 강남과 송파 양쪽의 기지국 안테나 각도를 땅바닥 쪽으로 기계적/전자적으로 숙이게 만들어(Tilt down), 전파가 다른 셀 경계 너머로 날아가지 않고 자기 구역(발밑)에만 꽂히도록 물리적 방화벽을 쳤다.
   - **송신 전력 제어 (Power Control)**: 기지국의 출력을 20W에서 10W로 반토막 내어 재사용 거리(Reuse Distance) 밖으로 나가는 전파 에너지를 소멸시켰다. 줄어든 커버리지는 빈 공간에 초소형 스몰 셀(Pico Cell)을 2개 덧대어 메웠다.
   - **결과**: 간섭(Noise)이 소멸하며 신호 대 잡음비(SINR)가 급상승했고, 끊김 현상이 완전히 사라짐과 동시에 다운로드 속도가 30% 증가했다.

### 도입 체크리스트 및 안티패턴
- **재사용 거리비(D/R Ratio) 계산 검증**: 셀 반경을 $R$, 같은 주파수를 다시 쓰는 곳까지의 거리를 $D$라 할 때, 클러스터 크기 $K$에 따라 $D/R = \sqrt{3K}$ 라는 황금 공식이 성립한다. 이 $D/R$ 비율을 무시하고 "가입자가 많으니 그냥 출력만 높이자"라고 세팅하면 망이 스스로 자신의 목을 조르는 셀프 간섭 지옥에 빠진다. 전파 출력 설계는 철저히 이 수학적 비율 안에서 통제되어야 한다.
- **안티패턴 (셀 가장자리 방치)**: K=1(모두 같은 주파수 사용) 환경인 4G/5G에서, 기지국 A와 B 사이에 서 있는 단말기(셀 엣지 가입자)에 대해 아무 조치도 취하지 않는 것. 이 단말기는 A 전파와 B 전파를 50:50으로 받아 통신이 완전히 박살 난다. 최신 아키텍처에서는 기지국 A와 B가 X2(또는 Xn) 광케이블 인터페이스로 0.1밀리초마다 대화하며 "야, 쟤 셀 경계에 있으니 내가 왼쪽 주파수 블록 쓸 테니 넌 오른쪽 블록으로 쏴줘(FFR, Fractional Frequency Reuse)"라며 동적으로 자원을 회피하는 지능형 스케줄링을 반드시 가동해야 한다.

- 📢 **섹션 요약 비유**: 옆 동네와 같은 멜로디의 노래를 틀어야 할 때, 스피커 볼륨을 무식하게 키우면 동네 전체가 시끄러워져 아무 노래도 못 듣습니다(전파 오버슈팅). 스피커 각도를 땅으로 향하게 꺾고(다운 틸팅), 볼륨을 확 낮춘 뒤 군데군데 작은 스피커를 여러 개 다는 것이 이동통신 공학자의 진짜 실력입니다.

### 정량/정성 기대효과

| 구분 | 주파수 재사용 미적용 (과거 방송 모델) | 클러스터 분할 및 K=1 융합 최적화 시 | 개선 효과 |
|:---|:---|:---|:---|
| **정량 (주파수 효율성)** | 국가 경매로 할당받은 대역폭만큼만 가입자 수용 | 셀 분할과 공간 재활용을 통해 수만 배 뻥튀기 | 할당 스펙트럼 대비 **트래픽 수용 한계 극복** |
| **정량 (기지국 간섭률)** | 안테나 무지향성 방사로 셀 경계 간섭률 50% 육박 | 120도 섹터링 및 다운 틸팅 적용 | 동일 채널 간섭(CCI) 및 노이즈 **70% 이상 소멸** |
| **정성 (통신 품질)** | 인구가 몰리는 강남역에서 매일 밤 통화 불통 | 마이크로셀(스몰셀) 분할을 통한 트래픽 오프로딩 | 혼잡 시간대(Busy Hour)에도 끊김 없는 데이터 체감 속도 보장 |

### 미래 전망 및 진화 방향
- **Massive MIMO를 통한 3차원(3D) 주파수 재사용**: 4G까지는 전파를 2차원(땅바닥) 부채꼴로 쏘는 섹터링이 한계였다. 5G에 이르러 128개의 안테나가 달린 Massive MIMO가 등장하면서, 전파를 상하좌우 고도까지 조절하여 "고층 빌딩 10층의 A직원과 1층의 B직원에게 동일한 주파수를 동시에 쏘아도(3D Beamforming) 간섭이 없는" 궁극의 3차원 공간 재사용 시대가 열렸다.
- **셀 경계가 사라지는 Cell-Free 아키텍처 (6G)**: 현재는 내가 A 기지국에 속해 있는지 B 기지국에 속해 있는지(셀 경계)가 명확하다. 6G 시대가 되면 무수한 스몰 셀 안테나들이 하나의 거대한 구름 덩어리처럼 동작하여, 단말기를 향해 가장 가까운 안테나 3~4개가 묻지도 따지지도 않고 동시에 데이터를 쏘아주는 '셀 경계 없는(Cell-Free) 통신'이 주파수 재사용 패러다임의 종착지가 될 것이다.

### 참고 표준
- **3GPP TS 36.211 / 38.211**: 물리 계층 규격 (LTE 및 5G NR 환경에서 K=1 주파수 재사용 시 간섭을 피하기 위한 직교성 자원 블록 할당 표준)
- **전파법 및 ITU-R 규정**: 국가별 기지국 송신 전력 한계치 및 대역 외 발사(인접 대역 간섭) 허용 기준.

이동통신 공학은 '물리학의 결핍을 기하학의 아이디어로 극복한' 위대한 학문이다. 허공에 떠도는 전파는 돈을 주고 살 수도, 복사할 수도 없는 절대적인 희소 자원이지만, 공학자들은 벌집 모양의 클러스터(Cluster)를 그리고 안테나의 고개를 꺾어 이 희소 자원을 무한한 금광으로 재창조해 냈다.

- 📢 **섹션 요약 비유**: 주파수 재사용은 물감(주파수)이 딱 3가지 색밖에 없는데, 수만 명이 그림을 그려야 할 때의 해결책입니다. 한 사람이 캔버스의 구석에 빨간색을 아주 작게 칠하게 하고, 그 빨간 물감이 마를 만큼 멀리 떨어진 저 반대편 구석에서 다른 사람이 또 빨간색을 칠하게 룰을 정하면 3가지 색으로도 세상을 다 칠할 수 있는 위대한 경제학입니다.

## 📌 관련 개념 맵 (Knowledge Graph)

| 개념 명칭 | 관계 및 시너지 설명 |
|:---|:---|
| **클러스터 (Cluster)** | 전파 간섭을 일으키지 않기 위해 전체 주파수를 나누어 가진 셀들의 최소 묶음 단위로, K=1, 3, 4, 7 등의 수학적 크기를 갖는다. |
| **셀 분할 (Cell Splitting)** | 동네에 사람이 너무 많아져 주파수(K) 재사용만으로 부족할 때, 커다란 기지국 반경을 강제로 쪼개서 작은 기지국 여러 개로 촘촘히 구겨 넣는 물리적 용량 확장술이다. |
| **섹터링 (Sectoring)** | 사방팔방으로 퍼지는 전파의 빛 공해(간섭)를 줄이기 위해, 기지국 안테나를 120도 각도의 3면으로 나누어 쏘게 만들어 부작용 없이 용량을 늘리는 알짜 기술이다. |
| **동일 채널 간섭 (CCI)** | 멀리 떨어진 옆 동네에서 나와 완벽히 똑같은 주파수를 재사용(Reuse)할 때, 그 전파가 우연히 내 앞까지 날아와 내 통신을 찢어버리는 주파수 재사용의 가장 치명적인 부작용이다. |
| **FFR (Fractional Frequency Reuse)** | 모든 기지국이 100% 동일한 주파수를 쓰는(K=1) 4G/5G 시대에, 셀 가장자리 끄트머리에 있는 폰들이 벼락(간섭)을 맞지 않도록 주파수 조각을 지능적으로 엇갈려 배정해 주는 스케줄링 기술이다. |

## 👶 어린이를 위한 3줄 비유 설명
1. 텔레비전 채널 1번 방송국이 서울에도 있고 부산에도 있으면, 서로 너무 멀어서 전파가 부딪히지 않기 때문에 각자 재미있게 방송할 수 있죠?
2. 주파수 재사용(Frequency Reuse)은 이 원리를 이용해, 서울 안에서도 강남과 강북처럼 멀리 떨어진 곳의 안테나들이 똑같은 전파(채널)를 또 쓰게 만드는 마법이에요.
3. 우리가 쓰는 전파(차선)는 몇 개 안 되지만, 안테나들이 멀찍이 떨어져서 목소리를 작게 낮춘 채 전파를 계속 '재활용'하기 때문에 전 국민이 끊김 없이 통화할 수 있는 거랍니다.

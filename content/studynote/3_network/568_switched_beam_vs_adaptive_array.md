# 568. 스위칭 빔 (Switched Beam) vs 적응형 어레이 (Adaptive Array 안테나)

> **핵심 인사이트**: 스마트 안테나가 핀 조명을 쏠 때, 무대 위에 미리 설치해 둔 5개의 조명 중 하나를 골라 켜고 끄는 것이 '스위칭 빔'이라면, 감독이 손에 조명을 직접 들고 배우의 움직임을 1mm 단위로 부드럽게 끝까지 쫓아다니는 것이 완벽한 기술인 '적응형 어레이'다.

## Ⅰ. 빔을 조향하는 두 가지 기술
스마트 안테나는 여러 개의 안테나 배열을 사용하여 좁은 전파 빔(Beam)을 형성합니다. 이 빔을 이동하는 사용자에게 어떻게 따라붙게 만들 것인가에 따라 시스템이 크게 두 가지로 나뉩니다.

## Ⅱ. 스위칭 빔 (Switched Beam) 방식
- **개념**: 기지국 주변 360도를 예를 들어 15도 간격으로 쪼개어, **미리 정해진 여러 개의 고정된 빔 패턴(방향)을 만들어 놓고 준비**합니다.
- **동작 방식**: 사용자의 신호가 가장 세게 들어오는 방향을 파악한 뒤, 그 위치와 가장 가까운 번호의 고정된 빔 하나를 선택(Switching)하여 전파를 쏩니다. 사용자가 이동하면 다음 번호의 빔으로 스위치를 툭 넘깁니다.
- **특징**: 
  - 알고리즘이 단순하고 저렴하게 구현할 수 있습니다.
  - 사용자가 두 빔의 경계선에 서 있으면 신호가 약해지는 사각지대가 발생합니다.
  - 방해 전파(간섭)가 사용자와 같은 빔 영역 안에 들어오면 간섭을 피할 방법이 없습니다.

## Ⅲ. 적응형 어레이 (Adaptive Array Antenna) 방식 🌟
- **개념**: 고정된 빔 번호가 없습니다. 고성능 디지털 신호 처리기(DSP)의 연산을 통해 **가입자가 이동하는 방향과 속도에 맞춰 빔의 모양과 방향을 실시간으로 자유자재로 변경(성형)**하는 진정한 의미의 스마트 안테나입니다.
- **동작 방식 (도래각과 영점 조절)**:
  - 수신된 전파의 위상차를 계산하여 사용자의 정확한 방향(도래각, DOA)을 알아내어 빔을 조준합니다.
  - 가장 무서운 기능은 **영점(Nulling) 조절**입니다. 방해 전파(간섭)가 날아오는 각도를 파악하면, 그쪽 방향으로만 안테나 수신 감도를 수학적으로 완벽한 '0'으로 찌그러뜨려 노이즈를 원천 차단합니다.
- **특징**:
  - 사용자만을 정확히 끝까지 쫓아가며(Tracking), 간섭 신호는 완벽히 회피합니다.
  - 어마어마한 양의 행렬 연산을 실시간으로 수행해야 하므로 컴퓨팅 파워가 많이 필요하고 구축 비용이 매우 비쌉니다. (현대 4G/5G MIMO, 빔포밍의 핵심 기반)

## Ⅳ. 요약 비교표

| 구분 | 스위칭 빔 (Switched Beam) | 적응형 어레이 (Adaptive Array) |
| :--- | :--- | :--- |
| **빔의 형태** | 여러 개의 **고정된 방향**을 미리 세팅 | 사용자를 따라 빔을 **실시간 자유 성형** |
| **이동자 추적** | 빔 번호를 갈아타며 대략적인 추적 | 1도 단위의 정밀한 연속 추적 (Tracking) |
| **간섭파 회피** | 간섭 방향을 피하는 능력이 취약함 | 간섭파 방향으로 **수신 감도 0(Nulling)** 생성 |
| **시스템 복잡도/가격**| 낮음 (저렴함) | **매우 높음 (비쌈, 고성능 DSP 필요)** |
| **성능 (통화 품질)** | 중간 | **최상 (진정한 스마트 안테나)** |

> 📢 **섹션 요약 비유**: 스위칭 빔은 아파트 복도의 센서등입니다. 내가 1번 전등 구역에 가면 1번이 켜지고, 2번 구역으로 걸어가면 2번이 켜지지만 구역 사이사이는 어둡습니다. 반면 적응형 어레이는 내가 복도에 들어서자마자 천장의 카메라가 나를 인식하고, 레이저 빔이 내 발걸음을 1cm 오차 없이 계속 따라다니면서 내 앞길만 환하게 비춰주는 최첨단 추적 조명입니다.

# 551. 이동통신망(Cellular Network) 통신 개념

> **핵심 인사이트**: 옛날 무전기는 남산 꼭대기에 거대한 안테나 하나를 세워 서울 전체를 커버하려다 보니 가입자를 몇 명 못 받았다. 이동통신망(셀룰러)은 서울을 수천 개의 벌집(Cell)으로 쪼개고, 각 벌집마다 작은 기지국을 세운 뒤 "멀리 떨어진 벌집끼리는 같은 주파수를 재사용"하여 수천만 명이 동시에 통화할 수 있게 만든 기적의 발상이다.

## Ⅰ. 셀룰러 네트워크(Cellular Network)의 등장 배경
- **단일 대형 안테나 방식의 한계**: 과거 카폰(Car Phone) 시절에는 고출력 기지국 하나가 반경 수십 km를 커버했습니다. 그러나 사용 가능한 라디오 주파수 대역은 매우 한정적이기 때문에, 동시에 통화할 수 있는 채널 수가 극도로 부족했습니다.
- **셀룰러의 탄생**: 서비스 지역을 작은 6각형의 셀(Cell, 벌집 모양) 여러 개로 분할하고, 각 셀의 중앙에 소출력 기지국(Base Station)을 설치하는 아키텍처가 고안되었습니다.

## Ⅱ. 핵심 기술 요소 2가지 🌟
셀룰러 네트워크가 폭발적인 가입자 수용 용량을 가질 수 있게 된 두 가지 핵심 개념입니다.

### 1. 주파수 재사용 (Frequency Reuse)
서로 인접한 셀끼리는 전파 간섭(혼선)을 막기 위해 다른 주파수(예: F1, F2, F3)를 사용하지만, **충분히 멀리 떨어져서 전파가 닿지 않는 다른 셀에서는 앞서 썼던 주파수(F1)를 다시 재사용**하는 기술입니다. 이 '공간적 재사용' 덕분에 한정된 주파수 자원으로 이론상 무한대에 가까운 가입자를 수용할 수 있습니다.

### 2. 핸드오버 (Handover / Handoff)
사용자가 통화를 하면서 A 기지국(셀) 영역에서 B 기지국(셀) 영역으로 이동할 때, 통화가 끊기지 않도록 **통신 채널을 A에서 B로 부드럽게 넘겨주는(이어달리기) 기술**입니다. 이를 위해 단말기와 기지국, 그리고 백본의 교환기(MSC)가 끊임없이 신호 세기를 측정하고 제어 신호를 주고받습니다.

## Ⅲ. 셀룰러 네트워크의 기본 구성 (구조)
1. **단말기 (MS/UE)**: 스마트폰
2. **기지국 (BTS / eNodeB / gNB)**: 각 셀의 중심에서 무선 전파를 쏘고 받는 안테나 탑
3. **기지국 제어기 (BSC)**: 여러 기지국을 묶어서 주파수 할당 및 핸드오버를 제어하는 중간 관리자
4. **이동 통신 교환국 (MSC)**: 유선 전화망(PSTN)이나 인터넷망과 연결해주고, 가입자 위치 정보(HLR/VLR)를 관리하는 중앙 교환기

> 📢 **섹션 요약 비유**: 단일 안테나가 '서울 한복판에서 확성기를 들고 소리치는 한 명의 마을 이장님(동시에 한 사람과만 대화 가능)'이라면, 셀룰러 네트워크는 '서울을 수천 개의 방(Cell)으로 쪼개고 방마다 칸막이를 친 뒤(주파수 재사용), 손님이 방을 옮겨 다닐 때마다 담당 웨이터를 자연스럽게 교체해주는(핸드오버) 거대한 레스토랑'입니다.

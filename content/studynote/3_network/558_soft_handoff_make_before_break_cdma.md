# 558. 소프트 핸드오버 (Soft Handoff) - Make before Break, CDMA

> **핵심 인사이트**: 공중그네를 탈 때 무서워서 기존 밧줄을 절대 놓지 못하는 사람이 있다. 한 손은 A 밧줄을 꽉 쥔 채로, 다른 한 손으로 새로 날아온 B 밧줄을 꽉 잡은(Make) 뒤에야, 안심하고 A 밧줄을 놓아버리는(Break) 완벽한 무중단 이어달리기. 이것이 과거 3G CDMA 음성 통화를 빛나게 했던 소프트 핸드오버다.

## Ⅰ. 소프트 핸드오버 (Soft Handover)의 개념
단말기가 새로운 기지국으로 넘어갈 때, **새로운 기지국과 먼저 무선 채널 연결을 맺은(Make) 후, 두 기지국의 전파를 동시에 수신하다가 기존 기지국과의 연결을 천천히 끊는(Break) 무단절 방식**입니다.

## Ⅱ. 원리와 특징 (Make before Break)
- **메커니즘 (CDMA의 마법)**: 이 기술은 모든 인접한 기지국들이 **완전히 동일한 주파수 대역**을 사용하고, 오직 '암호 코드(PN Code)'로만 구분되는 **CDMA(코드 분할 다중 접속) 기술에서만 구현 가능**한 특권이었습니다.
- 단말기 내부에는 레이크 수신기(Rake Receiver)라는 특수 안테나 장비가 있어, A 기지국과 B 기지국에서 날아오는 똑같은 주파수의 신호를 '동시에 2~3개' 수신하여 하나로 합성(결합 다이버시티)할 수 있습니다.
- **장점 (무단절)**: 물리적인 끊김이 0.1초도 발생하지 않으므로 음성 통화 품질이 매우 부드럽고 핑퐁 효과(Call Drop)가 완벽하게 방지됩니다.
- **단점 (자원 낭비)**: 교차로(셀 경계지역)에 있는 하나의 스마트폰이 2개 이상의 기지국 무선 채널 자원을 꽉 붙잡고(점유하고) 있어야 하므로, **기지국 시스템 전체의 채널 용량 낭비가 극심해집니다.** 단말기 배터리도 빨리 닳습니다.

## Ⅲ. 소프터 핸드오버 (Softer Handover) - 헷갈림 주의!
소프트 핸드오버와 이름이 비슷하지만 다릅니다.
- **소프트(Soft)**: 'A 기지국'과 'B 기지국'처럼 아예 **물리적으로 다른 철탑(기지국) 장비 간**에 2개의 채널을 잡고 넘어가는 것입니다.
- **소프터(Softer)**: **하나의 동일한 기지국 철탑 내부**에서, 안테나 방향(섹터)만 다르게 쪼개놓은 구역(예: 북쪽 섹터 -> 남쪽 섹터)을 이동할 때 발생하는, 기지국 내부의 채널 갈아타기 방식입니다. 훨씬 빠르고 쉽게 처리됩니다.

## Ⅳ. 하드 핸드오버 vs 소프트 핸드오버 요약표

| 구분 | 하드 핸드오버 (Hard Handoff) | 소프트 핸드오버 (Soft Handoff) |
| :--- | :--- | :--- |
| **방식** | **Break** before Make (선 단절 후 접속) | **Make** before Break (선 접속 후 단절) |
| **주파수 특성** | 주로 서로 **다른 주파수** 사이에서 발생 | 반드시 **동일한 주파수** (CDMA 코드 기반) 환경 |
| **연결 끊김 (Call Drop)** | 찰나의 순간 발생 가능성 있음 | 전혀 없음 (무단절) |
| **무선 자원(채널) 효율성**| 낭비 없음 (**매우 높음**) | 경계지역에서 2배 점유로 **낭비 극심** |
| **주요 적용 통신망** | **4G LTE, 5G NR (현재 주력)** | 2G/3G CDMA (과거 음성 주력) |

> 📢 **섹션 요약 비유**: 지하철을 갈아탈 때, 하드 핸드오버는 1호선에서 완전히 내려서 승강장을 걸어간 뒤 2호선으로 갈아타는 찰나의 공백기(Break)가 있는 환승입니다. 반면 소프트 핸드오버는 마주 보고 달리는 1호선과 2호선 열차 문 사이에 다리를 양쪽으로 걸치고 서 있다가(동시 점유), 1호선 쪽 다리를 마저 당겨 2호선으로 쏙 들어가는 완벽한 연속 환승입니다.

# B8ZS 및 HDB3 스크램블링 기술 분석

#### 핵심 인사이트 (3줄 요약)
> 1. **본질**: AMI 코딩의 고질적 문제인 **연속된 '0'**으로 인한 동기화 상실을 방지하기 위해, 특정 개수 이상의 0이 나오면 이를 의도적인 **비트 패턴 위반(Violation)**으로 대체하는 기술.
> 2. **가치**: 데이터 선로를 그대로 쓰면서도 클럭 정보를 유지할 수 있게 하며, 북미(B8ZS)와 유럽(HDB3) 표준으로 정립되어 전 세계 디지털 전용망의 신뢰성을 보장한다.
> 3. **융합**: 물리 계층의 코딩 위반 규칙과 논리 계층의 패턴 복원 알고리즘이 결합된 형태이며, 최근 고속 통신의 데이터 무작위화(Randomization) 사상의 원조가 된다.

---

### Ⅰ. 스크램블링 기술의 등장 배경

- **문제점**: AMI 방식에서 '0'이 계속되면 전압 변화가 없어 수신측 클럭이 멈춤 (Sync Loss).
- **해결 원리**: 
  - 0이 너무 길어지면 미리 약속된 **'특수한 1'**들을 끼워 넣어 전압을 강제로 변화시킴.
  - 이때 삽입된 1은 정상적인 AMI 규칙을 어기는 **Bipolar Violation (BPV)** 형태로 만들어, 수신측이 "아, 이건 진짜 데이터 1이 아니라 동기용 가짜 패턴이구나!"라고 알아채게 함.

---

### Ⅱ. B8ZS vs HDB3 비교 분석

| 구분 | B8ZS (Bipolar 8-Zero Substitution) | HDB3 (High Density Bipolar 3) |
|:---|:---|:---|
| **표준 지역** | 북미, 한국, 일본 (T1 급) | 유럽 (E1 급) |
| **치환 조건** | **0이 8개** 연속될 때 | **0이 4개** 연속될 때 |
| **작동 방식** | 00000000 -> **000VB0VB** 패턴 삽입 | 0000 -> **000V** 또는 **B00V** 삽입 |
| **특징** | 패턴이 고정되어 단순함. | DC Balance를 위해 패턴을 가변적으로 선택. |

* **V (Violation)**: 이전 1과 **같은** 극성.
* **B (Bipolar)**: 이전 1과 **반대** 극성 (AMI 규칙 준수).

---

### Ⅲ. 치환 패턴 시각화 (ASCII)

HDB3에서 0이 4개 올 때의 변화 예시다.

```ascii
    [ Original Data ] :  1 0 0 0 0 1
    [ AMI 코딩 ]      : +V 0 0 0 0 -V  (동기화 위험!)
    
    [ HDB3 스크램블 ]  : +V 0 0 0 +V -V  (Violation 발생)
                           |_____|
                        (가짜 1 삽입하여 전압 변화 유도)
```

---

### Ⅳ. 실무 적용 및 기술사적 판단 (Strategy & Decision)

#### 1. 전용 회선 규격 설계
- **현상**: 해외 지사와 통신망 연동 시 데이터가 깨지거나 끊김.
- **기술사적 결단**: 
  - 양단의 장비가 동일한 스크램블링 방식(B8ZS vs HDB3)을 쓰는지 확인한다. 
  - 특히 T1(1.544Mbps)과 E1(2.048Mbps)은 속도뿐만 아니라 이 스크램블링 방식이 다르므로, 국가 간 연동 시 프로토콜 변환기(Converter) 설정이 필수임을 명시해야 한다.

#### 2. 기술사적 인사이트: 투명성 (Transparency)
- B8ZS와 HDB3는 사용자의 데이터 내용이 무엇이든(0이 수백만 개여도) 상관없이 통신을 유지해주는 **'비트 투명성'**을 제공하는 핵심 기술임을 강조해야 한다.

---

### Ⅴ. 기대효과 및 결론 (Future & Standard)

#### 1. 정량/정성 기대효과
- **동기화 안정성 극대화**: 어떤 데이터 패턴에서도 클럭 유지 가능.
- **대역폭 효율 유지**: 별도의 제어 비트 추가 없이 물리적 규칙 위반만으로 동기화 해결.

#### 2. 미래 전망
현재의 초고속 이더넷이나 무선 통신에서는 B8ZS 같은 단순 치환보다는, 데이터를 수학적으로 무작위화하는 **LFSR (Linear Feedback Shift Register)** 기반의 스크램블러를 사용한다. 하지만 '데이터 패턴에 상관없이 물리적 특성을 일정하게 유지한다'는 철학은 모든 현대 통신 표준의 근간으로 이어지고 있다.

3. 이 비밀 암호 덕분에 친구는 잠들지 않고 내 말을 끝까지 잘 들을 수 있게 된답니다!

# 555. Co-channel Interference (동일 채널 간섭) / Adjacent Channel Interference (인접 채널 간섭)

> **핵심 인사이트**: 무선 통신은 모두가 같은 공기를 매질로 소리치는 것과 같다. 간섭은 두 가지다. 멀리 있는 다른 클러스터에서 나와 완전히 '같은 주파수'를 쓰는 사람이 소리쳐서 들리는 노이즈(동일 채널 간섭)와, 바로 내 옆 셀에서 '주파수 번호는 다르지만 너무 비슷한' 주파수로 소리쳐서 전파가 번져 들어오는 노이즈(인접 채널 간섭)다.

## Ⅰ. 무선 통신 간섭의 원인
셀룰러 이동통신은 한정된 주파수를 공간적으로 쪼개어 재사용(Frequency Reuse)하기 때문에, 필연적으로 다른 기지국이나 다른 단말기가 쏘는 전파가 내 신호에 섞여 들어오는 **간섭(Interference)** 현상이 발생하며, 이는 통화 끊김이나 속도 저하의 주원인이 됩니다.

## Ⅱ. Co-channel Interference (동일 채널 간섭) 🌟
- **개념**: 셀룰러망에서 주파수 재사용을 위해, **물리적으로 멀리 떨어진 다른 셀에서 나와 완전히 동일한 주파수 채널을 사용함으로써 발생하는 간섭**입니다.
- **원인**: 전파는 딱 정해진 경계선에서 사라지지 않고 서서히 약해지면서 퍼져나갑니다. A셀에서 쏜 주파수 F1의 전파 꼬리가, 멀리 떨어진 다른 클러스터의 A셀(주파수 F1 사용)까지 닿아버릴 때 발생합니다.
- **해결 방안 (대책)**:
  1. **주파수 재사용 거리(D) 확보**: 동일 주파수를 쓰는 셀 사이의 거리를 최대한 멀리 띄워놓습니다. 즉, 클러스터 크기(N=재사용 계수)를 크게 잡습니다. (N=3 대신 N=7 사용)
  2. **지향성 안테나 (Sectoring)**: 360도로 퍼지는 전방향(Omni) 안테나 대신, 120도씩 3개로 방향을 쪼갠 섹터 안테나를 사용하여 전파가 다른 셀로 침범하는 각도를 줄입니다.
  3. **스마트 전력 제어**: 단말기가 기지국과 가까울 때는 전파 출력을 최소로 낮춰서 불필요하게 멀리 날아가지 못하게 합니다.

## Ⅲ. Adjacent Channel Interference (인접 채널 간섭)
- **개념**: 완전히 동일한 주파수는 아니지만, 주파수 대역표 상에서 **내 주파수(예: 800MHz)와 바로 옆에 붙어있는 주파수(예: 802MHz) 대역의 신호가 새어 들어와(Leakage) 발생하는 간섭**입니다.
- **원인**: 수신기 내부의 필터(Filter)가 완벽한 직사각형 모양으로 특정 주파수만 딱 잘라내지 못하고, 옆 차선의 주파수 신호 끄트머리를 조금씩 같이 수신해버리기 때문입니다. (도로에서 옆 차선 버스의 소음이 들리는 것과 같음)
- **해결 방안 (대책)**:
  1. **보호 대역 (Guard Band) 설정**: 두 주파수 채널 사이에 데이터를 싣지 않는 빈 공간(여백 주파수)을 두어 완충 지대를 만듭니다.
  2. **주파수 교차 할당**: 물리적으로 바로 옆에 붙어있는 셀(A와 B)에게는 주파수 번호 1번과 2번을 주지 않고, 일부러 1번과 7번처럼 멀리 떨어진 주파수를 할당합니다.
  3. **고성능 대역 통과 필터(BPF)**: 수신기의 필터 품질을 높여 예리하게 신호를 걸러냅니다.

## Ⅳ. 요약 비교표

| 구분 | Co-channel Interference (동일 채널 간섭) | Adjacent Channel Interference (인접 채널 간섭) |
| :--- | :--- | :--- |
| **발생 주파수** | **완전히 똑같은 주파수** | **대역이 맞닿아 있는 바로 옆 주파수** |
| **발생 위치 (원인)** | 멀리 떨어진 **다른 클러스터의 동일 셀** | 주로 **동일 셀 내부** 또는 **바로 옆 셀** |
| **핵심 해결책** | 셀 간 거리(D) 이격, 안테나 섹터링 분할 | **Guard Band(보호대역) 확보**, 채널 번호 띄워 할당 |

> 📢 **섹션 요약 비유**: 시험장에서 똑같이 생긴 두 명의 학생이 시험을 치는데, 동일 채널 간섭은 "저 멀리 뒷자리에서 나와 완전히 똑같은 수학 시험을 치는 애가 중얼거리는 소리가 내 귀에 들리는 것(재사용 거리 부족)"이고, 인접 채널 간섭은 "바로 내 옆자리 짝꿍이 나와는 다른 국어 시험을 치고 있지만 목소리가 너무 커서 내 수학 계산을 방해하는 것(필터링 부족)"입니다.
